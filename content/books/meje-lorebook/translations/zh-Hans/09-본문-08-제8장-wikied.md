# 第8章　双重叙述——外部比喻与内部法则的散文

假设读者的 IP 有一种名为“维度移动”的机制：人物从自己的维度前往另一个维度的运行原理。LOREBOOK 应如何说明它？有两种写法。

一种是直接向读者讲授机制本身。

```
[直接讲授]

维度移动是本 IP 的核心机制，是人物从自己的维度前往另一维度的运行原理。
维度之间的移动通过名为“棺舱”的装置发生；移动时人物的意识转移，身体仍留在
原来的维度。这一机制是本 IP 所有维度事件的基础。
```

这并没有错误，信息也准确；但它少了一样东西：读者没有在脑中描绘维度移动的踏脚石。“棺舱”“意识转移”等名称不属于读者已有的经验，读者可以记住名称，却无法亲手触及它的运作。

另一种是让读者从熟悉的外部比喻进入。

```
[外部比喻]

躺进棺舱并转动把手的瞬间，对跑者本人而言，与戴上 VR 头显没有不同。
然而实际发生的是，他的意识从一个维度跨越到另一个维度的物理过程。没有人称它
为“连接”；人们只说“进去了”。
```

这就是**双重叙述**。熟悉的外部比喻（VR 头显）让读者从踏脚石进入，其下的 IP 法则（意识跨越维度）在同一条叙述中自然流出。读者于是能够在脑中描绘机制。

这种方法把外语作品韩语翻译中形成的模式应用到散文工作。当 IP 有黑洞、神器、维度移动、附身等独有机制时，它使读者以熟悉的方式进入，同时不破坏 IP 的氛围。

## 为什么不能用表格并列

双重叙述的一个陷阱，是将外部比喻与内部法则并列在表中。

```
[表格并列：应避免的例子]

| 外部比喻 | 内部法则 |
|---|---|
| VR 头显 | 意识转移 |
| 连接 Matrix | 跨越维度 |
| 登录 | 潜入 |
```

作为资料它是准确的，但在 LOREBOOK 散文中会带来两个问题。第一，它降低可读性：读者读到表格时会中断，扫视表格后再回到正文。

第二，它破坏散文的氛围。比喻与法则在表中一一对应时，读者会把它们当成彼此分离的资料。比喻变成单纯的说明工具，法则也在表中被固定，读者无法直接触及。

应避免表格并列，而让两者自然融入散文。正如上面的外部比喻例子，在同一段中让比喻与法则一起流动。

## 散文化的三段节奏——体验、实际、余韵

双重叙述推荐三段节奏。

> **体验。** 使用该机制的人以自己的感官感受到什么；这是接近外部比喻的部分。

> **实际。** 该体验之下的 IP 法则是什么，实际发生了什么；这是内部法则的部分。

> **余韵。** 两者的差异塑造何种性质，或两者在哪里自然相遇。

前述例子可以分为：体验是“像戴上 VR 头显”；实际是“意识跨越维度的物理过程”；余韵是“人们不说连接，只说进去了”。三段在同一流动中自然相连。读者通过熟悉的比喻进入，在第二段接触 IP 法则，在第三段感到两者的微妙差异。这是初次书写机制散文时的标准建议，并非每篇散文都必须遵守的强制格式。

## 虚构 IP“○○”的案例

以虚构 IP“○○”的巫俗机制为例。将附身写成双重叙述，可以是这样：

```
附身发生的瞬间，对男巫而言与一辆车坐着两个人没有不同。他握着方向盘，
但副驾驶座上有人同坐，偶尔补上一句话。那个人的称呼就是“神”。然而实际
发生的是，巫师意识的一侧打开了另一个自我的领域；在本 IP 的巫俗体系中，
它被视为支撑巫师工作的基础。读者看见男巫金美善在附身状态下工作时，会觉得
他一个人在做两个人的工作。这种感觉是准确的。
```

三段节奏分别是“车里有两个人”的汽车比喻、意识一侧打开另一个自我领域的 IP 法则，以及“一个人做两个人的工作”的余韵。读者从汽车比喻进入，接触意识的变化，最后确认比喻的准确。

同一 IP 的神降也可以用另一种比喻：它像一个人一生一次的入学典礼。人们聚集，一名学生在那里正式获得一种身份。但在本 IP 的巫俗中，这个身份不是学校赋予的，而是与神订立的契约；正式登记发生在此世之外。仪式结束后，巫师作为神的通道回到家中，此后那条通道终生不会断绝。

入学典礼的比喻亲切，神之契约的法则自然流动，而终生不断的通道传达出机制的分量。读者的 IP 有独有机制时，也可用同样方式构造。

## 选择比喻的三项标准

双重叙述中最大的决定是选择哪一种外部比喻。比喻合适，读者立刻进入；比喻失准，整篇散文都会变得别扭。

第一项标准是**读者日常中存在的东西**。好的比喻来自读者经常接触的事物：开车、入学典礼、咖啡馆相会、吃饭、读书、散步等。若 IP 是科幻，日常比喻与 IP 法则之间距离很大；能缩短这种距离的比喻就是好比喻。将维度移动比作 VR 头显便是案例。

第二项是**是否照亮机制的本质**。比喻不必复现全部特征。“一辆车坐两个人”照亮的是附身的本质：意识的一侧存在另一个自我的领域。AI 工作伙伴可以提出候选，但真正准确照亮 IP 性格的比喻，仍需熟悉该 IP 的人决定。

最后是**不违背本 IP 的性格**。即使熟悉的比喻也不能与 IP 冲突。若“○○”接近巫俗、料理与日常，宇宙飞船、机器人、人工智能等科幻比喻便不自然，汽车、入学典礼、餐食等日常比喻更自然；若 IP 是科幻，机械性的比喻则更合适。应使比喻的语调符合 IP 的语调。

## 不要在正文引用法则代码

双重叙述有一项绝对禁止：不可在正文引用法则代码。资料内部可能将机制整理为如下代码：

```
PHYS-M01 附身的意识转移原理
PHYS-M02 附身时的身体变化
PHYS-M03 附身的限制与危险
```

这些是内部整理用的名称，不能原样写进 LOREBOOK 正文。PHYS-M01 对读者并不熟悉，内部整理的名称一旦露出，会破坏氛围。应将代码所指的资料——附身的意识转移原理——自然融入散文。代码只可出现在脚注、附录一术语表或 HTML 注释中的非公开编辑笔记里，绝不进入正文。

## 市场中的范例

**托尔金《精灵宝钻》的伊露维塔之乐。** 它以合唱的比喻处理创世机制。神性存在创作一段音乐、和声与不协和构成世界的形而上学，并不直接讲授，而是从“歌”“音乐”“乐章”等熟悉的概念进入。

**厄休拉·勒古恩《黑暗的左手》中的格森人。** 它通过恋爱、家庭与日常人际关系来呈现两性种族这一科幻机制，而不是直接暴露生物学资料。人物在凯默期相遇、家庭共餐等场景使机制自然显露。

**沃卓斯基《黑客帝国》的连接。** 它由电话、头显与梦的比喻让读者进入虚拟现实。在接触“神经接口”的科幻资料前，读者已通过拿起话筒、沉入睡眠等熟悉动作接受了机制。

**ZA/UM《极乐迪斯科》的“思想柜”。** 它以**把书放进书架**的比喻处理人物的思考过程。抽象的思想内化，经由读者熟悉的日常动作变得可以一次接受。

**FromSoftware《只狼》的不死恢复。** 它用医学和外伤的比喻让读者进入不死能力。伤口愈合的时间与疲劳累积的极限等日常感受，成为科幻奇幻机制的踏脚石。

五个范例都让外部比喻从熟悉的踏脚石引入读者，使内部法则在下方自然流动。这正是 LOREBOOK 散文应遵循的方法。

## 没有机制的 IP

若读者的 IP 没有独有机制，本章方法只能部分适用；但社会制度、职业日常、文化风俗仍可采用相似做法。虚构 IP“○○”的限定继承制，可以将“家族企业选择继承人”的外部比喻与该制度在 IP 内部的运行法则配对。巫师的日常可将咨询师、治疗师、宗教人士的比喻与 IP 中巫师的角色配对。风俗、节庆和日常仪式也可轻度应用。

## 审校的五条合格线

> **比喻是否为读者所熟悉？** 确认读者是否在日常中见过该外部比喻；例如化学实验这类过于专门的比喻未必熟悉。

> **法则是否自然流动？** 确认从比喻转向法则的桥梁自然，读者能在同一流动中跟随。

> **三段节奏是否自然？** 体验、实际与余韵应连成一个流动；若其中一项缺失或过长，散文会显得别扭。

> **法则代码是否未暴露在正文？** 用自动 grep 检查 `PHYS-*` 等代码模式，正文必须为零项。

> **正文中是否没有五项绝对禁止？** 检查是否没有讲课体、夸张的命运、游戏元话语、流行习语和断定性决定论。

双重叙述构建得当时，读者会在 VR 头显、车里的两个人、入学典礼等日常场景中遇见 IP 的机制，而 IP 的法则在其下自然流动。这是一本书在处理科幻、奇幻、巫俗等陌生机制时，仍能为读者铺下熟悉踏脚石的方法。
