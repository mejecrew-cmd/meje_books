# 第7章　三种声调——编年体、现场体、图鉴体

想象书架上并列司马迁《史记》、托尔金《霍比特人》与林奈《自然系统》。《史记》冷静记录事件的时间、人物、结果；《霍比特人》铺陈草地颜色与烟囱气味；《自然系统》用卡片整理分类代码和分布。LOREBOOK把三本书汇入一册。

同一件“水平47年春，七岁女孩在Meryton郊外接受神灵，成为家族第一名巫堂”的事，用三种声调会完全不同。

```
[编年体]
水平47年春，一名女孩在Meryton郊外巫堂家接受神灵。她当时七岁，成为家族第一名巫堂。此事开启三代世袭。

[现场体]
那天黎明也与往日无异。窄巷尽头的院中，香长久燃烧。七岁女孩跪在神坛前，指尖触到香炉温度。就在短暂一刻，某种东西进入她的身体。

[图鉴体]
巫堂入门通过案例047。日期：水平47年春。地点：Meryton郊外。
入门者：7岁女童。家族：第一名巫堂。平均入门年龄：6至9岁。平均通过率：约60%。
```

资料相同，印象却分别成为记录、现场和分类表。450页若只有一种声调会令人疲倦，因此把三种核心声调与四种辅助声调分配到12部。

## 编年体——史

这是历史记录者的声音，不给事件附加感情，把悲剧或荣耀的判断交给读者。它明确时间、地点、人物，避免“某时”“某处”“某人”，并保留判断。“开启三代世袭”是事实，“沉重命运”“辉煌开端”则是叙述者评价。

它适合第3部历史与编年史、附录编年史总览。《史记》《三国史记》《朝鲜王朝实录》以及《Forgotten Realms》编年附录、《Warhammer》帝国史都是范例。

## 现场体——场

这是身在现场者的声音，以气味、声音、距离、人群和日常动作展开。它不看抽象城市，而看一条巷子；不看一般人物，而看一名路人。

```
Hip Street深夜两点后，油味更浓。巷内小餐厅制作夜宵，一两名行人把手插在口袋里匆匆走过。咖啡馆窗后，一名男巫喝着淡茶，杯中蒸汽碰到玻璃，使窗面模糊。
```

它适合第4部地理、第6部日常·文化、第7部类型行动者。感官过多会降低信息密度，一段放一两项感官资料和一两项事实最自然。

## 图鉴体——鉴

这是分类学家的声音，以分类代码、测量值、分布和卡片形式整理，不判断好坏。

```
男巫（学名: Mudang masculus）
分布：城市50%、乡村30%、游走20%
频率：总人口约0.05%
入门：通过神降仪式
细分：神坛守护者、法师、游走巫堂
相关机制：附身、神降、郁结
```

它适合第8部维度·空间、第9部存在、第10部势力。《自然系统》《Monster Manual》、宝可梦图鉴、*Star Wars Visual Dictionary*都是范例。卡片之间应插入短散文，维持通读脉络。

## 四种辅助声调

**序言体**直接向读者问候，**随笔体**在第1部或后记跟随思考流动，**法则解说体**在第2部解释机制原理，**工具体**用于术语表、索引、地图图注与编年史行。它们与核心三种组成七声调。

## 分部声调映射

每部主声调占60%至70%，辅助占30%至40%。序言用序言体；第1部随笔＋现场；第2部法则解说＋图鉴；第3部编年＋随笔；第4部现场＋图鉴；第5部图鉴＋编年；第6部现场＋随笔；第7部现场＋图鉴；第8部图鉴＋编年；第9部图鉴＋现场；第10部图鉴＋编年；附录用工具体。

## 一部内部也分散声调

第9部人物卡可先用图鉴体分类，再用现场体描写日常，最后用编年体概括家族履历。分类登记人物，现场赋予生命，编年赋予时间；主声调中的辅助变化避免单调。

## 五项绝对禁用

七种声调都要避开五种写法：居高临下的**讲课体**；把资料膨胀成“永恒命运”的**夸张宿命**；玩家、骰子、角色生成等**游戏元叙述**；很快过时的**流行俚语**；抹去例外的**断言式决定论**。LOREBOOK应坐在读者身旁，从IP内部展示资料，以能持续五年、十年的语言保留例外。

## 声调审查合格线

第一，主声调应占一部70%以上，辅助不超过30%。第二，读者跨部时应明显感到语调变化。第三，一部内部不能分成50对50，否则需要整体重写。

自动检查能抓住明显讲课体和夸张宿命，但图鉴段落中混入少量讲课语气等细微偏差，以及按IP调整标准映射，仍需人的判断。七声调自然流动时，读者会在同一本书里追随时间、站在巷中、走过分类卡片。
