# 第7章　三種聲調——編年體、現場體、圖鑑體

想象書架上並列司馬遷《史記》、托爾金《霍位元人》與林奈《自然系統》。《史記》冷靜記錄事件的時間、人物、結果；《霍位元人》鋪陳草地顏色與煙囪氣味；《自然系統》用卡片整理分類程式碼和分佈。LOREBOOK把三本書匯入一冊。

同一件“水平47年春，七歲女孩在Meryton郊外接受神靈，成為家族第一名巫堂”的事，用三種聲調會完全不同。

```
[編年體]
水平47年春，一名女孩在Meryton郊外巫堂家接受神靈。她當時七歲，成為家族第一名巫堂。此事開啟三代世襲。

[現場體]
那天黎明也與往日無異。窄巷盡頭的院中，香長久燃燒。七歲女孩跪在神壇前，指尖觸到香爐溫度。就在短暫一刻，某種東西進入她的身體。

[圖鑑體]
巫堂入門透過案例047。日期：水平47年春。地點：Meryton郊外。
入門者：7歲女童。家族：第一名巫堂。平均入門年齡：6至9歲。平均透過率：約60%。
```

資料相同，印象卻分別成為記錄、現場和分類表。450頁若只有一種聲調會令人疲倦，因此把三種核心聲調與四種輔助聲調分配到12部。

## 編年體——史

這是歷史記錄者的聲音，不給事件附加感情，把悲劇或榮耀的判斷交給讀者。它明確時間、地點、人物，避免“某時”“某處”“某人”，並保留判斷。“開啟三代世襲”是事實，“沉重命運”“輝煌開端”則是敘述者評價。

它適合第3部歷史與編年史、附錄編年史總覽。《史記》《三國史記》《朝鮮王朝實錄》以及《Forgotten Realms》編年附錄、《Warhammer》帝國史都是範例。

## 現場體——場

這是身在現場者的聲音，以氣味、聲音、距離、人群和日常動作展開。它不看抽象城市，而看一條巷子；不看一般人物，而看一名路人。

```
Hip Street深夜兩點後，油味更濃。巷內小餐廳製作夜宵，一兩名行人把手插在口袋裡匆匆走過。咖啡館窗後，一名男巫喝著淡茶，杯中蒸汽碰到玻璃，使窗面模糊。
```

它適合第4部地理、第6部日常·文化、第7部型別行動者。感官過多會降低資訊密度，一段放一兩項感官資料和一兩項事實最自然。

## 圖鑑體——鑑

這是分類學家的聲音，以分類程式碼、測量值、分佈和卡片形式整理，不判斷好壞。

```
男巫（學名: Mudang masculus）
分佈：城市50%、鄉村30%、遊走20%
頻率：總人口約0.05%
入門：透過神降儀式
細分：神壇守護者、法師、遊走巫堂
相關機制：附身、神降、鬱結
```

它適合第8部維度·空間、第9部存在、第10部勢力。《自然系統》《Monster Manual》、寶可夢圖鑑、*Star Wars Visual Dictionary*都是範例。卡片之間應插入短散文，維持通讀脈絡。

## 四種輔助聲調

**序言體**直接向讀者問候，**隨筆體**在第1部或後記跟隨思考流動，**法則解說體**在第2部解釋機制原理，**工具體**用於術語表、索引、地圖圖注與編年史行。它們與核心三種組成七聲調。

## 分部聲調對映

每部主聲調佔60%至70%，輔助佔30%至40%。序言用序言體；第1部隨筆＋現場；第2部法則解說＋圖鑑；第3部編年＋隨筆；第4部現場＋圖鑑；第5部圖鑑＋編年；第6部現場＋隨筆；第7部現場＋圖鑑；第8部圖鑑＋編年；第9部圖鑑＋現場；第10部圖鑑＋編年；附錄用工具體。

## 一部內部也分散聲調

第9部人物卡可先用圖鑑體分類，再用現場體描寫日常，最後用編年體概括家族履歷。分類登記人物，現場賦予生命，編年賦予時間；主聲調中的輔助變化避免單調。

## 五項絕對禁用

七種聲調都要避開五種寫法：居高臨下的**講課體**；把資料膨脹成“永恆命運”的**誇張宿命**；玩家、骰子、角色生成等**遊戲元敘述**；很快過時的**流行俚語**；抹去例外的**斷言式決定論**。LOREBOOK應坐在讀者身旁，從IP內部展示資料，以能持續五年、十年的語言保留例外。

## 聲調審查合格線

第一，主聲調應占一部70%以上，輔助不超過30%。第二，讀者跨部時應明顯感到語調變化。第三，一部內部不能分成50對50，否則需要整體重寫。

自動檢查能抓住明顯講課體和誇張宿命，但圖鑑段落中混入少量講課語氣等細微偏差，以及按IP調整標準對映，仍需人的判斷。七聲調自然流動時，讀者會在同一本書裡追隨時間、站在巷中、走過分類卡片。
