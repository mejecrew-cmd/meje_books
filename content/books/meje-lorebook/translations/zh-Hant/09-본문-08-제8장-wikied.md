# 第8章　雙重敘述——外部比喻與內部法則的散文

假設讀者的 IP 有一種名為“維度移動”的機制：人物從自己的維度前往另一個維度的執行原理。LOREBOOK 應如何說明它？有兩種寫法。

一種是直接向讀者講授機制本身。

```
[直接講授]

維度移動是本 IP 的核心機制，是人物從自己的維度前往另一維度的執行原理。
維度之間的移動透過名為“棺艙”的裝置發生；移動時人物的意識轉移，身體仍留在
原來的維度。這一機制是本 IP 所有維度事件的基礎。
```

這並沒有錯誤，資訊也準確；但它少了一樣東西：讀者沒有在腦中描繪維度移動的踏腳石。“棺艙”“意識轉移”等名稱不屬於讀者已有的經驗，讀者可以記住名稱，卻無法親手觸及它的運作。

另一種是讓讀者從熟悉的外部比喻進入。

```
[外部比喻]

躺進棺艙並轉動把手的瞬間，對跑者本人而言，與戴上 VR 頭顯沒有不同。
然而實際發生的是，他的意識從一個維度跨越到另一個維度的物理過程。沒有人稱它
為“連線”；人們只說“進去了”。
```

這就是**雙重敘述**。熟悉的外部比喻（VR 頭顯）讓讀者從踏腳石進入，其下的 IP 法則（意識跨越維度）在同一條敘述中自然流出。讀者於是能夠在腦中描繪機制。

這種方法把外語作品韓語翻譯中形成的模式應用到散文工作。當 IP 有黑洞、神器、維度移動、附身等獨有機制時，它使讀者以熟悉的方式進入，同時不破壞 IP 的氛圍。

## 為什麼不能用表格並列

雙重敘述的一個陷阱，是將外部比喻與內部法則並列在表中。

```
[表格並列：應避免的例子]

| 外部比喻 | 內部法則 |
|---|---|
| VR 頭顯 | 意識轉移 |
| 連線 Matrix | 跨越維度 |
| 登入 | 潛入 |
```

作為資料它是準確的，但在 LOREBOOK 散文中會帶來兩個問題。第一，它降低可讀性：讀者讀到表格時會中斷，掃視表格後再回到正文。

第二，它破壞散文的氛圍。比喻與法則在表中一一對應時，讀者會把它們當成彼此分離的資料。比喻變成單純的說明工具，法則也在表中被固定，讀者無法直接觸及。

應避免表格並列，而讓兩者自然融入散文。正如上面的外部比喻例子，在同一段中讓比喻與法則一起流動。

## 散文化的三段節奏——體驗、實際、餘韻

雙重敘述推薦三段節奏。

> **體驗。** 使用該機制的人以自己的感官感受到什麼；這是接近外部比喻的部分。

> **實際。** 該體驗之下的 IP 法則是什麼，實際發生了什麼；這是內部法則的部分。

> **餘韻。** 兩者的差異塑造何種性質，或兩者在哪裡自然相遇。

前述例子可以分為：體驗是“像戴上 VR 頭顯”；實際是“意識跨越維度的物理過程”；餘韻是“人們不說連線，只說進去了”。三段在同一流動中自然相連。讀者透過熟悉的比喻進入，在第二段接觸 IP 法則，在第三段感到兩者的微妙差異。這是初次書寫機制散文時的標準建議，並非每篇散文都必須遵守的強制格式。

## 虛構 IP“○○”的案例

以虛構 IP“○○”的巫俗機制為例。將附身寫成雙重敘述，可以是這樣：

```
附身發生的瞬間，對男巫而言與一輛車坐著兩個人沒有不同。他握著方向盤，
但副駕駛座上有人同坐，偶爾補上一句話。那個人的稱呼就是“神”。然而實際
發生的是，巫師意識的一側開啟了另一個自我的領域；在本 IP 的巫俗體系中，
它被視為支撐巫師工作的基礎。讀者看見男巫金美善在附身狀態下工作時，會覺得
他一個人在做兩個人的工作。這種感覺是準確的。
```

三段節奏分別是“車裡有兩個人”的汽車比喻、意識一側開啟另一個自我領域的 IP 法則，以及“一個人做兩個人的工作”的餘韻。讀者從汽車比喻進入，接觸意識的變化，最後確認比喻的準確。

同一 IP 的神降也可以用另一種比喻：它像一個人一生一次的入學典禮。人們聚集，一名學生在那裡正式獲得一種身份。但在本 IP 的巫俗中，這個身份不是學校賦予的，而是與神訂立的契約；正式登記發生在此世之外。儀式結束後，巫師作為神的通道回到家中，此後那條通道終生不會斷絕。

入學典禮的比喻親切，神之契約的法則自然流動，而終生不斷的通道傳達出機制的分量。讀者的 IP 有獨有機制時，也可用同樣方式構造。

## 選擇比喻的三項標準

雙重敘述中最大的決定是選擇哪一種外部比喻。比喻合適，讀者立刻進入；比喻失準，整篇散文都會變得彆扭。

第一項標準是**讀者日常中存在的東西**。好的比喻來自讀者經常接觸的事物：開車、入學典禮、咖啡館相會、吃飯、讀書、散步等。若 IP 是科幻，日常比喻與 IP 法則之間距離很大；能縮短這種距離的比喻就是好比喻。將維度移動比作 VR 頭顯便是案例。

第二項是**是否照亮機制的本質**。比喻不必復現全部特徵。“一輛車坐兩個人”照亮的是附身的本質：意識的一側存在另一個自我的領域。AI 工作夥伴可以提出候選，但真正準確照亮 IP 性格的比喻，仍需熟悉該 IP 的人決定。

最後是**不違背本 IP 的性格**。即使熟悉的比喻也不能與 IP 衝突。若“○○”接近巫俗、料理與日常，宇宙飛船、機器人、人工智慧等科幻比喻便不自然，汽車、入學典禮、餐食等日常比喻更自然；若 IP 是科幻，機械性的比喻則更合適。應使比喻的語調符合 IP 的語調。

## 不要在正文引用法則程式碼

雙重敘述有一項絕對禁止：不可在正文引用法則程式碼。資料內部可能將機制整理為如下程式碼：

```
PHYS-M01 附身的意識轉移原理
PHYS-M02 附身時的身體變化
PHYS-M03 附身的限制與危險
```

這些是內部整理用的名稱，不能原樣寫進 LOREBOOK 正文。PHYS-M01 對讀者並不熟悉，內部整理的名稱一旦露出，會破壞氛圍。應將程式碼所指的資料——附身的意識轉移原理——自然融入散文。程式碼只可出現在腳註、附錄一術語表或 HTML 註釋中的非公開編輯筆記裡，絕不進入正文。

## 市場中的範例

**托爾金《精靈寶鑽》的伊露維塔之樂。** 它以合唱的比喻處理創世機制。神性存在創作一段音樂、和聲與不協和構成世界的形而上學，並不直接講授，而是從“歌”“音樂”“樂章”等熟悉的概念進入。

**厄休拉·勒古恩《黑暗的左手》中的格森人。** 它透過戀愛、家庭與日常人際關係來呈現兩性種族這一科幻機制，而不是直接暴露生物學資料。人物在凱默期相遇、家庭共餐等場景使機制自然顯露。

**沃卓斯基《駭客帝國》的連線。** 它由電話、頭顯與夢的比喻讓讀者進入虛擬現實。在接觸“神經介面”的科幻資料前，讀者已透過拿起話筒、沉入睡眠等熟悉動作接受了機制。

**ZA/UM《極樂迪斯科》的“思想櫃”。** 它以**把書放進書架**的比喻處理人物的思考過程。抽象的思想內化，經由讀者熟悉的日常動作變得可以一次接受。

**FromSoftware《只狼》的不死恢復。** 它用醫學和外傷的比喻讓讀者進入不死能力。傷口癒合的時間與疲勞累積的極限等日常感受，成為科幻奇幻機制的踏腳石。

五個範例都讓外部比喻從熟悉的踏腳石引入讀者，使內部法則在下方自然流動。這正是 LOREBOOK 散文應遵循的方法。

## 沒有機制的 IP

若讀者的 IP 沒有獨有機制，本章方法只能部分適用；但社會制度、職業日常、文化風俗仍可採用相似做法。虛構 IP“○○”的限定繼承製，可以將“家族企業選擇繼承人”的外部比喻與該制度在 IP 內部的執行法則配對。巫師的日常可將諮詢師、治療師、宗教人士的比喻與 IP 中巫師的角色配對。風俗、節慶和日常儀式也可輕度應用。

## 審校的五條合格線

> **比喻是否為讀者所熟悉？** 確認讀者是否在日常中見過該外部比喻；例如化學實驗這類過於專門的比喻未必熟悉。

> **法則是否自然流動？** 確認從比喻轉向法則的橋樑自然，讀者能在同一流動中跟隨。

> **三段節奏是否自然？** 體驗、實際與餘韻應連成一個流動；若其中一項缺失或過長，散文會顯得彆扭。

> **法則程式碼是否未暴露在正文？** 用自動 grep 檢查 `PHYS-*` 等程式碼模式，正文必須為零項。

> **正文中是否沒有五項絕對禁止？** 檢查是否沒有講課體、誇張的命運、遊戲元話語、流行習語和斷定性決定論。

雙重敘述構建得當時，讀者會在 VR 頭顯、車裡的兩個人、入學典禮等日常場景中遇見 IP 的機制，而 IP 的法則在其下自然流動。這是一本書在處理科幻、奇幻、巫俗等陌生機制時，仍能為讀者鋪下熟悉踏腳石的方法。
