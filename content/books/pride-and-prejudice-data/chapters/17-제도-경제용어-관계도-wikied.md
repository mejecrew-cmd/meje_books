# Pride and Prejudice — 제도 경제용어 관계도

> 이 문서는 작중 줄거리를 움직이는 **법률·경제·사회 제도**를 체계적으로 정리한다.
> 이 용어들을 이해하지 못하면 인물들의 행동 동기와 위기의 본질을 번역할 수 없다.

---

## 1. 결혼 경제학 — 왜 모두가 결혼에 목숨을 거는가

### 핵심 구조: 지참금(Fortune) → 정착금(Settlement) → 용돈(Pin-money)

```
[신부 측]                              [신랑 측]
Fortune (지참금/재산)                  Estate / Income (영지/수입)
    │                                      │
    └──→ Marriage Articles (혼인 계약) ←──┘
              │
              ├──→ Settlement (결혼 정착금)
              │      남편 사후 미망인에게 보장되는 연금
              │
              ├──→ Pin-money (품위 유지비)
              │      아내가 남편 간섭 없이 쓸 수 있는 연간 고정금
              │
              └──→ Jointure (과부 연금)
                     Settlement의 구체적 형태
```

### 작중 적용

| 인물 | 지참금/재산 | 결혼 정착금 | 번역 시 핵심 |
|---|---|---|---|
| **베넷 자매 각자** | 약 1,000파운드 (어머니 사후 5,000파운드의 1/5) | — | "4% 공채(4 per cents)"로 운용. 연 이자 약 40파운드 — 독립 불가능한 금액 |
| **리디아-위컴** | 리디아 몫 1,000파운드 + 연 100파운드(아버지 생전) | 다아시가 위컴에게 추가 비용 지불 | 베넷 씨 발언: "이렇게 적은 돈으로 해결되다니" — 실제로는 다아시가 대부분 부담 |
| **조지아나 다아시** | 30,000파운드 | — | 위컴이 조지아나를 유혹한 진짜 이유 |
| **드 버그 양** | 로징스 영지 전체 | — | 다아시와의 정략결혼이 "두 영지의 통합"을 의미 |
| **미스 킹** | 10,000파운드 (조부 유산) | — | 위컴이 엘리자베스에서 전향한 이유 |

### 용어 번역 가이드

| 원문 | 추천 번역 | 절대 안 되는 번역 | 이유 |
|---|---|---|---|
| Fortune | 재산 / 지참금 | 행운 | 결혼 맥락에서는 반드시 "재산" |
| Settlement | 결혼 정착금 / 재산 합의서 | 정착, 합의 | 법적 구속력 있는 재산 계약 |
| Pin-money | 품위 유지비 / 아내의 용돈 | 핀 돈 | 혼인 계약에 명시된 공식 비용 |
| 4 per cents | 4퍼센트 공채 | 4퍼센트 | 영국 정부 채권의 고유명사 |
| Portion | 지분 / 몫 | 부분 | 유산에서의 법적 배분 |

## 2. 한정상속제 (Entail) — 소설의 핵심 갈등 동인

### 구조

```
[영지 소유자: Mr. Bennet]
    │
    │  한정상속제(Entail)에 의해
    │  딸에게는 상속 불가
    │
    └──→ [법적 상속인: Mr. Collins]
              │
              ├── Mr. Bennet 사망 시 → 롱번 영지 전체 계승
              ├── Mrs. Bennet + 딸 5명 → 퇴거해야 함
              └── 대안: 아들이 태어나면 Entail을 공동으로 해제(cut off) 가능했으나
                         딸만 5명 연속 → 해제 불가능
```

### 사건별 Entail의 서사적 기능

| 사건 | Entail의 역할 |
|---|---|
| EVT-011 (콜린스 도착) | 추상적 법률이 구체적 인물로 체현 — "나의 사랑스러운 사촌들에게 보상하겠다" |
| EVT-017 (콜린스 청혼) | 엘리자베스의 거절 = 한정상속의 "보상"을 거부 |
| EVT-020 (샬럿 수락) | 샬럿이 콜린스를 수락 = 롱번의 미래 안주인 확정 → 베넷 부인의 적대 |
| Chap 50 | 베넷 씨의 후회: "아들이 태어나리라 믿고 저축하지 않았다" |

### 번역 포인트
- **"entail"을 매번 같은 단어로 번역**: "한정상속(제)" 통일. "영지 상속 제한", "한정 상속법" 등으로 흔들리지 않아야 함.
- 베넷 부인이 "I cannot bear to hear that mentioned"이라 할 때의 "that" = entail. 번역에서도 이 단어를 직접 쓰지 않고 회피하는 뉘앙스를 살려야 함.

## 3. 성직 경제학 — 교구 목사는 어떻게 먹고사는가

### 핵심 구조

```
[후원자(Patron)]
    │
    ├──[Patronage: 성직 추천권]──→ 교구 목사 임명
    │
    ├──[Presentation of the living: 성직록 부임권]──→ 특정 교구의 수입 제공
    │
    └──[Advowson: 성직 임명권 소유]──→ 영구적 권리 (영지에 부속)
    
[교구 목사(Rector/Vicar)]
    │
    ├── 수입원 1: Tithes (십일조) — 교구민이 내는 세금/농산물
    ├── 수입원 2: Glebe land (교구 토지) — 목사관에 딸린 농지
    └── 수입원 3: Fees (수수료) — 세례, 결혼, 장례 등

[서품(Taking orders)]
    └── 대학 졸업 후 주교에게 서품을 받아야 목사가 될 수 있음
```

### 작중 적용

| 인물/사건 | 성직 경제학의 역할 |
|---|---|
| **콜린스** | 레이디 캐서린의 후원(Patronage)으로 헌스포드 교구 목사직 획득. 십일조를 "후원자 심기를 거스르지 않는 선에서" 합의해야 한다고 주장 → 아첨의 경제적 근거 |
| **위컴-다아시 갈등의 본질** | 선대 다아시가 위컴에게 최고 교구(the best living)를 유언으로 약속. 위컴은 서품을 포기하고(resolved against taking orders) 대신 3,000파운드를 받아 탕진. 3년 후 현직 목사(incumbent)가 사망하자 다시 성직록을 요구 → 거절당함 |
| **다아시의 권력** | 다수의 교구에 성직 추천권을 보유. 피츠윌리엄 대령에게는 이 권한이 없음 → 샬럿이 엘리자베스를 다아시와 맺어주려 한 은밀한 이유 |

### 용어 번역 가이드

| 원문 | 추천 번역 | 설명 |
|---|---|---|
| Living | 성직록 / 교구 수입 | 단순한 "생활"이 아니라 교구에서 나오는 공식 수입 |
| Patronage | 후원 / 성직 추천권 | 목사를 임명하고 해임할 수 있는 귀족의 권리 |
| Presentation | 성직록 부임권 / 교구 임명 | 특정 교구에 목사를 "보내는" 행위 |
| Taking orders | 서품을 받다 / 성직에 나아가다 | 주교에게 공식 성직자로 서품받는 의식 |
| Incumbent | 현직 목사 / 재임자 | 현재 그 교구를 맡고 있는 사람 |
| Rector | (교구) 목사 | 십일조 전부를 받는 정식 목사 (Vicar는 일부만 받음) |
| Tithes | 십일조 (세입) | 교구민이 내는 세금/농산물 — 목사의 주 수입원 |

## 4. 군대 경제학 — 장교직은 어떻게 얻는가

### 핵심 구조

```
[민병대(Militia)]                    [정규군(Regulars)]
  │                                    │
  ├── 국내 치안/방위 담당               ├── 해외 파병 포함 실전 투입
  ├── 각 지역 순회 주둔                 ├── 고정 주둔지
  ├── 장교직: 비교적 저렴               ├── 장교직: 매관제(Purchase system)
  │                                    │     └── 돈을 내고 계급을 사고팖
  └── 위컴의 소속 (도주 전)             └── 위컴의 소속 (도주 후, 다아시가 매입)

[계급 서열 (하→상)]
  Ensign (보병 소위) → Lieutenant (중위) → Captain (대위) → Colonel (대령)
  
  ※ Ensigncy(소위직)는 정규군 최하위 장교 계급
  ※ 다아시가 위컴을 위해 산 것이 바로 이 소위직
```

### 작중 적용

| 용어 | 작중 맥락 | 번역 포인트 |
|---|---|---|
| **Militia regiment** | 메리턴에 겨울 주둔. 리디아-키티의 광란의 대상. 장교들의 붉은 코트(red coats)가 시각적 상징 | "민병대 연대" 통일 |
| **Commission** | 장교직 자체를 의미. "accepted a commission" = 장교가 되었다 | "임관" 또는 "장교직" |
| **Commission purchased** | 돈으로 장교직을 구매. 다아시가 위컴의 소위직을 사줌 | "장교직 매입 / 임관 비용 대납" |
| **Ensigncy** | 정규군 최하위 장교 계급. 가장 저렴한 장교직 | "보병 소위직" |
| **Colonel** | 연대 지휘관. 포스터 대령, 피츠윌리엄 대령 | "대령" — 작중 두 명의 대령이 전혀 다른 성격 |

## 5. 사교 제도 — 결혼 시장의 규칙

### 핵심 용어

| 원문 | 번역 | 설명 | 작중 맥락 |
|---|---|---|---|
| **Assembly** | 어셈블리 / 지역 무도회 | 지역 유지가 모여 춤추는 공적 사교 행사. 누가 누구와 춤추는지가 사회적 신호 | 메리턴 무도회(EVT-003)가 모든 것의 출발점 |
| **Ball** | (정식) 무도회 | Assembly보다 격식. 개인 저택에서 주최 | 네더필드 무도회(EVT-016) |
| **Coming out** | 데뷔 / 사교계 진출 | 어린 딸이 성인으로서 무도회에 나갈 자격을 얻는 것 | 레이디 캐서린이 "다섯 딸이 모두 데뷔?"라며 비난 |
| **Accomplishment** | 교양 / 숙녀의 기예 | 음악, 그림, 춤, 언어 등 결혼 시장에서의 가치를 높이는 기술 | Chap 8의 "교양 논쟁" 참조 (사물 사전 §5) |
| **Special licence** | 특별 혼인 허가증 | 캔터베리 대주교청 발급. 시간·장소 제한 없이 결혼 가능. 매우 비쌈 | 베넷 부인이 다아시의 부에 취해 요구 |
| **Banns** | 결혼 예고 | 교구 교회에서 3주 연속 공고 후 결혼. 일반적 절차 | (작중 직접 언급은 적으나 Special licence의 대비 개념) |

## 6. 제도 간 연결 관계도 — 전체 구조

```
━━━━━━━━━━━━━━━━ 영지 경제 ━━━━━━━━━━━━━━━━
│                                              │
│  영지(Estate) ──→ 연간 수입(Income)          │
│      │                                       │
│      ├──[Entail]──→ 남성 상속인에게만 상속    │
│      │                                       │
│      ├──[Manor]──→ 수렵권·어업권 등 부속 권리 │
│      │                                       │
│      └──[Patronage]──→ 교구 목사 임명권       │
│                                              │
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

         ↓ 연간 수입이 곧 결혼 시장에서의 가치

━━━━━━━━━━━━━━━━ 결혼 경제 ━━━━━━━━━━━━━━━━
│                                              │
│  남성의 수입(Income) + 여성의 지참금(Fortune) │
│      │                                       │
│      └──→ Marriage Articles (혼인 계약)       │
│              │                               │
│              ├──→ Settlement (정착금)         │
│              ├──→ Pin-money (품위유지비)      │
│              └──→ Jointure (과부 연금)        │
│                                              │
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

         ↓ 결혼이 불가능한 경우의 대안

━━━━━━━━━━━━━━━━ 직업 경제 ━━━━━━━━━━━━━━━━
│                                              │
│  성직(Church)                                │
│    └── Patronage → Living → Tithes           │
│                                              │
│  군대(Army)                                  │
│    └── Commission purchased → Ensigncy       │
│                                              │
│  법률(Law)                                   │
│    └── Attorney / Solicitor (사무변호사)      │
│        (필립스 씨, 가디너 씨 부친)            │
│                                              │
│  상업(Trade)                                 │
│    └── 가디너 씨 — 존경받으나 계급적 편견 대상│
│                                              │
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 이 구조가 줄거리에 미치는 영향

| 제도 | 줄거리 동인 |
|---|---|
| **Entail** | 베넷 자매의 결혼이 절박한 이유. 콜린스의 존재 이유. 샬럿의 결혼 동기. |
| **Fortune/Settlement** | 위컴이 조지아나(3만 파운드)를 노린 이유. 리디아 결혼의 경제적 조건. 콜린스가 "당신의 재산이 너무 적어 다른 기회가 없다"며 모욕한 근거. |
| **Patronage/Living** | 다아시-위컴 갈등의 핵심. 콜린스의 레이디 캐서린 아첨의 경제적 근거. |
| **Commission/Ensigncy** | 다아시가 위컴을 '처리'한 방법 — 소위직을 사주고 북부로 전출시킴(사실상 추방). |
| **Trade(상업)** | 빙리 가문의 재산이 상업에서 유래한 콤플렉스. 가디너 삼촌이 칩사이드에 산다는 이유로 빙리 자매의 경멸 대상. 그러나 다아시가 가디너 부부를 기꺼이 받아들이는 것이 그의 변화를 증명. |

## 7. 작중 금액 환산 참고표

| 작중 금액 | 의미 | 현대 환산 (매우 대략적) |
|---|---|---|
| 연 10,000파운드 | 다아시의 수입 — 최상위 젠트리/준귀족 | 약 연 10-15억 원 |
| 연 4-5,000파운드 | 빙리의 수입 — 상위 젠트리 | 약 연 4-7억 원 |
| 연 2,000파운드 | 베넷 씨의 수입 — 중위 젠트리 | 약 연 2-3억 원 |
| 100,000파운드 | 빙리의 상속 재산 총액 | 약 100-150억 원 |
| 30,000파운드 | 조지아나의 지참금 | 약 30-50억 원 |
| 10,000파운드 | 미스 킹의 유산 | 약 10-15억 원 |
| 5,000파운드 | 베넷 부인의 지참금 (딸 5명이 나눔) | 약 5-7억 원 (전체) |
| 3,000파운드 | 위컴이 성직록 대신 받은 금액 | 약 3-5억 원 |
