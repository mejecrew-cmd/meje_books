# Pride and Prejudice — 건물 평면도 (추정/창작)

> ⚠️ **이 문서는 추정/창작입니다.** 원문에 등장하는 방·동선·창문·정원의 단편적 묘사를 종합하여 합리적 평면도를 재구성한 것입니다. 영국 조지안/팔라디오 양식의 표준 + 작품 내 디테일을 결합.

## 1. 펨벌리 (Pemberley) — 다아시 영지

### 외부 배치 (Ch.43 묘사 기반)

```
                       [북쪽: 숲이 우거진 언덕 능선]
                              ┃
   [정원 + 잔디밭]  ━━━ [본관: 석조 대저택] ━━━ [마구간]
        ┃                     ┃                    ┃
   [강 (Stream)]              ┃              [후문 도로]
        ┃                     ┃ (← 가디너 일행 진입)
   [다리 (Bridge)]      [정문 + 자갈 진입로]
        ┃
   [협곡 + 산책로 (10마일 둘레)]

  [영지 입구 lodge]  ━━━ [반 마일 숲길] ━━━ [언덕 위 첫 시야]
```

### 본관 내부 추정

#### 1층 (Ground Floor)
```
┌─────────────────────────────────────────────────────┐
│  Entrance Hall (입구 홀, "fine proportion")         │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ │
│                                                     │
│  [Dining Parlour]  [Drawing Room]  [Library]        │
│  강가 전망 +        [Saloon (대응접실)]               │
│  대형 식탁          [Music Room]                    │
│                                                     │
│  [Breakfast Parlour]  [Mr. Darcy's Study]           │
└─────────────────────────────────────────────────────┘
```
> **명시 단서:** "Dining-parlour... large, well-proportioned room" / "lofty and handsome rooms" / 모든 창문에서 다른 풍경

#### 2층 (First Floor)
```
┌─────────────────────────────────────────────────────┐
│  [Picture Gallery (그림 갤러리)]                    │
│  ━━━ 가족 초상화들 + 다아시 대형 초상 ━━━           │
│                                                     │
│  [Georgiana's Sitting Room]  ← 새로 단장됨          │
│  "lately fitted up with greater elegance"           │
│                                                     │
│  [Principal Bedrooms 2-3개]                         │
│  [Mr. Darcy's Bedroom]                              │
│  [Georgiana's Bedroom]                              │
└─────────────────────────────────────────────────────┘
```
> **명시 단서:** "spacious lobby above" / "picture gallery" / "principal bed-rooms"

#### 부속 (Outbuildings)
- 마구간 (Stables) — 다아시가 후문에서 등장 (Ch.43)
- 온실 (Hothouse) — 포도/넥타린/복숭아 재배 추정
- 가축 우리, 곡물 창고
- 작은 사원/정자 (decorative folly) 가능성

## 2. 로징스 (Rosings) — 켄트

### 외부 배치 (Ch.28-29 기반)

```
   [Rosings Park 영지 - 거대한 공원]
            ┃
   [본관: handsome modern building, on rising ground]
            ┃
   [전면: 유리창 다수 (창문세 = 부의 상징)]
            ┃
   [Park 경계 = paling (나무 울타리)]
            ┃
   [좁은 lane (도로)]
            ┃
   [Hunsford Parsonage 정원]
```

### 본관 내부 추정 (콜린스 묘사 + 첫 방문 묘사)

#### 1층
```
┌─────────────────────────────────────────────────────┐
│  Entrance Hall (입구 홀)                            │
│  → Antechamber (대기실) → ━━━━━━━━━━━━━━━━━━━━━━━ │
│                                                     │
│  [큰 응접실 (드 버그 양 포함 가족 거실)]              │
│  [Drawing Room]                                     │
│                                                     │
│  [Dining Room - 거대 식탁, 화려한 은식기]            │
│  [Small Summer Breakfast Parlour]                   │
│  ← 콜린스가 필립스 가 응접실을 이것에 비교           │
│                                                     │
│  [Music Room with capital instrument]               │
└─────────────────────────────────────────────────────┘
```

#### 부속
- **Mrs. Jenkinson's Room** — 피아노 비치. 엘리자베스에게 "와서 연습하라" 권한 곳
- **Anne de Bourgh's Sitting Room** (병약함을 고려)
- **Sir Lewis de Bourgh의 옛 집무실** (현재 비어있음)
- 마차 보관소 (carriage-house) — 마차 여러 대
- 온실 (광활한 영지 → 필수)

### 정원
- **자갈 산책로** (gravel walks)
- **공원 (park)** — 헌스포드 목사관과 paling으로 분리
- **숲 (grove)** — 엘리자베스가 산책하던 곳

## 5. 헌스포드 목사관 (Hunsford Parsonage)

### 외부 배치 (Ch.28 기반)

```
   [Rosings Park 경계 (paling)]
              ┃
   [Lane (좁은 도로)]
              ┃
   [Small Gate (작은 문)]
              ┃
   [Garden + Gravel Walk]
              ┃
   [본관: 작지만 잘 지어진 집]
              ┃
   [뒷마당 + 두 개의 목초지 (meadows)]
```

### 본관 내부 추정

#### 1층
```
┌─────────────────────────────────────────────────────┐
│  Entrance + Hall                                    │
│                                                     │
│  [Parlour (응접실)] ← 후면. 정원 안 보임            │
│  ↳ 샬럿이 의도적으로 후면 방 선택                   │
│  ↳ Ch.34: 다아시의 첫 청혼 장면                     │
│                                                     │
│  [Dining Room]                                      │
│                                                     │
│  [Mr. Collins's Book Room (서재)] ← 도로 쪽         │
│  ↳ 콜린스가 마차 감시 + 정원 일                     │
└─────────────────────────────────────────────────────┘
```

> **샬럿의 영리한 배치:** 콜린스의 서재가 도로 쪽 + 응접실이 후면. 콜린스가 자기 방에 머물러 부인을 덜 괴롭힘.

#### 2층
- 콜린스 부부 침실
- 손님 침실 (엘리자베스, 마리아용)

#### 부속
- 정원 (콜린스 직접 관리)
- 두 개의 목초지

## 번역/각색 시 활용

| 시나리오 | 활용 |
|---|---|
| **드라마/영화 미술팀** | 각 저택의 방 구성을 일관되게 디자인 |
| **무대 각색** | 핵심 장면(Ch.34 청혼, Ch.43 펨벌리 갤러리, Ch.56 wilderness 협박)의 공간 재현 |
| **번역 묘사 보완** | 원문의 모호한 공간 묘사를 자연스럽게 풀 때 참조 |
| **소설 각색** | 인물 동선 추가 시 공간적 합리성 확보 |
