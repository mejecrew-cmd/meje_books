# Pride and Prejudice — 번역/각색 레퍼런스 패키지

Jane Austen의 *Pride and Prejudice* (1813) 원문을 번역하거나 각색할 때 필요한 **모든 참조 자료**를 체계적으로 정리한 패키지.

**작업 표준**: `../MEJE서연각작업skill_v3.md` (v3.0, 2026-04-21)
