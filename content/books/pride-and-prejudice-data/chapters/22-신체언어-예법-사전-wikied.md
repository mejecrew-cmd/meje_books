# Pride and Prejudice — 신체언어 예법 사전

> 리젠시 시대의 신체적 매너 + 예법 + 동작을 정리. 영상/무대 디렉팅의 절대적 기준.
> 잘못된 동작 = 시대극의 신뢰성 붕괴.

---

## 1. 인사 (Greetings)

### Bow (남성의 인사)

#### 격식 정도

| 종류 | 동작 | 사용 |
|---|---|---|
| **Slight bow** | 머리 살짝 숙임. 4-5도 | 일상적 인사. 동등한 신분 |
| **Bow from the neck** | 목만 숙임. 10-15도 | 표준 인사 |
| **Bow from the waist** | 허리 숙임. 30-45도 | 격식 (귀부인, 상관) |
| **Deep bow** | 60도 이상 + 한쪽 발 약간 뒤로 | 최고 격식 (왕족, 첫 만남) |

#### 작중 예시
- **다아시 → 콜린스** (Ch.18 무도회): "slight bow" — 격하 의도
- **콜린스 → 모든 사람**: 깊은 격식 — 그의 위선
- **위컴 → 다아시** (Ch.15 거리): "touched his hat" — 모자 만지기 정도, 거의 무시

#### 번역 포인트
- "bowed" = 단순 "인사했다"가 아니라 **신체적 동작**으로 시각화
- 격식 정도가 관계 신호

### Curtsey (여성의 인사)

#### 종류

| 종류 | 동작 | 사용 |
|---|---|---|
| **Bob curtsey** | 무릎 가볍게 굽힘 | 일상 |
| **Half curtsey** | 무릎 더 굽히고 머리 숙임 | 표준 |
| **Full court curtsey** | 한쪽 발 깊이 뒤로, 무릎 깊이 굽히기 | 궁정/최고 격식 |

#### 동작 디테일
- **양손**: 드레스 옆을 살짝 잡거나 자연스럽게 둠
- **눈**: 시선은 살짝 아래로 (격식 클수록)
- **시간**: 1-2초 유지

#### 작중 예시
- **엘리자베스 → 다아시** (Ch.30): "merely courtesied to him, without saying a word" — **단순 curtsey + 침묵 = 냉담의 신호**
- **마리아 루카스 → 레이디 캐서린**: 깊은 curtsey (긴장)

### Handshake — **거의 안 함**

- 리젠시 시대에는 악수가 **친밀의 표시** (현대처럼 일반적 인사 아님)
- 처음 만난 사이에는 절대 안 함
- 작중: Ch.61에서 빙리 양이 엘리자베스에게 "even shook hands" — **양보의 표시**

## 2. 호칭 부르는 동작

### 첫 만남 후 정식 소개 (Introduction)

```
[규칙]
- 신분 낮은 자가 높은 자에게 소개됨
- 남성이 여성에게 소개됨 (여성이 결정권)
- 소개 없이 말 거는 것 = 무례

[절차]
1. 매개자(중매인)가 두 사람 사이에 섬
2. "Mr. Darcy, may I introduce Miss Elizabeth Bennet?"
   - 신분 높은 사람 이름 먼저
3. 양쪽이 인사 (bow + curtsey)
4. 짧은 대화 가능

[작중 예시]
- 콜린스가 다아시에게 자기소개 (Ch.18) — 무례. 다아시 경멸
- 사회자(Master of Ceremonies)가 무도회에서 소개 가능
```

### 모자 (Hat) 매너

| 상황 | 동작 |
|---|---|
| 여성에게 인사 | **모자 들기** (touch + lift) |
| 실내 진입 | **모자 벗기** |
| 지인 만남 | "touched his hat" — 위컴-다아시 (Ch.15) |
| 여성과 대화 시 | 모자 들고 시작 → 들고 있거나 옆구리 |

## 3. 무도회 매너

### 입장

```
[순서]
1. 외투/모자/장갑을 클로크룸에 맡김
2. 응접실(antechamber)에서 잠시 대기
3. 사회자에게 도착 통보 ("Mr. and Mrs. Bennet, with their daughters")
4. 메인 홀 입장
5. 호스트/호스티스에게 인사
```

### 춤 신청 절차

```
[남성이 여성에게]
1. 정식으로 소개되어 있어야 함
2. 다가가서 bow + "May I have the honour of the next dance, Miss Bennet?"
3. 여성이 수락 → 곡이 시작될 때까지 옆에 서서 대화
4. 곡 시작 → 함께 무도 라인으로 이동
```

### 거절 규칙

```
[규칙]
- 한 남성을 거절하면 그 곡은 누구와도 안 춤 (관례)
- 표면 이유: "I am engaged" (이미 약속 있음) 또는 "I am too tired"
- 노골적 거절 = 사회적 모욕

[작중 예시]
- Ch.6: 엘리자베스가 다아시 거절 → "I have not the least intention of dancing"
   = 모욕적 거절. 다아시가 오히려 매혹됨
- Ch.18: 엘리자베스가 다아시 신청 수락 — "without knowing what she did"
   = 무의식적 수락
```

### 춤 동작

#### Country Dance (가장 흔함)

```
[배치]
- 두 줄 (남자 줄 + 여자 줄)이 마주 봄
- 한 쌍씩 양 끝에서 시작 → 점차 라인 따라 이동

[동작]
1. 마주 보고 인사 (bow + curtsey)
2. 손 잡고 한 바퀴
3. 4번 스텝 앞으로 → 4번 뒤로
4. 도시도(do-si-do) — 등 맞대고 한 바퀴
5. 라인 따라 다음 쌍과 반복
6. 30분-1시간 한 곡 (여러 번 반복)
```

#### 손 잡는 방식

```
[격식 — 두 사람]
- 손바닥 마주 + 손가락은 살짝만 닿음
- 절대 깍지 끼지 않음
- 장갑 착용 (스킨십 차단)

[더 격식적 — 손가락 끝만]
- 검지+중지만 살짝 잡음
- 격식 무도회

[작중 의미]
- 손가락 끝만 잡으면 = 거리감
- 손바닥 전체 잡으면 = 친밀
- 다아시-엘리자베스의 춤 (Ch.18) — 어떤 식이었을지가 화면에서 의미
```

### Reel (스코틀랜드)

- 더 빠르고 활기참
- 4명 또는 더 많은 인원
- 회전 동작 많음
- 다아시가 엘리자베스에게 권한 춤 (Ch.10) — **활기찬 춤 = 도발적 제안**

### Quadrille (격식 4인 춤)

- 네 쌍이 사각형 배치
- 복잡한 스텝
- 1815년경부터 영국 인기 (작중 시점에는 막 들어옴)

### Boulanger (마지막 춤)

- 무도회 마지막 곡
- 모든 인원 원형 배치
- 활기차고 친근

## 4. 식탁 매너

### 자리 배치

```
[격식 정찬]

[호스트(주인)]                    [호스티스(안주인)]
   ↓                                   ↓
   ━━━━━━━ 식탁 ━━━━━━━
   
   호스티스 옆 = 주빈 (가장 중요한 손님)
   호스트 옆 = 부빈 (두 번째 중요)
   끝자리 = 가장 낮은 신분 / 친척

[작중 예시]
- Ch.29 로징스: 콜린스가 식탁 끝(bottom) — 가장 낮은 자리
- Ch.18 무도회 supper: 엘리자베스 vs 다아시가 마주 앉음 — 의도적 배치
```

### 음식 서빙 (à la française)

```
[방식]
1. 모든 음식을 식탁에 한꺼번에 펼침
2. 호스트가 메인 요리 자르기 시작
3. 각자 가까운 음식을 덜어 먹음
4. 멀리 있는 음식은 옆 사람에게 부탁
   "Could I trouble you for the salt?"
5. 1-2시간 식사 진행
```

### 기본 매너

| 행동 | 매너 |
|---|---|
| 냅킨 | 무릎에 (옷에 안 걸침) |
| 포크 | **왼손** (영국식) |
| 칼 | 오른손 |
| 음식 자를 때 | 한 입 분량씩만 |
| 입에 음식 있을 때 | **말하지 않음** |
| 와인 | 옆 사람과 함께 마시는 의례 ("I drink to you") |
| 식사 중 떠남 | 매우 무례 |

### 디저트 후

```
1. 호스티스가 일어나면 모든 여성 일어남
2. 여성들 응접실로 (Drawing Room)
3. 남성들은 식탁에 남아 포트와인 + 정치 대화
4. 30분-1시간 후 남성들 응접실 합류 → 차/커피
```

## 5. 응접실 매너

### 자리 배치

```
[표준 배치]
- 호스티스: 차 테이블 옆 (티팟 옆)
- 손님 여성: 호스티스 가까이
- 남성: 자유롭게
- 노인: 안락의자 (벽난로 가까이)

[활동]
- 차 마시기
- 카드 게임
- 음악 (피아노)
- 자수, 바느질
- 책 읽기 (가능하나 사교 무시)
- 산책 (정원으로)
```

### 차 따르기

```
[호스티스의 의례]
1. 작은 컵에 우유 먼저 (일부)
2. 차를 따르기
3. 설탕은 손님이 직접
4. 컵 + 받침을 손님에게 전달

[손님 매너]
- 받침을 왼손에 + 컵을 오른손에
- 한 모금씩
- 시끄러운 소리 안 냄
```

## 6. 방문(Calling) 매너

### Morning Call

```
[규칙]
- "Morning call" = 실제로는 오전 11시-오후 3시
- 15-30분만 머묾 (장기 체류 무례)
- 응접실에서 받음 (침실/식당 X)

[순서]
1. 도착: 명함(visiting card) 전달 또는 직접 안내
2. 응접실로 안내됨
3. 호스티스가 들어와 인사
4. 짧은 대화 (날씨, 안부, 가십)
5. 차 권유 (수락 또는 거절)
6. 짧은 후 일어나서 인사 + 퇴장

[명함의 의미]
- "I have called" — 방문 사실 통보
- 모서리 접음 = 본인이 직접 옴 (하인 시킨 게 아님)
- 답방 의무 발생 (일주일 내)
```

### 작중 적용
- Ch.3: 빙리가 베넷 씨에게 답방 — 10분 (격식)
- Ch.55: 빙리가 한 시간 머묾 — **친밀 표시**
- Ch.55: 빙리 양이 답방 짧게 + 차가움 — **거리 두기**

## 7. 마차 타고 내리기

### 마차 매너

```
[탑승 순서]
1. 가장 신분 높은 여성 먼저
2. 다른 여성들
3. 남성들 마지막 (또는 마부석/말 위)

[자리]
- "Forward seat" (앞을 보는 좌석) = 격식 자리
- "Back seat" (뒤를 보는 좌석) = 낮은 자리

[하차]
1. 남성이 먼저 내려서
2. 여성에게 손을 내밀어 도와줌
3. 손을 잡고 내림 — 신체 접촉의 드문 기회
```

### 작중 의미

- Ch.43 펨벌리 떠날 때: "Mr. Darcy handed the ladies into the carriage" — **다아시의 손**이 엘리자베스/가디너 부인의 손과 닿음
- 신체 접촉이 거의 없는 시대에 **마차 보조 = 친밀의 합법적 순간**

## 8. 편지 쓰기/접기/봉인

### 편지 쓰는 자세

```
[책상 자세]
1. 편지지 비스듬히
2. 깃펜은 오른손 (잉크통 가까이)
3. 왼손은 종이 누르기
4. 한 장당 약 20-30분 소요

[잘못된 자세 = 매너 부족]
```

### 편지 접기

```
[전통 방식 — 봉투 없음]
1. 한 면에 글씨 (다른 면은 비워둠)
2. 가로로 3등분 접기
3. 다시 세로로 접기
4. 외부에 주소 적기
5. 봉인 밀랍 + 도장 (가능시 가문 인장)

[작중 예시]
- 다아시 편지(Ch.35): "two sheets of letter paper, written quite through, in a very close hand"
- 다아시는 예외적으로 **봉투 사용** ("envelope containing two sheets")
- = 비밀 + 격조의 표시
```

### 편지 받기

```
[매너]
- 즉시 읽지 않고 잠시 보류 (감정 통제)
- 사적 편지는 혼자 있을 때 읽음
- 가족 앞에서 읽을 때는 짧게만
- 답장은 즉시 (예의)
```

### 작중 적용
- Ch.21 빙리 양 편지: 제인이 위층으로 가서 혼자 읽음 — **나쁜 소식 직감**
- Ch.46 제인의 편지: 엘리자베스가 즉시 읽음 — **긴급 상황**

## 9. 침실/사적 공간 예법

### 들어가는 규칙

```
[방문 가능]
- 침실: 가족만 (자매, 부모)
- 책상이 있는 boudoir: 친한 여성 친구
- 서재: 가족 + 친한 친구

[금지]
- 미혼 남녀가 단둘이 방에 있는 것 = 스캔들
- 침실에 이성 = 절대 금지

[작중 예시]
- 다아시 + 엘리자베스가 단둘이 응접실(Ch.32, 34) = 이미 매우 격식 위반
- 다아시가 응접실 들어왔을 때 빙리 양/허스트 부인 등의 시선
```

### 옷 갈아입기

```
[Lady's Maid의 도움]
1. 가운 풀기
2. 코르셋 풀기
3. 페티코트 갈아입기
4. 새 가운 입기
5. 머리 손질
6. 신발 갈아 신기

[혼자 할 경우]
- 자매끼리 서로 도움 (베넷 자매)
- "Sarah, never mind Miss Lizzy's hair" (Ch.55) — 시간 부족
```

## 10. 손/팔 사용

### 팔 잡기 (Walking arm-in-arm)

```
[규칙]
- 여성이 남성의 오른팔 옆에 섬
- 여성이 남성의 팔에 자기 손을 얹음 (잡지 않음)
- 친밀의 표시이나 격식 내

[작중]
- Ch.43 펨벌리: 가디너 부인-엘리자베스 → 다아시-가디너 부인 → 다아시-엘리자베스
   = 짝 변경 = 의미 있는 순간
```

### 손 키스 — **거의 안 함**

- 영국에서는 손 키스 거의 없음 (대륙 풍습)
- 위컴 같은 인물이 시도하면 **느끼함**으로 비쳐짐

### 부채 사용 (여성)

```
[부채의 언어 — 의식하지 않게 사용되나 의미 전달]
- 펼치고 흔들기: 평온함
- 빠른 부채질: 긴장/흥분
- 부채로 얼굴 가리기: 부끄러움
- 부채 닫기: 대화 종료 신호

[작중 가능 장면]
- 무도회에서 빙리 양의 부채질 = 다아시 관찰 도구
```

## 11. 시선 매너

### 응시 규칙

```
[허용]
- 대화 상대 눈 보기 (1-3초씩)
- 호스티스/호스트 시선 받기

[금지]
- 노골적 응시 (특히 여성에게)
- 위아래로 훑어보기 = 매우 무례
- 식사 중 다른 사람 음식 보기

[작중 예시]
- 다아시가 엘리자베스를 자주 응시 — Ch.10에서 엘리자베스가 인지
- "She hardly knew how to suppose that she could be an object of admiration"
- = 시선이 무례한지 호감인지 모호
```

### 미소

```
[격식 정도]
- 입 약간 올림 = 표준
- 이를 보이는 미소 = 매우 친밀
- 큰 웃음(소리) = 무례 (특히 여성)

[엘리자베스의 특이성]
- "lively, playful disposition, which delighted in anything ridiculous"
- 자주 웃음 = 일반 여성보다 활기참 = 사회적 코드 미세 위반
```

```
[격식 인사]
```

## 13. 신체 접촉 빈도 (낮음)

### 허용되는 접촉

| 접촉 | 상황 |
|---|---|
| 악수 | 친밀 표시 (드물게) |
| 팔 잡기 | 산책 시 |
| 마차 보조 | 손 잡고 내림 |
| 춤 | 손 잡기 |
| 가족 | 자매끼리 포옹, 키스 |

### 작중 신체 접촉 순간 (희소)

- 다아시 + 엘리자베스: **춤 (Ch.18)** + **마차 보조 (Ch.43)** + **재청혼 산책 (Ch.58)**
- 그 외 거의 없음 → 매 접촉이 **감정적으로 폭발적**

## 14. 슬픔/분노 표현 (절제)

### 격식 사회의 감정 통제

```
[허용]

[작중 위반]
```

- Ch.46 엘리자베스: 다아시 앞에서 울음 — 위기로 자제 무너짐

## 16. 인물별 특수 신체언어

### Elizabeth — 활동적 + 약간의 코드 위반

- 빠른 걸음
- 큰 웃음 가끔
- 손짓 활기참
- 시선 직접적 (도전적)

### Darcy — 절제 + 침묵

- 느린 동작
- 침묵의 시간 길게
- 시선 깊고 길게
- 손 거의 안 움직임 (말할 때도)

### Mrs. Bennet — 과장

- 자주 부채질
- 큰 한숨
- 손짓 과장
- "신경" 발작 = 의자에 기댐, 부채로 얼굴 가림

### Mr. Bennet — 무관심한 자세

- 의자에 깊이 앉음
- 책에서 시선 거의 안 듦
- 가족 대화에 등 돌리기도

### Lydia — 무절제

- 뛰어다님 ("flew about the house")
- 큰 소리 웃음
- 매너 무시

### Lady Catherine — 위압

- 똑바른 자세
- 직접 응시 (압박)
- 손짓 명령조

### Collins — 과잉

- 깊은 bow 반복
- 손 비비기 (아첨)
- 머리 자주 숙임

### Wickham — 매혹적 매너

- 부드러운 미소
- 친밀한 거리 (관습 위반)
- 시선 길게 (유혹적)

## 17. 핵심 장면 신체언어 가이드

### Ch.34 청혼 장면

```
[다아시 동선]
- 응접실 입장 (당당함이나 긴장)
- 앉음 → 일어남 → 방 안 서성임
- 엘리자베스에게 다가감 ("agitated manner")
- 청혼 시 약간 격하게 ("dwelt on with a warmth")
- 거절 후 벽난로에 기대 (충격 흡수)
- 마지막에 빠른 걸음으로 방을 가로지름

[엘리자베스 동선]
- 앉아 있음 (놀라서)
- 일어나지 않음 (자세 유지)
- 거절 시에도 자세 유지
- 분노로 얼굴 붉어짐
- 다아시 퇴장 후 무너짐
```

### Ch.46 도주 소식

```
[엘리자베스]
- 편지 들고 의자에서 벌떡 일어남
- 문으로 달려감
- 다아시와 마주침 - 멈춤
- 다리에 힘 풀림 → 다시 앉음
- 손수건으로 얼굴 가림
- 흐느낌

[다아시]
- 입장 시 놀람
- 다가감 → 하인 부르려 함
- 그녀를 위로하려 함
- 방안 서성임 (혼자만의 결심)
- 마지막 눈빛 → 퇴장
```

### Ch.58 재청혼

```
[양쪽]
- 산책 중 단둘이 됨
- 엘리자베스가 먼저 입을 엶 (감사)
- 다아시 멈춤
- 두 사람 마주 봄
- 천천히 걸으며 대화
- 손은 잡지 않음 (격식 유지)
- 그러나 시선은 깊어짐
- 마지막에 자연스럽게 손 잡고 귀가 가능
```
