# Pride and Prejudice — 인물관계도

## 1. 가문 구조

### 베넷 가 (Bennet Family) — 롱번 (Longbourn)
```
Mr. Bennet (베넷 씨) ─── Mrs. Bennet (베넷 부인, 구성: Gardiner)
    │
    ├── Jane Bennet (제인, 첫째)     ──→ Mr. Bingley와 결혼
    ├── Elizabeth Bennet (엘리자베스, 둘째) ──→ Mr. Darcy와 결혼
    ├── Mary Bennet (메리, 셋째)
    ├── Kitty Bennet (키티/캐서린, 넷째)
    └── Lydia Bennet (리디아, 막내)   ──→ Mr. Wickham과 결혼
```

### 베넷 부인의 친척
```
Mrs. Bennet의 남동생 ─── Mr. Gardiner (가디너 씨, 런던 상인)
                         │
                         └── Mrs. Gardiner (가디너 부인, 더비셔 연고)
                              + 자녀 4명 (딸 둘, 아들 둘)

Mrs. Bennet의 여동생 ─── Mrs. Philips (필립스 부인, 메리턴 거주)
                         │
                         └── Mr. Philips (필립스 씨, 사무변호사)
```

### 다아시 가 (Darcy Family) — 펨벌리 (Pemberley)
```
고(故) Mr. Darcy (선대 다아시 씨) ─── Lady Anne Darcy (앤 다아시 부인)
    │                                    │
    │                                    └── 자매: Lady Catherine de Bourgh
    │
    ├── Fitzwilliam Darcy (다아시 씨)  ──→ Elizabeth와 결혼
    └── Georgiana Darcy (조지아나, 16세)
         [공동 후견인: 다아시 + Colonel Fitzwilliam]
```

### 드 버그 가 (de Bourgh Family) — 로징스 (Rosings)
```
고(故) Sir Lewis de Bourgh ─── Lady Catherine de Bourgh (레이디 캐서린)
                                │
                                └── Miss Anne de Bourgh (앤 드 버그)
                                     + Mrs. Jenkinson (동반자/가정교사)
```

### 빙리 가 (Bingley Family) — 네더필드 (Netherfield)
```
고(故) Mr. Bingley (부친, 북부 상인)
    │
    ├── Charles Bingley (빙리 씨)  ──→ Jane과 결혼
    ├── Caroline Bingley (빙리 양)
    └── Louisa Bingley ─── Mr. Hurst (허스트 씨)
         (허스트 부인)
```

### 루카스 가 (Lucas Family) — 루카스 로지 (Lucas Lodge)
```
Sir William Lucas (윌리엄 루카스 경) ─── Lady Lucas (루카스 부인)
    │
    ├── Charlotte Lucas (샬럿, 27세) ──→ Mr. Collins와 결혼
    ├── Maria Lucas (마리아)
    └── 어린 아들들
```

### 콜린스 (Collins)
```
Mr. Collins (콜린스 목사, 베넷 씨의 먼 친척)
    │
    └── 한정상속제(Entail)에 의해 롱번 영지의 법적 상속인
         헌스포드(Hunsford) 교구 목사
         레이디 캐서린의 후원(patronage)을 받음
```

## 2. 핵심 관계 변천 (사건순)

### 다아시 ↔ 엘리자베스 (소설의 중심축)
```
EVT-003  [반감 시작]  다아시의 "tolerable" 모욕 → 엘리자베스의 반감
EVT-004  [일방적 끌림]  다아시가 "fine eyes"에 끌림 (엘리자베스는 모름)
EVT-009  [bewitched]  다아시가 완전히 매혹됨 → 의도적 거리두기
EVT-016  [대결]  무도회에서 춤+위컴 건 정면 대결
EVT-028  [파열]  첫 청혼/거절 "last man in the world"
EVT-029  [전환]  다아시의 편지 → 진실 폭로
EVT-030  [자각]  "Till this moment, I never knew myself"
EVT-035  [재평가]  펨벌리에서 다아시의 인격 재발견
EVT-036  [재회]  완전히 변한 다아시의 태도
EVT-039  [사랑 자각]  "all love must be vain" — 처음으로 사랑을 명명
EVT-042  [감사]  다아시의 비밀 중재 폭로
EVT-046  [결합]  두 번째 청혼 수락
EVT-048  [결혼]  펨벌리의 안주인
```

### 빙리 ↔ 제인
```
EVT-003  [첫 만남]  무도회에서 두 번 춤 → 상호 호감
EVT-006  [심화]  제인 병환에 진심 어린 걱정
EVT-010  [이별]  제인 귀가, 빙리는 아쉬움
EVT-019  [분리]  다아시+빙리 양의 공모로 런던 철수
EVT-021  [절망]  빙리 양의 편지로 제인의 희망 소멸
EVT-023  [위선 폭로]  빙리 양의 런던 냉대 → 우정 종료
EVT-037  [재회]  펨벌리에서 빙리가 "11월 26일 이후 못 봤다"며 그리움
EVT-043  [귀환]  다아시의 이간 해제 → 네더필드 복귀
EVT-044  [약혼]  재청혼 + 수락
EVT-048  [결혼]
```

### 위컴 ↔ 여러 인물
```
위컴 ↔ 다아시:
  - 유년: 같은 영지에서 함께 자람 (선대 다아시의 대자)
  - 성직록 3천 파운드로 자발 포기 → 방탕 → 재요구 → 거절
  - 조지아나 유혹 미수 (EVT-029에서 폭로)
  - 리디아 도주로 최종 대결 (EVT-038~040)

위컴 ↔ 엘리자베스:
  EVT-014  거짓 증언으로 신뢰 획득
  EVT-022  미스 킹으로 전향 → 엘리자베스의 환상 소멸
  EVT-030  편지로 실체 폭로 → 신뢰 완전 붕괴
  EVT-033  마지막 만남 — 위선 직접 감지

위컴 ↔ 리디아:
  EVT-038  브라이튼에서 도주 (결혼 의도 없음)
  EVT-040  다아시의 비밀 중재로 결혼 성사
```

### 콜린스 ↔ 베넷 가 ↔ 샬럿
```
EVT-011  콜린스 롱번 방문 (한정상속제의 인격화)
EVT-012  청혼 대상: 제인 → 엘리자베스 전환
EVT-017  엘리자베스 청혼 거절
EVT-018  베넷 씨가 엘리자베스 편
EVT-020  샬럿이 콜린스 수락 → 엘리자베스와 우정 균열
```

## 3. 계급/경제적 관계도

### 재산 서열
```
[최상위]
  Mr. Darcy          연 10,000파운드 + 펨벌리 영지 + 런던 타운하우스
  Lady Catherine      로징스 영지 + 막대한 재산

[상위]
  Mr. Bingley         약 100,000파운드 상속 (연 4-5천) / 영지 미구매
  Miss de Bourgh       로징스 상속녀 (금액 불명, 막대)
  Georgiana Darcy      30,000파운드 지참금

[중위]
  Mr. Bennet           연 2,000파운드 (한정상속으로 딸에게 불리)
  Mr. Collins          헌스포드 교구 수입 (상당히 안정적)

[하위]
  Mr. Wickham          무일푼 + 빚 (다아시가 구제)
  베넷 자매 각자       어머니 사후 약 200파운드씩 (4% 공채)
```

### 후원(Patronage) 관계
```
Lady Catherine ──[성직 추천권]──→ Mr. Collins
Mr. Darcy ──[성직 추천권, 다수 교구]──→ (여러 목사)
Mr. Darcy ──[소위직 매입]──→ Mr. Wickham (비밀 중재)
```

### 한정상속제(Entail) 체인
```
Mr. Bennet의 영지 (롱번)
    │
    └──[한정상속]──→ Mr. Collins (남성 상속인)
                      │
                      └──→ Charlotte Lucas Collins (미래 안주인)
                      
※ 베넷 부인과 딸 5명은 상속에서 배제됨
```

## 5. 감정/관계 유형별 분류

### 사랑/결혼 관계
| 관계 | 성격 | 결과 |
|---|---|---|
| 다아시 ↔ 엘리자베스 | 오만과 편견을 넘은 진정한 사랑 | 결혼 |
| 빙리 ↔ 제인 | 순수한 상호 호감, 외부 개입으로 시련 | 결혼 |
| 위컴 ↔ 리디아 | 충동과 무분별함, 돈으로 성사 | 결혼 (불행) |
| 콜린스 ↔ 샬럿 | 현실적 타협, 감정 없는 계약 | 결혼 (안정적이나 무미건조) |
| 베넷 씨 ↔ 베넷 부인 | 젊은 시절의 착각, 오래전에 소멸 | 결혼 (냉소적 공존) |

### 적대/경쟁 관계
| 관계 | 성격 |
|---|---|
| 빙리 양 → 엘리자베스 | 다아시를 둘러싼 일방적 질투와 적대 |
| 빙리 양 → 제인 | 표면적 우정 + 이면적 이간 |
| 레이디 캐서린 → 엘리자베스 | 계급적 경멸 + 드 버그 양과의 정략결혼 수호 |
| 다아시 ↔ 위컴 | 유년의 동료에서 원수로 (조지아나 사건) |

### 우정/신뢰 관계
| 관계 | 성격 |
|---|---|
| 엘리자베스 ↔ 제인 | 가장 깊은 자매 유대, 비밀의 동반자 |
| 엘리자베스 ↔ 샬럿 | 결혼관 충돌로 균열, 부분 회복 |
| 엘리자베스 ↔ 가디너 부인 | 현명한 조언자이자 정서적 지지자 |
| 다아시 ↔ 빙리 | 성격은 반대이나 가장 가까운 친구 |
| 다아시 ↔ 피츠윌리엄 대령 | 사촌이자 조지아나의 공동 후견인 |

### 후원/종속 관계
| 관계 | 성격 |
|---|---|
| 레이디 캐서린 → 콜린스 | 성직 추천권에 기반한 절대적 후원-아첨 |
| 다아시 → 위컴 | 선대의 유산 → 배신 → 비밀 구제 |
| 다아시 → 빙리 | 판단력의 우위를 점하는 우정 (이간/해제) |

## 6. 번역자를 위한 관계 변화 핵심 타임라인

번역 시 인물 간 호칭, 어조, 태도가 사건에 따라 변하므로 아래 타임라인을 참고:

| 시점 | 다아시→엘리자베스 | 엘리자베스→다아시 | 빙리→제인 | 위컴→엘리자베스 |
|---|---|---|---|---|
| Chap 1-6 | 무관심→은밀한 끌림 | 단순 반감 | 호감 시작 | (미등장) |
| Chap 7-13 | bewitched+억제 | 반감 유지 | 진심 어린 걱정 | (미등장) |
| Chap 14-17 | 거리두기 | 편견 고착(위컴 증언) | 호감 절정 | 매력적 피해자 |
| Chap 18-23 | 고통스러운 끌림 | 적대감 극대화 | 분리됨 | 미스 킹 전향 |
| Chap 24-34 | 억제 실패→청혼 | "last man in the world" | 분리+절망 | 관계 소멸 |
| Chap 35-40 | 편지로 자기변호 | 편견 자각+수치 | (변화 없음) | 환멸 완료 |
| Chap 41-45 | 완전히 변한 태도 | 존경+감사+미명명 끌림 | 그리움 유지 | (위컴 경멸) |
| Chap 46-54 | 비밀 구출 | 사랑 자각+"all vain" | 귀환+재접근 | 도주범/사기꾼 |
| Chap 55-61 | 재청혼 | 수락 | 약혼+결혼 | 가문의 수치 |
