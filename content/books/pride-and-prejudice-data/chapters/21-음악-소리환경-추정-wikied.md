## 2. 리젠시 시대 음악 종류

### 가정 음악 (Drawing Room Music)

| 종류 | 설명 |
|---|---|
| **Ballads (발라드)** | 짧은 서정 노래. 여성 솔로 |
| **Italian Songs** | 오페라 아리아. 빙리 양 같은 패셔너블 여성 |
| **Scotch/Irish Airs** | 민요풍. 가벼운 분위기. Mary 연주 |
| **English Folk Songs** | "Greensleeves" 등 전통곡 |
| **Glees** | 3-4성부 합창 |
| **Sonatas** | 클래식 (베토벤, 모차르트, 하이든) |

### 무도회 음악 (Ball Music)

| 춤 종류 | 음악 |
|---|---|
| **Country Dance** | 활기찬 2-4박자 |
| **Cotillion** | 4쌍 춤 |
| **Quadrille** | 4쌍 춤 (1815년 유행) |
| **Reel** | 스코틀랜드 풍 빠른 춤 |
| **Boulanger** | 마지막 전원 춤 |
| **Minuet** | 격식, 노인용 (이미 구식) |
| **Waltz** | 1812년 영국 도입 — 작중 시점에 막 등장. 작중 미언급 |

### 작곡가 추정
- **모차르트** (1791년 사망, 인기 정점)
- **하이든** (1809년 사망, 영국 방문 경력)
- **베토벤** (현역, 새 음악)
- **헨델** (이미 고전)
- **Thomas Arne** (영국 작곡가)
- **Charles Dibdin** (영국 발라드)

## 3. 인물별 음악 성향

### Elizabeth Bennet
- **피아노:** "by no means capital" (Ch.6) — 적당한 수준
- **노래:** 함께. "pleasing"
- **선호곡:** 가벼운 ballads
- **태도:** "rather not sit down before those who must be in the habit of hearing the very best performers" (Ch.6) — 겸손
- **번역 포인트:** 음악을 즐기되 자랑하지 않음. 기교보다 표현

### Mary Bennet
- **피아노:** **연습 많이 하나 재능 부족** — "neither genius nor taste" (Ch.6)
- **노래:** 약한 목소리 + 어색한 매너
- **선호곡:** 긴 협주곡, 종교 음악, 도덕적 가사
- **장면:** Ch.6 루카스 로지 (Scotch/Irish airs로 자매들 춤추게), Ch.18 네더필드 무도회 (장시간 노래로 망신)
- **번역 포인트:** 진지하나 우스꽝스러운 콤비

### Caroline Bingley
- **피아노:** 잘 침. 격식 있는 연주
- **선호곡:** Italian songs, Scotch airs (다아시 관심 끌기용)
- **장면:** Ch.10 다아시 관심 끌려 일부러 변주
- **하프:** 가능 (가능성)

### Georgiana Darcy
- **피아노:** **탁월** — "her performance on the pianoforte is exquisite" (Ch.8 Caroline)
- "she practises very constantly" (Ch.31 다아시)
- **하프:** "delighted to hear of her improvement on the harp" (Ch.10)
- **선호곡:** 진지한 클래식
- **새 피아노:** Ch.43 — 다아시 선물

### Anne de Bourgh
- **불가** — "her health had not allowed her to apply" (Ch.31 레이디 캐서린)
- 만약 건강했다면 잘했을 것이라고 어머니 자랑

### Lady Catherine
- **본인 안 침** — "If I had ever learnt, I should have been a great proficient" (Ch.31)
- **잘난 척** — 음악 평론은 함

### Lydia Bennet
- **음악 거의 안 함** — 흥미 부족
- 무도회 춤만 좋아함

### 남성들
- **다아시:** 음악 안 함 (당시 남성 표준). 듣기는 함
- **빙리:** 안 함. 듣기는 즐김
- **콜린스:** "사랑할 만한 여흥"이라며 칭찬만
- **Mr. Bennet:** 안 함 (서재에 있음)
- **Mr. Hurst:** 무관심

## 4. 무도회 음악 운영

### 메리턴 어셈블리 (Ch.3)
- 외부 악단 (소규모, 4-6명)
- 바이올린, 첼로, 플루트, 하프시코드 또는 피아노
- 댄스 마스터(Master of Ceremonies)가 곡 지정

### 네더필드 사적 무도회 (Ch.18)
- 더 큰 악단 (6-10명)
- 별도 무대 또는 갤러리에서 연주
- 곡 사이 짧은 휴식

### 무도회 진행
```
1. Country Dance (2-3곡)
2. 휴식 + 음료
3. Quadrille 또는 Cotillion
4. 휴식
5. Reel
6. 야식 (supper)
7. 다시 Country Dance
8. 마지막: Boulanger (전원)
```

## 6. 일상 소리 환경 (추정)

### 롱번 (시골 저택)
**아침:**
- 닭 울음 (5-6시)
- 새 지저귐
- 부엌에서 그릇 소리 + 화로 소리
- 하녀가 침실 문 두드림 + 따뜻한 물 부음

**오전:**
- 마차 바퀴 소리 (방문객)
- 자매들 대화, 베넷 부인 호출
- 피아노 연습 (간헐)
- Mr. Bennet의 책장 넘기는 소리 (서재)

**낮:**
- 베넷 부인의 외출 마차
- 정원에서 새소리
- 농장 일꾼들 소리 (멀리서)

**저녁:**
- 식기 소리 (정찬)
- 응접실 대화
- 카드 섞는 소리
- 시계 소리

**밤:**
- 벽난로 장작 소리

## 7. 핵심 장면별 사운드 (각색용)

### Ch.3 메리턴 무도회
- **음악:** 활기찬 country dance
- **배경:** 수다, 발걸음, 부채
- **다아시 "tolerable" 순간:** 음악이 잠시 줄어드는 효과 가능

### Ch.10 네더필드 응접실
- **빙리 양의 피아노:** 처음엔 이탈리안 → 활기찬 Scotch
- **다아시 "Reel" 제안 순간:** 음악 강조
- **펜 긁는 소리** (다아시 편지 작성)

### Ch.18 네더필드 무도회
- **풀 사운드:** 큰 악단 + 인파
- **다아시-엘리자베스 춤:** 음악 속 그들의 대화
- **Mary 노래:** 어색한 분위기 — 음악 톤 조정 (실제로 음정이 약간 어긋나도록 연출 가능)

### Ch.34 헌스포드 청혼
- **고요** — 음악 없음
- **벽난로 장작 소리**
- **다아시 발걸음** (방 안 서성임)
- **엘리자베스 호흡**

### Ch.36 편지 후 산책
- **새소리**, **나뭇잎 바스락**
- **엘리자베스 호흡** (감정의 격동)
- 외부 음악 없음 — 내면 침묵

### Ch.43 펨벌리
- **마차 바퀴** (자갈 진입로)
- **강물** (Ch.43 명시)
- **갤러리 발소리** (석조 바닥)
- **정원사의 갈퀴** (다아시 등장 직전)

### Ch.46 도주 소식
- **편지 부스럭**
- **엘리자베스 흐느낌**
- **다아시 발걸음** (방 안 서성)

### Ch.58 두 번째 청혼 산책
- **새소리** (가을 아침)
- **자갈 밟는 소리**
- **두 사람 목소리** — 부드러움

### Ch.61 결혼식
- **교회 종**
- **오르간** (있다면)
- **하객들의 웅성거림**

## 8. 음악적 모티프 (각색용)

### 인물별 테마 제안 (영상 음악)

| 인물 | 테마 분위기 |
|---|---|
| **Elizabeth** | 경쾌하고 위트 있는 (Mozart 풍) |
| **Darcy** | 깊고 절제된 (Beethoven 초기 풍) |
| **Jane** | 부드럽고 서정적 |
| **Bingley** | 밝고 활기찬 |
| **Wickham** | 매혹적이나 표면적 (속이는 화려함) |
| **Collins** | 코믹한 행진곡풍 |
| **Lady Catherine** | 위압적 행진곡 |
| **Lydia** | 무모한 댄스풍 |

### 사건별 음악 변화
- **첫 청혼 (Ch.34):** 긴장 + 폭발
- **편지 (Ch.35):** 가라앉은 슬픔
- **자각 (Ch.36):** 사색적
- **펨벌리 (Ch.43):** 경이 + 발견
- **도주 소식 (Ch.46):** 충격
- **재청혼 (Ch.58):** 평화 + 결합
