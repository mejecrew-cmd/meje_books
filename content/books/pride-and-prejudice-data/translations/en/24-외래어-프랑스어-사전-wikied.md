# *Pride and Prejudice* — Foreign and French Expressions Glossary

> A collection, classification, and translation guide to every non-English expression in the novel, chiefly French.
> In Regency English, the use of French marks education, class, and refinement.
> This guide provides criteria for choosing between transliteration and translation.

---

## 1. The Social Meaning of French Usage

### Historical Context (The 1810s)

- Britain is fighting the **Napoleonic Wars** (1803–1815), yet French culture remains the model for the upper classes.
- French is **an essential language for an educated woman**, as implied by Darcy’s definition of an “accomplished woman” in Ch.8.
- Georgiana may be presumed to study French.
- **Italian** also appears through music and opera.

### Patterns of Usage in the Novel

| User | Frequency | Meaning |
|---|---|---|
| **Narrator** | Frequent | Refinement + subtle expression |
| **Darcy** | Occasional | Unforced education |
| **Miss Bingley** | Frequent | Fashionable display |
| **Collins** | Never | Lack of cultivation |
| **Elizabeth** | Occasional | Natural ease |
| **Mrs. Bennet** | Never | Ignorance |

#### 1. **hauteur** (Arrogance, Haughtiness)

| Item | Details |
|---|---|
| Pronunciation | [oh-TUR] in French, or an anglicized pronunciation |
| Location | Ch.18 ball—“A deeper shade of *hauteur* overspread his features” |

| Meaning | A haughty bearing; more physical and facial than simple pride |
| Translation | “arrogance” / “haughtiness”; preserving the italics is recommended |

## 3. Character Analysis Through Foreign-Language Usage

### Darcy—Unforced Cultivation

```
Pattern of Use:
- Precise in formal letters
- Appropriate even in ordinary conversation
- Never ostentatious
- Capable of French, Latin, and classical quotation

In Translation:
- Natural diction + appropriate retention of foreign words
- Preserve refinement
```

### Miss Bingley—Fashionable Display

```
Pattern of Use:
- Frequent
- Pronunciation is accurate, but chosen for effect
- Prefers “fashionable” words

In Translation:
- Preserve the foreign words with a slight air of affectation
- Additional expressions such as “à la mode” may be appropriate
```

### Mr. Collins—A Lack of Cultivation

```
Pattern of Use:
- Almost never uses foreign words
- Awkward enough even in English
- Uses Latin only in a religious context

In Translation:
- Avoid foreign words; compensate with excessive formal diction
- Phrases equivalent to “I dare flatter myself” and “presumptuous though it may be”
```

### Lydia—Ignorance

```
Pattern of Use:
- No foreign expressions
- Even a word such as “Boulanger” is pronounced by her mother instead

In Translation:
- Use awkward pronunciation or avoid the foreign word
- Paraphrase it as “that last dance at the ball”
```

### Elizabeth—Wit + Self-Mockery

```
Pattern of Use:
- Uses “éclat,” “adieu,” and similar words in self-mockery
- Uses them for humor, not display

In Translation:
- Retain the foreign word + a lightly distanced tone
- Give exclamations such as “Farewell!” a touch of exaggeration
```

## 4. Translation Decision Matrix

### Criteria for Transliteration vs. Translation

```
[Transliteration Recommended]
- Proper nouns: places, people, games, and dances
- Words emphasized in italics: hauteur, éclat
- No exact equivalent exists in the target language
- Period refinement needs to be preserved

[Translation Recommended]
- Ordinary narrative terms: per annum → annually
- A natural equivalent exists in the target language
- The foreign word would harm readability

[Combined Method]
- At first occurrence, translate and add the transliteration in parentheses
- Use only the transliteration thereafter
- Alternatively, explain it in a note at first occurrence
```

### Recommended Standard Decisions

| Original | Recommended Rendering | Reason |
|---|---|---|
| hauteur | arrogance (*hauteur*) | Preserves the italic emphasis |
| ennui | ennui / weariness | Natural target-language equivalent may be used |
| tête-à-tête | private conversation / tête-à-tête | Translate + retain at first occurrence |
| ragout | ragout | Retain the food name |
| vingt-un | vingt-un (twenty-one) | Retain + add the number |
| piquet | piquet | Retain the game name |
| quadrille | quadrille | Retain the game/dance name |
| loo | loo | Retain the game name |
| éclat | brilliance (*éclat*) | Translate + preserve the original |
| adieu | farewell! / adieu! | Preserve the emotional intensity |
| toilette | dressing | Translate |
| antechamber | waiting room | Translate |
| Boulanger | Boulanger | Retain the dance name |
| per annum | annually | Translate |
| postscript | postscript / P.S. | Translate |
| olive branch | gesture of peace / offspring | Translate according to context |

## 5. Consistency in Handling Foreign Words in Translation

### Standards for the Whole Work

```
[Principle 1: Consistency]
- Render the same foreign word in the same way throughout one work

[Principle 2: Notes Where Necessary]
- At first occurrence, explain the meaning and cultural implication
- Thereafter, use only the transliteration or translation

[Principle 3: Period Accuracy]
- Do not use loanwords that entered the target language much later
- Replace modern terms such as “style” with a period-appropriate “form” or “manner”

[Principle 4: Refinement]
- A foreign word is a marker of refinement
- Use it only where it sounds natural
- Use more for Miss Bingley and less for Collins
```

## 6. Beyond French—Latin, Italian, and Greek

### Latin (A Marker of Learning)

- per annum (yearly)
- post script (postscript)
- ad libitum (at will)
- carte blanche (blank authority)
- modus operandi (method of operation)

These imply the education of Darcy and Mr. Bennet.

### Italian (A Marker of Musical Education)

- aria
- duet—naturalized in English
- forte / piano—musical directions
- soprano / alto / tenor / bass

Miss Bingley’s “Italian songs” in Ch.10 mark musical cultivation.

### Greek (Myth and Philosophy)

- olive branch—indirectly, a Greek symbol of peace
- Hermitage—a hermit’s dwelling, with a Greek root

## 8. Pronouncing Foreign Words on Screen and Stage

### Guide for Actors and Voice Performers

```
[French]
- Use an anglicized British pronunciation, modeled on British actors
- Overly exact French pronunciation sounds unnatural
- Example: “hauteur” in an English pronunciation rather than fully French [oh-TUR]

[Latin]
- Use British school pronunciation: Etonian Latin
- “per annum” = [per AN-um]

[Italian]
- Give musical terms their correct Italian pronunciation
- Miss Bingley’s “Italian songs” should be genuine Italian pieces
```

### Foreign Words That Existed in the Novel’s Period

### Modern Loanwords That Also Create Anachronism in Korean
- Terms equivalent to “stress,” “cool,” “mindset,” and “table” must never be used in their modern loanword forms.

## 10. Conclusion: Five Principles for Translating Foreign Words

1. **Preserve italics as italics**—emphasize what the original emphasizes.
2. **Transliteration + a note is often best**—explain the meaning at first occurrence, then retain the term.
3. **Preserve differences between characters**—Miss Bingley uses foreign words often; Collins does not.
4. **Use only period foreign words**—exclude expressions that entered after the nineteenth century.
5. **Maintain consistency**—render the same word the same way throughout one work.
