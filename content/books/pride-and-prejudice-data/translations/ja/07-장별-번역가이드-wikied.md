# 『高慢と偏見』――章別翻訳ガイド

> 61章それぞれについて、**要約、登場人物、場所、時間、重要な台詞、誤訳の危険、参照すべき設定シート**を整理する。
> 翻訳者が各章を作業する際、**この文書だけを開けば**必要な文脈をすべて得られる。
>
> **凡例：**
> - 🔑 = 小説全体で重要な核心的台詞（翻訳時に特に注意）
> - ⚠️ = 誤訳の危険がある表現
> - 📎 = 参照すべき設定シート

## Volume I（Chapters 1～23）

### Chapter 1 — Bingleyの知らせ＋Bennet夫妻の最初の会話

| 項目 | 内容 |
|---|---|
| **要約** | Netherfieldに裕福な青年が入居するとの知らせ。Mrs Bennetは夫に訪問を求め、Mr Bennetは皮肉を交えてかわす。 |
| **場所** | Longbournの応接間 |
| **人物** | Mr Bennet、Mrs Bennet（会話のみ。娘たちは言及される） |
| **EVT** | EVT-001 |
| **🔑 重要な台詞** | “It is a truth universally acknowledged, that a single man in possession of a good fortune must be in want of a wife.” — 小説全体のアイロニーを定める冒頭。表面と実際の意味が反転している。 |
| | “You want to tell me, and I have no objection to hearing it.” — Mr Bennetの皮肉な譲歩 |
| | “I must throw in a good word for my little Lizzy.” — えこひいきの最初の兆候 |
| **⚠️ 誤訳の危険** | “mean understanding” = 知性が**乏しい**（卑劣ではない）／“fancied herself nervous” = 神経症だと**自称した**（実際の一時的な緊張ではない）／“business of her life” = 人生の**事業**という経営的比喩 |
| **📎 参照** | 時代語辞典（nervous, mean）、ニュアンスリスト（NUA-001, NUA-002） |

### Chapter 3 — Meryton舞踏会：“tolerable”

| 項目 | 内容 |
|---|---|
| **要約** | Merytonの集会舞踏会。Bingley一行5人が入場。Darcyは当初注目を集めるが、傲慢な態度で嫌悪される。Elizabethを“tolerable”と侮辱。BingleyはJaneと二度踊る。 |
| **場所** | Merytonの集会場 |
| **人物** | ほぼ全員（Wickhamは未登場） |
| **EVT** | EVT-003 |
| **🔑 重要な台詞** | “She is tolerable; but not handsome enough to tempt me; and I am in no humour at present to give consequence to young ladies who are slighted by other men.” — 小説の起点となる文。三層の傲慢を含む。 |
| **⚠️ 誤訳の危険** | “tolerable” = まずまず／我慢できる程度（褒め言葉ではない）／“to tempt” = 彼を惹きつける。自分には惹きつけられる価値があるとの前提／“give consequence” = 重要人物として扱う／“discovered to be proud” = 「発見」ではなく「そう判定された」。即断のアイロニー／“His character was decided” = 性格ではなく**評判が決定した** |
| **📎 参照** | 時代語辞典（tolerable, consequence, character）、ニュアンスリスト（NUA-005, NUA-006）、反復モチーフ（“tolerable”） |
