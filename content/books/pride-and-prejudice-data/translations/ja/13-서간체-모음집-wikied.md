## 手紙の文体類型

### 類型1：コリンズ書簡体（L-01、L-06、L-18、L-19）
```
特徴：
- 一つの文が5～8行に及ぶ
- セミコロンと挿入節を過剰に入れ子にする
- “I flatter myself”／“I venture to say”など、うぬぼれと謙遜の矛盾
- すべての段落でレディ・キャサリンに言及
- “olive branch”を反復使用（和解→妊娠へ意味が転換）

翻訳原則：
- 長さを絶対に縮めない。長さ＝愚かさ
- 「あえて自負いたしますれば」／「僭越ながら」など、過剰に格式ばった語尾を使う
- 同じ手紙の中で謙遜とうぬぼれが交互に現れることを生かす
```

### 類型2：ダーシー書簡体（L-10）
```
特徴：
- 最高の格調をもつ複文。譲歩節→条件節→主節を精緻に構築
- 感情の抑制が次第に崩れる構造（ビングリー部分＝冷静 → ウィカム部分＝激昂）
- “Be not alarmed, madam”——冒頭の一文が読者の緊張を誘う
- 自己弁護でありながら、相手を責めない節度
- “God bless you”で終わる——唯一の感情表出

翻訳原則：
- 口語と明確に異なる書簡体：「～していただきたく存じます」／「～であることをここに申し述べます」
- ビングリー部分（第1部）とウィカム部分（第2部）の調子の変化を最大化
- “I must now mention a circumstance which I would wish to forget myself”——苦痛を明かす直前のためらいを生かす
```

### 類型3：ビングリー嬢書簡体（L-04、L-05）
```
特徴：
- 誇張した愛情表現：“dearest friend”、“I depend on you”
- 離間の核心を柔らかな表現で包む
- “dare to entertain the hope”——謙遜を装いながら刃を刺す
- 格式的でありながら親密さを装う二重性

翻訳原則：
- 誇張した親密さをそのまま保つ：「最愛の友へ」
- 再読時に偽善が見えるようにする——初読者には真心らしく、再読者には巧妙に
```

### 類型4：ジェイン書簡体（L-03、L-07～09、L-11～12）
```
特徴：
- 平常時：落ち着き、抑制、矮小化表現
- L-11→L-12：平静から混乱への急激な変化
- L-12の“I hardly know what I have written”＝統制を失ったことの自認
- “I take up my pen again to do, what I have just told you I would not”＝もう書かないと言いながら再び書く

翻訳原則：
- L-11とL-12の調子の差を最大化
- L-12の途切れては続く呼吸を生かす
```

### 類型5：ガーディナー家書簡体（L-13～14、L-17、L-21）
```
叔父（L-13～14）：
- 事務的、簡潔で、法律・財務用語が正確
- 感情を抑制。“I hope it will give you satisfaction”程度の乾いた表現

夫人（L-17）：
- 夫より温かく、おしゃべり
- ユーモアと暗示を巧妙に混ぜる：“If he had another motive, I am sure it would never disgrace him”
- 出来事を時系列で語る物語的な手紙

翻訳原則：
- 叔父：「～する次第です」（実業家の実務文体）
- 夫人：「～のように思います」（親しみがありながら教養ある女性文体）
```

### 類型6：リディア書簡体（L-15、L-16）
```
特徴：
- 文法を無視し、下線（イタリック）を乱用
- “was obliged to leave off in a violent hurry”——急に書きやめる
- 内容がないのに長い（キティ宛ては「語句の下に線がいっぱい」）
- L-15（駆け落ちの手紙）：“Don’t be alarmed”→直後に驚愕すべき内容

翻訳原則：
- 10代の無責任な手紙文体をそのまま保つ
- 下線／強調を、イタリック・下線・太字などの**表記**で生かす
- 急に途切れる箇所を「——」で表す
```
