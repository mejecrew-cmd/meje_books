# 『高慢と偏見』——人物の隠れた関係網（推定／創作）

> ⚠️ **本書には推定と創作が含まれる。** 作中で明示された関係に加え、時代背景・人物の動線・社会的合理性から、**暗示された関係や省略された関係**を追跡する。翻案時に人物を立体化するために利用できる。

---

## 1. 作中で明示されない家族関係網

### ベネット姉妹の微細な同盟
| 同盟 | 強度 | 根拠 |
|---|---|---|
| **ジェイン–エリザベス** | ★★★★★ | 明示。秘密を共有 |
| **キティ–リディア** | ★★★★ | 明示。キティがリディアに追随 |
| **メアリー単独** | ★ | 誰とも深い同盟を結ばず、音楽と本へ逃避 |
| **エリザベス–メアリー（潜在）** | ★ | 二人とも知的だが、メアリーは衒学的。結びつきは浅い |
| **ジェイン–メアリー（潜在）** | ★★ | ジェインの優しさならメアリーを包み込める可能性 |
| **ジェイン–リディア** | ★ | ほぼ皆無。性格が異なる |

### ベネット家内部の情報の流れ
```
[秘密の共有]
エリザベス ↔ ジェイン（大半の秘密）
キティ → リディア（士官の情報）
リディア → ベネット夫人（噂話）
メアリー → 誰にも話さない（ひそかに日記をつけると推定）

[遮断された情報]
- ダーシーの最初の求婚：ジェインだけが知る
- ウィカムの正体：ジェインだけが知る。ジョージアナの件は除く
- ダーシーの秘密の仲介：まずエリザベスだけが知り、後にジェインへ
- ベネット夫人は常に最後に知る
```

## 2. 使用人–主人の関係網（推定）

### ヒル夫人 ↔ ベネット夫人
- ベネット夫妻の結婚時期から見て、**20年以上の付き合い**
- ベネット夫人の神経発作に最も多く付き合う人物
- ベネット夫人がヒルに秘密を打ち明けることも多い可能性
- 第49章：“Have you heard the good news?”——親密な呼びかけ

### レイノルズ夫人 ↔ ダーシー家
- **ダーシーが4歳の頃から知る**。第43章で明示
- 実質的にダーシーの**第二の母**
- 母レディ・アンの死後、母性的役割を代替した可能性
- ジョージアナの養育にも影響
- **訪問者にダーシーを自慢する言葉**は本心

### セアラ ↔ ベネット姉妹
- **姉妹の日常の身の回りを世話**
- 舞踏会の支度で誰を先に助けるか選ぶ。第55章ではジェインを優先
- 噂の潜在的な伝達経路

### ペンバリーの庭師たち ↔ ダーシー
- ダーシーが領地経営に直接関わるため、庭師たちとも一定の関係を持つ
- ガーディナー夫妻がペンバリーに滞在中、庭師がガーディナー氏の釣りに同行

## 3. メリトンの社会ネットワーク（推定）

### 噂の流れ
```
ロング夫人 → ベネット夫人 → ベネット姉妹 → メリトン
           ↓
フィリップス夫人 ← ベネット夫人（姉妹）
           ↓
フィリップス家の客（士官たち）→ メリトン全体

[別の経路]
ルーカス夫人 → ベネット夫人（競争＋情報）
サー・ウィリアム・ルーカス → 社交界全般（“first circles”を自慢）
```

### ロング夫人の位置
- **メリトンの噂の中心**——第1章でビングリーの情報を最初に伝える
- 未婚の姪二人と同居し、結婚市場でベネット姉妹と競合
- 第5章で**ダーシーが彼女と30分間沈黙**し、ベネット夫人の反感を強める

### 士官同士の関係
- **ウィカム–デニー：** 親密。ロンドンから一緒に来る
- **ウィカム–カーター：** 同僚
- **ウィカム–フォースター大佐：** 部下と上官。第46章で大佐がウィカムの逃亡意図を漏らしており、**ウィカムがフォースターの信頼を得ていなかった**ことを示す
- **ウィカムは隊内で人気：** “captivating”だが、第50章では賭博債務も判明

## 4. コリンズの庇護ネットワーク

### レディ・キャサリンの影響網
```
レディ・キャサリン
├── コリンズ氏（ハンスフォード牧師、絶対的従属）
│   └── シャーロット（コリンズを通じて間接的に従属）
├── ジェンキンソン夫人（アン・ド・バーグの付添人、雇用）
├── ヤング夫人（元家庭教師、ダーシーが推薦した可能性）
├── ポープ嬢（家庭教師職を斡旋）
└── レディ・メトカーフなど（社交界での庇護）

[ダーシーは甥だが従属しない]
```

### コリンズの社会的位置
- **上層のレディ・キャサリン**と**下層の教区民**の双方に忠実
- ベネット家には両価的感情。親族である一方、その財産はいずれ自分のもの
- シャーロットとの結婚後は、サー・ウィリアムを含む妻の実家にも庇護を求める

## 5. ダーシーの階級的人脈（推定）

### 直接明示
- ペンバリー近隣の紳士たち
- ロンドン社交界。名前は非公開
- ビングリー。最も親しい友人
- フィッツウィリアム大佐。従兄弟
- レディ・キャサリン。伯母

### 推定人脈
- **フィッツウィリアム大佐の父である伯爵**——ダーシーの母方の伯父
  - ダーシーが訪ねるロンドンのどこかにいる
  - ジョージアナの後見人の一人である可能性
- **ダーシーの事務弁護士**——領地管理とウィカムとの交渉
- **ペンバリー近隣の医師**——ジョージアナの健康管理
- **ケンブリッジ大学の同窓**——ビングリーもその一人かもしれない
- **ロンドンのクラブ会員**。White's、Brooks'sなど

## 6. ウィカムの暗い人脈（推定）

### 明示
- ヤング夫人。ジョージアナ事件の共謀者で、ロンドンの下宿屋を営む
- デニー氏。メリトンでウィカムを紹介した人物

### 推定
- **ロンドンの賭博場の人脈**——借金の出所
- **メリトンとブライトンの飲み仲間**
- **ほかに誘惑を試みた女性たち**。ジョージアナ、キング嬢、リディア以外にもいる可能性
- **チェンバレン**。第51章でリディアが、ウィカムに女装させられた友人として言及

## 7. ビングリー姉妹の社交的野心（推定）

### キャロラインの結婚市場
- **ダーシーが第一候補**で、絶対に諦められない
- ほかの男性は渋々検討するのみ
- 30代に入れば焦燥する可能性
- 結末の第61章では、エリザベスへ“arrears of civility”を返し、**ダーシーの従兄弟などとのつながりを維持**する

### ルイーザ・ハーストの結婚
- ハースト氏との結婚は“fashion rather than fortune”。第4章
- **事実上の失敗した結婚**——夫は食事とカードにしか関心がない
- キャロラインの同盟者であり慰め役

## 8. 地方社会の結婚市場（推定）

### メリトンの未婚女性（ベネット姉妹を除く）
- **ルーカス姉妹、4～5人**。マライア、シャーロットなど
- **ロング夫人の姪二人**
- **グールディング家の令嬢たち**。ヘイ・パーク在住で、第50章にベネット夫人が言及

- **ウェッブ家の令嬢たち**。第29章で言及され、ピアノが得意

- **数十人の士官**

- **グールディング氏**。近隣住民

### 結婚可能性マトリクス
- ビングリー：すべての母親の標的
- ダーシー：身分が高すぎ、しかも高慢なので敬遠
- ウィカム：魅力的だが貧しい。キング嬢のような相続人だけが選択肢
- コリンズ：生活は安定しているが滑稽。シャーロットのような現実主義者だけが検討

## 10. ウィカムとリディアの逃走経路追跡（再構成）

### 作中の手がかりから再構成した逃走順序

```
[日曜深夜] 駅馬車でブライトンを出発
     ↓
[月曜未明] エプソム到着、駅馬車を乗り換え
     ↓
[月曜朝] クラパム到着 → **駅馬車を捨て、ハックニー・コーチへ乗り換え**
     ↓
[月曜午前] ロンドンへ入る
     ↓
[行動不明] まずウィカムがヤング夫人を訪ねる。第52章のガーディナー夫人の手紙
     ↓
[ヤング夫人が受け入れず] ----街の別の住居へ移動
     ↓
[数日後] ダーシーがヤング夫人を買収し、居場所を把握
     ↓
[2～3日の交渉後] 結婚に合意
     ↓
[8月初旬] ガーディナー家で結婚式。ダーシーも出席
```

### ウィカムの本当の意図（推定）
- 第46章のデニーの証言によれば、**結婚する意図は皆無**
- リディアを一時的に愛人として連れ歩き、その後捨てる計画
- 借金を清算して再出発することが目的
- **ダーシーの資金がなければ結婚しなかった**

### ダーシーとガーディナー叔父の秘密協議（EVT-039-005）
```
- 土曜日にガーディナー家を訪問
- ダーシー：「費用はすべて私が負担した。ガーディナー氏がしたように見せてほしい」
- ガーディナー氏は当初、自尊心から拒否
- ダーシーの説得：「私の過ちから始まったことです」
- 合意：表向きはガーディナーの功績とする
```

### ダーシーがビングリーへ離間を告白（EVT-042-001）
```
- ペンバリーでダーシーがビングリーに告白
- 「ジェインが君を愛していたかどうか、私は判断を誤った」
- 「ジェインがロンドンに滞在していた事実を君に隠した」
- ビングリーの反応：怒り → 許し。性格に合う
- ただちにネザーフィールドへ戻ると決める
```

### ベネット氏のロンドン捜索（EVT-038後）
```
- フォースター大佐と同行
- ロンドンの捜査員Bow Street Runnersを雇った可能性
- 成果なく、捜索をガーディナーへ委ねて帰宅
- 自尊心が大きく傷つき、その後さらに引きこもる
```

## 12. 結末後（第61章）に見えない関係変化

### ペンバリーの日常（推定）
- エリザベスが女主人となり、レイノルズ夫人と協力
- ジョージアナが姉妹のようにエリザベスを慕う
- ビングリーとジェインは30マイル以内へ移住し、頻繁に訪問

### ウィカムとリディアの慢性的依存
- 毎年姉たちに資金を求める
- **エリザベスが自分の小遣いから密かに援助**すると推定
- ダーシーは知らないふりをするか、知りながら黙認

### キティの変化
- ペンバリーとネザーフィールドを頻繁に訪問
- “much improved”——社交作法が改善
- 結婚の可能性が開く

### メアリーの変化
- 姉妹が家を出た後、母の唯一の同伴者となる
- ビングリー夫人となった姉の社交に加わる可能性
- 外見への劣等感が減る

### ベネット夫妻
- 二人の娘が裕福な家へ嫁ぎ、ベネット夫人の神経も安定する可能性
- ベネット氏の引きこもりは変わらない
- メアリーだけが家に残る

### レディ・キャサリンの和解
- 好奇心に屈する
- ペンバリーを訪問
- エリザベスへやむなく礼儀を尽くす
- 真の友情は生まれない

### ビングリー嬢の偽善的交際
- 「ペンバリーへ出入りするための埋め合わせの親切」
- 結婚市場での最後の切り札
