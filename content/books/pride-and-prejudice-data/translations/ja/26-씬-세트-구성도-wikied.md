# 『高慢と偏見』シーン・セット構成図

> 各重要場面の**空間配置、視覚情報、照明、天候、人物位置、音、感覚的細部**を原作から抽出して整理する。
> 翻訳では場面描写の正確さを保証し、翻案ではセット構成の根拠となる。

---

## SCENE 01：ロングボーン応接間—ビングリーの知らせ（第1章）

| 項目 | 原文の根拠 |
|---|---|
| **空間** | 応接間と推定。ベネット夫妻だけがいる |
| **時間** | 昼間—“one day” |
| **雰囲気** | 日常的な夫婦の会話。ベネット氏は本か新聞、ベネット夫人は針仕事をしていると推定 |
| **動線** | ベネット夫人が一方的に話し、ベネット氏はほとんど動かない。最後に“he left the room, fatigued with the raptures of his wife” |
| **主要な視覚** | ベネット氏の無表情とベネット夫人の興奮の対比 |

## SCENE 02：メリトン集会場—舞踏会（第3章）

| 項目 | 原文の根拠 |
|---|---|
| **空間** | 公共の集会場。一方で踊り、他方で会話と見物 |
| **時間** | 夕方から夜。蝋燭の照明 |
| **人数** | 近隣全体、数十人。ビングリー一行5人が入場：ビングリー氏、姉妹二人、姉の夫、もう一人の青年 |
| **音** | 楽団の音楽、踊る足音、話し声 |
| **主要配置** | ダーシーは部屋の隅を歩き、誰とも踊らない。エリザベスは二曲の間、相手なしで座る。ビングリーがダーシーを誘いに来て、エリザベスは二人の会話を**聞き取れる距離**にいる。 |
| **主要な視覚** | ダーシーの孤立とビングリーの活発さ。“She is tolerable”の瞬間、侮辱されても愉快に処理するエリザベスの表情 |
| **衣装** | ビングリー姉妹：“fine women, with an air of decided fashion”。ダーシー：“fine, tall person, handsome features, noble mien” |

## SCENE 03：ネザーフィールドの朝—エリザベス到着（第7～8章）

| 項目 | 原文の根拠 |
|---|---|
| **空間** | ネザーフィールドの朝食室。全員が朝食中 |
| **時間** | 朝 |
| **天候** | 前夜の大雨で、朝の野原は泥だらけ |
| **エリザベスの状態** | “weary ancles, dirty stockings, and a face glowing with the warmth of exercise”。ペティコートに6インチの泥、髪も乱れている |
| **反応** | ビングリー嬢＋ハースト夫人：軽蔑、陰で“almost wild”と非難／ビングリー：好意、“remarkably well”／ダーシー：“divided between admiration and doubt”／ハースト：朝食しか考えない |
| **主要な視覚** | 泥だらけのエリザベスの輝く顔と、身なりの整ったビングリー姉妹の冷たい視線 |

## SCENE 04：ネザーフィールドの夜—教養論争＋リール（第8、10章）

| 項目 | 原文の根拠 |
|---|---|
| **空間** | ネザーフィールド応接間。暖炉、ピアノ、カード卓、ソファ |
| **照明** | 夕食後の蝋燭 |
| **配置—第8章** | ルー卓にハースト夫妻、ビングリー嬢、ビングリー。エリザベスは断って読書し、その後カード卓のそばで見物し、ダーシーと教養を論じる。 |
| **配置—第10章** | ダーシーが手紙を書き、ビングリー嬢が口を挟む。ビングリーとハーストはピケ。エリザベスは針仕事。ビングリー嬢がピアノでスコットランド民謡を弾き、ダーシーがリールを提案する。 |
| **音** | ピアノ、ペンの走る音、ビングリー嬢の絶え間ない追従 |
| **主要な視覚** | ダーシーの視線が繰り返しエリザベスへ戻ることにビングリー嬢が気づく瞬間 |

## SCENE 05：フィリップス家の夜—ウィカムの証言（第16章）

| 項目 | 原文の根拠 |
|---|---|
| **空間** | フィリップス家の応接間。カード卓二つ：ホイストはコリンズと大人、くじ札は若者たち |
| **時間** | 夜。蝋燭 |
| **感覚** | フィリップス叔父のポートワインの匂い。コーヒーとマフィン |
| **主要配置** | ウィカムがエリザベスの隣に座る。リディアはくじに夢中で彼を独占できず、ウィカムとエリザベスの長い会話が可能になる。 |
| **雰囲気** | 台所からの温かい料理の匂い＋カードの音＋ウィカムの低い声 |
| **主要な視覚** | ウィカムの端正な顔。他の士官は彼と比べると“broad-faced stuffy uncle Philips” |

## SCENE 06：ネザーフィールド舞踏会（第18章）

| 項目 | 原文の根拠 |
|---|---|
| **空間** | ネザーフィールド大舞踏室。別に夜食室 |
| **時間** | 夕方から深夜。ホワイトスープ、晩餐、カード。ベネット家は最後まで残る |
| **照明** | 大量の蝋燭による華やかな照明 |
| **人数** | 多数の士官“cluster of red coats”＋近隣の家々＋ビングリー一行。ウィカムは**不在**。 |
| **主要場面の配置** | (1) コリンズとエリザベスの最初の二曲：“dances of mortification”。コリンズは逆方向へ動く。(2) ダーシーとエリザベスは向かい合う二列で踊りながら会話し、“hauteur”の瞬間、ルーカス卿が割り込む。(3) 夜食では二人が**向かい合って**座り、ダーシーはベネット夫人の予言を聞く。(4) メアリーがピアノ前で歌い、全員が聴く。(5) 帰りを待つ間、ビングリー嬢とハースト夫人は疲れ、ビングリーとジェインだけが離れて話す。 |
| **主要な視覚** | 最も混雑し華やかな場面。その中にダーシーの孤独とエリザベスの羞恥がある |

## SCENE 07：ロングボーン朝食室—コリンズの求婚（第19章）

| 項目 | 原文の根拠 |
|---|---|
| **空間** | 朝食室。朝食直後 |
| **人物動線** | コリンズがベネット夫人に“private audience”を求める → ベネット夫人とキティ退室 → エリザベスだけ残る → コリンズの長広舌 → エリザベスが逃げようとする → コリンズが追う |
| **エリザベスの行動** | “incessant employment”で感情を隠し、針仕事に集中するふりをする |
| **主要な視覚** | 扉の外で盗み聞きするため**うろつく**ベネット夫人：“dawdled about in the vestibule to watch” |

## SCENE 08：ロングボーン書斎—「母上が……私が……」（第20章）

| 項目 | 原文の根拠 |
|---|---|
| **空間** | 書斎。ベネット氏専用の、本に囲まれた空間 |
| **配置** | ベネット氏は読書から顔を上げ、“calm unconcern”で妻を見る |
| **重要台詞の位置** | 呼ばれたエリザベスは書斎で両親の間に立ち、裁定を聞く |
| **雰囲気** | ベネット氏の冷笑的ユーモアが支配する場所。この部屋だけで彼は完全に自分自身である |

## SCENE 09：ロージングズの大晩餐—キャサリン令夫人との初対面（第29章）

| 項目 | 原文の根拠 |
|---|---|
| **空間** | ロージングズ食堂 → 応接間。玄関ホールには“fine proportion and finished ornaments” |
| **規模** | “all the servants, and all the articles of plate”。大規模な晩餐 |
| **公園横断** | ハンスフォードから公園を徒歩約半マイル：“every park has its beauty” |
| **配置** | キャサリン令夫人が中心。コリンズは食卓末席。エリザベスはシャーロットとド・バーグ嬢の間。ド・バーグ嬢は一言も話さない。 |
| **視覚** | キャサリン令夫人：“tall, large woman, strongly-marked features”。ド・バーグ嬢：“pale and sickly... insignificant”。ジェンキンソン夫人は令嬢の前に遮光スクリーンを置く。 |
| **音** | キャサリン令夫人の絶え間ない独白＋コリンズの同調＋ルーカス卿の畏敬に満ちた沈黙 |

## SCENE 10：ハンスフォード牧師館—最初の求婚（第34章）

| 項目 | 原文の根拠 |
|---|---|
| **空間** | 牧師館応接間。エリザベスは一人で、他の家族はロージングズへ行っている |
| **時間** | 夜。蝋燭一本、または最低限の照明 |
| **天候** | 明示なし。エリザベスは頭痛で留守番するほど気力が落ちている |
| **ダーシーの動線** | 扉が開き一人で入る → 体調を尋ねる → 座って立ち、“walked about the room” → “in an agitated manner”で近づく → 求婚 → 暖炉棚に寄りかかり拒絶を聞く → “walked with quick steps”で部屋を横切る → 退室 |
| **エリザベス** | 座ったまま立たない → 拒絶後“grew more angry every moment” → ダーシー退室後“sat down and cried for half an hour” |
| **主要な視覚** | 暗い部屋で暖炉の光に照らされるダーシーの“pale with anger”の顔と、エリザベスの紅潮した頬 |
| **音** | 二人の声、ダーシーの足音、閉まる扉、玄関扉が開く音だけ |

## SCENE 11：ロージングズ公園の朝—手紙の受け渡し（第35章）

| 項目 | 原文の根拠 |
|---|---|
| **空間** | 公園の木柵沿いの散歩道 → 門 |
| **時間** | 春の早朝 |
| **天候** | “pleasantness of the morning”—晴れた春の朝 |
| **動線** | エリザベスが門から公園を覗く → 木立から紳士が近づく → ダーシーが“holding out a letter” → エリザベスが“instinctively”受け取る → ダーシーは軽く挨拶し木立へ消える |
| **主要な視覚** | “a look of haughty composure”—ダーシーの顔に抑えられた傷。手紙を差し出す手。木の葉の間へ消える後ろ姿 |
| **その後** | エリザベスは歩きながら読み始める：“two sheets of letter paper, written quite through, in a very close hand” |

## SCENE 12：ペンバリー初見（第43章）

| 項目 | 原文の根拠 |
|---|---|
| **接近** | 馬車でロッジから領地へ入り、“a beautiful wood stretching over a wide extent”を通って半マイル登る |
| **最初の眺望** | 丘上から谷の向こうに**ペンバリー邸**：“a large, handsome stone building, standing well on rising ground, and backed by a ridge of high woody hills; and in front a stream of some natural importance was swelled into greater, but without any artificial appearance” |
| **主要な視覚** | 自然美を最大限に生かす造園：“neither formal nor falsely adorned”。ロージングズの誇示的な華美と正反対 |
| **内部** | 食堂：“large, well-proportioned”。家具：“neither gaudy nor uselessly fine—with less of splendour, and more real elegance, than the furniture of Rosings” |
| **画廊** | 大型肖像画。エリザベスはダーシーの等身大肖像の前に長く立つ：“such a smile over the face, as she remembered to have sometimes seen, when he looked at her” |
| **庭園での再会** | 芝生を横切り川へ向かう途中、“the owner of it himself suddenly came forward from the road which led behind it to the stables”。20ヤードで向かい合い、二人とも深く赤面する。 |

## SCENE 13：ラムトンの宿—駆け落ちの知らせ（第46章）

| 項目 | 原文の根拠 |
|---|---|
| **空間** | 宿の居間。エリザベスが一人で手紙を読む |
| **時間** | 朝。叔父夫妻は散歩中 |
| **動線** | 読み終えて飛び上がる → 扉へ走る → 扉が開き**ダーシー登場** → 青ざめた顔と慌ただしさに驚く → 彼女は座って泣く → ダーシーは眉を寄せ歩き回る → 優しく慰める → “one serious parting look”を残して退室 |
| **主要な視覚** | 手紙を握る震える手。歩き回るダーシーの影。“I have not an instant to lose” |
| **音** | エリザベスのすすり泣き、ダーシーの慎重な足音、扉が閉まる音 |

## SCENE 14：ロングボーンのwilderness—キャサリン令夫人との対決（第56章）

| 項目 | 原文の根拠 |
|---|---|
| **空間** | ロングボーン庭園の人工自然林＋装飾用hermitage |
| **接近** | キャサリン令夫人が四頭立て馬車で到着 → ベネット夫人が応対 → エリザベスを庭へ連れ出す |
| **時間** | 朝 |
| **配置** | 砂利道 → 小林の中で二人だけが対峙。馬車は玄関で待ち、侍女は車内にいる |
| **主要な視覚** | キャサリン令夫人の威圧的な体格とエリザベスの断固たる顔。二人の間に落ちる葉影 |
| **音** | 砂利を踏む音、令夫人の命令口調、皮肉なほど平穏な鳥の声 |

## SCENE 15：ロングボーン散歩道—二度目の求婚（第58章）

| 項目 | 原文の根拠 |
|---|---|
| **空間** | ロングボーン近郊、ルーカス・ロッジ方面の田舎道 |
| **時間** | 秋の朝 |
| **動線** | 5人—ビングリー、ジェイン、エリザベス、キティ、ダーシー—で出発 → ビングリーとジェインが遅れる → キティがルーカス家で離脱 → **エリザベスとダーシーだけになる** |
| **風景** | 秋の田舎道、木の葉、野原：“They walked on without knowing in what direction” |
| **主要な視覚** | 並んで歩き話す二人の横顔。エリザベスは表情を見られず、“she could not look, she could listen”—声だけが感情を運ぶ。 |
| **雰囲気** | 最初の求婚の暗く緊張した室内と正反対。明るく開かれた屋外、自由な選択 |
