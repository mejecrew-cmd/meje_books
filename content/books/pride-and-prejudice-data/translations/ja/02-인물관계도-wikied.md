# 『高慢と偏見』――人物相関図

## 1. 家系構造

### Bennet家――Longbourn
```
Mr Bennet ─── Mrs Bennet（旧姓Gardiner）
    │
    ├── Jane Bennet（長女）       ──→ Mr Bingleyと結婚
    ├── Elizabeth Bennet（次女）  ──→ Mr Darcyと結婚
    ├── Mary Bennet（三女）
    ├── Kitty／Catherine Bennet（四女）
    └── Lydia Bennet（末娘）      ──→ Mr Wickhamと結婚
```

### Mrs Bennetの親族
```
Mrs Bennetの弟 ─── Mr Gardiner（ロンドンの商人）
                    │
                    └── Mrs Gardiner（Derbyshireに縁がある）
                         ＋子ども4人（娘2人、息子2人）

Mrs Bennetの妹 ─── Mrs Philips（Meryton在住）
                    │
                    └── Mr Philips（事務弁護士）
```

### Darcy家――Pemberley
```
故Mr Darcy ─── Lady Anne Darcy
    │             │
    │             └── 姉妹：Lady Catherine de Bourgh
    │
    ├── Fitzwilliam Darcy       ──→ Elizabethと結婚
    └── Georgiana Darcy（16歳）
         ［共同後見人：Darcy＋Colonel Fitzwilliam］
```

### de Bourgh家――Rosings
```
故Sir Lewis de Bourgh ─── Lady Catherine de Bourgh
                           │
                           └── Miss Anne de Bourgh
                                ＋Mrs Jenkinson（付添人／家庭教師）
```

### Bingley家――Netherfield
```
故Mr Bingley（父、北部の商人）
    │
    ├── Charles Bingley      ──→ Janeと結婚
    ├── Caroline Bingley
    └── Louisa Bingley ─── Mr Hurst
         （Mrs Hurst）
```

### Lucas家――Lucas Lodge
```
Sir William Lucas ─── Lady Lucas
    │
    ├── Charlotte Lucas（27歳） ──→ Mr Collinsと結婚
    ├── Maria Lucas
    └── 年少の息子たち
```

### Collins
```
Mr Collins（牧師、Mr Bennetの遠縁）
    │
    └── 限嗣相続（Entail）によるLongbourn領地の法定相続人
         Hunsford教区の牧師
         Lady Catherineの後援（patronage）を受ける
```

## 2. 中核関係の変遷（事件順）

### Darcy ↔ Elizabeth（小説の中心軸）
```
EVT-003  ［反感の始まり］Darcyの「tolerable」という侮辱 → Elizabethの反感
EVT-004  ［一方的な魅了］Darcyが「fine eyes」に惹かれる（Elizabethは知らない）
EVT-009  ［bewitched］Darcyが完全に魅了される → 意図的に距離を置く
EVT-016  ［対決］舞踏会でのダンス＋Wickhamをめぐる正面衝突
EVT-028  ［決裂］最初の求婚／拒絶「last man in the world」
EVT-029  ［転換］Darcyの手紙 → 真実の暴露
EVT-030  ［自覚］「Till this moment, I never knew myself」
EVT-035  ［再評価］PemberleyでDarcyの人格を再発見
EVT-036  ［再会］完全に変わったDarcyの態度
EVT-039  ［愛の自覚］「all love must be vain」――初めて愛と名づける
EVT-042  ［感謝］Darcyによる秘密の仲介が明らかになる
EVT-046  ［結合］二度目の求婚を受諾
EVT-048  ［結婚］Pemberleyの女主人となる
```

### Bingley ↔ Jane
```
EVT-003  ［初対面］舞踏会で2度踊る → 相互の好意
EVT-006  ［深化］Janeの病気を心から心配
EVT-010  ［別れ］Janeが帰宅し、Bingleyは名残惜しむ
EVT-019  ［分離］Darcy＋Miss Bingleyの共謀でロンドンへ撤収
EVT-021  ［絶望］Miss Bingleyの手紙でJaneの希望が消える
EVT-023  ［偽善の露呈］ロンドンでのMiss Bingleyの冷遇 → 友情の終わり
EVT-037  ［再会］PemberleyでBingleyが「11月26日以来会っていない」と恋しがる
EVT-043  ［帰還］Darcyが引き離しを撤回 → Netherfieldへ戻る
EVT-044  ［婚約］再求婚＋受諾
EVT-048  ［結婚］
```

### Wickham ↔ 複数の人物
```
Wickham ↔ Darcy：
  - 幼少期：同じ領地で共に育つ（故Mr Darcyの名付け子）
  - 3,000ポンド相当の聖職禄を自ら放棄 → 放蕩 → 再要求 → 拒絶
  - Georgiana誘惑未遂（EVT-029で発覚）
  - Lydiaとの逃亡をめぐる最後の対決（EVT-038～040）

Wickham ↔ Elizabeth：
  EVT-014  偽りの証言で信頼を得る
  EVT-022  Miss Kingへ乗り換える → Elizabethの幻想が消え始める
  EVT-030  手紙で正体が暴かれる → 信頼が完全に崩壊
  EVT-033  最後の対面――偽善を直接見抜く

Wickham ↔ Lydia：
  EVT-038  Brightonから逃亡（結婚の意思なし）
  EVT-040  Darcyの秘密の仲介で結婚成立
```

### Collins ↔ Bennet家 ↔ Charlotte
```
EVT-011  CollinsがLongbournを訪問（限嗣相続の人格化）
EVT-012  求婚相手：Jane → Elizabethへ変更
EVT-017  Elizabethが求婚を拒絶
EVT-018  Mr BennetがElizabethの味方をする
EVT-020  CharlotteがCollinsを受け入れる → Elizabethとの友情に亀裂
```

## 3. 階級／経済関係図

### 財産順位
```
［最上位］
  Mr Darcy           年1万ポンド＋Pemberley領地＋ロンドンのタウンハウス
  Lady Catherine     Rosings領地＋莫大な財産

［上位］
  Mr Bingley         約10万ポンドを相続（年4,000～5,000）／領地は未購入
  Miss de Bourgh     Rosingsの相続人（金額不明、莫大）
  Georgiana Darcy    持参金3万ポンド

［中位］
  Mr Bennet          年2,000ポンド（限嗣相続により娘たちに不利）
  Mr Collins         Hunsford教区からの収入（かなり安定）

［下位］
  Mr Wickham         無一文＋借金（Darcyが救済）
  Bennet姉妹各人     母の死後約200ポンドずつ（利率4％の国債）
```

### 後援（Patronage）関係
```
Lady Catherine ──［聖職推薦権］──→ Mr Collins
Mr Darcy ──［複数教区の聖職推薦権］──→ 複数の牧師
Mr Darcy ──［少尉職を購入］──→ Mr Wickham（秘密の仲介）
```

### 限嗣相続（Entail）の連鎖
```
Mr Bennetの領地（Longbourn）
    │
    └──［限嗣相続］──→ Mr Collins（男性相続人）
                         │
                         └──→ Charlotte Lucas Collins（将来の女主人）

※ Mrs Bennetと5人の娘は相続から除外される
```

## 5. 感情／関係類型別の分類

### 恋愛／結婚関係
| 関係 | 性格 | 結果 |
|---|---|---|
| Darcy ↔ Elizabeth | 高慢と偏見を乗り越えた真実の愛 | 結婚 |
| Bingley ↔ Jane | 純粋な相思相愛、外部介入による試練 | 結婚 |
| Wickham ↔ Lydia | 衝動と無分別、金銭で成立 | 結婚（不幸） |
| Collins ↔ Charlotte | 現実的妥協、感情のない契約 | 結婚（安定しているが無味乾燥） |
| Mr Bennet ↔ Mrs Bennet | 若い頃の錯覚、とうに消滅 | 結婚（冷笑的な共存） |

### 敵対／競争関係
| 関係 | 性格 |
|---|---|
| Miss Bingley → Elizabeth | Darcyをめぐる一方的な嫉妬と敵意 |
| Miss Bingley → Jane | 表面的友情＋裏での引き離し |
| Lady Catherine → Elizabeth | 階級的侮蔑＋Miss de Bourghとの政略結婚を守ろうとする |
| Darcy ↔ Wickham | 幼少期の仲間から敵へ（Georgiana事件） |

### 友情／信頼関係
| 関係 | 性格 |
|---|---|
| Elizabeth ↔ Jane | 最も深い姉妹の絆、秘密を共有する相手 |
| Elizabeth ↔ Charlotte | 結婚観の衝突で亀裂、部分的に回復 |
| Elizabeth ↔ Mrs Gardiner | 賢明な助言者であり情緒的な支え |
| Darcy ↔ Bingley | 性格は正反対だが最も親しい友人 |
| Darcy ↔ Colonel Fitzwilliam | 従兄弟でGeorgianaの共同後見人 |

### 後援／従属関係
| 関係 | 性格 |
|---|---|
| Lady Catherine → Collins | 聖職推薦権に基づく絶対的な後援と追従 |
| Darcy → Wickham | 先代の遺産 → 裏切り → 秘密の救済 |
| Darcy → Bingley | 判断力で優位に立つ友情（引き離し／撤回） |

## 6. 翻訳者向け・関係変化の中核タイムライン

人物間の呼称、口調、態度は事件に応じて変化するため、翻訳時には以下のタイムラインを参照すること：

| 時点 | Darcy → Elizabeth | Elizabeth → Darcy | Bingley → Jane | Wickham → Elizabeth |
|---|---|---|---|---|
| 第1～6章 | 無関心 → 秘かな魅了 | 単純な反感 | 好意の始まり | 未登場 |
| 第7～13章 | bewitched＋抑制 | 反感を維持 | 心からの心配 | 未登場 |
| 第14～17章 | 距離を置く | 偏見が固定（Wickhamの証言） | 好意の頂点 | 魅力的な被害者 |
| 第18～23章 | 苦しい魅了 | 敵意が最大化 | 引き離される | Miss Kingへ乗り換え |
| 第24～34章 | 抑制失敗 → 求婚 | 「last man in the world」 | 分離＋絶望 | 関係消滅 |
| 第35～40章 | 手紙による自己弁護 | 偏見を自覚＋羞恥 | 変化なし | 幻滅が完了 |
| 第41～45章 | 完全に変わった態度 | 尊敬＋感謝＋まだ名づけられない魅了 | 恋しさを維持 | Wickhamへの軽蔑 |
| 第46～54章 | 秘密の救出 | 愛を自覚＋「all vain」 | 帰還＋再接近 | 逃亡者／詐欺師 |
| 第55～61章 | 再求婚 | 受諾 | 婚約＋結婚 | 一族の恥 |
