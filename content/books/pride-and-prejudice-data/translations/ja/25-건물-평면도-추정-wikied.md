# 『高慢と偏見』建物平面図（推定・創作）

> ⚠️ **本書は推定と創作による再構成である。** 原作に断片的に現れる部屋、動線、窓、庭園の描写を総合して合理的な平面図を再構成し、英国ジョージアン／パラディアン様式の標準と作中の細部を組み合わせた。

---


## 翻訳・翻案での活用

| 用途 | 活用方法 |
|---|---|
| **ドラマ／映画の美術部門** | 各屋敷の部屋構成を一貫して設計する |
| **舞台翻案** | 主要場面の空間を再現する：第34章の求婚、第43章のペンバリー画廊、第56章のwildernessでの対決 |
| **翻訳での描写補足** | 原文の曖昧な空間描写を自然に補う際の参考 |
| **小説翻案** | 人物の動線を加える際に空間的合理性を確保する |
