# 『高慢と偏見』―メニューと食文化（推定・創作）

> ⚠️ **本書は推定・創作に基づく文書である。** 作中で明記される料理（ラグー、鹿肉、温室果物、ホワイト・スープ、ポートワイン）に、摂政時代の標準的な食事と各家の経済力を組み合わせ、実際の食卓構成を再現した。

---

## 摂政時代の食事の基本構成

### 正餐（Dinner）のコース構成

| コース | 内容 |
|---|---|
| **First Course** | スープ＋魚＋肉を一度に食卓へ並べる |
| **Second Course** | 家禽＋野菜＋ペイストリーを一度に並べる |
| **Dessert** | 果物＋木の実＋小さなケーキ。食卓を整え直して別に供する |

> **方式：** “à la française”―すべての料理を一度に食卓へ広げ、各人が近くの料理を取り分ける。

### 家ごとの水準
- **ペンバリー／ロージングズ：** 毎食が正式なコースで、装飾も華やか
- **ネザーフィールド：** 正式なコースだが、ビングリーの素朴な好みを反映
- **ロングボーン：** 平日は簡素、客を迎えるときは正式
- **ハンスフォード：** 質素でコースを削減
- **ウィカムとリディア：** きわめて質素

## 1. ペンバリー（Pemberley）―ダーシー家の正餐（推定メニュー）

### 平日の家族正餐（ダーシー＋ジョージアナ）

```
[First Course]
- White Soup（ホワイト・スープ、白身の家禽がベース）
- Roasted Pheasant（キジのロースト、領地での狩猟）
- Boiled Salmon with Lobster Sauce（ロブスターソース添えのゆでサケ）
- Vegetable Side: Asparagus, Cauliflower

[Second Course]
- Roast Beef
- Stewed Mushrooms
- Apple Tart
- Plum Pudding

[Dessert]
- Fruit Pyramid: Grapes, Nectarines, Peaches（★ 温室果物）
- Walnuts, Almonds
- Madeira Wine
```

### 第45章 ガーディナー夫妻＋エリザベスとの正餐（推定）

```
[First Course]
- Turtle Soup（正式な客へのもてなし）
- Glazed Ham
- Stewed Trout

[Second Course]
- Roast Capon
- Savoury Pie
- French Beans

[Dessert]
- Grapes, Nectarines, and Peaches ★（作中に明記）
- Cheese, Biscuits

[Wine]
- Claret（ボルドーの高級赤ワイン）
- Madeira（デザート用）
- Port（男性が食後に飲む）
```

### 飲み物
- **Tea**（午後、インド産の高級茶）
- **Coffee**（正餐後）
- **Hot Chocolate**（朝食に出た可能性）

## 2. ロージングズ（Rosings）―レディ・キャサリンの正餐

### 第29章 最初の正餐の推定メニュー

原作の明記：“exceedingly handsome dinner... all the servants, and all the articles of plate”

```
[First Course]
- Spring Soup with Green Peas
- Salmon with Sauce Hollandaise
- Boiled Turkey with Oyster Sauce
- Side Dishes: Spinach, Carrots, Pickles

[Second Course]
- Roast Lamb（季節は春）
- Sweetbreads
- Veal Olives
- Lemon Cheesecakes
- Apricot Tart

[Dessert]
- Pyramid of Fruits（温室果物）
- Italian Cream
- Sweet Biscuits
- Various Sugared Confections
```

### 特徴
- **レディ・キャサリンの誇示**―コリンズが料理を一品ずつ褒める
- **アン・ド・バーグはほとんど食べない**―ジェンキンソン夫人が見守り、別の料理を勧める
- **コリンズとサー・ウィリアム・ルーカスの過剰な称賛**

## 3. ネザーフィールド（Netherfield）―ビングリー家の正餐

### 第8章 エリザベス到着日の正餐（推定）

原作の手がかり：ビングリー姉妹が好む“ragout”が登場する。

```
[First Course]
- Soup à la Reine
- Boiled Carp
- Ragoût of Veal（★ 作中に明記。フランス風シチュー）
- Glazed Onions

[Second Course]
- Roast Partridge
- Pigeon Pie
- French Beans
- Charlotte Russe（実質的にはデザート）

[Dessert]
- Fruit, Cheese, Wine

※ ハースト氏はragoutだけを好み、素朴な料理を拒む
※ エリザベスはplain dishを好む
```

### 第18章 ネザーフィールド舞踏会の夜食（Supper）

原作に明記される“white soup”は、舞踏会に欠かせない料理だった。

```
[Supper Spread―真夜中ごろ]
- White Soup ★（家政婦長ニコルズが作る）
- Cold Ham
- Cold Chicken（作中に明記：“cold ham and chicken”）
- Trifle（イギリス式デザート）
- Jellies, Creams
- Fruit
- Wine, Negus（ワイン＋水＋砂糖＋香辛料）
```

### 第13章 コリンズ歓迎の正餐―正式な支度

原作ではコリンズが食事を“highly admired”したとある。

```
[格上げしたメニュー]
- White Soup
- Boiled Fish
- Roast Beef + Yorkshire Pudding
```

> **ベネット夫人の自尊心：** 第13章でコリンズが誰が料理したのかと尋ねると、夫人は“her daughters had nothing to do in the kitchen”と強調する。専門の料理人がいることへの誇りである。

### 第55章 ダーシーとビングリーを迎える正餐

## 10. 飲み物ごとの社会的意味

| 飲み物 | 階級・文脈 | 作中 |
|---|---|---|
| **Champagne** | 最上流 | ペンバリー、ロージングズ |
| **Claret（Bordeaux）** | 上流 | ペンバリーの正餐に出たと推定 |
| **Madeira** | 上流のデザートワイン | すべての正式な食事 |
| **Port Wine** | 中流の男性 | フィリップス叔父の体臭、食後 |
| **Sherry** | 中流、女性も飲む | 軽い食前酒 |
| **Negus** | 温かいワイン飲料 | 舞踏会の夜食 |
| **Tea（Indian）** | 普遍的だが等級差がある | すべての家庭。等級が富を示す |
| **Coffee** | 正餐後 | すべての正式な食事 |
| **Beer/Ale** | 下層の日常 | 使用人 |
| **Cider** | 田舎 | 下層 |
| **Hot Chocolate** | 朝食の贅沢 | ペンバリーと推定 |

## 11. 季節ごとの食材

### 秋（小説冒頭、9～11月）
- 狩猟期の始まり（キジ、ヤマウズラ、野ウサギ）
- イギリス産のリンゴ、ナシ、プラム
- カボチャ、キャベツ

### 冬（12～2月）
- 保存肉（塩漬け、燻製）
- 冬野菜（カブ、キャベツ、タマネギ）
- 保存果物（ジャム、酢漬け）
- **鹿肉（venison）**―狩猟期

### 春（3～5月）―ハンスフォード期
- 春野菜（アスパラガス、エンドウ豆）
- 子羊肉
- 産みたての卵

### 夏（6～8月）―ペンバリー期
- 豊富な新鮮野菜
- **温室果物**（ブドウ、モモ、ネクタリン）
- 豊富な魚類
- イチゴ、ラズベリー

## 12. 食卓作法（翻訳・翻案時の参考）

| 項目 | 規則 |
|---|---|
| **席順** | 女主人が食卓の端に座り、主賓はその右隣 |
| **給仕** | すべての料理を食卓に同時に並べ、各自が近くのものを取り分ける |
| **女性が先に退出** | デザート後、女性は応接間へ移り、男性はポートワインを飲む |
| **再合流** | 約1時間後、男性が応接間で女性と合流する（ティータイム） |
| **コリンズの過剰さ** | コースごとに褒めるのは品位に欠ける行為 |
| **ハースト氏の無礼** | 食べ物にしか関心がなく、エリザベスが“plain dish”を好むと言うと会話を打ち切る |
