# 『高慢と偏見』――時代語誤訳防止辞典

> オースティンの時代（摂政期、1810年代頃）の英語には、現代英語と**綴りは同じでも意味が異なる**語が数多く存在する。
> 本文書は、**現代的な意味で訳すと誤訳になる語**を整理し、翻訳上の事故を防ぐためのものである。
>
> **危険度の分類：**
> - **［危険］**――現代の意味と正反対、または完全に異なる。誤訳すると文全体の意味が逆転する。
> - **［注意］**――現代の意味と一部重なるが、中心的なニュアンスが異なる。
> - **［参考］**――現代では消えた用法、または自然な翻訳のために理解すべき用法。

---

## A. ［危険］――現代の意味と正反対／完全に異なる語

### 1. sensible
| 項目 | 内容 |
|---|---|
| **オースティン時代の意味** | (1) 分別のある、賢明な（= having good sense）／(2) ～を認識している（= aware of） |
| **現代英語の意味** | 敏感な（= sensitive）← **この意味ではない** |
| **作中例** | "a sensible, intelligent young woman"（第5章、シャーロット）= **分別があり聡明な**女性 |
| | "I am very sensible of the honour"（第19章、エリザベス）= その栄誉を**よく承知しております** |
| **訳語** | 「分別のある」／「賢明な」／「～をよく認識している」 |
| **誤訳例** | ~~「敏感な若い女性」~~ → 人物像がまったく別物になる |

### 2. condescension / condescend
| 項目 | 内容 |
|---|---|
| **オースティン時代の意味** | 身分の高い者が**自ら身を低くして好意を施すこと**。褒め言葉。 |
| **現代英語の意味** | 上から目線で見下す態度 ← **正反対** |
| **作中例** | "such affability and condescension"（第14章、コリンズがレディ・キャサリンを称賛）= レディ・キャサリンが**身を低くしてくださるご厚意** |
| | "She had even condescended to advise him to marry"（第14章）= **わざわざ身を低くして**結婚を勧めてくださった |
| **訳語** | 「身を低くしてくださる」／「ありがたくも」／「わざわざ」 |
| **誤訳例** | ~~「そのような傲慢さと上から目線」~~ → コリンズの追従が非難に変わる |
| **注意** | オースティンはこの語を**アイロニー**としても用いる。コリンズは心から褒めているが、読者には滑稽に映る――翻訳ではこの二重性を残す必要がある。 |

### 3. nervous / nerves
| 項目 | 内容 |
|---|---|
| **オースティン時代の意味** | 神経が**衰弱している**／神経疾患がある（医学的概念） |
| **現代英語の意味** | 緊張している、不安な ← ニュアンスが異なる |
| **作中例** | "You have no compassion on my poor nerves."（第1章、ベネット夫人）= 私の**哀れな神経衰弱**に同情もないのですね |
| | "she fancied herself nervous"（第1章、語り手）= 自分は**神経が弱いと思い込んでいた** |
| **訳語** | 「神経（が弱い）」／「神経衰弱」――単なる「緊張」ではない |
| **要点** | ベネット夫人の “nerves” は医学的な自己診断（実際には思い込み）であり、単純な不安ではない。 |

### 4. nice
| 項目 | 内容 |
|---|---|
| **オースティン時代の意味** | 精密な、気難しい、微妙な区別をする（= precise, discriminating） |
| **現代英語の意味** | 良い、親切な ← ほとんど関係がない |
| **作中例** | 直接の使用は少ないが、"a nice distinction" = **微細な**区別 |
| **訳語** | 「精密な」／「気難しい」／「微妙な」 |

### 5. wanting
| 項目 | 内容 |
|---|---|
| **オースティン時代の意味** | **不足している**、欠けている（= lacking） |
| **現代英語の意味** | 欲している（= desiring）← 完全に異なる |
| **作中例** | "No one admitted to the privilege of hearing you can think anything wanting."（第31章、ダーシー）= あなたの演奏を聴く幸運に恵まれた者なら、**何か不足がある**とは思わないでしょう |
| **訳語** | 「不足している」／「欠けている」 |
| **誤訳例** | ~~「何かを欲しているとは思わないでしょう」~~ |

### 6. address（名詞）
| 項目 | 内容 |
|---|---|
| **オースティン時代の意味** | (1) 話しぶり、物腰、接し方／(2) 手紙の宛先／(3) 求愛行為 |
| **現代英語の意味** | ほぼ「住所」という単一の意味 |
| **作中例** | "very pleasing address"（第15章、ウィカム）= とても好感の持てる**話しぶり／物腰** |
| | "my addresses"（第20章、コリンズ）= 私の**求愛** |
| | "Jane had written the direction remarkably ill"（第46章）= ジェインは**宛先を**ひどく読みにくく書いていた（ここでは direction = 宛先） |
| **訳語** | 文脈に応じて「話しぶり」／「物腰」／「求愛」／「宛先」 |

### 7. direction
| 項目 | 内容 |
|---|---|
| **オースティン時代の意味** | 手紙の**宛先／送り先**（= address on a letter） |
| **現代英語の意味** | 方向、指示 |
| **作中例** | "the wished-for direction"（第52章、ウィカムの住所）= 待ち望んでいた**宛先** |
| **訳語** | 「宛先」／「居所」 |

## B. ［注意］――中心的なニュアンスが異なる語

### 8. handsome
| 項目 | 内容 |
|---|---|
| **オースティン時代の意味** | (1) 人：容姿のよい／美しい（男女共用）／(2) 行為・金額：**気前のよい、寛大な**／(3) 建物：堂々とした |
| **現代との違い** | 現代ではほぼ男性の容姿にのみ用いる |
| **作中例** | "a handsome young man"（ビングリー）= 容姿のよい／"handsome modern building"（ロージングズ）= **堂々とした**建物／"a very handsome thought"（コリンズがダーシーを称賛）= **立派な**考え |
| **訳語** | 人：「容姿のよい／美しい」／建物：「堂々とした／立派な」／行為：「立派な／寛大な」 |

### 9. civil / civility
| 項目 | 内容 |
|---|---|
| **オースティン時代の意味** | 礼儀正しい――ただし**最低限の礼儀**。真心があるとは限らない。 |
| **現代との違い** | 現代も近いが、オースティンでは “civil” と “warm/cordial” が明確に区別される |
| **作中例** | "with cold civility"（第46章、エリザベス→ダーシー）= **冷ややかな礼儀**をもって |
| | "civil disdain"（第18章、ビングリー嬢）= **丁重な侮蔑**――意図的な撞着語法 |
| **訳語** | 「礼儀正しい」。真心のなさを示す場合は「形式的な」／「儀礼的な」 |
| **段階** | civil（最低限）< polite（標準）< courteous（丁重）< warm（温かい）< cordial（心からの） |

### 10. consequence
| 項目 | 内容 |
|---|---|
| **オースティン時代の意味** | **社会的重要性／体面／地位**（= social importance） |
| **現代英語の意味** | 結果（= result） |
| **作中例** | "give consequence to young ladies who are slighted by other men"（第3章、ダーシー）= ほかの男たちが相手にしない娘たちを**重要人物のように扱う**気はない |
| | "a man of consequence" = **体面／地位のある**人物 |
| **訳語** | 文脈に応じて「体面」／「重要性」／「地位」 |

### 11. intelligence
| 項目 | 内容 |
|---|---|
| **オースティン時代の意味** | **知らせ、情報**（= news, information） |
| **現代英語の意味** | 知能 |
| **作中例** | "the great intelligence"（第30章、コリンズがダーシー到着を告げる）= この**重大な知らせ** |
| | "fresher intelligence of her former friends"（第25章）= 以前の友人たちについての**より新しい知らせ** |
| **訳語** | 「知らせ」／「情報」／「便り」 |

### 12. design
| 項目 | 内容 |
|---|---|
| **オースティン時代の意味** | **意図、計画、企み**（= intention, plan, scheme） |
| **現代英語の意味** | デザイン |
| **作中例** | "Is that his design in settling here?"（第1章、ベネット氏）= ここに住み着くのは彼の**意図**なのか？ |
| | "without any design"（第22章、ジェイン）= 何の**企み**もなく |
| **訳語** | 「意図」／「計画」／「下心」 |

### 13. countenance
| 項目 | 内容 |
|---|---|
| **オースティン時代の意味** | (1) **顔／顔色／表情**／(2) 支持する、是認する |
| **現代英語の意味** | 顔（現在はまれ） |
| **作中例** | "a fine countenance"（ウィカム）= 立派な**顔立ち** |
| | "Bingley could keep his countenance"（第9章）= ビングリーは**平静な表情を保つ**ことができた |
| **訳語** | 「表情」／「顔色」／「顔立ち」 |

### 14. character
| 項目 | 内容 |
|---|---|
| **オースティン時代の意味** | (1) 性格／(2) **評判、世間の評価**――二つの意味が同時に働くことがある |
| **作中例** | "His character was decided."（第3章）= 彼の**評判は決まった**（= 傲慢な男と断定された） |
| | "Wickham's character" = ウィカムの**実像／本性**（後半で反転する） |
| **訳語** | 文脈により「性格」と「評判」を区別。両義が働く場合は「人物像」「人となり」 |

### 15. establishment
| 項目 | 内容 |
|---|---|
| **オースティン時代の意味** | (1) **嫁入り／身の落ち着き先**（結婚によって独立した家庭を築くこと）／(2) 独立生活の基盤全般 |
| **作中例** | "what an establishment it would be for one of them"（第1章）= 娘の一人にとってどれほどよい**縁組／落ち着き先**になることか |
| | "the pure and disinterested desire of an establishment"（第22章、シャーロット）= ひたすら**安定した住まい**を得たいという望み |
| **訳語** | 「嫁入り」／「身の落ち着き先」／「安定した家庭」――「施設」ではない |

### 16. connexion(s) / connection(s)
| 項目 | 内容 |
|---|---|
| **オースティン時代の意味** | **親族関係／人脈**（= relatives, family ties） |
| **現代英語の意味** | 一般的なつながり |
| **作中例** | "low connexions"（ダーシー）= **身分の低い縁者／人脈**（= 商人の叔父、弁護士の叔父） |
| | "the connexion was a most unhappy one" = その**姻戚関係**はきわめて不幸なものだった |
| **訳語** | 「人脈」／「姻戚」／「親族関係」 |

### 17. amiable
| 項目 | 内容 |
|---|---|
| **オースティン時代の意味** | **愛すべき、好感を抱かせる、善良な**――現代の “amicable”（友好的な）より強い称賛 |
| **作中例** | "she is one of the most tractable creatures... a very great favourite"（第33章、エリザベスによるジョージアナ評） |
| | "the most amiable man of my acquaintance"（第24章、ジェインがビングリーを評して）= 私の知るもっとも**愛すべき**人 |
| **訳語** | 「愛すべき」／「優しい」／「好感の持てる」 |

### 18. agreeable
| 項目 | 内容 |
|---|---|
| **オースティン時代の意味** | **一緒にいて楽しい人、愉快な人**（= pleasant company） |
| **作中例** | "If he had been so very agreeable, he would have talked to Mrs. Long"（第5章、ベネット夫人）= それほど**感じのよい**人ならロング夫人と話したはず |
| **訳語** | 「愉快な」／「好感の持てる」／「感じのよい」 |
| **段階** | disagreeable（不愉快）< tolerable（我慢できる）< agreeable（愉快）< amiable（愛すべき） |

## C. ［参考］――現代では消えた用法

### 19. complaisance / complaisant
| 項目 | 内容 |
|---|---|
| **意味** | 人を喜ばせようとする**従順で協力的な態度**／快く応じること |
| **混同注意** | “complacency”（自己満足）とは異なる |
| **作中例** | "his complaisance"（第6章）= 彼の**人に合わせる態度** |

### 20. propriety / impropriety
| 項目 | 内容 |
|---|---|
| **意味** | **適切さ／礼儀作法／身分にふさわしい行動**――単なる「適切」ではなく、社会規範の遵守 |
| **作中例** | "want of propriety"（ダーシーの手紙、ベネット家への批判）= **礼儀作法の欠如** |
| **訳語** | 「礼節」／「体面」／「品行」 |

### 21. scheme
| 項目 | 内容 |
|---|---|
| **意味** | **計画**（現代の否定的な「陰謀」より中立的） |
| **作中例** | "No scheme could have been more agreeable"（第27章、旅行計画）= これ以上楽しい**計画**はなかった |
| **訳語** | 「計画」／「案」――必ずしも否定的ではない |

### 22. apprehension
| 項目 | 内容 |
|---|---|
| **意味** | (1) **理解、把握**／(2) 不安――オースティンでは (1) のほうが多い |
| **作中例** | "I had no apprehension of his feeling a serious attachment"（第35章、ダーシーの手紙）= 彼が真剣な愛情を抱いているとは**認識していなかった** |
| **訳語** | 「認識」／「把握」（不安の意味の場合のみ「不安」） |

### 23. complaint
| 項目 | 内容 |
|---|---|
| **意味** | (1) 不平／(2) **病気、病症** |
| **作中例** | "nervous complaints"（ベネット夫人）= **神経の病** |
| **訳語** | 文脈を確認。ベネット夫人の場合は常に「病気」 |

### 24. prospect
| 項目 | 内容 |
|---|---|
| **意味** | (1) **眺望／景色**（literal view）／(2) 将来の見通し |
| **作中例** | "a charming prospect over that gravel walk"（第9章）= 砂利道の向こうに広がる見事な**眺め** |
| **訳語** | 建物・窓の文脈では「眺望／景色」、未来の文脈では「見通し／可能性」 |

### 25. natural
| 項目 | 内容 |
|---|---|
| **意味** | (1) 自然な／(2) 自然主義的な（造園）／(3) **庶出の**（まれ）／(4) 生来の |
| **作中例** | "natural beauty"（ペンバリーの造園）= 人工的でない、**自然美を生かした**美しさ |
| | "a sort of natural self-consequence"（リディア）= 一種の**生来の**うぬぼれ |
| **訳語** | 文脈に応じて区別。ペンバリーの “natural” は**自然主義美学**の用語。 |

## D. 好感度形容詞の段階表――翻訳時の強度調整

オースティンが人物描写に用いる形容詞には、**精密な段階**がある。

### 肯定的段階（弱→強）
```
tolerable（我慢できる、まあまあ）← 最低――ダーシーによるエリザベス評
  ↓
civil（礼儀正しい）← 儀礼的な最低限
  ↓
agreeable（愉快な、一緒にいて心地よい）
  ↓
pleasing（好感の持てる）
  ↓
amiable（愛すべき、善良な）
  ↓
charming（魅力的な）
  ↓
delightful（楽しい、喜びを与える）
  ↓
excellent（すばらしい）← 最高
```

### 否定的段階（弱→強）
```
not very agreeable（あまり愉快ではない）← 英国式の婉曲表現
  ↓
disagreeable（不愉快な）
  ↓
unpleasant（不快な、より直接的）
  ↓
ill-natured（意地の悪い）
  ↓
proud（高慢な）← ダーシーに適用
  ↓
insolent（無礼な）
  ↓
abominable（忌まわしい）
  ↓
despicable（軽蔑すべき）← 最強――エリザベスの自己批判
```

### 翻訳原則
- この段階を**訳文でも同じ強度差**で表す。
- “tolerable” と “agreeable” をどちらも「悪くない」と訳すと段階が崩れる。
- “civil” と “amiable” をどちらも「親切な」と訳すと、関係性の温度差が消える。

## E. 英国式婉曲表現（Understatement）――誤訳がもっとも多い領域

オースティンは**二重否定、縮小表現、婉曲法**をきわめて頻繁に用いる。直訳するとニュアンスが消失する。

| 原文パターン | 表面的意味 | 実際の意味 | 翻訳指針 |
|---|---|---|---|
| "not unpleasing" | 不快ではない | **かなりよい** | 二重否定をほどきつつ強度を弱く保つ：「悪くない」 |
| "not without" | ～がないわけではない | **明らかにある** | 「たしかに～があった」 |
| "she could not help" | ～せずにはいられなかった | **～した**（強い衝動） | 自然なら二重否定を保つ：「～せずにはいられなかった」 |
| "I dare say" | あえて言えば | **たぶんそうだろう**（軽い推測） | 「たぶん」／「そうでしょう」 |
| "I am sure" | 確実に | **私が思うには**（確信ではなく習慣的な修辞） | ベネット夫人の場合は「きっとそうですとも」（根拠のない確信） |
| "something like regret" | 後悔のようなもの | 後悔だが**認めたくない** | 「後悔のようなもの」――「後悔」と断定しない |
| "not entirely hopeless" | 完全に望みがないわけではない | **少しは望みがある** | 「まったく望みがないわけではない」 |
| "I fancy" | 私は想像する | **～のように思う**（軽い推測） | 「～と思う」／「～のようだ」 |
| "by no means" | どの手段によっても～でない | **決して～でない**（強い否定） | 「決して～ない」 |
| "tolerable/tolerably" | 我慢できる | **かろうじて悪くない**（褒め言葉ではない） | 「まあまあ」／「どうにか」 |

## F. 動詞の時代的ニュアンス

| 動詞 | オースティン時代 | 現代 | 作中例 | 訳語 |
|---|---|---|---|---|
| **to own** | 認める、告白する | 所有する | "I own I was surprised" = 驚いたことを**認める** | 「認める」／「告白する」 |
| **to engage** | (1) 約束する (2) 雇う (3) 関心を引く | 主に婚約する | "she was engaged elsewhere" = ほかに**先約があった** | 文脈を確認 |
| **to want** | **不足する**／必要とする | 欲する | "want of propriety" = 礼節の**不足** | 「不足」／「欠如」 |
| **to wait on** | **訪問する**（儀礼的） | 待つ／給仕する | "to wait on Mr. Bingley" = ビングリー氏を**訪問する** | 「訪問する」／「ご挨拶に伺う」 |
| **to make out** | **理解する、把握する** | 見分ける／キスをする | "I cannot make him out" = 彼を**理解できない** | 「理解する」／「把握する」 |
| **to recollect** | **意識的に思い出す** | 現代では remember とほぼ同義 | オースティンでは「意識して記憶を呼び起こす」ニュアンス | 「思い起こす」／「想起する」 |
| **to flatter oneself** | **自負する**（しばしば思い違い） | 自分にお世辞を言う | "I flatter myself that..." = **僭越ながら自負するに…** | 「自負する」――しばしばアイロニー |
| **to trifle with** | **もてあそぶ、軽く扱う** | — | "You are too generous to trifle with me" = あなたは私を**もてあそぶ**ほど卑劣ではない | 「もてあそぶ」／「弄ぶ」 |

## G. 翻訳で絶対に混同してはならない対語（Pair）

| 対語 | 違い | 混同した場合 |
|---|---|---|
| **pride** vs **vanity** | pride = 自己評価、vanity = 他人からどう見られたいか（第5章、メアリーの定義） | 小説の題名と主題に関わる核心的区別が崩れる |
| **sense** vs **sensibility** | sense = 理性／分別、sensibility = 感受性／感情 | ジェイン（sensibility）とエリザベス（sense）の対比が消える |
| **civil** vs **polite** vs **amiable** | civil = 最低限の礼儀、polite = 標準、amiable = 心からの好意 | 関係性の温度差が消える |
| **fortune** vs **income** | fortune = 財産総額、income = 年収 | 経済規模を誤読する。ビングリーの10万ポンド対年4,000～5,000ポンド |
| **trade** vs **profession** | trade = 商業（格が低い）、profession = 専門職（格が高い） | 階級偏見の微妙な差が消える |
| **living** vs **life** | living = 聖職禄／教区収入、life = 人生 | ダーシーとウィカムの対立の核心が逆転する |
| **interesting** | オースティン：関心を引く／心を動かす（現代より強い） | “most interesting” を単に「面白い」と訳すと感情の強度が落ちる |
