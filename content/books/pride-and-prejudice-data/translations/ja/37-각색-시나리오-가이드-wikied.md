# Pride and Prejudice — 翻案・脚本制作ガイド

> ⚠️ **本書は創作支援用の文書である。** 媒体、時代、視点ごとに翻案する際、どの要素をどう変更できるか、そしてどの要素には決して手を加えてはならないかを整理する。

---

## 1. 絶対に変更してはならないもの――小説の精髄

### 中核構造――変えれば作品のアイデンティティを失う

| 要素 | 理由 |
|---|---|
| **最初の求婚 → 拒絶 → 手紙 → 自覚**という流れ | 小説の構造的な背骨 |
| **ダーシーが二度求婚する構造** | 変化が中心主題だから |
| **ウィカムの嘘 → 暴露** | 偏見が働く仕組みだから |
| **エリザベスの自己認識**（「I never knew myself」） | 主題の頂点 |
| **ペンバリー訪問 → ダーシーへの印象の反転** | 空間を通して人格を発見する装置 |
| **リディアの駆け落ち → ダーシーの秘密の仲介** | 行動によって愛を証明する展開 |
| **二組の結婚で終わる結末** | ジャンル上の約束 |

### 慎重に扱うべき要素

| 要素 | 注意点 |
|---|---|
| **階級意識** | 現代化すると動機が失われかねないため、設定時代に合わせること |
| **結婚が持つ経済的な切迫感** | 現代の視点で単純な恋愛物語にしてしまう危険がある |
| **女性に対する社会的制約** | 弱めれば対立も弱くなる |
| **コリンズのおかしさ** | 単なる喜劇的人物に堕する危険がある |

## 2. 媒体別の翻案ガイド

### A. 映画（90～150分）

#### 圧縮戦略
- **第I巻（第1～23章）**＝30～40分、全体の3分の1
- **第II巻（第24～42章）**＝30～40分
- **第III巻（第43～61章）**＝30～50分

#### 優先して残すもの
- メリトン舞踏会（EVT-003）
- 最初の求婚（EVT-028）
- 手紙（EVT-029）
- 自覚へ至る散歩（EVT-030）
- ペンバリー（EVT-035、EVT-036）
- 駆け落ち（EVT-038）
- 二度目の求婚（EVT-046）

#### 圧縮・省略できるもの
- コリンズの求婚の細部（EVT-017）――短く処理する
- 第5～6章の一部を含む社交場面
- ガーディナー夫妻の登場場面の一部
- メアリーとキティの一部の場面

#### 視覚的に強調するもの
- ペンバリーを初めて望む景観（第43章）
- ハンスフォードでの求婚を包む閉ざされた空間
- 舞踏会の華やかさ

### B. ミニシリーズ（5～6時間）

#### 1995年BBC版のモデル――全6話、約5時間
- **ほぼすべての章を収録できる**
- 自由間接話法をボイスオーバーで処理できる
- ロンドンでの捜索など、描かれないダーシーの行動を画面に追加できる

#### 優先して追加・拡張するもの
- EVT-039-001～005、ダーシーの秘密の仲介――実際に映像化する
- EVT-019-001、ビングリーとジェインを引き離す協議――映像化する
- 人物の日課の細部

### C. 舞台劇（2時間30分～3時間）

#### 圧縮と象徴化
- **回り舞台またはミニマルなセット**
- 登場人物の数を調整し、端役は統合できる
- コーラスやナレーションで出来事を圧縮する

#### 必須の中核場面
- メリトン舞踏会をオープニングに置く
- ハンスフォードの求婚を第1幕のクライマックスにする
- ペンバリーから第2幕を始める
- 駆け落ちの危機
- 二度目の求婚をフィナーレにする

### D. ミュージカル

#### 歌に適した場面
- 「It is a truth universally acknowledged」――オープニングの群舞
- メアリーの歌――喜劇場面
- 舞踏会――アンサンブル・ダンス
- 「I never knew myself」――エリザベスのソロ
- 「In vain have I struggled」――ダーシーのソロ
- 「All love must be vain」――エリザベスのソロ
- 二度目の求婚――デュエット
- 結婚式――フィナーレ

### E. ウェブトゥーン／漫画

#### コマ割り戦略
- 第一印象＝大きなコマ
- 会話＝小さなコマを連続させる
- 求婚＝人物の表情をクローズアップした一ページ

#### 視覚的モチーフ
- 繰り返されるダーシーの視線
- 実物としてデザインされた手紙
- ワイド画面で見せるペンバリーの風景

## 3. 時代別の翻案

### A. 時代に忠実な翻案――1811年
- すべての設定シートをそのまま活用する
- 衣装、建築、作法を正確に再現する
- **現代の観客にとっての障壁：**階級制度、結婚の切迫感、敬称体系

### B. ヴィクトリア朝への移行――1850～1860年代
- 衣装：クリノリンを用い、より格式高くする
- 婚姻法：条件が変化するが、1882年既婚女性財産法より前
- ダーシー：伯爵以上の爵位も可能
- ウィカム：植民地軍へ送る

### C. エドワード朝への移行――1900年代
- 衣装：ホブルスカート、自動車が登場
- 舞踏会 → カントリーハウス・パーティー
- ダーシー：最後の世代の領主

### D. 1920～1930年代
- モダニズム
- 衣装：フラッパー対伝統
- 自動車と電話が登場
- 結婚の意味が弱まるため、別の葛藤が必要

### E. 現代韓国への翻案――例

#### 変換マトリクス
| 原作 | 現代韓国版 |
|---|---|
| ベネット家 | 5人姉妹を抱える中産階級の自営業者家庭 |
| 限嗣相続 | 事業の失敗など別の形の経済危機 |
| ダーシー | 財閥企業の後継者／江南の大富豪 |
| ビングリー | ダーシーの親友 |
| ウィカム | 魅力的な詐欺師 |
| コリンズ | 宗教と現代韓国の隔たり――弁護士、教授、保険外交員など |
| レディ・キャサリン | ダーシーの母親または年長の母方のおば |
| 舞踏会 | クラブ・パーティー／結婚式／企業パーティー |
| 軍隊 | 職場または別の魅力的な外部集団 |

#### 必ず残すべき中核
- 第一印象の罠
- 手紙またはメッセージによる真相の暴露
- 家族が愛に及ぼす影響
- 自己認識の瞬間
- 二度告白する構造

#### 現代化の危険
- **結婚の切迫感が弱まる** → キャリア、家族、ビザなど別の切迫した利害が必要
- **階級差が弱まる** → 学歴、出身地、家庭背景など別の差が必要
- **伝達手段が変化する** → カカオトークやメッセンジャーが手紙を代替する

### F. SF翻案
- 宇宙植民地の家門
- ダーシー＝大艦隊の後継者
- ウィカム＝宇宙海賊
- 限嗣相続＝家門の惑星領地を継ぐ制度

### G. ファンタジー翻案
- 魔法の家門
- ダーシー＝強大な魔法使い一族の一員
- 結婚＝家門の魔法の結合
- 舞踏会＝魔法の儀式

## 4. 視点別の翻案

### A. ダーシー視点――リテリング
- **長所：**ダーシーの内面をすべて明かし、秘密の仲介を直接見せられる
- **危険：**謎が消え、エリザベスが真相を発見する過程が弱まる
- **先例：***Mr. Darcy's Diary*のような作品

### B. ウィカム視点
- アンチヒーロー物語または自己正当化
- ジョージアナ事件とリディアの駆け落ちにおけるウィカム側の動機
- **危険：**魅力的な語り手になりやすいため、道徳的な均衡が必要

### C. シャーロット視点
- 結婚をめぐる現実主義者の視点
- ハンスフォードの日常の細部
- コリンズとの結婚生活

### D. メアリー視点
- 「見えない三女」の視点
- 姉妹たちの観察者
- 音楽と本への逃避

### E. ジョージアナ視点
- ライトノベルに適する
- トラウマからの回復と、エリザベスを姉として受け入れる過程

### F. 使用人視点――レイノルズ夫人など
- 『ダウントン・アビー』風
- 「上の階で起こること」を下から観察する

### G. ビングリー視点
- 優柔不断な友情の視点
- ダーシーに頼る姿
- ジェインを忘れられない一年

## 5. ジャンル別の翻案

### A. 喜劇性を強める
- コリンズ、ベネット夫人、レディ・キャサリンの比重を増やす
- ウィカムのおかしな面を強調する
- ベネット氏の機知を強調する

### B. ドラマ／メロドラマ性を強める
- ダーシーとエリザベスの感情を細かく描く
- ジェインとビングリーが一年離れる苦しみ
- リディアの悲劇性

### C. ミステリー性を強める
- ウィカムの正体をよりゆっくり暴く
- ダーシーの秘密の仲介を推理劇にする
- 手紙が届くタイミングでサスペンスを作る

### D. ホラー／ゴシック――実験的
- ペンバリーを恐ろしい場所にする
- ウィカムを真の悪役にする
- レディ・キャサリンを脅威として描く

### E. アクション／スリラー
- ウィカム追跡をアクション・シークエンスにする
- 原作では起こらず、ベネット夫人が心配しただけの決闘場面を追加する
- 逃亡劇を強調する

## 6. 中核場面別の翻案案

### 第34章――最初の求婚

#### 正統版
- 暗い応接間で二人きり
- ダーシーの噴き出すような告白
- エリザベスの怒り
- 激しい対峙

#### 現代化案1――職場
- 会社のオフィスで告白する
- 同じ強度の衝突を起こす

#### 現代化案2――カカオトーク
- 長いメッセージ → 否定的な返信
- 携帯電話の画面を視覚表現に用いる

#### 映像化案
- クローズアップを中心にする
- 二人の視線を交差させる
- 音楽を使わない

### 第43章――ペンバリーを初めて望む場面

#### 正統版
- 馬車から本館の全景を初めて見せる
- 音楽が高まる

#### 現代化
- 江南のペントハウスを初めて訪れる
- または田舎の別荘へ入る

#### 映像化
- ワイドショット → クローズアップ
- ダーシーの肖像画、または写真

### 第46章――駆け落ちの知らせ

#### 正統版
- 二通の手紙とダーシーの登場
- エリザベスが崩れ落ちる

#### 現代化
- 電話がかかり、たまたまダーシーが隣にいる
- SNS上で暴露される

## 7. 絶対に避けるべき変更

### 人物改変の危険
- ❌ **エリザベスが最初からダーシーを好き**――偏見という主題が消える
- ❌ **ダーシーが最初から親切**――変化という主題が消える
- ❌ **ウィカムを同情的に描く**――道徳的な曖昧さが主題をぼかす
- ❌ **コリンズを真面目に描く**――風刺が弱まる
- ❌ **リディアを悲劇的な犠牲者だけにする**――無責任という主題が弱まる

### 構造改変の危険
- ❌ **ダーシーが一度求婚して承諾される**――作品の核心を否定する
- ❌ **手紙なしで自覚する**――もっとも重要な装置を取り除く
- ❌ **ペンバリー訪問を省く**――人格を発見する装置を取り除く
- ❌ **リディアの危機を省く**――ダーシーの変化を証明する装置を取り除く

### 雰囲気改変の危険
- ❌ **暗すぎる**――オースティンの機知が失われる
- ❌ **軽すぎる**――社会批判が失われる
- ❌ **激情的すぎる**――抑制された感情が失われる

## 9. 新しい時代へ移す際のチェックリスト

```
[ ] 結婚が今も切迫している理由があるか？
[ ] 階級や地位の差に意味があるか？
[ ] 第一印象を誤りうる仕組みがあるか？
[ ] 秘密の情報に価値があるか？（手紙／メッセージ）
[ ] 家族が重荷になりうるか？
[ ] 外見と実体が異なりうるか？
[ ] 自己認識の瞬間が成立するか？
[ ] 行動によって愛を証明できるか？
```

8項目すべてがYESなら、優れた翻案が可能である。

## 10. 作品の中核メッセージ――変更不可

1. **第一印象はしばしば間違う。** 傲慢と偏見は双方に存在する。
2. **結婚は社会的・経済的な行為であると同時に、愛の行為でもある。** 両者は衝突する。
3. **自己認識はもっとも難しい悟りである。** 「Till this moment, I never knew myself.」
4. **愛は外見ではなく行動で証明される。** ダーシーの秘密の仲介がその証拠である。
5. **家族は誰を愛するかを決めはしないが、その決断に影響を与える。**
6. **時間が外見と実体を明らかにする。** 「One has all the goodness, the other all the appearance of it.」

この六つのメッセージを守るなら、どのような翻案もなお*Pride and Prejudice*である。
