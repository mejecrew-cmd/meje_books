# 『高慢と偏見』感情語彙対応ガイド

> 作中で使われる感情表現の英語語彙、その対応語、強度等級をまとめたもの。
> 同じ感情でも、強度と色合いによって用いる語は異なる。翻訳における最難関の領域である。

---

## 1. 感情語彙の分類体系

### 強度の5段階
- ⓟ ごくわずか（faint）
- ① 弱い（mild）
- ② 中程度（moderate）
- ③ 強い（strong）
- ④ 激しい（intense）
- ⑤ 極端（extreme）

### 色合い（感情の質感）
- 🌸 温かい（warm）
- ❄️ 冷たい（cold）
- 🔥 激情的（passionate）
- 🌧️ 憂鬱（melancholy）
- ⚡ 激怒（furious）
- 🌫️ 曖昧（ambiguous）

| **esteem** | 尊敬、高い評価 | ② | 🌸 | “she respected, she esteemed” |
| **affection** | 愛情、優しさ | ② | 🌸 | “tender affection” |
| **fondness** | 優しい思い、愛着 | ② | 🌸 | “fond of his sister’s society” |

### 対応語ガイド

```
[弱い好意]
acquaintance：「面識」／「知り合い」
liking：「好ましさ」／「好意」
partiality：「偏った好意」／「ひいき」
inclination：「惹かれる心」／「心が傾く」

[中程度の好意]
regard：「情」／「思い」／「敬意」
esteem：「尊敬」／「敬慕」
affection：「愛情」／「優しさ」
fondness：「優しい思い」
attachment：「深い情」／「愛着」

[強い愛]
admiration：「敬慕」／「感嘆」
love：「愛」／「恋慕」
passion：「情熱」／「激情」
ardour：「熱烈な思い」
adoration：「崇拝」／「思慕」
```

### 作中における段階的な変化（ダーシー → エリザベス）

```
第6章   admiration（感嘆）—“fine eyes”
第10章  bewitched（魅了）—魔法のような惹かれ方
第34章  ardent love（熱烈な愛）—“how ardently I admire and love you”
第58章  unchanged affections（変わらぬ愛情）—静かな深み
```

```

vexation：「煩わしさ」／「苛立ち」—現代的になりすぎないよう、「心を乱される」が安全
annoyance：「煩わしさ」
irritation：「癇に障ること」／「引っ掛かり」

[強]
resentment：「恨み」／「憤り」／「遺恨」
anger：「怒り」／「憤怒」
indignation：「義憤」
abhorrence：「嫌悪」／「忌避」

[激烈]
detestation：「憎悪」／「嫌悪」
fury：「激怒」／「怒りが込み上げる」
wrath：「憤怒」—ほとんど使われない
```

### 作中におけるダーシーとエリザベスの怒りの強度

```
第5章   dislike（嫌悪）—弱い
第16章  resentment（恨み）—ウィカムの証言後
第33章  indignation（憤り）—フィッツウィリアムの暴露
第34章  ardent rejection（激しい拒絶）
第36章  shame（羞恥）—自分へ向けた怒り

## 11. 人物別の感情語彙の頻度

### Elizabeth（活気＋機知）

```
[頻出語彙]
- delight, amusement, vexation
- mortification（自尊心に関して）
- “lively, playful disposition”

[翻訳の調子]
- 活発＋わずかな風刺
- 「興味深い」「面白い」「煩わしい」
```

### Darcy（抑制＋強度）

```
[頻出語彙]
- regard, esteem, ardour
- shame, mortification
- “in vain have I struggled”

[翻訳の調子]
- 抑制された深い感情
- 「情」「敬慕」「羞恥」
```

### Mrs. Bennet（激しい表層感情）

```
[頻出語彙]
- joy, rapture, transport
- vexation, distress
- nerves（神経）

[翻訳の調子]
- 激情的だが浅い
- 「歓喜」「煩わしさ」「心労」
```

### Mr. Bennet（冷笑的な無関心）

```
[頻出語彙]
- amusement（他人の愚かさに対して）
- 感情をほとんど直接表さない

[翻訳の調子]
- 距離を置いた機知
- 「面白いと思うがね」／「興味深いな」
```

### Jane（抑制された優しさ）

```
[頻出語彙]
- regard, esteem, attachment
- 感情を抑える

[翻訳の調子]
- 優しいが格式を保つ
- 「情」「心にかける」
```

### Lydia（抑制なし）

```
[頻出語彙]
- joy, rapture, ecstasy
- すべてが表層的

[翻訳の調子]
- 浮ついているが、下品にはしない
- 「歓喜！」／「本当に！」
```

### Lady Catherine（怒り＋威圧）

```
[頻出語彙]
- displeasure, indignation
- arrogance, condescension

[翻訳の調子]
- 権威的な怒り
- 「不快」「憤怒」
```

### Collins（過剰な格式）

```
[頻出語彙]
- gratification, satisfaction
- humility, civility（外形だけ）

[翻訳の調子]
- 過剰に格式張った語彙
- 「満悦」「恐悦」
```

## 12. 主要場面における感情強度の推移

### 第34章：求婚場面

```
[ダーシー]
1. agitation（動揺）—入室
2. ardent admiration（熱烈な敬慕）—求婚
3. astonishment（驚愕）—拒絶される
4. anger（怒り）—“pale with anger”
5. mortification（屈辱）—“ashamed of what my own have been”

[エリザベス]
1. astonishment（驚愕）—求婚
2. compassion（同情）—一瞬
3. resentment（憤り）—彼の言い方を聞いて
4. anger（怒り）—激しい拒絶
5. agitation（動揺）—拒絶後30分間泣く
```

### 第36章：自己認識の場面

```
1. astonishment, apprehension, horror（手紙を読みながら）
2. shame（自己認識）
3. humiliation（自己批判）
4. wretchedness（自分自身の惨めさ）
5. resolution（決意）—“till this moment, I never knew myself”
```

### 第46章：駆け落ちの知らせ

```
[エリザベス]
1. anguish（悲痛）
2. wretchedness（惨めさ）
3. self-reproach（自責）
4. love realization（愛の自覚）—最も残酷な時点

[ダーシー]
1. astonishment（驚愕）
2. concern（深い気遣い）
3. resolve（決意）—口にはしない
```

### 第58章：二度目の求婚

```
[双方]
1. embarrassment（気まずさ）
2. relief（安堵）
3. gratitude（感謝）

## 13. 翻訳で微妙な違いを生かす

### “regard” vs. “esteem” vs. “affection” vs. “love”

```
[作中のダーシー]
第34章：“ardently admire and love”—愛（強）
第46章：“real, though unavailing concern”—深い気遣い（家族の危機）
第58章：“affections and wishes are unchanged”—変わらぬ愛情

[作中のエリザベス]
第44章：“She respected, she esteemed, she was grateful to him”
       → 尊敬、敬慕、感謝（愛の直前の段階）
第46章：“could have loved him”
       → 初めて“love”を使う
第58章：愛を受け入れる
```

### 訳し分け

```
原文：“regard”     → 「情」／「思い」（中立）
原文：“esteem”     → 「尊敬」／「敬慕」
原文：“affection”  → 「愛情」／「優しさ」
原文：“love”       → 「愛」／「恋慕」
原文：“passion”    → 「情熱」
原文：“ardour”     → 「熱烈な思い」
```

すべてを同じ「愛する」と訳せば、微妙な段階が崩れてしまう。

## 15. 翻訳の黄金原則

### 1. 段階を保つ
- “regard → esteem → affection → love”の推移を、訳文でも段階として保つ。

### 2. 人物ごとの語彙を差別化する
- ダーシーの“ardent” ≠ コリンズの“ardent”。
- 同じ語でも、人物の格調に合わせる。

### 3. 強度を一貫させる
- “rapture”を単なる「喜び」と訳さない—「恍惚」。
- “fury”を単なる「腹立ち」と訳さない—「激怒」。

### 4. 色合いを保つ
- 温かい語は温かく。
- 冷たい語は冷たく。
- “esteem”＝温かい尊敬、“respect”＝中立的な敬意—訳し分ける。

### 5. 時代の格調を保つ
- 「愛する」→「恋い慕う」／「思慕する」も可能。
- 「喜ぶ」→「満悦する」／「満ち足りる」も可能。
