## 4. 場所が持つ物語上の意味の変遷

各場所が物語の中でどのような意味を担い、事件に応じてどう変化するかを追跡する。

### ネザーフィールド・パーク（Netherfield Park）
| 時点 | 物語上の意味 |
|---|---|
| Chap 1 (EVT-001) | **期待の場所** — 裕福な青年の入居によって、結婚の可能性を象徴する |
| Chap 3 (EVT-003) | **社交の舞台** — 舞踏会で二組の関係が始まる |
| Chap 7–12 (EVT-006–010) | **試練の場所** — 看病のための滞在で関係が深まり、階級対立が露呈する |
| Chap 18 (EVT-016) | **対決の場所** — 舞踏会であらゆる葛藤が爆発する |
| Chap 21–23 (EVT-019–021) | **不在の象徴** — ビングリーが去った後の空き屋敷が、ジェインの喪失を体現する |
| Chap 53–55 (EVT-043–044) | **帰還の場所** — ビングリーの帰還と再求婚によって希望がよみがえる |

### ペンバリー（Pemberley）
| 時点 | 物語上の意味 |
|---|---|
| Chap 8（ビングリー嬢のお世辞） | **誇示の道具** — ビングリー嬢がダーシーに取り入るために持ち出す虚栄の題材 |
| Chap 43 (EVT-035) | **人格の鏡** — 自然美を生かした庭園、真に優雅な家具、レイノルズ夫人の証言がダーシーの内面を映す |
| Chap 43 (EVT-036) | **変化の舞台** — すっかり変わったダーシーの態度が、この空間で初めて示される |
| Chap 44–45 (EVT-037) | **統合の空間** — ダーシー、ジョージアナ、ビングリー、ガーディナー夫妻が一堂に会する |
| Chap 61 (EVT-048–049) | **結実の場所** — エリザベスが女主人となる最終的な帰着地 |

### ハンスフォード牧師館（Hunsford Parsonage）
| 時点 | 物語上の意味 |
