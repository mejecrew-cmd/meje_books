# 『高慢と偏見』— 人物別一日の予定表（推定／創作）

> ⚠️ **本資料は推定・創作に基づく。** 摂政時代のジェントリ・中産階級の標準的日課に、人物ごとの性格と作中の行動パターンを組み合わせ、日常の時間割を再構成した。脚色で人物の日常を描く際に利用できる。

---

## 摂政時代ジェントリの標準的な一日

| 時刻 | 活動 |
|---|---|
| 午前7～8時 | 起床。使用人が寝室へ温水を運ぶ。モーニング・ドレスを着る。 |
| 8～9時 | 散歩または私的な時間（手紙、読書） |
| 9～10時 | Breakfast（朝食）—家族全員 |
| 10～12時 | 散歩、訪問（morning calls）、家事、読書、音楽練習 |
| 12～15時 | 外出、買い物、散歩、来客対応 |
| 15時30分～16時 | 着替え（dressing for dinner） |
| 16～17時 | Dinner（正餐、一日の主な食事） |
| 17～19時 | 食後の休息（男性：ポートワイン、女性：客間） |
| 19～21時 | Tea、カード、音楽、読書、会話 |
| 21～22時 | Supper（軽い夜食、主に舞踏会後） |
| 22～23時 | 就寝 |

## 活用

| 状況 | 活用法 |
|---|---|
| **ドラマ／映画** | 日常場面を追加する際の時間帯の妥当性 |
| **翻訳** | "morning"が正確に何時かなど、時点描写を補完 |
| **小説への脚色** | 日常の細部を加えて人物に立体感を与える |
| **二次創作** | 時間の衝突なしに事件を配置 |
