# 《傲慢与偏见》——时代词语误译防范辞典

> 奥斯汀时代（摄政时期，约1810年代）的英语中，有许多词与现代英语**拼写相同，含义却不同**。
> 本文整理了那些**按照现代含义翻译就会造成误译的词语**，以防止翻译事故。
>
> **风险等级：**
> - **［危险］**——与现代含义相反或完全不同。误译会颠倒整个句子的意思。
> - **［注意］**——与现代含义部分重叠，但核心语感不同。
> - **［参考］**——现代已经消失的用法，或为自然翻译而必须了解的用法。

---

## A. ［危险］级——与现代含义相反／完全不同

### 1. sensible
| 项目 | 内容 |
|---|---|
| **奥斯汀时代的含义** | (1) 有判断力的、明智的（= having good sense）／(2) 意识到……的（= aware of） |
| **现代英语含义** | 敏感的（= sensitive）← **不是这个意思** |
| **书中例句** | "a sensible, intelligent young woman"（第5章，夏洛特）= **有判断力且聪明的**年轻女性 |
| | "I am very sensible of the honour"（第19章，伊丽莎白）= 我**非常清楚**这份荣幸 |
| **译法** | “有判断力的”／“明智的”／“很清楚……” |
| **误译示例** | ~~“敏感的年轻女性”~~ → 塑造出完全不同的人物形象 |

### 2. condescension / condescend
| 项目 | 内容 |
|---|---|
| **奥斯汀时代的含义** | 身份高贵者**降低身段给予恩惠**。属于赞美用语。 |
| **现代英语含义** | 自以为是、居高临下的态度 ← **正好相反** |
| **书中例句** | "such affability and condescension"（第14章，柯林斯赞美凯瑟琳夫人）= 凯瑟琳夫人**纡尊降贵的恩典** |
| | "She had even condescended to advise him to marry"（第14章）= 她甚至**屈尊亲自**劝他结婚 |
| **译法** | “纡尊降贵”／“恩慈地”／“亲自屈尊” |
| **误译示例** | ~~“那种傲慢与盛气凌人”~~ → 把柯林斯的奉承变成了批评 |
| **注意** | 奥斯汀也会以**反讽**方式使用这个词。柯林斯是真心称颂，读者却觉得可笑——译文必须保留这种双重性。 |

### 3. nervous / nerves
| 项目 | 内容 |
|---|---|
| **奥斯汀时代的含义** | 神经**衰弱**／患有神经性疾病（医学概念） |
| **现代英语含义** | 紧张、不安 ← 语感不同 |
| **书中例句** | "You have no compassion on my poor nerves."（第1章，班纳特太太）= 你对我**可怜的神经衰弱**一点同情也没有 |
| | "she fancied herself nervous"（第1章，叙述者）= 她自以为自己**神经脆弱** |
| **译法** | “神经（脆弱）”／“神经衰弱”——不是单纯的“紧张” |
| **要点** | 班纳特太太的 “nerves” 是医学式的自我诊断（其实是她的臆想），并非单纯的不安。 |

### 4. nice
| 项目 | 内容 |
|---|---|
| **奥斯汀时代的含义** | 精密的、挑剔的、能作细微辨别的（= precise, discriminating） |
| **现代英语含义** | 好的、友善的 ← 几乎无关 |
| **书中例句** | 虽然直接用例不多，但 "a nice distinction" = **细微的**区别 |
| **译法** | “精密的”／“挑剔的”／“细微的” |

### 5. wanting
| 项目 | 内容 |
|---|---|
| **奥斯汀时代的含义** | **不足的**、欠缺的（= lacking） |
| **现代英语含义** | 想要的（= desiring）← 完全不同 |
| **书中例句** | "No one admitted to the privilege of hearing you can think anything wanting."（第31章，达西）= 有幸听您演奏的人，不会觉得**有什么不足** |
| **译法** | “不足的”／“欠缺的” |
| **误译示例** | ~~“不会认为有什么东西是想要的”~~ |

### 6. address（名词）
| 项目 | 内容 |
|---|---|
| **奥斯汀时代的含义** | (1) 谈吐、举止、待人方式／(2) 信件地址／(3) 求爱行为 |
| **现代英语含义** | 几乎只指地址 |
| **书中例句** | "very pleasing address"（第15章，威克姆）= 十分讨人喜欢的**谈吐／举止** |
| | "my addresses"（第20章，柯林斯）= 我的**求爱** |
| | "Jane had written the direction remarkably ill"（第46章）= 简把**地址**写得格外难认（这里 direction = 地址） |
| **译法** | 按语境译作“谈吐”／“举止”／“求爱”／“地址” |

### 7. direction
| 项目 | 内容 |
|---|---|
| **奥斯汀时代的含义** | 信件的**地址／收件地点**（= address on a letter） |
| **现代英语含义** | 方向、指示 |
| **书中例句** | "the wished-for direction"（第52章，威克姆的地址）= 盼望已久的**地址** |
| **译法** | “地址”／“下落” |

## B. ［注意］级——核心语感不同

### 8. handsome
| 项目 | 内容 |
|---|---|
| **奥斯汀时代的含义** | (1) 人：英俊／美丽（男女通用）／(2) 行为、金额：**慷慨的、丰厚的**／(3) 建筑：宏伟的 |
| **与现代含义的差异** | 现代几乎只用于男性外貌 |
| **书中例句** | "a handsome young man"（宾利）= 英俊／"handsome modern building"（罗新斯）= **宏伟的**现代建筑／"a very handsome thought"（柯林斯赞美达西）= **高尚而慷慨的**想法 |
| **译法** | 人：“英俊／美丽”／建筑：“宏伟／气派”／行为：“高尚／慷慨” |

### 9. civil / civility
| 项目 | 内容 |
|---|---|
| **奥斯汀时代的含义** | 有礼貌——但只是**最低限度的礼貌**，未必出于真心 |
| **与现代含义的差异** | 现代用法也近似，但奥斯汀明确区分 “civil” 与 “warm/cordial” |
| **书中例句** | "with cold civility"（第46章，伊丽莎白对达西）= 带着**冷淡的礼貌** |
| | "civil disdain"（第18章，宾利小姐）= **礼貌的轻蔑**——有意的矛盾修辞 |
| **译法** | “有礼貌的”；需要表现不真诚时译作“形式上的”／“礼节性的” |
| **等级体系** | civil（最低）< polite（标准）< courteous（殷勤）< warm（热情）< cordial（真诚） |

### 10. consequence
| 项目 | 内容 |
|---|---|
| **奥斯汀时代的含义** | **社会重要性／体面／地位**（= social importance） |
| **现代英语含义** | 结果（= result） |
| **书中例句** | "give consequence to young ladies who are slighted by other men"（第3章，达西）= 把被其他男人冷落的小姐们**当成重要人物看待** |
| | "a man of consequence" = **有体面／有地位的**人 |
| **译法** | 按语境译作“体面”／“重要性”／“地位” |

### 11. intelligence
| 项目 | 内容 |
|---|---|
| **奥斯汀时代的含义** | **消息、情报**（= news, information） |
| **现代英语含义** | 智力 |
| **书中例句** | "the great intelligence"（第30章，柯林斯宣布达西到来）= 这个**重大消息** |
| | "fresher intelligence of her former friends"（第25章）= 关于旧友的**更新消息** |
| **译法** | “消息”／“情报”／“音讯” |

### 12. design
| 项目 | 内容 |
|---|---|
| **奥斯汀时代的含义** | **意图、计划、图谋**（= intention, plan, scheme） |
| **现代英语含义** | 设计 |
| **书中例句** | "Is that his design in settling here?"（第1章，班纳特先生）= 他在这里定居就是这个**意图**吗？ |
| | "without any design"（第22章，简）= 没有任何**图谋** |
| **译法** | “意图”／“计划”／“盘算” |

### 13. countenance
| 项目 | 内容 |
|---|---|
| **奥斯汀时代的含义** | (1) **脸、气色、表情**／(2) 支持、赞同 |
| **现代英语含义** | 面容（现已少用） |
| **书中例句** | "a fine countenance"（威克姆）= 出众的**容貌** |
| | "Bingley could keep his countenance"（第9章）= 宾利能够**保持表情不变** |
| **译法** | “表情”／“气色”／“容貌” |

### 14. character
| 项目 | 内容 |
|---|---|
| **奥斯汀时代的含义** | (1) 性格／(2) **名声、世人的评价**——两个含义可能同时发生作用 |
| **书中例句** | "His character was decided."（第3章）= 他的**名声已经定了**（= 被认定为傲慢的人） |
| | "Wickham's character" = 威克姆的**真面目／本性**（在后半部被颠覆） |
| **译法** | 按语境区分“性格”和“名声”。二义并存时可译为“为人”或“人物形象” |

### 15. establishment
| 项目 | 内容 |
|---|---|
| **奥斯汀时代的含义** | (1) **出嫁／安顿下来的归宿**（通过婚姻建立独立家庭）／(2) 独立生活的整体基础 |
| **书中例句** | "what an establishment it would be for one of them"（第1章）= 对其中一个女儿来说，那会是多么好的**归宿** |
| | "the pure and disinterested desire of an establishment"（第22章，夏洛特）= 纯粹为了获得**安稳住所**的愿望 |
| **译法** | “出嫁”／“归宿”／“安稳的家庭”——不是“设施” |

### 16. connexion(s) / connection(s)
| 项目 | 内容 |
|---|---|
| **奥斯汀时代的含义** | **亲属关系／人脉**（= relatives, family ties） |
| **现代英语含义** | 一般意义上的联系 |
| **书中例句** | "low connexions"（达西）= **地位低微的亲属／关系**（= 经商的舅舅、当律师的舅舅） |
| | "the connexion was a most unhappy one" = 那段**姻亲关系**极其不幸 |
| **译法** | “人脉”／“姻亲”／“亲属关系” |

### 17. amiable
| 项目 | 内容 |
|---|---|
| **奥斯汀时代的含义** | **可爱、讨人喜欢、善良**——比现代 “amicable”（友好的）更强烈的赞美 |
| **书中例句** | "she is one of the most tractable creatures... a very great favourite"（第33章，伊丽莎白评价乔治安娜） |
| | "the most amiable man of my acquaintance"（第24章，简评价宾利）= 我认识的人中最**可爱可亲的**男人 |
| **译法** | “可爱可亲的”／“温厚的”／“讨人喜欢的” |

### 18. agreeable
| 项目 | 内容 |
|---|---|
| **奥斯汀时代的含义** | **令人愉快的人，相处起来很舒服的人**（= pleasant company） |
| **书中例句** | "If he had been so very agreeable, he would have talked to Mrs. Long"（第5章，班纳特太太）= 如果他真那么**随和讨喜**，就会和朗太太说话 |
| **译法** | “令人愉快的”／“讨人喜欢的”／“随和的” |
| **等级** | disagreeable（讨厌）< tolerable（尚可忍受）< agreeable（令人愉快）< amiable（可爱可亲） |

## C. ［参考］级——现代已经消失的用法

### 19. complaisance / complaisant
| 项目 | 内容 |
|---|---|
| **含义** | 为取悦他人而表现出的**顺从、迁就态度**／乐意配合 |
| **注意区别** | 与 “complacency”（自满）不同 |
| **书中例句** | "his complaisance"（第6章）= 他那种**迁就配合的态度** |

### 20. propriety / impropriety
| 项目 | 内容 |
|---|---|
| **含义** | **得体／礼仪规范／合乎身份的行为**——不只是抽象的“适当”，而是遵守社会规范 |
| **书中例句** | "want of propriety"（达西的信中批评班纳特一家）= **缺乏礼仪规范** |
| **译法** | “礼仪”／“体面”／“端庄得体” |

### 21. scheme
| 项目 | 内容 |
|---|---|
| **含义** | **计划**（比现代带贬义的“阴谋”更中性） |
| **书中例句** | "No scheme could have been more agreeable"（第27章，旅行计划）= 再没有比这更令人愉快的**计划**了 |
| **译法** | “计划”／“方案”——不一定含贬义 |

### 22. apprehension
| 项目 | 内容 |
|---|---|
| **含义** | (1) **理解、察觉**／(2) 不安——奥斯汀更常使用含义 (1) |
| **书中例句** | "I had no apprehension of his feeling a serious attachment"（第35章，达西的信）= 我并**没有察觉**他怀有认真的感情 |
| **译法** | “察觉”／“理解”（只有表示不安时才译作“不安”） |

### 23. complaint
| 项目 | 内容 |
|---|---|
| **含义** | (1) 抱怨／(2) **疾病、病症** |
| **书中例句** | "nervous complaints"（班纳特太太）= **神经疾病** |
| **译法** | 应按语境判断；班纳特太太的用例一律是“疾病” |

### 24. prospect
| 项目 | 内容 |
|---|---|
| **含义** | (1) **景观／景色**（literal view）／(2) 未来前景 |
| **书中例句** | "a charming prospect over that gravel walk"（第9章）= 碎石路那边迷人的**景色** |
| **译法** | 建筑／窗户语境译作“景观／景色”，未来语境译作“前景／可能性” |

### 25. natural
| 项目 | 内容 |
|---|---|
| **含义** | (1) 自然的／(2) 自然主义的（园林）／(3) **非婚生的**（少见）／(4) 天生的 |
| **书中例句** | "natural beauty"（彭伯里园林）= 保留**自然之美的**美景（并非人工雕琢） |
| | "a sort of natural self-consequence"（莉迪亚）= 一种**天生的**自负 |
| **译法** | 按语境区分。彭伯里的 “natural” 是**自然主义美学**术语。 |

## D. 好感度形容词等级表——翻译时调整强度

奥斯汀描写人物时使用的形容词具有**精细的等级差异**：

### 正面等级（弱→强）
```
tolerable（尚可忍受、马马虎虎）← 最低——达西对伊丽莎白的评价
  ↓
civil（有礼貌的）← 礼节上的最低限度
  ↓
agreeable（令人愉快、相处舒服）
  ↓
pleasing（讨人喜欢）
  ↓
amiable（可爱可亲、善良）
  ↓
charming（迷人）
  ↓
delightful（令人愉悦、带来喜悦）
  ↓
excellent（极其出色）← 最高
```

### 负面等级（弱→强）
```
not very agreeable（不怎么讨人喜欢）← 英式委婉表达
  ↓
disagreeable（令人不快）
  ↓
unpleasant（不愉快，更直接）
  ↓
ill-natured（脾气坏、刻薄）
  ↓
proud（傲慢）← 用于达西
  ↓
insolent（无礼、放肆）
  ↓
abominable（可恶至极）
  ↓
despicable（卑劣可鄙）← 最强——伊丽莎白的自我批评
```

### 翻译原则
- 在译文中也要保持**同样的强度差异**。
- 把 “tolerable” 和 “agreeable” 都译作“还不错”，会破坏等级。
- 把 “civil” 和 “amiable” 都译作“亲切”，会抹去关系温度的差异。

## E. 英式委婉表达（Understatement）——误译最多的领域

奥斯汀极其频繁地使用**双重否定、弱化表达和委婉语**。直译会使语感消失。

| 原文模式 | 表面含义 | 实际含义 | 翻译指南 |
|---|---|---|---|
| "not unpleasing" | 并非令人不快 | **相当不错** | 化解双重否定，但保持较弱强度：“还不错” |
| "not without" | 并非没有 | **确实存在** | “确实有……” |
| "she could not help" | 她忍不住不…… | **她做了……**（强烈冲动） | 目标语言自然时保留否定结构：“她忍不住……” |
| "I dare say" | 我敢说 | **大概如此**（轻微推测） | “大概”／“我想是吧” |
| "I am sure" | 我确信 | **依我看**（常是习惯性修辞，并非真正确信） | 班纳特太太使用时译成没有根据的确信：“准是这样” |
| "something like regret" | 类似后悔的东西 | 是后悔，但**不愿承认** | 保留“近似后悔的感觉”，不能直接断定为“后悔” |
| "not entirely hopeless" | 并非完全没有希望 | **还有一点希望** | “并非全无希望” |
| "I fancy" | 我想象 | **我觉得似乎……**（轻微推测） | “我看”／“似乎” |
| "by no means" | 无论如何都不 | **绝对不**（强烈否定） | “决不”／“绝非” |
| "tolerable/tolerably" | 可以忍受 | **勉强过得去**（不是赞美） | “马马虎虎”／“勉强可以” |

## F. 动词的时代语感

| 动词 | 奥斯汀时代 | 现代 | 书中例句 | 译法 |
|---|---|---|---|---|
| **to own** | 承认、坦白 | 拥有 | "I own I was surprised" = 我**承认**自己很惊讶 | “承认”／“坦白” |
| **to engage** | (1) 约定 (2) 雇用 (3) 吸引注意 | 主要指订婚 | "she was engaged elsewhere" = 她在别处**已有约会** | 按语境判断 |
| **to want** | **缺少**／需要 | 想要 | "want of propriety" = **缺乏**礼仪 | “缺少”／“欠缺” |
| **to wait on** | **拜访**（礼节性） | 等待／侍候 | "to wait on Mr. Bingley" = **拜访**宾利先生 | “拜访”／“登门致意” |
| **to make out** | **理解、弄清** | 辨认／接吻 | "I cannot make him out" = 我**无法理解他** | “理解”／“弄清” |
| **to recollect** | **有意识地回想起来** | 现代与 remember 近义 | 在奥斯汀笔下有“主动唤起记忆”的语感 | “回想”／“忆起” |
| **to flatter oneself** | **自以为、自负**（往往是误判） | 奉承自己 | "I flatter myself that..." = **恕我自信地说……** | “自以为”／“自负”——常带反讽 |
| **to trifle with** | **玩弄、轻率对待** | — | "You are too generous to trifle with me" = 你不会卑劣到**玩弄我的感情** | “玩弄”／“戏弄” |

## G. 翻译时绝对不能混淆的词语组（Pair）

| 词语组 | 区别 | 混淆的后果 |
|---|---|---|
| **pride** vs **vanity** | pride = 自我评价；vanity = 希望别人如何看待自己（第5章，玛丽的定义） | 小说标题与主题的核心区分崩塌 |
| **sense** vs **sensibility** | sense = 理性／判断力；sensibility = 感受力／情感 | 简（sensibility）与伊丽莎白（sense）的对照消失 |
| **civil** vs **polite** vs **amiable** | civil = 最低限度礼貌；polite = 标准礼貌；amiable = 真诚好感 | 关系温度的差异消失 |
| **fortune** vs **income** | fortune = 财产总额；income = 年收入 | 误读经济规模：宾利的10万英镑财产与每年4,000～5,000英镑收入 |
| **trade** vs **profession** | trade = 商业（地位较低）；profession = 专业职业（地位较高） | 阶级偏见中的微妙差异消失 |
| **living** vs **life** | living = 圣职俸禄／教区收入；life = 生命、人生 | 颠倒达西与威克姆矛盾的核心 |
| **interesting** | 奥斯汀时代：引人关注／令人感动（比现代用法更强） | 把 “most interesting” 只译成“有趣”，会削弱情感强度 |
