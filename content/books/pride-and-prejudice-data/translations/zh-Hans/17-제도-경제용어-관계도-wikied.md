# 《傲慢与偏见》——制度与经济术语关系图

> 本文系统梳理推动小说情节发展的**法律、经济与社会制度**。
> 若不理解这些术语，就无法在翻译中准确呈现人物的行为动机与危机的本质。

---

## 1. 婚姻经济学——为什么人人都把婚姻视为生死大事

### 核心结构：财产／嫁妆（Fortune）→婚姻财产安排（Settlement）→妻子零用金（Pin-money）

```
[新娘一方]                            [新郎一方]
Fortune（财产／嫁妆）                 Estate / Income（地产／收入）
    │                                      │
    └──→ Marriage Articles（婚姻契约）←──┘
              │
              ├──→ Settlement（婚姻财产安排）
              │      丈夫去世后为遗孀保障的年金
              │
              ├──→ Pin-money（妻子零用金）
              │      妻子不受丈夫干涉即可支配的年度固定款项
              │
              └──→ Jointure（寡妇年金）
                     Settlement的一种具体形式
```

### 小说中的应用

| 人物 | 财产／嫁妆 | 婚姻财产安排 | 翻译要点 |
|---|---|---|---|
| **班纳特家每位女儿** | 约1,000英镑（母亲去世后，5,000英镑的五分之一） | — | 投资于“4厘公债（4 per cents）”，每年利息约40英镑——远不足以独立生活 |
| **莉迪亚—威克姆** | 莉迪亚的1,000英镑份额＋父亲在世期间每年100英镑 | 达西为威克姆支付额外费用 | 班纳特先生感叹“这么少的钱竟然就能解决”；实际上大部分费用由达西承担 |
| **乔治安娜·达西** | 30,000英镑 | — | 威克姆引诱乔治安娜的真正原因 |
| **德·包尔小姐** | 罗新斯庄园的全部地产 | — | 与达西的策略婚姻意味着“两个地产的合并” |
| **金小姐** | 10,000英镑（祖父遗产） | — | 威克姆转而放弃伊丽莎白、追求她的原因 |

### 术语翻译指南

| 原文 | 推荐译法 | 绝不能采用的译法 | 原因 |
|---|---|---|---|
| Fortune | 财产／嫁妆 | 幸运 | 在婚姻语境中必指财产 |
| Settlement | 婚姻财产安排／财产协议 | 定居、一般协议 | 具有法律约束力的财产契约 |
| Pin-money | 妻子零用金／体面开支 | 别针钱 | 婚姻契约明确规定的正式费用 |
| 4 per cents | 4厘公债／百分之四公债 | 百分之四 | 英国政府债券的专有名称 |
| Portion | 遗产份额／应得份额 | 部分 | 遗产中的法定分配 |

## 2. 限定继承制（Entail）——小说的核心冲突动力

### 结构

```
[地产所有人：班纳特先生]
    │
    │  受限定继承制（Entail）约束
    │  女儿无权继承
    │
    └──→ [法定继承人：柯林斯先生]
              │
              ├── 班纳特先生去世时 → 继承整个浪搏恩地产
              ├── 班纳特太太＋5个女儿 → 必须迁出
              └── 另一种可能：若生下儿子，本可由父子共同解除限定继承；但
                         连续生下5个女儿 → 无法解除
```

### 各事件中限定继承制的叙事功能

| 事件 | 限定继承制的作用 |
|---|---|
| EVT-011（柯林斯抵达） | 抽象法律化为具体人物——“我要补偿我亲爱的表妹们” |
| EVT-017（柯林斯求婚） | 伊丽莎白的拒绝＝拒绝限定继承制提供的所谓“补偿” |
| EVT-020（夏洛特接受） | 夏洛特接受柯林斯＝确定成为浪搏恩未来的女主人，因而激起班纳特太太的敌意 |
| 第50章 | 班纳特先生的悔恨：“我相信会生下儿子，所以没有储蓄。” |

### 翻译要点
- **“entail”每次都须采用同一译法**：统一译为“限定继承（制）”，不可在“地产继承限制”“限定继承法”等表达之间摇摆。
- 班纳特太太说“I cannot bear to hear that mentioned”时，“that”指限定继承制。译文也应保留她不愿直呼其名、刻意回避的语气。

## 3. 教会经济学——堂区牧师如何维持生计

### 核心结构

```
[赞助人（Patron）]
    │
    ├──[Patronage：圣职推荐权]──→ 任命堂区牧师
    │
    ├──[Presentation of the living：圣职俸禄授职权]──→ 提供特定堂区的收入
    │
    └──[Advowson：圣职任命权的所有权]──→ 附属于地产的永久权利
    
[堂区牧师（Rector/Vicar）]
    │
    ├── 收入来源1：Tithes（什一税）——堂区居民缴纳的税款／农产品
    ├── 收入来源2：Glebe land（教会附属地）——牧师住宅附带的农地
    └── 收入来源3：Fees（手续费）——洗礼、婚礼、葬礼等

[领受圣职（Taking orders）]
    └── 大学毕业后必须由主教授予圣职，方可成为牧师
```

### 小说中的应用

| 人物／事件 | 教会经济的作用 |
|---|---|
| **柯林斯** | 凭借凯瑟琳夫人的赞助（Patronage）获得汉斯福堂区牧师职位。他声称什一税必须在“不惹赞助人不快”的范围内商定，揭示其奉承行为的经济基础。 |
| **威克姆—达西冲突的本质** | 老达西在遗嘱中许诺把最优厚的圣职俸禄（the best living）留给威克姆。威克姆决定不领受圣职（resolved against taking orders），改收3,000英镑后挥霍殆尽。三年后，现任牧师（incumbent）去世，他又要求获得圣职俸禄，遭到拒绝。 |
| **达西的权力** | 拥有多个堂区的圣职推荐权。费茨威廉上校没有这种权力——这正是夏洛特试图撮合伊丽莎白与达西的隐秘原因。 |

### 术语翻译指南

| 原文 | 推荐译法 | 说明 |
|---|---|---|
| Living | 圣职俸禄／堂区收入 | 并非普通的“生活”，而是来自堂区的正式收入 |
| Patronage | 赞助／圣职推荐权 | 贵族任命牧师的权利 |
| Presentation | 圣职俸禄授职／堂区任命 | 将牧师“推荐”到特定堂区的行为 |
| Taking orders | 领受圣职／进入教会 | 由主教正式授予神职的仪式 |
| Incumbent | 现任牧师／现任圣职人员 | 当前负责该堂区的人 |
| Rector | 堂区牧师 | 领取全部什一税的正式牧师；Vicar只领取其中一部分 |
| Tithes | 什一税 | 堂区居民缴纳的税款或农产品，是牧师的主要收入来源 |

## 4. 军队经济学——军官职位如何取得

### 核心结构

```
[民兵（Militia）]                    [正规军（Regular Army）]
  │                                    │
  ├── 负责国内治安／防卫               ├── 投入实战，包括海外派遣
  ├── 在各地区轮换驻扎                 ├── 固定驻地
  ├── 军官职位：相对便宜               ├── 军官职位：捐官制（Purchase system）
  │                                    │     └── 花钱买卖军衔
  └── 威克姆私奔前所属部队             └── 私奔后所属部队，由达西出资购买职位

[军衔次序（低→高）]
  Ensign（步兵少尉）→ Lieutenant（中尉）→ Captain（上尉）→ Colonel（上校）
  
  ※ Ensigncy（步兵少尉职位）是正规军最低的军官等级
  ※ 达西替威克姆购买的正是这个少尉职位
```

### 小说中的应用

| 术语 | 小说语境 | 翻译要点 |
|---|---|---|
| **Militia regiment** | 冬季驻扎在麦里屯；莉迪亚与吉蒂狂热追逐的对象；军官们的红制服（red coats）是视觉象征 | 统一译为“民兵团” |
| **Commission** | 指军官职位本身；“accepted a commission”＝成为军官 | “任官”或“军官职位” |
| **Commission purchased** | 花钱购买军官职位；达西替威克姆购买步兵少尉职位 | “购买军官职位／代付任官费用” |
| **Ensigncy** | 正规军最低的军官等级，也是最便宜的军官职位 | “步兵少尉职位” |
| **Colonel** | 团级指挥官；福斯特上校与费茨威廉上校 | 统一译为“上校”，但小说中的两位上校性格截然不同 |

## 5. 社交制度——婚姻市场的规则

### 核心术语

| 原文 | 译法 | 说明 | 小说语境 |
|---|---|---|---|
| **Assembly** | 地区舞会／公共舞会 | 当地名流聚集跳舞的公共社交活动；谁与谁跳舞会传递社会信号 | 麦里屯舞会（EVT-003）是一切的起点 |
| **Ball** | 正式舞会 | 比Assembly更正式，由私人宅邸主办 | 尼日斐花园舞会（EVT-016） |
| **Coming out** | 社交界初次亮相 | 年轻女儿取得以成年人身份参加舞会的资格 | 凯瑟琳夫人斥责“5个女儿竟然全都进入社交界了？” |
| **Accomplishment** | 教养／淑女才艺 | 音乐、绘画、舞蹈、语言等提高婚姻市场价值的技能 | 参见第8章“才艺之争”（事物词典 §5） |
| **Special licence** | 特别结婚许可证 | 由坎特伯雷大主教署签发，可不受时间与地点限制结婚，费用极高 | 班纳特太太陶醉于达西的财富而提出要求 |
| **Banns** | 结婚预告 | 在堂区教堂连续三周公告后结婚，是一般程序 | 小说中直接提及较少，但它是Special licence的对照概念 |

### 这一结构如何推动情节

| 制度 | 情节动力 |
|---|---|
| **Entail** | 班纳特姐妹急于结婚的原因；柯林斯存在的原因；夏洛特结婚的动机。 |
| **Fortune/Settlement** | 威克姆盯上乔治安娜（30,000英镑）的原因；莉迪亚婚姻的经济条件；柯林斯以“你的财产太少，不会再有别的机会”侮辱伊丽莎白的依据。 |
| **Patronage/Living** | 达西—威克姆冲突的核心；柯林斯奉承凯瑟琳夫人的经济基础。 |
| **Commission/Ensigncy** | 达西“处置”威克姆的方法——替他购买步兵少尉职位并调往北方，实际上等同放逐。 |
| **Trade（商业）** | 宾利家族对财富源自商业的自卑；宾利姐妹因嘉丁纳舅父住在齐普赛街而轻视他；但达西欣然接纳嘉丁纳夫妇，证明了他的转变。 |
