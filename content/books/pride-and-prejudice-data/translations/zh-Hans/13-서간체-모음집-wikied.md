## 书信文体类型分类

### 类型1：柯林斯书信体（L-01、L-06、L-18、L-19）
```
特点：
- 一个句子长达5—8行
- 分号和插入从句过度嵌套
- “I flatter myself”／“I venture to say”等自负与谦逊的矛盾
- 每一段都提到凯瑟琳夫人
- 反复使用“olive branch”（意义从和解转为怀孕）

翻译原则：
- 绝不可缩短篇幅。长度＝愚蠢
- 使用过度正式的语气：“鄙人斗胆自许”／“虽属冒昧，仍敢……”
- 保留同一封信中谦逊与自负交替出现的特点
```

### 类型2：达西书信体（L-10）
```
特点：
- 格调最高的复合句，精密构成让步从句→条件从句→主句
- 情绪克制逐渐崩解（宾利部分＝冷静 → 威克姆部分＝激动）
- “Be not alarmed, madam”——开头第一句制造读者的紧张
- 虽是自我辩护，却有所节制，不责怪对方
- 以“God bless you”结尾——唯一一次流露情感

翻译原则：
- 使用与口语明确区分的书信体：“谨望您……”／“现特此说明……”
- 最大限度表现宾利部分（第一部分）与威克姆部分（第二部分）的语调变化
- 保留“I must now mention a circumstance which I would wish to forget myself”在揭开痛苦前的迟疑
```

### 类型3：宾利小姐书信体（L-04、L-05）
```
特点：
- 夸张的爱意修辞：“dearest friend”、“I depend on you”
- 用柔和表达包装离间的核心内容
- “dare to entertain the hope”——假装谦逊，同时把刀刺下
- 既正式又假装亲密的双重性

翻译原则：
- 原样保留夸张的亲密：“致我最亲爱的朋友”
- 让虚伪在重读时显现——初读像真心，重读则显得工于心计
```

### 类型4：简的书信体（L-03、L-07—09、L-11—12）
```
特点：
- 平时：平静、克制、使用淡化表达
- L-11→L-12：从镇定骤然转为混乱
- L-12的“I hardly know what I have written”＝承认自己失去控制
- “I take up my pen again to do, what I have just told you I would not”＝刚说不再写，却又提笔

翻译原则：
- 最大限度表现L-11与L-12的语调差异
- 保留L-12断裂又接续的呼吸
```

### 类型5：加德纳家的书信体（L-13—14、L-17、L-21）
```
加德纳先生（L-13—14）：
- 事务性、简洁，法律和财务用语准确
- 克制情感。即使“I hope it will give you satisfaction”也相当干燥

加德纳太太（L-17）：
- 比丈夫温暖，也更健谈
- 巧妙混合幽默与暗示：“If he had another motive, I am sure it would never disgrace him”
- 按时间顺序展开故事的叙事性书信

翻译原则：
- 加德纳先生：“现将……告知如下”（商人实务文体）
- 加德纳太太：“我想大概是……”（亲切而有教养的女性文体）
```

### 类型6：莉迪亚书信体（L-15、L-16）
```
特点：
- 无视语法，滥用下划线（斜体）
- “was obliged to leave off in a violent hurry”——仓促中断
- 没有内容却很长（寄给吉蒂的信“到处都在词语下面画线”）
- L-15（私奔信）：“Don’t be alarmed”→马上接着令人震惊的内容

翻译原则：
- 原样保留青少年不负责任的书信语气
- 用斜体、下划线、粗体等**排版标记**保留下划线／强调
- 用破折号“——”表现仓促中断
```
