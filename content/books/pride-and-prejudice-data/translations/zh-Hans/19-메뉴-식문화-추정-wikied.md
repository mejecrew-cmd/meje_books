# 《傲慢与偏见》——菜单与饮食文化（推测／创作）

> ⚠️ **本文属于推测与创作。** 文中结合小说明确提及的食物（法式炖肉、鹿肉、温室水果、白汤、波特酒）、摄政时期的标准饮食以及各家的经济实力，重构可能出现的实际餐桌配置。

---

## 摄政时期餐食的基本结构

### 正餐（Dinner）的上菜结构

| 次序 | 内容 |
|---|---|
| **First Course** | 汤＋鱼＋肉，同时摆上餐桌 |
| **Second Course** | 禽肉＋蔬菜＋酥皮点心，同时摆上 |
| **Dessert** | 水果＋坚果＋小蛋糕，重新布置餐桌后另行端上 |

> **方式：** “à la française”——所有菜肴同时铺陈在餐桌上，各人取用离自己较近的菜。

### 各家的水准
- **彭伯里／罗新斯：** 每餐都有完整正式的上菜次序和华丽装饰
- **内瑟菲尔德：** 采用正式次序，但反映宾利朴素的口味
- **浪博恩：** 平日简单，待客时讲究礼仪
- **汉斯福德：** 节俭，减少上菜轮次
- **威克姆与莉迪亚：** 极为节俭

## 1. 彭伯里（Pemberley）——达西家的正餐（推测菜单）

### 平日家庭正餐（达西＋乔治安娜）

```
[First Course]
- White Soup（白汤，以白肉禽类为底）
- Roasted Pheasant（烤野鸡，来自庄园狩猎）
- Boiled Salmon with Lobster Sauce（龙虾酱汁水煮鲑鱼）
- Vegetable Side: Asparagus, Cauliflower

[Second Course]
- Roast Beef
- Stewed Mushrooms
- Apple Tart
- Plum Pudding

[Dessert]
- Fruit Pyramid: Grapes, Nectarines, Peaches（★ 温室水果）
- Walnuts, Almonds
- Madeira Wine
```

### 第45章 嘉丁纳夫妇＋伊丽莎白的正餐（推测）

```
[First Course]
- Turtle Soup（正式款待宾客）
- Glazed Ham
- Stewed Trout

[Second Course]
- Roast Capon
- Savoury Pie
- French Beans

[Dessert]
- Grapes, Nectarines, and Peaches ★（小说明确提及）
- Cheese, Biscuits

[Wine]
- Claret（波尔多高级红葡萄酒）
- Madeira（配甜点）
- Port（男士餐后饮用）
```

### 饮料
- **Tea**（下午饮用，印度产高级茶）
- **Coffee**（正餐后）
- **Hot Chocolate**（早餐时可能饮用）

## 2. 罗新斯（Rosings）——凯瑟琳夫人的正餐

### 第29章首次正餐的推测菜单

原著明确写道：“exceedingly handsome dinner... all the servants, and all the articles of plate”。

```
[First Course]
- Spring Soup with Green Peas
- Salmon with Sauce Hollandaise
- Boiled Turkey with Oyster Sauce
- Side Dishes: Spinach, Carrots, Pickles

[Second Course]
- Roast Lamb（时令为春季）
- Sweetbreads
- Veal Olives
- Lemon Cheesecakes
- Apricot Tart

[Dessert]
- Pyramid of Fruits（温室水果）
- Italian Cream
- Sweet Biscuits
- Various Sugared Confections
```

### 特点
- **凯瑟琳夫人的炫耀**——柯林斯逐一称赞每道菜
- **安妮·德·包尔几乎不吃东西**——詹金森太太监看并劝她尝别的菜
- **柯林斯与卢卡斯爵士过度称赞**

## 3. 内瑟菲尔德（Netherfield）——宾利家的正餐

### 第8章伊丽莎白抵达当天的正餐（推测）

原著线索：出现宾利姐妹喜欢的“ragout”。

```
[First Course]
- Soup à la Reine
- Boiled Carp
- Ragoût of Veal（★ 小说明确提及，法式炖小牛肉）
- Glazed Onions

[Second Course]
- Roast Partridge
- Pigeon Pie
- French Beans
- Charlotte Russe（相当于甜点）

[Dessert]
- Fruit, Cheese, Wine

※ 赫斯特先生只喜欢ragout，拒绝朴素菜肴
※ 伊丽莎白偏爱plain dish
```

### 第18章内瑟菲尔德舞会夜宵（Supper）

原著明确提及“white soup”，这是舞会夜宵不可或缺的菜。

```
[Supper Spread——午夜前后]
- White Soup ★（由女管家尼科尔斯制作）
- Cold Ham
- Cold Chicken（小说明确提及：“cold ham and chicken”）
- Trifle（英式甜点）
- Jellies, Creams
- Fruit
- Wine, Negus（葡萄酒＋水＋糖＋香料）
```

## 4. 浪博恩（Longbourn）——班纳特家的正餐

### 平日家庭正餐（推测）

```
[First Course]
- Pea Soup（依季节而定）
- Boiled Mutton with Caper Sauce（羊肉＋刺山柑酱）
- Side: Mashed Potatoes, Greens

[Second Course]
- Roast Chicken
- Apple Pudding

[Dessert]
- Cheese
- Seasonal Fruit（本地产）
- Port Wine（供男士饮用）
```

### 第13章欢迎柯林斯的正餐——正式摆设

原著说这顿饭受到柯林斯“highly admired”。

```
[升级菜单]
- White Soup
- Boiled Fish
- Roast Beef + Yorkshire Pudding
- Pigeon Pie
- Mince Pies（柯林斯在11月抵达，因此可能出现）
- Apple Pie

[Dessert]
- Fruit, Cheese
- Madeira, Port
```

> **班纳特太太的自尊：** 第13章中，柯林斯询问是谁做的饭，她强调“her daughters had nothing to do in the kitchen”——这是家中雇有专业厨师所带来的自豪。

### 第55章欢迎达西与宾利的正餐

原著明确提及“venison”。

```
[最高级摆设]
[First Course]
- Vermicelli Soup
- Boiled Trout
- Fricassée of Chicken

[Second Course]
- Roast Venison ★（小说明确提及，鹿后腿肉＝haunch）
- Roast Duck
- Asparagus
- French Beans
- Game Pie

[Dessert]
- Apple Tart
- Custards
- Fruit（英国产苹果、梨）
- Madeira, Port

※ 班纳特太太承诺“two full courses”（第39章）——完整的正式餐食
```

## 5. 菲利普斯家（Mrs. Philips）——麦里屯之夜（第16章）

原著明确提及“coffee and muffin”、“hot supper”以及威克姆喝的酒等。

```
[Hot Supper——打牌后、午夜前]
- Cold Roast Pork
- Tongue
- Cold Meat Pies
- Pickled Vegetables
- Jelly
- Cheese

[Drinks]
- Port Wine ★（与菲利普斯姨父身上的气味相联系）

- Negus（供女士饮用）

[游戏间歇]
- Almonds, Raisins
- Sweet Biscuits
```

## 10. 各类饮料的社会意义

| 饮料 | 阶级／语境 | 小说中 |
|---|---|---|
| **Champagne** | 最上流社会 | 彭伯里、罗新斯 |
| **Claret（Bordeaux）** | 上流社会 | 推测用于彭伯里正餐 |
| **Madeira** | 上流社会的甜点酒 | 所有正式餐食 |
| **Port Wine** | 中层男士 | 与菲利普斯姨父的气味相联系；餐后饮用 |
| **Sherry** | 中层，女士也饮用 | 清淡开胃酒 |
| **Negus** | 温热葡萄酒饮料 | 舞会夜宵 |
| **Tea（Indian）** | 各阶层普遍饮用，但有等级差异 | 所有家庭；品质体现财富 |
| **Coffee** | 正餐后 | 所有正式餐食 |
| **Beer/Ale** | 下层日常饮料 | 仆役 |
| **Cider** | 乡村 | 下层 |
| **Hot Chocolate** | 奢侈早餐饮品 | 推测用于彭伯里 |

## 11. 各季食材

### 秋季（小说开篇，9月至11月）
- 狩猎季开始：野鸡、鹧鸪、野兔
- 英国产苹果、梨、李子
- 南瓜、卷心菜

### 冬季（12月至2月）
- 保存肉类：盐腌、熏制
- 冬季蔬菜：芜菁、卷心菜、洋葱
- 保存水果：果酱、腌渍品
- **鹿肉（venison）**——狩猎季

### 春季（3月至5月）——汉斯福德时期
- 春季蔬菜：芦笋、豌豆
- 羔羊肉
- 新鲜鸡蛋

### 夏季（6月至8月）——彭伯里时期
- 新鲜蔬菜丰富
- **温室水果：** 葡萄、桃子、油桃
- 鱼类丰富
- 草莓、覆盆子

## 12. 餐桌礼仪（翻译／改编参考）

| 项目 | 规则 |
|---|---|
| **座次** | 女主人坐在餐桌一端，主宾坐在她右侧 |
| **上菜** | 所有菜肴同时摆在桌上，各人取用附近的菜 |
| **女士先离席** | 甜点后女士们前往客厅，男士们留下喝波特酒 |
| **重新会合** | 约一小时后，男士到客厅与女士会合，进入茶点时间 |
| **柯林斯的过度表现** | 每一道菜都称赞，是有失体面的行为 |
| **赫斯特先生的无礼** | 只关心食物；伊丽莎白说喜欢“plain dish”后便中断交谈 |
