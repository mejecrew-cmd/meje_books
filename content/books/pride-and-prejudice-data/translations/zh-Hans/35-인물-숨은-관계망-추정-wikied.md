# 《傲慢与偏见》——人物隐性关系网（推定/创作）

> ⚠️ **本文件包含推定与创作。** 除作品明确写出的关系外，还根据时代背景、人物动线与社会合理性，追踪**受到暗示或被省略的关系**。可用于改编时增加人物立体感。

---

## 1. 作品未明确写出的家庭关系网

### 贝内特姐妹间的细微同盟
| 同盟 | 强度 | 依据 |
|---|---|---|
| **简—伊丽莎白** | ★★★★★ | 明确写出；共享秘密 |
| **凯蒂—莉迪亚** | ★★★★ | 明确写出；凯蒂追随莉迪亚 |
| **玛丽独自一人** | ★ | 与任何人都没有深厚同盟；逃入音乐与书本 |
| **伊丽莎白—玛丽（潜在）** | ★ | 两人都重视才智，但玛丽卖弄学问；关系不深 |
| **简—玛丽（潜在）** | ★★ | 简的温柔可能包容玛丽 |
| **简—莉迪亚** | ★ | 几乎没有；性格差异太大 |

### 贝内特家内部的信息流动
```
[共享秘密]
伊丽莎白 ↔ 简（大多数秘密）
凯蒂 → 莉迪亚（军官消息）
莉迪亚 → 贝内特太太（流言）
玛丽 → 无人（推定独自写日记）

[被阻断的信息]
- 达西第一次求婚：只有简得知
- 威克姆真相：只有简得知，不包括乔治安娜部分
- 达西的秘密斡旋：起初只有伊丽莎白，后来告诉简
- 贝内特太太总是最后才知道
```

## 2. 仆人—主人关系网（推定）

### 希尔太太 ↔ 贝内特太太
- **相伴二十年以上**，按贝内特夫妇结婚时期推算
- 最常承受贝内特太太神经发作的人
- 贝内特太太可能经常向希尔吐露秘密
- 第49章：“Have you heard the good news?”——亲密的称呼方式

### 雷诺兹太太 ↔ 达西家
- **从达西四岁起便认识他**，第43章明确写出
- 实际上承担达西**第二位母亲**的角色
- 安妮夫人去世后，可能替代了母亲的位置
- 也影响乔治安娜的成长
- **向访客称赞达西**出自真心

### 莎拉 ↔ 贝内特姐妹
- **服侍姐妹们的日常生活**
- 舞会准备时决定先帮助谁；第55章优先帮助简
- 潜在的流言通道

### 彭伯里的园丁 ↔ 达西
- 达西亲自参与庄园事务，因此与园丁保持一定关系
- 加德纳夫妇在彭伯里时，有园丁陪加德纳先生钓鱼

## 3. 梅里屯社会网络（推定）

### 流言的传播
```
朗太太 → 贝内特太太 → 贝内特姐妹 → 梅里屯
       ↓
菲利普斯太太 ← 贝内特太太（姐妹）
       ↓
菲利普斯家的客人（军官）→ 整个梅里屯

[其他渠道]
卢卡斯夫人 → 贝内特太太（竞争＋信息）
威廉爵士 → 整个社交界（夸耀“first circles”）
```

### 朗太太的位置
- **梅里屯流言中心**——第1章最先传来宾利消息的人
- 与两个未婚侄女同住，在婚姻市场上与贝内特姐妹竞争
- **达西与她沉默相对半小时**，第5章；进一步加深贝内特太太对他的反感

### 军官之间的关系
- **威克姆—丹尼：** 亲近；一同从伦敦来到梅里屯
- **威克姆—卡特：** 同僚
- **威克姆—福斯特上校：** 下属与上司。第46章福斯特上校泄露威克姆本想逃走的秘密，说明**威克姆未能取得福斯特信任**
- **威克姆在部队中很受欢迎：** “captivating”，同时在第50章留下赌债

## 4. 柯林斯的庇护网络

### 凯瑟琳夫人的影响力网络
```
凯瑟琳夫人
├── 柯林斯先生（汉斯福德牧师，绝对依附）
│   └── 夏洛特（通过柯林斯间接依附）
├── 詹金森太太（安妮·德包尔的女伴，受雇）
├── 杨太太（前家庭教师，可能由达西推荐）
├── 波普小姐（由其介绍家庭教师职位）
└── 梅特卡夫夫人等（社交界庇护）

[达西虽是她的外甥，却不受她支配]
```

### 柯林斯的社会位置
- 同时效忠**上层的凯瑟琳夫人**与**下层的教区居民**
- 对贝内特家感情矛盾：既是亲戚，也是他未来将继承的产业
- 与夏洛特结婚后，也试图争取包括威廉爵士在内的岳家支持

## 5. 达西的阶级人脉（推定）

### 明确写出
- 彭伯里附近的绅士
- 伦敦社交界，姓名未公开
- 宾利，最亲近的朋友
- 菲茨威廉上校，表兄弟
- 凯瑟琳夫人，姨母

### 推定人脉
- **伯爵，菲茨威廉上校之父**——达西的舅父
  - 达西前往伦敦拜访的某处
  - 可能是乔治安娜的监护人之一
- **达西的律师**——管理庄园并参与威克姆谈判
- **彭伯里附近的医生**——照料乔治安娜的健康
- **剑桥大学同窗**——宾利也可能是其中之一
- **伦敦俱乐部成员**，如White's、Brooks's等

## 6. 威克姆的黑暗人脉（推定）

### 明确写出
- 杨太太，乔治安娜事件的共谋者、伦敦寄宿屋经营者
- 丹尼先生，在梅里屯介绍威克姆的人

### 推定
- **伦敦赌场人脉**——债务来源
- **梅里屯与布莱顿的酒友**
- **其他曾试图诱惑的女性**，除乔治安娜、金小姐和莉迪亚外可能还有别人
- **钱伯莱恩**，莉迪亚在第51章提及的朋友；威克姆曾让他男扮女装

## 7. 宾利姐妹的社交野心（推定）

### 卡罗琳的婚姻市场
- **达西是第一选择**，绝不愿放弃
- 对其他人只能勉强考虑
- 进入三十岁后可能陷入恐慌
- 结尾第61章，她向伊丽莎白偿还“欠下的礼貌”，以**维持与达西表亲等人脉的联系**

### 路易莎·赫斯特的婚姻
- 与赫斯特先生的婚姻是“fashion rather than fortune”，第4章
- **实际上是失败的婚姻**——丈夫只关心吃饭和纸牌
- 她既是卡罗琳的盟友，也是安慰

## 8. 地方社会的婚姻市场（推定）

### 梅里屯未婚女性（贝内特姐妹除外）
- **卢卡斯姐妹，四至五人**，包括玛丽亚、夏洛特等
- **朗太太的两个侄女**
- **古尔丁家的小姐们**，住在海伊公园；第50章由贝内特太太提及
- **罗宾逊家的小姐们**
- **韦布家的小姐们**，第29章提及，擅长钢琴
- **沃森小姐**，第7章卡特上尉前去拜访

### 未婚男性
- **数十名军官**
- **罗宾逊先生**，第5章
- **古尔丁先生**，邻居

### 结婚可能性矩阵
- 宾利：所有母亲的目标
- 达西：地位太高且傲慢，因此众人回避
- 威克姆：迷人却贫穷，只有金小姐一类继承人可能选择他
- 柯林斯：生活稳定却可笑，只有夏洛特一类现实主义者会考虑

### 威克姆的真正意图（推定）

- 原计划把莉迪亚当作情妇带在身边一段时间，然后抛弃
- 目标是清偿债务、重新开始
- **若没有达西的钱，他不会结婚**

### 达西—加德纳舅舅的秘密协议（EVT-039-005）
```

- 加德纳先生起初因自尊拒绝

- 双方同意：表面上伪装成加德纳所为
```

### 达西向宾利坦白拆散两人（EVT-042-001）
```
- 达西在彭伯里向宾利坦白
- “我误判了简是否爱你。”
- “我向你隐瞒了简住在伦敦的事实。”
- 宾利的反应：愤怒 → 原谅，符合他的性格
- 立即决定返回内瑟菲尔德
```

### 贝内特先生赴伦敦搜寻（EVT-038之后）
```
- 与福斯特上校同行
- 可能雇用Bow Street Runners，即伦敦侦探
- 一无所获，将搜寻交给加德纳后返家
- 自尊遭受重创，此后更加闭门独处
```

## 12. 结局后（第61章）未呈现的关系变化

### 彭伯里日常生活（推定）
- 伊丽莎白成为女主人，与雷诺兹太太合作
- 乔治安娜像姐妹一样追随伊丽莎白
- 宾利与简搬到三十英里以内，经常来访

### 威克姆—莉迪亚的长期依赖
- 每年向姐姐们索要钱财
- 推定**伊丽莎白用自己的私用年金暗中支援**
- 达西假装不知情，或知情却默许

### 凯蒂的变化
- 经常访问彭伯里与内瑟菲尔德
- “much improved”——社交礼仪有所改善
- 结婚可能性打开

### 玛丽的变化
- 姐妹离家后，成为母亲唯一的陪伴者
- 可能加入“成为宾利太太的姐姐”的社交活动
- 对外貌的自卑减少

### 贝内特夫妇
- 两个女儿嫁入富裕家庭，贝内特太太的神经或许安定
- 贝内特先生依然闭居
- 只有玛丽留在家中

### 凯瑟琳夫人的和解
- 屈服于好奇心
- 访问彭伯里
- 不得不对伊丽莎白保持正式礼貌
- 但没有真正的友谊

### 宾利小姐虚伪的交往
- 为了出入彭伯里而表现“弥补性的亲切”
- 婚姻市场上的最后一张牌
