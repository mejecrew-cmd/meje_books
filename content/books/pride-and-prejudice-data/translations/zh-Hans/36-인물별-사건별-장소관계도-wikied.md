## 4. 地点叙事意义的变迁

追踪各地点在叙事中承载的意义，以及它如何随事件改变。

### 内瑟菲尔德庄园
| 时点 | 叙事意义 |
|---|---|
| 第1章，EVT-001 | **期待之地**——富有青年入住，象征结婚可能 |
| 第3章，EVT-003 | **社交舞台**——舞会开启两对人物的关系 |
| 第7至12章，EVT-006–010 | **考验之地**——照料病人的留宿加深关系，也暴露阶级冲突 |
| 第18章，EVT-016 | **对决之地**——所有矛盾在舞会爆发 |
| 第21至23章，EVT-019–021 | **缺席的象征**——宾利离开后的空宅体现简的失落 |
| 第53至55章，EVT-043–044 | **归来之地**——宾利返回并再次求婚，恢复希望 |

### 彭伯里
| 时点 | 叙事意义 |
|---|---|
| 第8章，宾利小姐奉承 | **炫耀工具**——宾利小姐为奉承达西而提及的虚荣题材 |
| 第43章，EVT-035 | **人格之镜**——顺应自然美的园林、真正优雅的家具与雷诺兹太太证言，反映达西内心 |
| 第43章，EVT-036 | **变化舞台**——达西彻底改变的态度首次在此展现 |
| 第44至45章，EVT-037 | **融合空间**——达西、乔治安娜、宾利与加德纳夫妇共处一地 |
| 第61章，EVT-048–049 | **结果之地**——伊丽莎白成为女主人的最终归宿 |

### 汉斯福德牧师住宅
| 时点 | 叙事意义 |
|---|---|
| 第28章，抵达 | **妥协空间**——夏洛特在与柯林斯婚姻中找到的“舒适家庭”的真相 |
| 第32章，EVT-026 | **紧张空间**——达西反复前来却沉默不语的奇异地点 |
| 第34章，EVT-028 | **破裂空间**——达西第一次求婚与伊丽莎白激烈拒绝 |

### 罗新斯公园步道
| 时点 | 叙事意义 |
|---|---|

## 5. 供译者使用的地点—氛围对应表

翻译中出现地点时，应反映该时点的叙事氛围：

| 地点 | 前期氛围 | 转折点 | 后期氛围 |
|---|---|---|---|
| 内瑟菲尔德 | 悸动、期待 | EVT-019，撤离 | 缺席、失落 → EVT-043恢复希望 |
| 彭伯里 | 尚未访问；宾利小姐奉承的题材 | EVT-035，访问 | 敬畏、重新发现、爱情结果 |
| 汉斯福德 | 夏洛特妥协后的安身处 | EVT-028，求婚／拒绝 | 破裂与转变现场 |
| 罗新斯 | 压迫性的权威 | EVT-025，初见 | 傲慢的展厅 → EVT-045反击的催化剂 |
| 梅里屯 | 活跃的社交中心 | EVT-014，威克姆证言 | 流言与偏见的温床 |
| 伦敦 | 分离与阴谋之城 | EVT-040，秘密斡旋 | 拯救与解决的舞台 |
| 布莱顿 | 莉迪亚幻想中的乐园 | EVT-038，私奔 | 毁灭的起点 |
| 浪搏恩 | 喧闹而不完美的家庭 | EVT-046，再次求婚 | 出发与归来的原点 |
