# 《傲慢与偏见》——人物作息表（推定/创作）

> ⚠️ **本文档包含推定与创作。** 结合摄政时期乡绅与中产阶级的标准作息、各人物性格和原作行为模式，重构人物日常时间表。可用于改编时描写人物日常生活。

---

## 摄政时期乡绅的标准一天（基本型）

| 时间 | 活动 |
|---|---|
| 早上7至8点 | 起床。仆人把温水送到卧室。穿晨间服。 |
| 8至9点 | 散步或私人时间（写信、阅读） |
| 9至10点 | Breakfast（早餐）——全家一起 |
| 10至12点 | 散步、拜访（morning calls）、家务、阅读、音乐练习 |
| 12至15点 | 外出、购物、散步、接待客人 |
| 15:30至16点 | 换装（dressing for dinner） |
| 16至17点 | Dinner（正餐，一天的主餐） |
| 17至19点 | 餐后休息（男性：波特酒；女性：客厅） |
| 19至21点 | Tea、纸牌、音乐、阅读、谈话 |
| 21至22点 | Supper（简单夜宵，通常仅在舞会后） |
| 22至23点 | 就寝 |

## 3. 贝内特先生——推定作息

| 时间 | 活动 | 依据 |
|---|---|---|
| 7至8点 | 不慌不忙地起床 | "indolent"的性格 |
| 8至9点 | **直奔书房** | 避免与妻子谈话 |
| 9点 | 早餐 | 与家人一起 |
| 9点以后 | **回到书房，整日独处** | "fond of the country and of books... his principal enjoyments" |
| 12至15点 | 散步、检查农场 | |
| 16点 | 参加家庭正餐 | |
| 17点以后 | **再次逃回书房** | 第20章：要求柯林斯"free use of my room" |
| 22点以后 | 阅读至深夜 | 第3章："He had rather hoped... He had a good deal of curiosity" |

### 日常模式
- **尽量少与妻子谈话**——只在用餐时
- **喜欢与伊丽莎白交谈**——在书房里
- **只在危机时行动**——平时旁观，EVT-038后才立即前往伦敦搜寻
