# 《傲慢与偏见》肢体语言与礼仪辞典

> 摄政时期身体礼仪、社交规范与动作方式指南，也是影视和舞台调度的最终基准。
> 一个错误的动作，就足以摧毁时代剧的可信度。

---

## 1. 问候礼

### Bow（男性鞠躬礼）

#### 正式程度

| 类型 | 动作 | 用途 |
|---|---|---|
| **轻微点头礼** | 头部微微下倾4～5度 | 社会地位相当者之间的日常问候 |
| **颈部鞠躬礼** | 仅将颈部下倾10～15度 | 标准问候 |
| **腰部鞠躬礼** | 从腰部弯身30～45度 | 正式场合；面向贵妇或上位者 |
| **深鞠躬礼** | 弯身60度以上，一只脚略向后撤 | 最高规格；面见王室或初次见面 |

#### 小说中的例子
- 第18章舞会上，**达西 → 柯林斯**：“slight bow（轻微鞠躬）”—刻意表现居高临下
- **柯林斯 → 所有人：**过分郑重的深鞠躬，显露他的虚伪
- 第15章街头，**威克姆 → 达西**：“touched his hat（碰了碰帽子）”—几乎等于不予理会

#### 翻译说明
- “Bowed”应当被呈现为一种**身体动作**，不能简化为“打了招呼”。
- 鞠躬的正式程度标示着人物关系。

### Curtsey（女性屈膝礼）

#### 类型

| 类型 | 动作 | 用途 |
|---|---|---|
| **Bob curtsey** | 轻轻屈膝 | 日常 |
| **Half curtsey** | 屈膝更深并低头 | 标准 |
| **Full court curtsey** | 一只脚大幅后撤，双膝深屈 | 宫廷或最高规格场合 |

#### 动作细节
- **双手：**轻提裙身两侧，或自然垂放
- **视线：**略微向下，礼节越正式，目光越低
- **时长：**保持1～2秒

#### 小说中的例子
- 第30章，**伊丽莎白 → 达西**：“merely courtesied to him, without saying a word（只是向他行了屈膝礼，一言不发）”—**最低限度的屈膝礼＋沉默，表示冷淡**
- **玛丽亚·卢卡斯 → 凯瑟琳夫人：**紧张地深行屈膝礼

### 握手—**几乎不用**

- 在摄政时期，握手表示**亲密**，并不像现代社会那样属于日常问候。
- 初次见面绝不握手。
- 第61章，宾利小姐与伊丽莎白“even shook hands（甚至握了手）”—**表示让步**。

## 2. 与人交谈时的行为规范

### 初次见面后的正式介绍

```
[规则]
- 把地位较低者介绍给地位较高者
- 把男性介绍给女性，由女性掌握是否交往的选择权
- 未经介绍便开口交谈，属于无礼

[程序]
1. 介绍人站在两人之间
2. “Mr. Darcy, may I introduce Miss Elizabeth Bennet?”
   - 先说地位较高者的名字
3. 双方互相致意：bow + curtsey
4. 随后可以进行简短交谈

[小说中的例子]
- 第18章，柯林斯自行向达西介绍自己—无礼；达西以轻蔑回应
- 舞会上，司仪可以为宾客进行介绍
```

### 帽子礼仪

| 情形 | 动作 |
|---|---|
| 向女性问候 | **抬帽致意：**触碰后提起 |
| 进入室内 | **脱帽** |
| 遇见熟人 | “Touched his hat（碰了碰帽子）”—第15章的威克姆与达西 |
| 与女性交谈 | 开始时抬帽，之后拿在手中或夹在腋下 |

## 3. 舞会礼仪

### 入场

```
[顺序]
1. 在衣帽间寄存外套、帽子和手套
2. 在前厅稍候片刻
3. 向司仪通报来宾：“Mr. and Mrs. Bennet, with their daughters”
4. 进入主舞厅
5. 向男女主人问候
```

### 邀舞程序

```
[男性走近女性]
1. 双方必须经过正式介绍
2. 上前鞠躬，并说：“May I have the honour of the next dance, Miss Bennet?”
3. 如果女性接受，男性站在她身旁，交谈至音乐响起
4. 音乐开始后，两人一同走入舞列
```

### 拒绝规则

```
[规则]
- 女性若拒绝一名男性，依照惯例，她在这一组舞中就不能再与任何人跳舞
- 得体的理由：“I am engaged（我已有舞伴）”或“I am too tired（我太累了）”
- 明确拒绝是一种社交侮辱

[小说中的例子]
- 第6章：伊丽莎白拒绝达西—“I have not the least intention of dancing”
  = 带有侮辱性的拒绝，反而令达西着迷
- 第18章：伊丽莎白“without knowing what she did（不知道自己做了什么）”便接受了达西的邀请
  = 无意识的同意
```

### 舞步

#### Country Dance（最常见的舞蹈）

```
[队形]
- 男士一列、女士一列，彼此相对而立
- 一对舞伴从队列末端开始，逐渐沿队列向前推进

[动作]
1. 面对彼此行礼：bow + curtsey
2. 双手相接，转一圈
3. 向前四步，再后退四步
4. Do-si-do—背对背绕行一圈
5. 与队列中的下一对重复动作
6. 一组舞会通过反复变换舞式持续30分钟至1小时
```

#### 牵手方式

```
[正式—两人]
- 掌心相对，只有指尖轻触
- 绝不十指相扣
- 戴着手套，皮肤不会直接接触

[更正式—仅用指尖]
- 只轻轻握住食指和中指
- 用于正式舞会

[小说中的含义]
- 仅触指尖 = 距离
- 整个掌心接触 = 亲密
- 第18章达西与伊丽莎白共舞时如何牵手，具有视觉意义
```

### Reel（苏格兰舞）

- 节奏更快、更活泼
- 由四人或更多人共舞
- 旋转动作很多
- 第10章达西提议与伊丽莎白共舞的舞种—**活泼的舞蹈令这一邀请带有挑衅意味**

### Quadrille（四对舞伴的正式舞蹈）

- 四对舞伴排成方形
- 舞步复杂
- 约1815年开始在英国流行；在小说所处年代才刚传入

### Boulanger（终场舞）

- 舞会的最后一曲
- 所有人围成圆圈
- 活泼而富有社交性

## 4. 餐桌礼仪

### 座位安排

```
[正式晚宴]

[男主人]                              [女主人]
   ↓                                     ↓
   ━━━━━━━ 餐桌 ━━━━━━━

   女主人身旁 = 首席宾客
   男主人身旁 = 第二位宾客
   餐桌下首 = 地位最低者或亲属

[小说中的例子]
- 第29章，罗新斯：柯林斯坐在桌尾的最低席位
- 第18章舞会的夜宵：伊丽莎白与达西相对而坐—刻意安排的位置
```

### 上菜方式（à la française）

```
[方法]
1. 一次把所有菜肴摆上餐桌
2. 男主人开始切分主菜
3. 每位用餐者从附近的菜盘自行取食
4. 需要远处的食物时，请邻座传递：
   “Could I trouble you for the salt?”
5. 一餐持续1～2小时
```

### 基本礼仪

| 动作 | 礼仪 |
|---|---|
| 餐巾 | 放在膝上，不塞进衣领 |
| 餐叉 | 英国用法为**左手**持叉 |
| 餐刀 | 右手 |
| 切食物 | 每次只切一口的分量 |
| 口中有食物 | **不要说话** |
| 葡萄酒 | 与邻座行共饮礼：“I drink to you” |
| 用餐中途离席 | 极其无礼 |

### 甜点之后

```
1. 女主人起身时，所有女性一同起身
2. 女性们退往客厅
3. 男性们留在餐桌旁，饮用波特酒并谈论政治
4. 30分钟至1小时后，男性前往客厅，与女性们共饮茶或咖啡
```

## 5. 客厅礼仪

### 座位安排

```
[标准安排]
- 女主人：茶桌和茶壶旁
- 女性宾客：女主人附近
- 男性：可自由站立或落座
- 老年人：壁炉旁的扶手椅

[活动]
- 饮茶
- 玩牌
- 钢琴演奏
- 刺绣和缝纫
- 可以阅读，但会显得脱离社交
- 在花园散步
```

### 倒茶

```
[女主人的仪式]
1. 有些家庭会先把牛奶倒入小茶杯
2. 倒茶
3. 宾客自行加糖
4. 将茶杯和茶碟递给宾客

[宾客礼仪]
- 左手托茶碟，右手持茶杯
- 小口啜饮
- 不要发出声响
```

## 6. 拜访礼仪

### Morning Call（上午拜访）

```
[规则]
- 虽称“morning call”，实际拜访时间为上午11点至下午3点
- 只停留15～30分钟；久留属于无礼
- 在客厅接待访客，绝不能在卧室或餐厅

[程序]
1. 到达时递上名片，或由仆人通报本人来访
2. 被引入客厅
3. 女主人入内，向访客问候
4. 简短谈论天气、健康和传闻
5. 女主人奉茶，访客接受或婉拒
6. 稍坐片刻后起身告辞

[名片的含义]
- “我曾来访”—来访通知
- 折起一角，表示访客本人到场，而非派仆人送来
- 产生一周内回访的义务
```

### 小说中的运用
- 第3章：宾利回访贝内特先生，停留十分钟—正式
- 第55章：宾利停留一小时—**亲密的标志**
- 第55章：宾利小姐进行短暂而冷淡的回访—**保持距离**

## 7. 上下马车

### 马车礼仪

```
[上车顺序]
1. 地位最高的女性先上车
2. 其他女性
3. 男性最后上车，或坐上车夫座／骑马随行

[座位]
- 面朝行进方向的前座 = 上座
- 背朝行进方向的后座 = 地位较低者的座位

[下车]
1. 男性先下车
2. 男性伸手协助女性
3. 女性握住他的手迈下车—难得的身体接触机会
```

### 小说中的含义

- 第43章离开彭伯里时，“Mr. Darcy handed the ladies into the carriage”—**达西的手**会触碰伊丽莎白或加德纳夫人的手。
- 在身体接触极少的时代，**协助上下马车是合乎礼法的亲密时刻**。

## 8. 书写、折叠与封缄书信

### 写信姿势

```
[在书桌前]
1. 将信纸斜放
2. 右手持羽毛笔，靠近墨水瓶
3. 左手压住纸张
4. 写满一张约需20～30分钟

[姿势错误 = 失礼]
```

### 折叠书信

```
[传统方法—不用信封]
1. 在一面书写，另一面留白
2. 横向折成三折
3. 再纵向折叠
4. 在外侧写上地址
5. 用封蜡和印章封缄，如有家族印章则使用家徽

[小说中的例子]
- 第35章达西的信：“two sheets of letter paper, written quite through, in a very close hand”
- 达西破例使用了**信封**：“envelope containing two sheets”
- 这表示秘密性＋考究
```

### 接收书信

```
[礼仪]
- 不要立刻阅读；稍候片刻以表现情绪克制
- 私人信件应在独处时阅读
- 在家人面前阅读时，只可简短查看
- 出于礼貌，应立即回信
```

### 小说中的运用
- 第21章，宾利小姐的信：简上楼独自阅读—**预感到坏消息**
- 第46章，简的信：伊丽莎白立即阅读—**紧急情况**

## 9. 卧室与私人空间礼仪

### 进入规则

```
[可以来访]
- 卧室：仅限家人（姐妹、父母）
- 设有书桌的boudoir：亲密的女性朋友
- 书房：家人＋亲密朋友

[禁止]
- 未婚男女单独共处一室 = 丑闻
- 异性进入卧室 = 绝对禁止

[小说中的例子]
- 达西＋伊丽莎白单独留在客厅（第32、34章）= 已经严重违反正式礼法
- 达西进入客厅时，宾利小姐、赫斯特太太等人的目光
```

### 更衣

```
[由Lady’s Maid协助]
1. 解开长袍
2. 解开紧身胸衣
3. 更换衬裙
4. 穿上新长袍
5. 整理头发
6. 换鞋

[独自更衣时]
- 姐妹互相帮助（贝内特姐妹）
- “Sarah, never mind Miss Lizzy’s hair”（第55章）—时间不足
```

## 10. 手与手臂的使用

### 挽臂同行（Walking Arm in Arm）

```
[规则]
- 女性站在男性右侧
- 女性把手搭在男性手臂上，而不是抓住他的手臂
- 表示亲密，但仍在正式礼法范围内

[小说中]
- 第43章，彭伯里：加德纳夫人–伊丽莎白 → 达西–加德纳夫人 → 达西–伊丽莎白
  = 同行组合的变换 = 意味深长的瞬间
```

### 吻手礼—**几乎不用**

- 英国几乎没有吻手礼；这是欧洲大陆的习俗。
- 若由威克姆一类人物尝试，会显得**油滑做作**。

### 扇子的使用（女性）

```
[扇子的语言—即使无意识使用，也会传达含义]
- 打开并轻轻摇动：平静
- 快速扇动：紧张／兴奋
- 用扇子遮脸：害羞
- 合起扇子：结束谈话的信号

[小说中可能出现的场面]
- 舞会上宾利小姐扇动扇子 = 观察达西的工具
```

## 11. 目光礼仪

### 注视规则

```
[允许]
- 每次与谈话对象对视1～3秒
- 接受女主人／男主人的目光

[禁止]
- 公然凝视，尤其是凝视女性
- 从头到脚打量他人 = 极其无礼
- 用餐时盯着别人的食物

[小说中的例子]
- 达西经常注视伊丽莎白—伊丽莎白在第10章察觉
- “She hardly knew how to suppose that she could be an object of admiration”
- 无法判断他的目光究竟无礼，还是出于爱慕
```

### 微笑

```
[正式程度]
- 嘴角微微上扬 = 标准
- 露齿微笑 = 非常亲密
- 放声大笑 = 无礼，尤其对女性而言

[伊丽莎白的独特之处]
- “lively, playful disposition, which delighted in anything ridiculous”
- 她经常发笑，比一般女性更活泼，因此轻微逾越了社会规范
```

## 12. 距离（Personal Space）

### 标准距离

```
[普通交谈]
- 1～1.5米（超过一臂长度）
- 距离更近 = 亲密

[正式问候]
- 2米（正式距离）
- 为行鞠躬礼先后退，再上前

[跳舞]
- 只有牵手时靠近
- 舞式之间至少相隔一步

[小说中]
- 达西把椅子移近伊丽莎白（第32章），之后又拉开距离
- 情绪迸发 → 自我克制
```

## 13. 身体接触频率（低）

### 允许的接触

| 接触 | 情形 |
|---|---|
| 握手 | 表示亲密（少见） |
| 挽臂 | 散步时 |
| 协助上下马车 | 握手扶下车 |
| 跳舞 | 牵手 |
| 家人 | 姐妹之间拥抱、亲吻 |

### 小说中的身体接触时刻（稀少）

- 达西＋伊丽莎白：**共舞（第18章）**＋**协助上下马车（第43章）**＋**再次求婚时的散步（第58章）**
- 除此之外几乎没有接触，因此每一次触碰都具有**情感上的爆发力**。

## 14. 悲伤与愤怒的表达（克制）

### 礼仪社会中的情绪控制

```
[允许]
- 轻轻叹息
- 转过头去
- 沉默
- 离开房间

[应当克制]
- 放声哭泣（仅可独处时）
- 愤怒地喊叫
- 夸张的手势

[小说中的违反情形]
- 贝内特太太的神经发作 = 缺乏克制
- 莉迪亚的兴奋 = 违反礼仪
```

### 真正的情绪爆发场面（克制崩溃）

- 第34章，达西：“walked with quick steps across the room”—愤怒爆发
- 第34章，伊丽莎白：拒绝后“cried for half an hour”—但只在独处时
- 第46章，伊丽莎白：在达西面前哭泣—危机使她失去自制

## 15. 影视与舞台调度检查表

### 每个场面都应确认

- [ ] 问候动作是否符合正式程度？
- [ ] 座位安排是否符合身份？
- [ ] 身体接触频率是否符合时代？
- [ ] 目光礼仪是否准确？
- [ ] 穿衣／脱衣时是否有lady’s maid协助？
- [ ] 上马车的顺序是否正确？
- [ ] 书信折叠与封缄是否正确？
- [ ] 舞步是否符合时代？
- [ ] 餐桌座位是否遵守正式礼法？
- [ ] 情绪表达是否克制？

### 现代影视剧中的常见错误

- ❌ 握手过于频繁
- ❌ 身体接触过多（现代人的感觉）
- ❌ 说话和笑声太大
- ❌ 目光过于直接
- ❌ 人与人之间距离过近
- ❌ 更衣太快，而且没有lady’s maid

## 16. 各人物的特殊肢体语言

### 伊丽莎白—活跃＋略微打破规范

- 步伐轻快
- 偶尔大声发笑
- 手势活泼
- 目光直接，带有挑战意味

### 达西—克制＋沉默

- 动作缓慢
- 长时间沉默
- 目光深沉而持久
- 即使说话，也几乎不动双手

### 贝内特太太—夸张

- 经常扇扇子
- 大声叹息
- 手势夸张
- “神经”发作时靠在椅背上，用扇子遮住脸

### 贝内特先生—漠然的姿态

- 深深坐进椅子
- 几乎不把目光从书本上移开
- 有时背对家人的谈话

### 莉迪亚—毫无克制

- 到处奔跑（“flew about the house”）
- 放声大笑
- 无视礼仪

### 凯瑟琳夫人—威压

- 姿势笔直
- 直视对方，施加压力
- 使用命令式手势

### 柯林斯—过度

- 反复深鞠躬
- 为了奉承而搓手
- 频繁低头

### 威克姆—迷人的礼仪

- 温柔微笑
- 站在亲密距离，违反惯例
- 长久注视，带有诱惑意味

## 17. 关键场面的肢体语言指南

### 第34章：求婚场面

```
[达西的行动路线]
- 进入客厅，显得自信却又紧张
- 坐下 → 起身 → 在房间里踱步
- 走近伊丽莎白（“agitated manner”）
- 求婚时语气略显激烈（“dwelt on with a warmth”）
- 遭到拒绝后倚靠壁炉架，承受震惊
- 最后快步穿过房间

[伊丽莎白的行动路线]
- 因惊讶而坐着不动
- 不起身，维持原有姿势
- 即使拒绝时也保持姿势
- 因愤怒而涨红脸
- 达西离开后才崩溃
```

### 第46章：私奔的消息

```
[伊丽莎白]
- 手拿信件，从椅子上一跃而起
- 奔向门口
- 遇见达西，停下脚步
- 双腿失去力气，再次坐下
- 用手帕遮住脸
- 啜泣

[达西]
- 进门时吃了一惊
- 走近她，并试图叫仆人来
- 试图安慰她
- 在房间里踱步，暗自下定决心
- 最后看她一眼，随后离开
```

### 第58章：再次求婚

```
[两人]
- 散步途中只剩彼此
- 伊丽莎白先开口表达感谢
- 达西停下脚步
- 两人面对彼此
- 一边缓慢行走，一边交谈
- 不牵手，以维持礼法
- 但目光逐渐加深
- 最后可以自然地牵起手，一同回家
```
