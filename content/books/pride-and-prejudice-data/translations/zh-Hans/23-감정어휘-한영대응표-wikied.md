# 《傲慢与偏见》情感词汇对应指南

> 小说中使用的英语情感词汇、对应含义与强度等级。
> 即使表达同一种情感，也要根据强度和色彩选用不同词语。这是翻译中最困难的领域。

---

## 1. 情感词汇分类体系

### 五级强度
- ⓟ 极轻（faint）
- ① 轻微（mild）
- ② 中等（moderate）
- ③ 强烈（strong）
- ④ 激烈（intense）
- ⑤ 极端（extreme）

### 色彩（情感的质地）
- 🌸 温暖（warm）
- ❄️ 冷淡（cold）
- 🔥 激情（passionate）
- 🌧️ 忧郁（melancholy）
- ⚡ 盛怒（furious）
- 🌫️ 模糊（ambiguous）

## 2. 好感与爱情系列（按强度排列）

| 英语 | 含义／语感 | 强度 | 色彩 | 小说中的例子 |
|---|---|---|---|---|
| **acquaintance** | 相识、熟人关系 | ⓟ | 🌫️ | “her acquaintance”—仅仅认识 |
| **liking** | 好意、喜欢 | ① | 🌸 | “I give you leave to like him” |
| **partiality** | 偏爱、带偏向的好感 | ① | 🌸 | “his partiality for Miss Bennet” |
| **inclination** | 吸引、内心倾向 | ① | 🌫️ | “a slight, thin sort of inclination” |
| **regard** | 尊重、情分 | ② | 🌸 | “his sincere, if not equal, regard” |
| **esteem** | 敬重、高度评价 | ② | 🌸 | “she respected, she esteemed” |
| **affection** | 爱意、温情 | ② | 🌸 | “tender affection” |
| **fondness** | 温情、爱意 | ② | 🌸 | “fond of his sister’s society” |
| **attachment** | 深情、依恋 | ③ | 🌸 | “stronger attachment than I had ever known” |
| **admiration** | 倾慕、赞叹 | ③ | 🔥 | “looked at with great admiration” |
| **love** | 爱 | ③～④ | 🔥 | “how ardently I admire and love you” |
| **passion** | 热情、激情 | ④ | 🔥 | “the utmost force of passion” |
| **ardour** | 热烈 | ④ | 🔥 | “spoken... with such ardour”（间接） |
| **adoration** | 仰慕、崇拜 | ⑤ | 🔥 | “the adoration of Mrs. Forster” |

### 对应词指南

```
[轻微好感]
acquaintance：“相识”／“熟人关系”
liking：“喜欢”／“好意”
partiality：“带偏向的好感”／“偏爱”
inclination：“吸引”／“内心倾向”

[中等好感]
regard：“情分”／“心意”／“尊重”
esteem：“敬重”／“倾慕”
affection：“爱意”／“温情”
fondness：“温柔的情意”
attachment：“深情”／“依恋”

[强烈爱情]
admiration：“倾慕”／“赞叹”
love：“爱”／“爱慕”
passion：“热情”／“激情”
ardour：“热烈的情感”
adoration：“崇拜”／“爱慕”
```

### 小说中的阶段性演变（达西 → 伊丽莎白）

```
第6章   admiration（赞叹）—“fine eyes”
第10章  bewitched（着迷）—如同魔法般的吸引
第34章  ardent love（热烈的爱）—“how ardently I admire and love you”
第58章  unchanged affections（不变的爱意）—平静的深度
```

## 3. 负面情感—愤怒与憎恶系列

| 英语 | 含义／语感 | 强度 | 色彩 |
|---|---|---|---|
| **dislike** | 不喜欢、反感 | ① | ❄️ |
| **disapprobation** | 不赞同、不满 | ① | ❄️ |
| **displeasure** | 不悦、不满 | ② | ⚡ |
| **vexation** | 烦恼、恼怒 | ② | ⚡ |
| **annoyance** | 烦恼 | ② | ⚡ |
| **irritation** | 不快、刺恼 | ② | ⚡ |
| **resentment** | 怨恨、愤慨 | ③ | ⚡ |
| **anger** | 愤怒 | ③ | ⚡ |
| **indignation** | 义愤 | ③ | ⚡ |
| **abhorrence** | 憎恶 | ④ | ❄️ |
| **detestation** | 痛恨 | ④ | ⚡ |
| **fury** | 暴怒 | ⑤ | 🔥 |
| **wrath** | 带有《圣经》色彩的愤怒 | ⑤ | 🔥 |

### 对应词指南

```
[轻]
dislike：“不喜欢”／“反感”
disapprobation：“不赞同”／“不以为然”

[中]
displeasure：“不悦”／“心生不快”
vexation：“烦恼”／“恼怒”—避免过于现代的说法，“心烦意乱”更稳妥
annoyance：“烦恼”
irritation：“刺恼”／“心有芥蒂”

[强]
resentment：“怨恨”／“愤慨”／“积怨”
anger：“愤怒”／“怒气”
indignation：“义愤”
abhorrence：“憎恶”／“厌恶”

[激烈]
detestation：“痛恨”／“憎恶”
fury：“暴怒”／“怒火上涌”
wrath：“盛怒”—几乎不用
```

### 小说中达西与伊丽莎白的愤怒强度

```
第5章   dislike（反感）—轻微
第16章  resentment（怨恨）—听过威克姆的陈述后
第33章  indignation（义愤）—菲茨威廉透露真相
第34章  ardent rejection（激烈拒绝）
第36章  shame（羞耻）—对自己的愤怒
```

## 11. 各人物的情感词汇使用频率

### Elizabeth（活力＋机智）

```
[常用词汇]
- delight, amusement, vexation
- mortification（涉及自尊时）
- “lively, playful disposition”

[翻译语调]
- 活泼＋略带讽刺
- “有意思”“有趣”“烦人”
```

### Darcy（克制＋强度）

```
[常用词汇]
- regard, esteem, ardour
- shame, mortification
- “in vain have I struggled”

[翻译语调]
- 克制而深沉
- “情分”“倾慕”“羞耻”
```

### Mrs. Bennet（激烈的表层情绪）

```
[常用词汇]
- joy, rapture, transport
- vexation, distress
- nerves（神经）

[翻译语调]
- 情绪激烈却流于表面
- “欢欣”“烦恼”“忧心”
```

### Mr. Bennet（讥诮的冷漠）

```
[常用词汇]
- amusement（来自他人的愚蠢）
- 几乎不直接表达情感

[翻译语调]
- 保持距离的机智
- “我觉得颇有意思”／“真有趣”
```

### Jane（克制的温柔）

```
[常用词汇]
- regard, esteem, attachment
- 克制情感

[翻译语调]
- 温柔却不失礼法
- “情分”“放在心上”
```

### Lydia（毫无克制）

```
[常用词汇]
- joy, rapture, ecstasy
- 全都流于表面

[翻译语调]
- 兴奋，但避免粗俗
- “太高兴了！”／“真是的！”
```

### Lady Catherine（愤怒＋威压）

```
[常用词汇]
- displeasure, indignation
- arrogance, condescension

[翻译语调]
- 权威式愤怒
- “不悦”“怒气”
```

### Collins（过度正式）

```
[常用词汇]
- gratification, satisfaction
- humility, civility（仅有表象）

[翻译语调]
- 过分正式的词汇
- “欣慰”“不胜感激”
```

## 12. 关键场面的情感强度追踪

### 第34章：求婚场面

```
[达西]
1. agitation（动摇）—进门
2. ardent admiration（热烈倾慕）—求婚
3. astonishment（惊愕）—遭到拒绝
4. anger（愤怒）—“pale with anger”
5. mortification（屈辱）—“ashamed of what my own have been”

[伊丽莎白]
1. astonishment（惊愕）—听到求婚
2. compassion（同情）—短暂出现
3. resentment（愤慨）—听到他的措辞后
4. anger（愤怒）—激烈拒绝
5. agitation（动摇）—拒绝后哭了半小时
```

### 第36章：自我认识场面

```
1. astonishment, apprehension, horror（阅读书信时）
2. shame（认识自己）
3. humiliation（自我批判）
4. wretchedness（自身的悲惨）
5. resolution（决意）—“till this moment, I never knew myself”
```

### 第46章：私奔的消息

```
[伊丽莎白]
1. anguish（悲痛）
2. wretchedness（悲惨）
3. self-reproach（自责）
4. love realization（意识到爱）—在最残酷的时刻

[达西]
1. astonishment（惊愕）
2. concern（深切关怀）
3. resolve（决意）—没有说出口
```

### 第58章：再次求婚

```
[双方]
1. embarrassment（尴尬）
2. relief（宽慰）
3. gratitude（感激）
4. happiness（幸福）
5. felicity（至福）
```

## 13. 在翻译中保留细微差别

### “regard” vs. “esteem” vs. “affection” vs. “love”

```
[小说中的达西]
第34章：“ardently admire and love”—爱（强）
第46章：“real, though unavailing concern”—深切关怀（家庭危机）
第58章：“affections and wishes are unchanged”—不变的爱意

[小说中的伊丽莎白]
第44章：“She respected, she esteemed, she was grateful to him”
       → 尊敬、倾慕、感激（爱情前一阶段）
第46章：“could have loved him”
       → 首次使用“love”
第58章：接受爱情
```

### 译法差异

```
原文：“regard”     → “情分”／“心意”（中性）
原文：“esteem”     → “尊敬”／“倾慕”
原文：“affection”  → “爱意”／“温情”
原文：“love”       → “爱”／“爱慕”
原文：“passion”    → “热情”
原文：“ardour”     → “热烈的情感”
```

若全部译成同一个“爱”，细微的情感阶段便会崩塌。

## 15. 翻译的黄金原则

### 1. 保留情感阶段
- “regard → esteem → affection → love”的变化，在译文中也必须呈现为递进阶段。

### 2. 区分人物用词
- 达西的“ardent”≠柯林斯的“ardent”。
- 即使是同一个词，也要符合人物的修养与格调。

### 3. 保持强度一致
- 不要把“rapture”弱化为普通的“喜悦”—应译为“狂喜”。
- 不要把“fury”弱化为“生气”—应译为“暴怒”。

### 4. 保留情感色彩
- 温暖的词要译得温暖。
- 冷淡的词要译得冷淡。
- “esteem”＝温暖的敬重；“respect”＝中性的尊重—必须区别。

### 5. 保持时代格调
- “爱”可以译为“爱慕”／“倾心”。
- “高兴”可以译为“欣慰”／“欢欣”。
