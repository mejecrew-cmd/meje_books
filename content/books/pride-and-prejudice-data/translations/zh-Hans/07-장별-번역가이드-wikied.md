> **图例：**
> - 🔑 = 对整部小说至关重要的关键台词（翻译时须特别谨慎）

## Volume I（Chapters 1～23）

### Chapter 1 — Bingley的消息＋Bennet夫妇的首次对话

| 条目 | 内容 |
|---|---|
| **摘要** | 消息传来：一位富有的年轻人即将入住Netherfield。Mrs Bennet要求丈夫登门拜访，Mr Bennet则以冷嘲回避。 |
| **地点** | Longbourn客厅 |
| **人物** | Mr Bennet、Mrs Bennet（只有两人对话；女儿们被提及） |
| **EVT** | EVT-001 |
| **🔑 关键台词** | “It is a truth universally acknowledged, that a single man in possession of a good fortune must be in want of a wife.” — 开篇奠定全书的反讽：表层与实际意义彼此倒置。 |
| | “You want to tell me, and I have no objection to hearing it.” — Mr Bennet带着讥讽的让步 |
| | “I must throw in a good word for my little Lizzy.” — 偏爱的第一个信号 |
| **⚠️ 误译风险** | “mean understanding” = 才智**贫乏**，不是卑鄙／“fancied herself nervous” = **自称**神经衰弱，并非当下真的紧张／“business of her life” = 她一生的**事业**，带经营隐喻 |
| **📎 参考** | 时代词典（nervous, mean）、语气清单（NUA-001, NUA-002） |

### Chapter 2 — Mr Bennet公开秘密拜访

| 条目 | 内容 |
|---|---|
| **摘要** | Mr Bennet迟迟才透露自己早已拜访Bingley，全家欢呼。 |
| **地点** | Longbourn客厅 |
| **人物** | Bennet全家 |
| **EVT** | EVT-002 |
| **🔑 关键台词** | “I am the tallest.” — Lydia的第一句台词；自信与欠缺判断力的第一个信号 |
| **⚠️ 误译风险** | “stout” = 当时指**健壮／有气派**，不是现代意义的肥胖 |
| **📎 参考** | 时代词典 |

### Chapter 3 — Meryton舞会：“tolerable”

| 条目 | 内容 |
|---|---|
| **摘要** | Meryton集会舞会。Bingley一行五人入场。Darcy起初备受瞩目，却因傲慢举止招致厌恶，并以“tolerable”羞辱Elizabeth。Bingley与Jane跳了两次舞。 |
| **地点** | Meryton集会厅 |
| **人物** | 几乎所有人（Wickham尚未登场） |
| **EVT** | EVT-003 |
| **🔑 关键台词** | “She is tolerable; but not handsome enough to tempt me; and I am in no humour at present to give consequence to young ladies who are slighted by other men.” — 小说的起点句，包含三层傲慢。 |
| **⚠️ 误译风险** | “tolerable” = 还过得去／勉强可以，不是赞美／“to tempt” = 吸引他，预设自己值得别人来吸引／“give consequence” = 当作重要人物对待／“discovered to be proud” = 不是“发现”，而是**被判定**为傲慢，带仓促裁决的反讽／“His character was decided” = 被定下的是**名声**，不是性格 |
| **📎 参考** | 时代词典（tolerable, consequence, character）、语气清单（NUA-005, NUA-006）、反复母题（“tolerable”） |

### Chapter 4 — Jane与Elizabeth评价Bingley／Darcy

| 条目 | 内容 |
|---|---|
| **摘要** | 姐妹回顾舞会。Jane称赞Bingley，对照Bingley与Darcy的性格。叙述者说明Bingley家族背景：商业出身，继承近10万英镑。 |
| **地点** | Longbourn（姐妹房间） |
| **人物** | Jane、Elizabeth；叙述者说明Bingley与Darcy的背景 |
| **⚠️ 误译风险** | “not been of age two years” = 成年（21岁）未满两年，即约23岁／“the liberty of a manor” = 不只是房屋，而是狩猎权等**庄园权利**／“acquired by trade” = 通过商业**赚得**的财产，“trade”带阶级烙印 |
| **📎 参考** | 制度与经济用语（manor, trade）、完整群体清单（trade的含义） |

### Chapter 5 — 定义“Pride vs Vanity”

| 条目 | 内容 |
|---|---|
| **摘要** | Lucas家来访，众人谈论舞会。Charlotte提出务实婚姻论，Mary定义“pride vs vanity”，一个年幼的Lucas幻想饲养猎狐犬。 |
| **地点** | Longbourn（Lucas家来访） |
| **人物** | Elizabeth、Charlotte、Mrs Bennet、Mary、Sir William Lucas |
| **🔑 关键台词** | “I could easily forgive his pride, if he had not mortified mine.” — 首次揭示Elizabeth自己也有pride。 |
| | “Pride relates more to our opinion of ourselves; vanity to what we would have others think of us.” — 对书名核心概念的定义。Mary虽卖弄学问，内容却准确。 |
| **⚠️ 误译风险** | “mortified” = **伤害**了她的自尊，比一般的令其难堪更强 |
| **📎 参考** | 反复母题（pride, vanity）、时代词典（pride vs vanity词组） |

### Chapter 6 — Darcy的“fine eyes”＋在Lucas Lodge拒舞

| 条目 | 内容 |
|---|---|
| **摘要** | Darcy不知不觉被Elizabeth吸引，并承认她有“fine eyes”。Sir William在Lucas Lodge撮合共舞，Elizabeth拒绝。Miss Bingley用“fine eyes”挤兑Darcy。 |
| **地点** | Lucas Lodge |
| **人物** | Darcy、Elizabeth、Miss Bingley、Sir William Lucas、Charlotte |
| **EVT** | EVT-004, EVT-005 |
| **🔑 关键台词** | “Miss Elizabeth Bennet.” — Darcy第一次向Miss Bingley说出Elizabeth姓名的瞬间 |
| | “Mr. Darcy is all politeness.” — Elizabeth的讽刺。表层是礼貌，实际是挖苦。 |
| **⚠️ 误译风险** | 自由间接引语：“Mr. Darcy had at first scarcely allowed her to be pretty”——叙述者用Darcy本人的语气传达其内心。“allowed”指他**允许自己承认**她漂亮。 |
| **📎 参考** | 语气清单（NUA-008）、反复母题（“fine eyes”）、称谓指南（Miss Eliza vs Miss Elizabeth） |

### Chapter 7 — Jane患病＋Elizabeth徒步前往

| 条目 | 内容 |
|---|---|
| **摘要** | 说明Bennet家的财务状况，首次提到Entail。民兵抵达。Miss Bingley邀请Jane晚餐，Mrs Bennet期盼下雨，故意让Jane骑马前往；Jane病倒。Elizabeth徒步三英里去照料她。 |
| **地点** | Longbourn → Netherfield |
| **人物** | Bennet家、Bingley一行 |
| **EVT** | EVT-006, EVT-007 |
| **🔑 关键台词** | Mrs Bennet：“you had better go on horseback, because it seems likely to rain” — 计策的核心句。表面是观察，实际是希望。 |
| **⚠️ 误译风险** | “entailed, in default of heirs male” = **没有男性继承人时**生效的限嗣继承／“apothecary” = 药剂师兼当地医疗人员，不只是普通药师／“petticoat, six inches deep in mud” = 衬裙沾了六英寸深的泥，是违反阶级规范的视觉证据 |
| **📎 参考** | 制度与经济用语（Entail）、时代词典（apothecary）、摄政时期百科（医疗体系）、场景图（SCENE 03） |

### Chapter 8 — 才艺之争（Accomplishment debate）

| 条目 | 内容 |
|---|---|
| **摘要** | Netherfield晚餐。Miss Bingley与Mrs Hurst背后批评Elizabeth。Darcy承认她的“fine eyes”因运动而明亮。众人嘲弄Cheapside的舅舅。Elizabeth、Darcy与Miss Bingley争论何谓“有才艺的女性”。 |
| **地点** | Netherfield餐厅 → 客厅 |
| **人物** | Elizabeth、Darcy、Miss Bingley、Mrs Hurst、Mr Bingley、Mr Hurst |
| **EVT** | EVT-007-001 |
| **🔑 关键台词** | “Not at all; they were brightened by the exercise.” — Darcy简短而坚决的反击，正面击破Miss Bingley对“fine eyes”的攻击。 |
| | “to all this she must yet add something more substantial, in the improvement of her mind by extensive reading.” — Darcy真正的标准在内在心智而非外表。 |
| **⚠️ 误译风险** | “playing high” = 下**高额赌注**／“ragout” = 法式炖菜，属于阶级信号；“plain dish”则是朴素英式菜／“Cheapside” = 不只是地名，更是**阶级鄙视的符码**／“accomplished” = 有教养，并非现代意义的“有成就” |
| **📎 参考** | 物品词典（ragout, loo）、时代词典（accomplished）、语气清单（NUA-010, NUA-011） |

### Chapter 9 — Mrs Bennet在Netherfield失态

| 条目 | 内容 |
|---|---|
| **摘要** | Mrs Bennet、Kitty与Lydia拜访Netherfield。Mrs Bennet炫耀、传闲话并攻击Darcy，令Elizabeth羞愧。随后讨论诗与爱情，Lydia催促Bingley兑现举办舞会的承诺。 |
| **地点** | Netherfield早餐室 → 客厅 |
| **人物** | Mrs Bennet、Lydia、Kitty、Elizabeth、Darcy、Bingley、Miss Bingley |
| **EVT** | EVT-008 |
| **🔑 关键台词** | Darcy：“I have been used to consider poetry as the food of love.”／Elizabeth：“of a fine, stout, healthy love it may. But if it be only a slight, thin sort of inclination, I am convinced that one good sonnet will starve it entirely away.” — 精彩的智性交锋 |
| | Mrs Bennet：“I cannot see that London has any great advantage over the country... I know we dine with four-and-twenty families.” — 建立在无知上的炫耀 |
| **⚠️ 误译风险** | “fancied she had gained a complete victory” = 她**自以为**获胜，“fancied”是信号／“stout” = **强健的**，不是肥胖 |
| **📎 参考** | 语气清单（NUA-012）、称谓指南（Mrs Bennet的说话方式） |

### Chapter 10 — 里尔舞＋“bewitched”＋Miss Bingley的散步计策

| 条目 | 内容 |
|---|---|
| **摘要** | Darcy写信，Miss Bingley不断插话。Bingley与Darcy争论性格。Darcy邀跳里尔舞，Elizabeth拒绝，他却已“bewitched”。Miss Bingley邀请Elizabeth同行以吸引Darcy注意，Darcy分析她们的“两种动机”。 |
| **地点** | Netherfield客厅 |
| **人物** | Darcy、Elizabeth、Miss Bingley、Bingley、Mr Hurst |
| **EVT** | EVT-009 |
| **🔑 关键台词** | “He had never been so bewitched by any woman as he was by her. He really believed that, were it not for the inferiority of her connections, he should be in some danger.” — 叙述者揭露Darcy的内心。“bewitched”（着魔般迷恋）＋“danger”把爱情定义为威胁。 |
| **⚠️ 误译风险** | “bewitched” = **仿佛中了魔法般着迷**，强于普通的吸引／“precipitance” = 急躁，Darcy用以批评Bingley的优柔寡断／“mend a pen” = **削**羽毛笔，不是修理 |
| **📎 参考** | 物品词典（mend a pen, piquet, reel）、场景图（SCENE 04）、语气清单（NUA-008） |

### Chapter 11 — 坦承缺点＋“wilfully to misunderstand”

| 条目 | 内容 |
|---|---|
| **摘要** | Jane开始康复，Bingley细心照料。晚间读书、打牌。Miss Bingley接近Darcy的尝试失败。Elizabeth与Darcy讨论彼此的缺点，Darcy察觉自己兴趣带来的“danger”，转而警戒。 |
| **地点** | Netherfield客厅 |
| **人物** | Darcy、Elizabeth、Bingley、Jane、Miss Bingley |
| **EVT** | EVT-009-001, EVT-009-002 |
| **🔑 关键台词** | “My good opinion once lost is lost for ever.” — 双重伏笔：（1）Wickham事件的基础；（2）Elizabeth后来把这句自白当作**武器**。 |
| | “And yours is wilfully to misunderstand them.” — 精准预言全书主题 |
| **⚠️ 误译风险** | “wilfully” = **故意地**，并非失误，而是选择性误解／“resentful” = 怀恨，Darcy的自我分析 |
| **📎 参考** | 语气清单（NUA-013, NUA-014）、反复母题（pride, “implacable resentment”） |

### Chapter 13 — Collins的信与来访

| 条目 | 内容 |
|---|---|
| **摘要** | 朗读Collins的信：提出和解、奉承Lady Catherine，并献上“olive branch”。Collins于11月18日星期一四点抵达，称赞Longbourn家具，暗含它们将来归自己的意味。 |
| **地点** | Longbourn |
| **人物** | Bennet全家、Mr Collins |
| **EVT** | EVT-011 |
| **🔑 关键台词** | “olive branch” — 和解姿态；Chapter 57中改指**后代／怀孕**。 |
| | “No, my dear; I think not. I have great hopes of finding him quite the reverse.” — Mr Bennet期待Collins恰恰是个蠢人。 |
| **⚠️ 误译风险** | “demean myself” = 当时指**行事／自处**，与现代“贬低自己”正相反／“condescension” = **屈尊施予的恩惠**，并非单纯傲慢／“rectory” = 牧师住宅与相关收入的整体 |
| **📎 参考** | 时代词典（condescension, demean）、制度与经济用语（Entail, Patronage, living）、书信体汇编（L-01） |

### Chapter 14 — Collins长篇大论＋Fordyce《布道集》

| 条目 | 内容 |
|---|---|
| **摘要** | Collins颂扬Lady Catherine，描述Rosings及其800英镑壁炉饰件，并介绍Miss de Bourgh。他朗读Fordyce《布道集》，被Lydia打断后改玩双陆棋。 |
| **地点** | Longbourn餐厅 → 客厅 |
| **人物** | Collins、Mr Bennet、Bennet姐妹、Mrs Bennet |
| **🔑 关键台词** | “They arise chiefly from what is passing at the time” — 回答Mr Bennet关于奉承是即兴还是预先计划的问题；毫无自觉的虚伪臻于完成。 |
| **⚠️ 误译风险** | “Fordyce's Sermons” = 1766年出版的真实书籍，需注释／“protested that he never read novels” = **绝不**读小说，属于牧师的道德姿态 |
| **📎 参考** | 物品词典（Fordyce's, backgammon）、称谓指南（Collins的长篇大论体） |

### Chapter 15 — Collins更换对象＋Wickham登场

| 条目 | 内容 |
|---|---|
| **摘要** | Collins背景：教育不足、自负且卑躬。受Mrs Bennet暗示，把求婚对象从Jane改为Elizabeth。前往Meryton散步时Wickham首次出现，并在街上与Darcy相遇，双方变了脸色。 |
| **地点** | Longbourn → Meryton街道 |
| **人物** | Collins、Mrs Bennet、Elizabeth、Lydia、Kitty、Wickham、Denny、Darcy、Bingley |
| **EVT** | EVT-012, EVT-013 |
| **🔑 关键台词** | “This was his plan of amends—of atonement—for inheriting their father's estate” — 叙述反讽：Collins把求婚自我合理化为**补偿**。 |
| | “Both changed colour, one looked white, the other red.” — Darcy与Wickham相遇时的视觉对照；文本未说明谁白谁红。 |
| **⚠️ 误译风险** | “wanting only regimentals to make him completely charming” = 只**缺少**军服便会魅力十足／“prepossession” = 先入之见 |
| **📎 参考** | 时代词典（wanting）、完整未指名人物清单（#22 “the stranger” = Wickham） |

| **📎 参考** | 制度与经济用语（living, presentation, patronage）、时代词典（living）、物品词典（whist, lottery tickets, port wine）、场景图（SCENE 05）、语气清单（NUA-016, NUA-017） |

### Chapter 18 — Netherfield舞会（前半部高潮）

| 条目 | 内容 |
|---|---|
| **摘要** | Wickham缺席。Elizabeth与Collins跳了屈辱的一舞，与Darcy共舞时就Wickham正面交锋。Miss Bingley警告她，Collins向Darcy自我介绍，Mrs Bennet公开预言，Mary唱歌后被Mr Bennet制止，Bennet家留到最后。 |
| **地点** | Netherfield大宴会厅＋餐厅 |
| **人物** | 除Wickham外几乎所有人 |
| **EVT** | EVT-016, EVT-016-001～004 |

| **📎 参考** | 场景图（SCENE 06）、语气清单（NUA-018, NUA-019）、称谓指南（Miss Bennet指Jane还是对话对象）、物品词典（white soup, red coats） |

### Chapter 19 — Collins求婚

| 条目 | 内容 |
|---|---|
| **摘要** | Collins正式求婚，列出义务、幸福、Lady Catherine建议三项理由。Elizabeth拒绝五次以上，Collins却把拒绝解释为“优雅女性的惯例”。 |
| **地点** | Longbourn早餐室 |
| **人物** | Collins、Elizabeth、Mrs Bennet |
| **EVT** | EVT-017 |
