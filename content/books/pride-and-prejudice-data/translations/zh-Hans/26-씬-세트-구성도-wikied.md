# 《傲慢与偏见》场景与布景构成图

> 从原作提取并整理各关键场景的**空间布局、视觉信息、照明、天气、人物位置、声音与感官细节**。
> 确保翻译中场景描写准确，并为改编时的布景构成提供依据。

---

## SCENE 01：浪搏恩客厅—宾利的消息（第1章）

| 项目 | 原文依据 |
|---|---|
| **空间** | 推测为客厅，只有贝内特夫妇 |
| **时间** | 白天—“one day” |
| **氛围** | 日常夫妻谈话。推测贝内特先生看书或报纸，贝内特太太做针线 |
| **动线** | 贝内特太太单方面说个不停，贝内特先生几乎不动。最后“he left the room, fatigued with the raptures of his wife” |
| **核心视觉** | 贝内特先生面无表情，与贝内特太太的兴奋形成对比 |

## SCENE 02：梅里屯公共舞厅—舞会（第3章）

| 项目 | 原文依据 |
|---|---|
| **空间** | 公共舞厅，一侧跳舞，另一侧交谈与观看 |
| **时间** | 傍晚至夜间，烛光照明 |
| **人数** | 整个邻里数十人。宾利一行五人入场：宾利先生、两个姐妹、姐姐的丈夫、另一名青年 |
| **声音** | 乐队、舞步、谈笑声 |
| **核心布局** | 达西在房间角落踱步，不与任何人跳舞。伊丽莎白有两支舞没有舞伴，只能坐着。宾利来劝达西，伊丽莎白在**足以偷听两人谈话的距离**。 |
| **核心视觉** | 达西的孤立与宾利的活跃。“She is tolerable”时伊丽莎白虽受辱，却以幽默化解的表情 |
| **服装** | 宾利姐妹：“fine women, with an air of decided fashion”。达西：“fine, tall person, handsome features, noble mien” |

## SCENE 03：内瑟菲尔德清晨—伊丽莎白抵达（第7～8章）

| 项目 | 原文依据 |
|---|---|
| **空间** | 内瑟菲尔德早餐室，众人正在用早餐 |
| **时间** | 清晨 |
| **天气** | 前夜暴雨，早晨田野泥泞 |
| **伊丽莎白的状态** | “weary ancles, dirty stockings, and a face glowing with the warmth of exercise”；衬裙沾了六英寸泥，头发凌乱 |
| **反应** | 宾利小姐＋赫斯特太太：轻蔑，背后说她“almost wild”／宾利：赞赏，“remarkably well”／达西：“divided between admiration and doubt”／赫斯特：只想着早餐 |
| **核心视觉** | 满身泥的伊丽莎白容光焕发，与衣着整洁的宾利姐妹冰冷目光形成对比 |

## SCENE 04：内瑟菲尔德夜晚—才艺争论＋里尔舞（第8、10章）

| 项目 | 原文依据 |
|---|---|
| **空间** | 内瑟菲尔德客厅：壁炉、钢琴、牌桌、沙发 |
| **照明** | 晚餐后的烛光 |
| **布局—第8章** | 赫斯特夫妇、宾利小姐、宾利坐在卢牌桌旁。伊丽莎白拒绝参加并读书，随后移到牌桌旁观看，与达西争论女性才艺。 |
| **布局—第10章** | 达西写信，宾利小姐不断插话。宾利与赫斯特玩皮克。伊丽莎白做针线。宾利小姐弹钢琴，又弹苏格兰小调，引出达西的里尔舞邀请。 |
| **声音** | 钢琴、笔尖刮纸、宾利小姐无休止的奉承 |
| **核心视觉** | 宾利小姐察觉达西的目光反复转向伊丽莎白的时刻 |

## SCENE 05：菲利普斯家夜晚—威克姆的陈述（第16章）

| 项目 | 原文依据 |
|---|---|
| **空间** | 菲利普斯家客厅。两张牌桌：柯林斯和成年人玩惠斯特，年轻人抽彩券 |
| **时间** | 夜晚，烛光 |
| **感官** | 菲利普斯姨父的波特酒气味，咖啡与松饼 |
| **核心布局** | 威克姆坐在伊丽莎白身旁。莉迪亚沉迷抽彩，无法独占威克姆，使他能与伊丽莎白长谈。 |
| **氛围** | 厨房暖食的香气＋纸牌声＋威克姆的低语 |
| **核心视觉** | 威克姆英俊的脸；其他军官相比之下像“broad-faced stuffy uncle Philips” |

## SCENE 06：内瑟菲尔德舞会（第18章）

| 项目 | 原文依据 |
|---|---|
| **空间** | 内瑟菲尔德大舞厅，另有夜宵室 |
| **时间** | 傍晚至深夜。白汤、晚餐、纸牌；贝内特一家最后离开 |
| **照明** | 大量烛光，华丽明亮 |
| **人数** | 众多军官“cluster of red coats”＋邻里各家＋宾利一行。威克姆**缺席**。 |
| **核心场面布局** | (1) 柯林斯与伊丽莎白头两支舞：“dances of mortification”，柯林斯走错方向；(2) 达西与伊丽莎白在相对两列中边舞边谈，出现“hauteur”时刻，威廉爵士插话；(3) 夜宵时二人**相对而坐**，达西听见贝内特太太的预言；(4) 玛丽在钢琴前唱歌，众人听着；(5) 等候离场时宾利小姐与赫斯特太太疲惫不堪，宾利与简单独站着交谈。 |
| **核心视觉** | 最拥挤华丽的场面，其中并存达西的孤独与伊丽莎白的羞耻 |

## SCENE 07：浪搏恩早餐室—柯林斯求婚（第19章）

| 项目 | 原文依据 |
|---|---|
| **空间** | 早餐室，早餐刚结束 |
| **人物动线** | 柯林斯请求与贝内特太太“private audience” → 贝内特太太和吉蒂离开 → 只剩伊丽莎白 → 柯林斯长篇大论 → 伊丽莎白试图逃走 → 柯林斯追赶 |
| **伊丽莎白的动作** | 用“incessant employment”隐藏情绪，假装专心做针线 |
| **核心视觉** | 贝内特太太在门外**徘徊偷听**：“dawdled about in the vestibule to watch” |

## SCENE 08：浪搏恩书房—“你母亲……我……”（第20章）

| 项目 | 原文依据 |
|---|---|
| **空间** | 书房，贝内特先生的专属空间，四周是书 |
| **布局** | 贝内特先生从书上抬头，以“calm unconcern”看着妻子 |
| **关键台词位置** | 伊丽莎白被叫进书房，站在父母之间听取裁决 |
| **氛围** | 贝内特先生讥诮幽默主宰的空间，只有在此他才真正是自己 |

## SCENE 09：罗新斯盛宴—初见凯瑟琳夫人（第29章）

| 项目 | 原文依据 |
|---|---|
| **空间** | 罗新斯餐厅 → 客厅。门厅有“fine proportion and finished ornaments” |
| **规模** | “all the servants, and all the articles of plate”；盛大晚餐 |
| **穿过公园** | 从汉斯福德步行约半英里：“every park has its beauty” |
| **布局** | 凯瑟琳夫人居中。柯林斯坐桌尾下席。伊丽莎白在夏洛特与德包尔小姐之间，德包尔小姐一言不发。 |
| **视觉** | 凯瑟琳夫人：“tall, large woman, strongly-marked features”。德包尔小姐：“pale and sickly... insignificant”。詹金森太太在她面前放遮光屏。 |
| **声音** | 凯瑟琳夫人不间断的独白＋柯林斯附和＋威廉爵士敬畏的沉默 |

## SCENE 10：汉斯福德牧师住宅—第一次求婚（第34章）

| 项目 | 原文依据 |
|---|---|
| **空间** | 牧师住宅客厅。伊丽莎白独自一人，其他人去了罗新斯 |
| **时间** | 夜晚，一支蜡烛或最低限度照明 |
| **天气** | 未明说；伊丽莎白因头痛而留下，精神不佳 |
| **达西动线** | 门开后独自进入 → 问候健康 → 坐下又起身，“walked about the room” → “in an agitated manner”走近 → 求婚 → 倚在壁炉架听她拒绝 → “walked with quick steps”穿过房间 → 离开 |
| **伊丽莎白** | 坐着不起身 → 拒绝后“grew more angry every moment” → 达西离开后“sat down and cried for half an hour” |
| **核心视觉** | 黑暗房间中火光照着达西“pale with anger”的脸，与伊丽莎白涨红的双颊相对 |
| **声音** | 只有两人的声音、达西脚步、门关上及前门开启的声音 |

## SCENE 11：罗新斯公园清晨—递信（第35章）

| 项目 | 原文依据 |
|---|---|
| **空间** | 公园木栅栏旁步道 → 入口门 |
| **时间** | 春日清晨 |
| **天气** | “pleasantness of the morning”—晴朗春晨 |
| **动线** | 伊丽莎白从栅栏门看向公园 → 一位绅士从树林走近 → 达西“holding out a letter” → 伊丽莎白“instinctively”接下 → 达西略行一礼，消失在树林中 |
| **核心视觉** | “a look of haughty composure”—达西脸上被克制的伤痛；递信的手；消失在树叶间的背影 |
| **之后** | 伊丽莎白边走边读：“two sheets of letter paper, written quite through, in a very close hand” |

## SCENE 12：初见彭伯里（第43章）

| 项目 | 原文依据 |
|---|---|
| **接近** | 马车经门房进入庄园，穿过“a beautiful wood stretching over a wide extent”，上行半英里 |
| **第一眼** | 山顶隔谷看见**彭伯里宅邸**：“a large, handsome stone building, standing well on rising ground, and backed by a ridge of high woody hills; and in front a stream of some natural importance was swelled into greater, but without any artificial appearance” |
| **核心视觉** | 最大限度保留自然美：“neither formal nor falsely adorned”，与罗新斯炫耀式华丽相反 |
| **内部** | 餐厅：“large, well-proportioned”。家具：“neither gaudy nor uselessly fine—with less of splendour, and more real elegance, than the furniture of Rosings” |
| **画廊** | 大幅肖像。伊丽莎白久久站在达西全身像前：“such a smile over the face, as she remembered to have sometimes seen, when he looked at her” |
| **花园重逢** | 穿过草坪走向河边时，“the owner of it himself suddenly came forward from the road which led behind it to the stables”。二人在二十码外相遇，都深深脸红。 |

## SCENE 13：兰姆顿旅店—私奔消息（第46章）

| 项目 | 原文依据 |
|---|---|
| **空间** | 旅店起居室，伊丽莎白独自读信 |
| **时间** | 早晨，姨父母外出散步 |
| **动线** | 伊丽莎白读信后跳起 → 奔向门口 → 门开，**达西出现** → 达西因她脸色苍白、神态急迫而吃惊 → 她坐下哭泣 → 达西皱眉踱步 → 温声安慰 → 留下“one serious parting look”后离开 |
| **核心视觉** | 紧握信件的颤抖手；达西踱步的影子；“I have not an instant to lose” |
| **声音** | 伊丽莎白啜泣、达西谨慎脚步、关门声 |

## SCENE 14：浪搏恩wilderness—与凯瑟琳夫人对峙（第56章）

| 项目 | 原文依据 |
|---|---|
| **空间** | 浪搏恩花园的人工自然林＋装饰性hermitage |
| **接近** | 凯瑟琳夫人乘四马马车抵达 → 贝内特太太接待 → 将伊丽莎白带到花园 |
| **时间** | 早晨 |
| **布局** | 碎石步道 → 两人在小树林中单独对峙。马车停在门前，女仆在车内等候。 |
| **核心视觉** | 凯瑟琳夫人的威严体格与伊丽莎白坚定的脸；两人之间的树叶阴影 |
| **声音** | 踩碎石声、凯瑟琳夫人的命令语气、反衬平静的鸟鸣 |

## SCENE 15：浪搏恩步道—第二次求婚（第58章）

| 项目 | 原文依据 |
|---|---|
| **空间** | 浪搏恩附近、通往卢卡斯小屋方向的乡间道路 |
| **时间** | 秋日早晨 |
| **动线** | 五人—宾利、简、伊丽莎白、吉蒂、达西—出发 → 宾利与简落后 → 吉蒂在卢卡斯家离开 → **只剩伊丽莎白与达西** |
| **风景** | 秋日乡路、落叶、田野：“They walked on without knowing in what direction” |
| **核心视觉** | 两人并肩走着交谈的侧影。伊丽莎白看不到他的表情：“she could not look, she could listen”，情感只由声音传达。 |
| **氛围** | 与第一次求婚黑暗紧张的室内完全相反：明亮开阔的户外，自由的选择 |
