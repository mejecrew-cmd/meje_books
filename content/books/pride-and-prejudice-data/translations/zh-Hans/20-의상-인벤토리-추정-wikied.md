# 《傲慢与偏见》——服装清单（推测／创作）

> ⚠️ **本文属于推测与创作。** 文中结合小说明确提及的服装片段（红制服、鞋花、阳伞、沾泥的衬裙等）、摄政时期的服装标准以及各人物的经济实力与性格，重构每个人的衣橱，供服装设计师参考。

---

## 摄政时期女性服装（1811—1812年）基础

### 按种类
- **Morning Dress（晨间裙）：** 室内服，朴素、长袖，也可用于接待访客
- **Walking Dress（步行裙）：** 外出服，搭配软帽与长外套
- **Day Dress / Carriage Dress：** 乘马车外出时的正式服装
- **Evening Dress（晚礼服）：** 晚餐服，短袖、低领口
- **Ball Gown（舞会礼服）：** 最华丽的服装，常为白色
- **Riding Habit（骑装）：** 仿男装设计，搭配帽子与马鞭

### 面料
- **Muslin（细棉布）：** 夏季使用，白色居多
- **Silk（丝绸）：** 晚礼服／舞会礼服
- **Wool / Bombazine：** 冬季外套
- **Cambric（细麻布）：** 高级亚麻布，用于衬衣
- **Calico（印花棉布）：** 平日棉布
- **Lace（蕾丝）：** 装饰

## 1. 伊丽莎白·班纳特——服装清单

### 推测衣橱（中层乡绅，活泼好动）

| 种类 | 数量 | 特点 |

| Morning Dress | 4–5件 | 细棉布、印花棉布，朴素 |
| Walking Dress | 2–3件 | **便于活动**，也有耐泥污的深色服装 |
| Pelisse（长外套） | 2件 | 棕色或深绿色 |
| Spencer Jacket | 2件 | 外出用短夹克 |
| Day Dress | 3–4件 | |
| Evening Dress | 2–3件 | 可用于舞会 |
| Ball Gown | 1件 | 白色细棉布，刺绣装饰 |
| Riding Habit | 1件 | 很少骑马 |
| Bonnets | 4–5顶 | 按季节区分 |
| Shawls | 3–4条 | |
| Gloves | 多副 | |
| Stockings | 多双 | |
| Petticoats | 6–7件 | ★ 第8章：泥污高达6英寸 |
| Shoes | 4–5双 | 舞会用缎面鞋＋日常皮鞋 |

### 小说明确提及的服装

| 场景 | 服装 |
|---|---|
| 第8章抵达内瑟菲尔德 | Walking dress＋沾了6英寸泥的petticoat＋dirty stockings |
| 第18章内瑟菲尔德舞会 | Ball Gown（推测为白色细棉布）＋shoe-roses（鞋花） |

### 翻译／改编要点
- **便于户外活动的服装**占比高
- 比宾利小姐少有“fashionable”服装
- 这体现她的性格

## 2. 简·班纳特——服装清单

### 推测衣橱（长女，最美丽的姐妹）

数量与伊丽莎白相近，但存在以下差异：

| 差异 | 说明 |
|---|---|
| 颜色 | 柔和的浅色：白色、淡蓝色 |
| 风格 | 更优雅端庄 |
| 面料 | 高级细棉布比例较高 |
| 外出服 | 便于活动的服装较少 |

### 小说明确提及
| 场景 | 服装 |
|---|---|
| 第7章前往内瑟菲尔德 | 适合骑马的walking dress，但淋了雨 |
| 生病期间 | 睡衣＋罩袍 |
| 第18章舞会 | Ball gown——作为长女最受瞩目 |

## 3. 莉迪亚·班纳特——服装清单

### 推测（最奢侈）

| 种类 | 特点 |
|---|---|

### 结婚服装（第49—51章）

原著明确提及“wedding clothes”，班纳特太太对此十分执着。

```
[班纳特太太订购的全套结婚服装]

- Silver/Pearl Veil
- Pelisse
- New Bonnets（多顶）
- New Gloves

※ 班纳特先生以“no mark of affection”为由拒绝；推测由嘉丁纳舅父承担费用
```

### 婚后（北方）
- 向姐姐们借衣服，或赊账购买

## 4. 玛丽与吉蒂——服装清单

### 玛丽（三女，不太在意外表）
- 只有普通晨间裙／晚礼服
- 以方便读书的室内服为主
- 舞会服装最少

### 吉蒂（四女，模仿莉迪亚）
- 模仿莉迪亚
- 但无法像莉迪亚那样华丽
- **莉迪亚离开后（第61章）：** 在姐妹影响下，衣着变得整洁得体

## 9. 班纳特先生——服装清单

### 推测（乡村乡绅，保守）

| 种类 | 特点 |
|---|---|
| Old-style Coats | 2–3件，略显过时 |
| Breeches | 推测不喜欢新式pantaloons |
| Powdering Gown ★ | 第50章明确提及；给假发扑粉时穿的罩袍——**过时的标志** |
| Nightcap ★ | 第50章明确提及 |

> **翻译要点：** Powdering gown是十八世纪假发时代的遗物。到1810年代，假发已基本消失，但班纳特先生以自嘲口吻提及它，表现出他**知道自己落后于时代**。

## 14. 服装所表现的关系变化

### 达西服装的变化
- 第3章舞会：带有压迫感的正装
- 第43章彭伯里：在家中显得更放松
- 第58章再次求婚：可能采用较明亮的颜色

### 伊丽莎白服装的变化
- 第7章沾泥的衬裙→自由的象征
- 婚后成为彭伯里的女主人→更正式的服装

### 莉迪亚的服装
- 没有变化；婚前婚后同样冲动
