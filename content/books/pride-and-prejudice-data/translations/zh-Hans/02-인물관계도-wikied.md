# 《傲慢与偏见》——人物关系图

## 1. 家族结构

### Bennet家——Longbourn
```
Mr Bennet ─── Mrs Bennet（原姓Gardiner）
    │
    ├── Jane Bennet（长女）       ──→ 与Mr Bingley结婚
    ├── Elizabeth Bennet（次女）  ──→ 与Mr Darcy结婚
    ├── Mary Bennet（三女）
    ├── Kitty／Catherine Bennet（四女）
    └── Lydia Bennet（幼女）      ──→ 与Mr Wickham结婚
```

### Mrs Bennet的亲属
```
Mrs Bennet的弟弟 ─── Mr Gardiner（伦敦商人）
                     │
                     └── Mrs Gardiner（与Derbyshire有渊源）
                          ＋四名子女（两女、两男）

Mrs Bennet的妹妹 ─── Mrs Philips（住在Meryton）
                     │
                     └── Mr Philips（事务律师）
```

### Darcy家——Pemberley
```
已故Mr Darcy ─── Lady Anne Darcy
    │               │
    │               └── 姐妹：Lady Catherine de Bourgh
    │
    ├── Fitzwilliam Darcy       ──→ 与Elizabeth结婚
    └── Georgiana Darcy（16岁）
         ［共同监护人：Darcy＋Colonel Fitzwilliam］
```

### de Bourgh家——Rosings
```
已故Sir Lewis de Bourgh ─── Lady Catherine de Bourgh
                             │
                             └── Miss Anne de Bourgh
                                  ＋Mrs Jenkinson（陪伴者／家庭教师）
```

### Bingley家——Netherfield
```
已故Mr Bingley（父亲，北方商人）
    │
    ├── Charles Bingley       ──→ 与Jane结婚
    ├── Caroline Bingley
    └── Louisa Bingley ─── Mr Hurst
         （Mrs Hurst）
```

### Lucas家——Lucas Lodge
```
Sir William Lucas ─── Lady Lucas
    │
    ├── Charlotte Lucas（27岁） ──→ 与Mr Collins结婚
    ├── Maria Lucas
    └── 年幼的儿子们
```

### Collins
```
Mr Collins（牧师，Mr Bennet的远亲）
    │
    └── 依据限嗣继承制（Entail）成为Longbourn庄园的法定继承人
         Hunsford教区牧师
         接受Lady Catherine的庇护（patronage）
```

## 2. 核心关系演变（按事件顺序）

### Darcy ↔ Elizabeth（小说的中心轴）
```
EVT-003  ［反感开始］Darcy以“tolerable”羞辱她 → Elizabeth产生反感
EVT-004  ［单向吸引］Darcy被她的“fine eyes”吸引（Elizabeth并不知情）
EVT-009  ［bewitched］Darcy完全着迷 → 刻意保持距离
EVT-016  ［对决］舞会共舞＋围绕Wickham正面交锋
EVT-028  ［破裂］第一次求婚／拒绝：“last man in the world”
EVT-029  ［转折］Darcy的信 → 真相揭露
EVT-030  ［自省］“Till this moment, I never knew myself”
EVT-035  ［重新评价］在Pemberley重新认识Darcy的人格
EVT-036  ［重逢］Darcy的态度彻底改变
EVT-039  ［意识到爱］“all love must be vain”——第一次把感情称为爱
EVT-042  ［感谢］Darcy秘密斡旋的事实曝光
EVT-046  ［结合］接受第二次求婚
EVT-048  ［结婚］成为Pemberley的女主人
```

### Bingley ↔ Jane
```
EVT-003  ［初见］在舞会上共舞两次 → 彼此有好感
EVT-006  ［加深］真心担忧生病的Jane
EVT-010  ［离别］Jane回家，Bingley依依不舍
EVT-019  ［分离］Darcy＋Miss Bingley合谋撤往伦敦
EVT-021  ［绝望］Miss Bingley的信打消Jane的希望
EVT-023  ［虚伪暴露］Miss Bingley在伦敦冷待Jane → 友情终结
EVT-037  ［重逢］在Pemberley，Bingley说“自11月26日以后”就没见过Jane，流露思念
EVT-043  ［归来］Darcy撤销阻挠 → 返回Netherfield
EVT-044  ［订婚］再次求婚＋接受
EVT-048  ［结婚］
```

### Wickham ↔ 多名人物
```
Wickham ↔ Darcy：
  - 童年：在同一庄园一起长大（已故Mr Darcy的教子）
  - 主动放弃价值3,000英镑的圣职俸禄 → 放荡 → 再次索要 → 遭拒
  - 企图诱骗Georgiana（在EVT-029中曝光）
  - 因Lydia出逃而最终对决（EVT-038～040）

Wickham ↔ Elizabeth：
  EVT-014  以虚假证词取得信任
  EVT-022  转向Miss King → Elizabeth的幻想开始破灭
  EVT-030  信件揭露真面目 → 信任彻底崩溃
  EVT-033  最后一次见面——直接识破他的虚伪

Wickham ↔ Lydia：
  EVT-038  从Brighton出逃（没有结婚打算）
  EVT-040  Darcy秘密斡旋，促成婚姻
```

### Collins ↔ Bennet家 ↔ Charlotte
```
EVT-011  Collins访问Longbourn（限嗣继承制的人格化）
EVT-012  求婚对象：Jane → 改为Elizabeth
EVT-017  Elizabeth拒绝求婚
EVT-018  Mr Bennet站在Elizabeth一边
EVT-020  Charlotte接受Collins → 与Elizabeth的友情出现裂痕
```

## 3. 阶级／经济关系图

### 财产排名
```
［最高层］
  Mr Darcy           年入10,000英镑＋Pemberley庄园＋伦敦联排住宅
  Lady Catherine     Rosings庄园＋巨额财产

［上层］
  Mr Bingley         继承约100,000英镑（年入4,000～5,000）／尚未购买庄园
  Miss de Bourgh     Rosings继承人（金额不详，极其丰厚）
  Georgiana Darcy    30,000英镑嫁妆

［中层］
  Mr Bennet          年入2,000英镑（限嗣继承对女儿们不利）
  Mr Collins         Hunsford教区收入（相当稳定）

［下层］
  Mr Wickham         身无分文＋负债（由Darcy救助）
  Bennet姐妹每人     母亲去世后约200英镑（4%公债）
```

### 庇护（Patronage）关系
```
Lady Catherine ──［圣职推荐权］──→ Mr Collins
Mr Darcy ──［多个教区的圣职推荐权］──→ 多名牧师
Mr Darcy ──［购买少尉军职］──→ Mr Wickham（秘密斡旋）
```

### 限嗣继承制（Entail）链条
```
Mr Bennet的庄园（Longbourn）
    │
    └──［限嗣继承］──→ Mr Collins（男性继承人）
                         │
                         └──→ Charlotte Lucas Collins（未来女主人）

※ Mrs Bennet及其五个女儿被排除在继承之外
```

## 5. 按情感／关系类型分类

### 爱情／婚姻关系
| 关系 | 性质 | 结果 |
|---|---|---|
| Darcy ↔ Elizabeth | 超越傲慢与偏见的真爱 | 结婚 |
| Bingley ↔ Jane | 纯粹的彼此好感，因外部干预受到考验 | 结婚 |
| Wickham ↔ Lydia | 冲动与轻率，以金钱促成 | 结婚（不幸） |
| Collins ↔ Charlotte | 现实妥协、没有感情的契约 | 结婚（稳定但乏味） |
| Mr Bennet ↔ Mrs Bennet | 年轻时的错觉，早已消失 | 结婚（冷嘲式共存） |

### 敌对／竞争关系
| 关系 | 性质 |
|---|---|
| Miss Bingley → Elizabeth | 围绕Darcy的单向嫉妒与敌意 |
| Miss Bingley → Jane | 表面友情＋暗中挑拨 |
| Lady Catherine → Elizabeth | 阶级蔑视＋维护与Miss de Bourgh的政治婚姻 |
| Darcy ↔ Wickham | 从童年伙伴变为仇敌（Georgiana事件） |

### 友情／信任关系
| 关系 | 性质 |
|---|---|
| Elizabeth ↔ Jane | 最深厚的姐妹情谊，彼此分享秘密 |
| Elizabeth ↔ Charlotte | 因婚姻观冲突产生裂痕，后来部分修复 |
| Elizabeth ↔ Mrs Gardiner | 明智的顾问与情感支持者 |
| Darcy ↔ Bingley | 性格相反，却是最亲近的朋友 |
| Darcy ↔ Colonel Fitzwilliam | 表兄弟，也是Georgiana的共同监护人 |

### 庇护／依附关系
| 关系 | 性质 |
|---|---|
| Lady Catherine → Collins | 以圣职推荐权为基础的绝对庇护与奉承 |
| Darcy → Wickham | 父辈遗产 → 背叛 → 秘密救助 |
| Darcy → Bingley | Darcy占据判断优势的友情（挑拨／撤销） |

## 6. 供译者参考的关系变化核心时间线

人物之间的称呼、语气和态度会随事件改变，翻译时请参考以下时间线：

| 时点 | Darcy → Elizabeth | Elizabeth → Darcy | Bingley → Jane | Wickham → Elizabeth |
|---|---|---|---|---|
| 第1～6章 | 漠不关心 → 暗中被吸引 | 单纯反感 | 好感开始 | 尚未登场 |
| 第7～13章 | bewitched＋克制 | 维持反感 | 真诚担忧 | 尚未登场 |
| 第14～17章 | 保持距离 | 偏见固化（Wickham的证词） | 好感达到顶点 | 迷人的受害者 |
| 第18～23章 | 痛苦的吸引 | 敌意最大化 | 被迫分开 | 转向Miss King |
| 第24～34章 | 克制失败 → 求婚 | “last man in the world” | 分离＋绝望 | 关系消失 |
| 第35～40章 | 通过信件自我辩护 | 意识到偏见＋羞愧 | 无变化 | 彻底幻灭 |
| 第41～45章 | 态度彻底改变 | 尊敬＋感谢＋尚未命名的吸引 | 保持思念 | 鄙视Wickham |
| 第46～54章 | 秘密营救 | 意识到爱＋“all vain” | 归来＋重新接近 | 逃亡者／骗子 |
| 第55～61章 | 再次求婚 | 接受 | 订婚＋结婚 | 家族之耻 |
