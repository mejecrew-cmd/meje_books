# 《傲慢与偏见》建筑平面图（推测／创作）

> ⚠️ **本文档属于推测与创作。** 它综合小说中关于房间、动线、窗户和花园的零散描写，重建合理的平面图，并结合英国乔治亚式／帕拉第奥式建筑标准与作品细节。

```
┌─────────────────────────────────────────────────────┐
```

## 翻译与改编中的用途

| 场景 | 用途 |
|---|---|
| **电视剧／电影美术部门** | 统一设计各座宅邸的房间布局 |
| **舞台改编** | 重现关键场面的空间：第34章求婚、第43章彭伯里画廊、第56章wilderness中的对峙 |
| **补充翻译描写** | 自然解释原文含糊的空间描写时作为参考 |
| **小说改编** | 增加人物动线时确保空间合理性 |
