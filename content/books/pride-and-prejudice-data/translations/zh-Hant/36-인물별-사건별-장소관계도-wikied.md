## 4. 地點敘事意義的變遷

追蹤各地點在敘事中承載的意義，以及它如何隨事件改變。

### 內瑟菲爾德莊園
| 時點 | 敘事意義 |
|---|---|
| 第1章，EVT-001 | **期待之地**——富有青年入住，象徵結婚可能 |
| 第3章，EVT-003 | **社交舞臺**——舞會開啟兩對人物的關係 |
| 第7至12章，EVT-006–010 | **考驗之地**——照料病人的留宿加深關係，也暴露階級衝突 |
| 第18章，EVT-016 | **對決之地**——所有矛盾在舞會爆發 |
| 第21至23章，EVT-019–021 | **缺席的象徵**——賓利離開後的空宅體現簡的失落 |
| 第53至55章，EVT-043–044 | **歸來之地**——賓利返回並再次求婚，恢復希望 |

### 彭伯里
| 時點 | 敘事意義 |
|---|---|
| 第8章，賓利小姐奉承 | **炫耀工具**——賓利小姐為奉承達西而提及的虛榮題材 |
| 第43章，EVT-035 | **人格之鏡**——順應自然美的園林、真正優雅的傢俱與雷諾茲太太證言，反映達西內心 |
| 第43章，EVT-036 | **變化舞臺**——達西徹底改變的態度首次在此展現 |

| 第61章，EVT-048–049 | **結果之地**——伊麗莎白成為女主人的最終歸宿 |
