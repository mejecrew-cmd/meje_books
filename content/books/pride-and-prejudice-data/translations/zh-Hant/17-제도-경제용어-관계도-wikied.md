# 《傲慢與偏見》——制度與經濟術語關係圖

> 本文系統梳理推動小說情節發展的**法律、經濟與社會制度**。
> 若不理解這些術語，就無法在翻譯中準確呈現人物的行為動機與危機的本質。

---

## 1. 婚姻經濟學——為什麼人人都把婚姻視為生死大事

### 核心結構：財產／嫁妝（Fortune）→婚姻財產安排（Settlement）→妻子零用金（Pin-money）

```
[新娘一方]                            [新郎一方]
Fortune（財產／嫁妝）                 Estate / Income（地產／收入）
    │                                      │
    └──→ Marriage Articles（婚姻契約）←──┘
              │
              ├──→ Settlement（婚姻財產安排）
              │      丈夫去世後為遺孀保障的年金
              │
              ├──→ Pin-money（妻子零用金）
              │      妻子不受丈夫干涉即可支配的年度固定款項
              │
              └──→ Jointure（寡婦年金）
                     Settlement的一種具體形式
```

### 小說中的應用

| 人物 | 財產／嫁妝 | 婚姻財產安排 | 翻譯要點 |
|---|---|---|---|
| **班奈特家每位女兒** | 約1,000英鎊（母親去世後，5,000英鎊的五分之一） | — | 投資於「百分之四公債（4 per cents）」，每年利息約40英鎊——遠不足以獨立生活 |
| **莉迪亞—威克姆** | 莉迪亞的1,000英鎊份額＋父親在世期間每年100英鎊 | 達西為威克姆支付額外費用 | 班奈特先生感嘆「這麼少的錢竟然就能解決」；實際上大部分費用由達西承擔 |
| **喬治安娜·達西** | 30,000英鎊 | — | 威克姆引誘喬治安娜的真正原因 |
| **德·包爾小姐** | 羅新斯莊園的全部地產 | — | 與達西的政治聯姻意味著「兩個地產的合併」 |
| **金小姐** | 10,000英鎊（祖父遺產） | — | 威克姆轉而放棄伊莉莎白、追求她的原因 |

### 術語翻譯指南

| 原文 | 推薦譯法 | 絕不能採用的譯法 | 原因 |
|---|---|---|---|
| Fortune | 財產／嫁妝 | 幸運 | 在婚姻語境中必指財產 |
| Settlement | 婚姻財產安排／財產協議 | 定居、一般協議 | 具有法律約束力的財產契約 |
| Pin-money | 妻子零用金／體面開支 | 別針錢 | 婚姻契約明確規定的正式費用 |
| 4 per cents | 百分之四公債 | 百分之四 | 英國政府債券的專有名稱 |
| Portion | 遺產份額／應得份額 | 部分 | 遺產中的法定分配 |

## 2. 限定繼承制（Entail）——小說的核心衝突動力

### 結構

```
[地產所有人：班奈特先生]
    │
    │  受限定繼承制（Entail）約束
    │  女兒無權繼承
    │
    └──→ [法定繼承人：柯林斯先生]
              │
              ├── 班奈特先生去世時 → 繼承整個浪博恩地產
              ├── 班奈特太太＋5個女兒 → 必須遷出
              └── 另一種可能：若生下兒子，本可由父子共同解除限定繼承；但
                         連續生下5個女兒 → 無法解除
```

### 各事件中限定繼承制的敘事功能

| 事件 | 限定繼承制的作用 |
|---|---|
| EVT-011（柯林斯抵達） | 抽象法律化為具體人物——“我要補償我親愛的表妹們” |
| EVT-017（柯林斯求婚） | 伊莉莎白的拒絕＝拒絕限定繼承制提供的所謂「補償」 |
| EVT-020（夏綠蒂接受） | 夏綠蒂接受柯林斯＝確定成為浪博恩未來的女主人，因而激起班奈特太太的敵意 |
| 第50章 | 班奈特先生的悔恨：「我相信會生下兒子，所以沒有儲蓄。」 |

### 翻譯要點
- **“entail”每次都須採用同一譯法**：統一譯為“限定繼承（制）”，不可在“地產繼承限制”“限定繼承法”等表達之間搖擺。
- 班奈特太太說「I cannot bear to hear that mentioned」時，「that」指限定繼承制。譯文也應保留她不願直呼其名、刻意迴避的語氣。

## 3. 教會經濟學——堂區牧師如何維持生計

### 核心結構

```
[贊助人（Patron）]
    │
    ├──[Patronage：聖職推薦權]──→ 任命堂區牧師
    │
    ├──[Presentation of the living：聖職俸祿授職權]──→ 提供特定堂區的收入
    │
    └──[Advowson：聖職任命權的所有權]──→ 附屬於地產的永久權利
    
[堂區牧師（Rector/Vicar）]
    │
    ├── 收入來源1：Tithes（什一稅）——堂區居民繳納的稅款／農產品
    ├── 收入來源2：Glebe land（教會附屬地）——牧師住宅附帶的農地
    └── 收入來源3：Fees（手續費）——洗禮、婚禮、葬禮等

[領受聖職（Taking orders）]
    └── 大學畢業後必須由主教授予聖職，方可成為牧師
```

### 小說中的應用

| 人物／事件 | 教會經濟的作用 |
|---|---|
| **柯林斯** | 憑藉凱瑟琳夫人的贊助（Patronage）獲得漢斯福德堂區牧師職位。他聲稱什一稅必須在「不惹贊助人不快」的範圍內商定，揭示其奉承行為的經濟基礎。 |
| **威克姆—達西衝突的本質** | 老達西在遺囑中許諾把最優厚的聖職俸祿（the best living）留給威克姆。威克姆決定不領受聖職（resolved against taking orders），改收3,000英鎊後揮霍殆盡。三年後，現任牧師（incumbent）去世，他又要求獲得聖職俸祿，遭到拒絕。 |
| **達西的權力** | 擁有多個堂區的聖職推薦權。費茲威廉上校沒有這種權力——這正是夏綠蒂試圖撮合伊莉莎白與達西的隱秘原因。 |

### 術語翻譯指南

| 原文 | 推薦譯法 | 說明 |
|---|---|---|
| Living | 聖職俸祿／堂區收入 | 並非普通的“生活”，而是來自堂區的正式收入 |
| Patronage | 贊助／聖職推薦權 | 貴族任命牧師的權利 |
| Presentation | 聖職俸祿授職／堂區任命 | 將牧師“推薦”到特定堂區的行為 |
| Taking orders | 領受聖職／進入教會 | 由主教正式授予神職的儀式 |
| Incumbent | 現任牧師／現任聖職人員 | 當前負責該堂區的人 |
| Rector | 堂區牧師 | 領取全部什一稅的正式牧師；Vicar只領取其中一部分 |
| Tithes | 什一稅 | 堂區居民繳納的稅款或農產品，是牧師的主要收入來源 |

## 4. 軍隊經濟學——軍官職位如何取得

### 核心結構

```
[民兵（Militia）]                    [正規軍（Regular Army）]
  │                                    │
  ├── 負責國內治安／防衛               ├── 投入實戰，包括海外派遣
  ├── 在各地區輪換駐紮                 ├── 固定駐地
  ├── 軍官職位：相對便宜               ├── 軍官職位：捐官制（Purchase system）
  │                                    │     └── 花錢買賣軍銜
  └── 威克姆私奔前所屬部隊             └── 私奔後所屬部隊，由達西出資購買職位

[軍銜次序（低→高）]
  Ensign（步兵少尉）→ Lieutenant（中尉）→ Captain（上尉）→ Colonel（上校）
  
  ※ Ensigncy（步兵少尉職位）是正規軍最低的軍官等級
  ※ 達西替威克姆購買的正是這個少尉職位
```

### 小說中的應用

| 術語 | 小說語境 | 翻譯要點 |
|---|---|---|
| **Militia regiment** | 冬季駐紮在梅里屯；莉迪亞與吉蒂狂熱追逐的對象；軍官們的紅制服（red coats）是視覺象徵 | 統一譯為「民兵團」 |
| **Commission** | 指軍官職位本身；「accepted a commission」＝成為軍官 | 「任官」或「軍官職位」 |
| **Commission purchased** | 花錢購買軍官職位；達西替威克姆購買步兵少尉職位 | 「購買軍官職位／代付任官費用」 |
| **Ensigncy** | 正規軍最低的軍官等級，也是最便宜的軍官職位 | 「步兵少尉職位」 |
| **Colonel** | 團級指揮官；福斯特上校與費茲威廉上校 | 統一譯為「上校」，但小說中的兩位上校性格截然不同 |

## 5. 社交制度——婚姻市場的規則

### 核心術語

| 原文 | 譯法 | 說明 | 小說語境 |
|---|---|---|---|
| **Assembly** | 地區舞會／公共舞會 | 當地名流聚集跳舞的公共社交活動；誰與誰跳舞會傳遞社會訊號 | 梅里屯舞會（EVT-003）是一切的起點 |
| **Ball** | 正式舞會 | 比Assembly更正式，由私人宅邸主辦 | 內瑟菲爾德舞會（EVT-016） |
| **Coming out** | 社交界初次亮相 | 年輕女兒取得以成年人身分參加舞會的資格 | 凱瑟琳夫人斥責「5個女兒竟然全都進入社交界了？」 |
| **Accomplishment** | 教養／淑女才藝 | 音樂、繪畫、舞蹈、語言等提高婚姻市場價值的技能 | 參見第8章“才藝之爭”（事物詞典 §5） |
| **Special licence** | 特別結婚許可證 | 由坎特伯雷大主教署簽發，可不受時間與地點限制結婚，費用極高 | 班奈特太太陶醉於達西的財富而提出要求 |
| **Banns** | 結婚預告 | 在堂區教堂連續三週公告後結婚，是一般程式 | 小說中直接提及較少，但它是Special licence的對照概念 |

## 6. 制度間關係圖——整體結構

```
━━━━━━━━━━━━━━━━ 地產經濟 ━━━━━━━━━━━━━━━━
│                                              │
│  地產（Estate）──→ 年收入（Income）           │
│      │                                       │
│      ├──[Entail]──→ 僅由男性繼承人繼承        │
│      │                                       │
│      ├──[Manor]──→ 狩獵權、捕魚權等附屬權利  │
│      │                                       │
│      └──[Patronage]──→ 堂區牧師任命權         │
│                                              │
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

         ↓ 年收入直接決定在婚姻市場中的價值

━━━━━━━━━━━━━━━━ 婚姻經濟 ━━━━━━━━━━━━━━━━
│                                              │
│  男方收入（Income）＋女方財產（Fortune）      │
│      │                                       │
│      └──→ Marriage Articles（婚姻契約）       │
│              │                               │
│              ├──→ Settlement（婚姻財產安排）  │
│              ├──→ Pin-money（妻子零用金）     │
│              └──→ Jointure（寡婦年金）        │
│                                              │
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

         ↓ 無法結婚時的替代出路

━━━━━━━━━━━━━━━━ 職業經濟 ━━━━━━━━━━━━━━━━
│                                              │
│  教會（Church）                              │
│    └── Patronage → Living → Tithes           │
│                                              │
│  軍隊（Army）                                │
│    └── Commission purchased → Ensigncy       │
│                                              │
│  法律（Law）                                 │
│    └── Attorney / Solicitor（事務律師）       │
│        （菲利普斯先生、嘉丁納先生的父親）     │
│                                              │
│  商業（Trade）                               │
│    └── 嘉丁納先生——受人敬重，卻遭受階級偏見  │
│                                              │
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 這一結構如何推動情節

| 制度 | 情節動力 |
|---|---|
| **Entail** | 班奈特姊妹急於結婚的原因；柯林斯存在的原因；夏綠蒂結婚的動機。 |
| **Fortune/Settlement** | 威克姆盯上喬治安娜（30,000英鎊）的原因；莉迪亞婚姻的經濟條件；柯林斯以「你的財產太少，不會再有別的機會」侮辱伊莉莎白的依據。 |
| **Patronage/Living** | 達西—威克姆衝突的核心；柯林斯奉承凱瑟琳夫人的經濟基礎。 |
| **Commission/Ensigncy** | 達西“處置”威克姆的方法——替他購買步兵少尉職位並調往北方，實際上等同放逐。 |
| **Trade（商業）** | 賓利家族對財富源自商業的自卑；賓利姊妹因嘉丁納舅父住在齊普賽街而輕視他；但達西欣然接納嘉丁納夫婦，證明了他的轉變。 |

## 7. 小說金額換算參考表
