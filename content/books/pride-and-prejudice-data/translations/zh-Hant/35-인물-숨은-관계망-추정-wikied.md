# 《傲慢與偏見》——人物隱性關係網（推定/創作）

> ⚠️ **本檔案包含推定與創作。** 除作品明確寫出的關係外，還根據時代背景、人物動線與社會合理性，追蹤**受到暗示或被省略的關係**。可用於改編時增加人物立體感。

---

## 1. 作品未明確寫出的家庭關係網

### 貝內特姊妹間的細微同盟
| 同盟 | 強度 | 依據 |
|---|---|---|
| **簡—伊麗莎白** | ★★★★★ | 明確寫出；共享秘密 |
| **凱蒂—莉迪亞** | ★★★★ | 明確寫出；凱蒂追隨莉迪亞 |
| **瑪麗獨自一人** | ★ | 與任何人都沒有深厚同盟；逃入音樂與書本 |
| **伊麗莎白—瑪麗（潛在）** | ★ | 兩人都重視才智，但瑪麗賣弄學問；關係不深 |
| **簡—瑪麗（潛在）** | ★★ | 簡的溫柔可能包容瑪麗 |
| **簡—莉迪亞** | ★ | 幾乎沒有；性格差異太大 |

### 貝內特家內部的資訊流動
```
[共享秘密]
伊麗莎白 ↔ 簡（大多數秘密）
凱蒂 → 莉迪亞（軍官訊息）
莉迪亞 → 貝內特太太（流言）
瑪麗 → 無人（推定獨自寫日記）

[被阻斷的資訊]
- 達西第一次求婚：只有簡得知
- 威克姆真相：只有簡得知，不包括喬治安娜部分
- 達西的秘密斡旋：起初只有伊麗莎白，後來告訴簡
- 貝內特太太總是最後才知道
```

## 2. 僕人—主人關係網（推定）

### 希爾太太 ↔ 貝內特太太
- **相伴二十年以上**，按貝內特夫婦結婚時期推算
- 最常承受貝內特太太神經發作的人
- 貝內特太太可能經常向希爾吐露秘密
- 第49章：“Have you heard the good news?”——親密的稱呼方式

### 雷諾茲太太 ↔ 達西家
- **從達西四歲起便認識他**，第43章明確寫出
- 實際上承擔達西**第二位母親**的角色
- 安妮夫人去世後，可能替代了母親的位置
- 也影響喬治安娜的成長
- **向訪客稱讚達西**出自真心

### 莎拉 ↔ 貝內特姊妹
- **服侍姊妹們的日常生活**
- 舞會準備時決定先幫助誰；第55章優先幫助簡
- 潛在的流言通道

### 彭伯里的園丁 ↔ 達西
- 達西親自參與莊園事務，因此與園丁保持一定關係
- 加德納夫婦在彭伯里時，有園丁陪加德納先生釣魚

## 3. 梅里屯社會網路（推定）

### 流言的傳播
```
朗太太 → 貝內特太太 → 貝內特姊妹 → 梅里屯
       ↓
菲利普斯太太 ← 貝內特太太（姊妹）
       ↓
菲利普斯家的客人（軍官）→ 整個梅里屯

[其他渠道]
盧卡斯夫人 → 貝內特太太（競爭＋資訊）
威廉爵士 → 整個社交界（誇耀“first circles”）
```

### 朗太太的位置
- **梅里屯流言中心**——第1章最先傳來賓利訊息的人
- 與兩個未婚姪女同住，在婚姻市場上與貝內特姊妹競爭
- **達西與她沉默相對半小時**，第5章；進一步加深貝內特太太對他的反感

### 軍官之間的關係
- **威克姆—丹尼：** 親近；一同從倫敦來到梅里屯
- **威克姆—卡特：** 同僚
- **威克姆—福斯特上校：** 下屬與上司。第46章福斯特上校洩露威克姆本想逃走的秘密，說明**威克姆未能取得福斯特信任**
- **威克姆在部隊中很受歡迎：** “captivating”，同時在第50章留下賭債

## 4. 柯林斯的庇護網路

### 凱瑟琳夫人的影響力網路
```
凱瑟琳夫人
├── 柯林斯先生（漢斯福德牧師，絕對依附）
│   └── 夏洛特（透過柯林斯間接依附）
├── 詹金森太太（安妮·德包爾的女伴，受僱）
├── 楊太太（前家庭教師，可能由達西推薦）
├── 波普小姐（由其介紹家庭教師職位）
└── 梅特卡夫夫人等（社交界庇護）

[達西雖是她的外甥，卻不受她支配]
```

### 柯林斯的社會位置
- 同時效忠**上層的凱瑟琳夫人**與**下層的教區居民**
- 對貝內特家感情矛盾：既是親戚，也是他未來將繼承的產業
- 與夏洛特結婚後，也試圖爭取包括威廉爵士在內的岳家支援

## 5. 達西的階級人脈（推定）

### 明確寫出
- 彭伯里附近的紳士
- 倫敦社交界，姓名未公開
- 賓利，最親近的朋友
- 菲茨威廉上校，表兄弟
- 凱瑟琳夫人，姨母

### 推定人脈
- **伯爵，菲茨威廉上校之父**——達西的舅父
  - 達西前往倫敦拜訪的某處
  - 可能是喬治安娜的監護人之一
- **達西的律師**——管理莊園並參與威克姆談判
- **彭伯里附近的醫生**——照料喬治安娜的健康
- **劍橋大學同窗**——賓利也可能是其中之一
- **倫敦俱樂部成員**，如White's、Brooks's等

## 6. 威克姆的黑暗人脈（推定）

### 明確寫出
- 楊太太，喬治安娜事件的共謀者、倫敦寄宿屋經營者
- 丹尼先生，在梅里屯介紹威克姆的人

### 推定
- **倫敦賭場人脈**——債務來源
- **梅里屯與布萊頓的酒友**
- **其他曾試圖誘惑的女性**，除喬治安娜、金小姐和莉迪亞外可能還有別人
- **錢伯萊恩**，莉迪亞在第51章提及的朋友；威克姆曾讓他男扮女裝

## 7. 賓利姊妹的社交野心（推定）

### 卡羅琳的婚姻市場
- **達西是第一選擇**，絕不願放棄
- 對其他人只能勉強考慮
- 進入三十歲後可能陷入恐慌
- 結尾第61章，她向伊麗莎白償還“欠下的禮貌”，以**維持與達西表親等人脈的聯絡**

### 路易莎·赫斯特的婚姻
- 與赫斯特先生的婚姻是“fashion rather than fortune”，第4章
- **實際上是失敗的婚姻**——丈夫只關心吃飯和紙牌
- 她既是卡羅琳的盟友，也是安慰

## 8. 地方社會的婚姻市場（推定）

### 梅里屯未婚女性（貝內特姊妹除外）
- **盧卡斯姊妹，四至五人**，包括瑪麗亞、夏洛特等
- **朗太太的兩個姪女**
- **古爾丁家的小姐們**，住在海伊公園；第50章由貝內特太太提及
- **羅賓遜家的小姐們**
- **韋布家的小姐們**，第29章提及，擅長鋼琴
- **沃森小姐**，第7章卡特上尉前去拜訪

### 未婚男性
- **數十名軍官**
- **羅賓遜先生**，第5章
- **古爾丁先生**，鄰居

### 結婚可能性矩陣
- 賓利：所有母親的目標
- 達西：地位太高且傲慢，因此眾人迴避
- 威克姆：迷人卻貧窮，只有金小姐一類繼承人可能選擇他
- 柯林斯：生活穩定卻可笑，只有夏洛特一類現實主義者會考慮

### 威克姆的真正意圖（推定）

- 目標是清償債務、重新開始
- **若沒有達西的錢，他不會結婚**

## 12. 結局後（第61章）未呈現的關係變化

### 彭伯里日常生活（推定）
- 伊麗莎白成為女主人，與雷諾茲太太合作
- 喬治安娜像姊妹一樣追隨伊麗莎白
- 賓利與簡搬到三十英里以內，經常來訪

### 威克姆—莉迪亞的長期依賴
- 每年向姊姊們索要錢財
- 推定**伊麗莎白用自己的私用年金暗中支援**
- 達西假裝不知情，或知情卻默許

### 凱蒂的變化
- 經常訪問彭伯里與內瑟菲爾德
- “much improved”——社交禮儀有所改善
- 結婚可能性開啟

### 瑪麗的變化
- 姊妹離家後，成為母親唯一的陪伴者
- 可能加入“成為賓利太太的姊姊”的社交活動
- 對外貌的自卑減少

### 貝內特夫婦
- 兩個女兒嫁入富裕家庭，貝內特太太的神經或許安定
- 貝內特先生依然閉居
- 只有瑪麗留在家中

### 凱瑟琳夫人的和解
- 屈服於好奇心
- 訪問彭伯里
- 不得不對伊麗莎白保持正式禮貌
- 但沒有真正的友誼

### 賓利小姐虛偽的交往
- 為了出入彭伯里而表現“彌補性的親切”
- 婚姻市場上的最後一張牌
