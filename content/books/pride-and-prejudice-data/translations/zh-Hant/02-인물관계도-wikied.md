# 《傲慢與偏見》——人物關係圖

## 1. 家族結構

### Bennet家——Longbourn
```
Mr Bennet ─── Mrs Bennet（原姓Gardiner）
    │
    ├── Jane Bennet（長女）       ──→ 與Mr Bingley結婚
    ├── Elizabeth Bennet（次女）  ──→ 與Mr Darcy結婚
    ├── Mary Bennet（三女）
    ├── Kitty／Catherine Bennet（四女）
    └── Lydia Bennet（么女）      ──→ 與Mr Wickham結婚
```

### Mrs Bennet的親屬
```
Mrs Bennet的弟弟 ─── Mr Gardiner（倫敦商人）
                     │
                     └── Mrs Gardiner（與Derbyshire有淵源）
                          ＋四名子女（兩女、兩男）

Mrs Bennet的妹妹 ─── Mrs Philips（住在Meryton）
                     │
                     └── Mr Philips（事務律師）
```

### Darcy家——Pemberley
```
已故Mr Darcy ─── Lady Anne Darcy
    │               │
    │               └── 姊妹：Lady Catherine de Bourgh
    │
    ├── Fitzwilliam Darcy       ──→ 與Elizabeth結婚
    └── Georgiana Darcy（16歲）
         ［共同監護人：Darcy＋Colonel Fitzwilliam］
```

### de Bourgh家——Rosings
```
已故Sir Lewis de Bourgh ─── Lady Catherine de Bourgh
                             │
                             └── Miss Anne de Bourgh
                                  ＋Mrs Jenkinson（陪伴者／家庭教師）
```

### Bingley家——Netherfield
```
已故Mr Bingley（父親，北方商人）
    │
    ├── Charles Bingley       ──→ 與Jane結婚
    ├── Caroline Bingley
    └── Louisa Bingley ─── Mr Hurst
         （Mrs Hurst）
```

### Lucas家——Lucas Lodge
```
Sir William Lucas ─── Lady Lucas
    │
    ├── Charlotte Lucas（27歲） ──→ 與Mr Collins結婚
    ├── Maria Lucas
    └── 年幼的兒子們
```

### Collins
```
Mr Collins（牧師，Mr Bennet的遠親）
    │
    └── 依據限嗣繼承制（Entail）成為Longbourn莊園的法定繼承人
         Hunsford教區牧師
         接受Lady Catherine的庇護（patronage）
```

## 2. 核心關係演變（按事件順序）

### Darcy ↔ Elizabeth（小說的中心軸）
```
EVT-003  ［反感開始］Darcy以「tolerable」羞辱她 → Elizabeth產生反感
EVT-004  ［單向吸引］Darcy被她的「fine eyes」吸引（Elizabeth並不知情）
EVT-009  ［bewitched］Darcy完全著迷 → 刻意保持距離
EVT-016  ［對決］舞會共舞＋圍繞Wickham正面交鋒
EVT-028  ［破裂］第一次求婚／拒絕：「last man in the world」
EVT-029  ［轉折］Darcy的信 → 真相揭露
EVT-030  ［自省］「Till this moment, I never knew myself」
EVT-035  ［重新評價］在Pemberley重新認識Darcy的人格
EVT-036  ［重逢］Darcy的態度徹底改變
EVT-039  ［意識到愛］「all love must be vain」——第一次把感情稱為愛
EVT-042  ［感謝］Darcy秘密斡旋的事實曝光
EVT-046  ［結合］接受第二次求婚
EVT-048  ［結婚］成為Pemberley的女主人
```

### Bingley ↔ Jane
```
EVT-003  ［初見］在舞會上共舞兩次 → 彼此有好感
EVT-006  ［加深］真心擔憂生病的Jane
EVT-010  ［離別］Jane回家，Bingley依依不捨
EVT-019  ［分離］Darcy＋Miss Bingley合謀撤往倫敦
EVT-021  ［絕望］Miss Bingley的信打消Jane的希望
EVT-023  ［虛偽暴露］Miss Bingley在倫敦冷待Jane → 友情終結
EVT-037  ［重逢］在Pemberley，Bingley說「自11月26日以後」就沒見過Jane，流露思念
EVT-043  ［歸來］Darcy撤銷阻撓 → 返回Netherfield
EVT-044  ［訂婚］再次求婚＋接受
EVT-048  ［結婚］
```

### Wickham ↔ 多名人物
```
Wickham ↔ Darcy：
  - 童年：在同一莊園一起長大（已故Mr Darcy的教子）
  - 主動放棄價值3,000英鎊的聖職俸祿 → 放蕩 → 再次索要 → 遭拒
  - 企圖誘騙Georgiana（在EVT-029中曝光）
  - 因Lydia出逃而最終對決（EVT-038～040）

Wickham ↔ Elizabeth：
  EVT-014  以虛假證詞取得信任
  EVT-022  轉向Miss King → Elizabeth的幻想開始破滅
  EVT-030  信件揭露真面目 → 信任徹底崩潰
  EVT-033  最後一次見面——直接識破他的虛偽

Wickham ↔ Lydia：
  EVT-038  從Brighton出逃（沒有結婚打算）
  EVT-040  Darcy秘密斡旋，促成婚姻
```

### Collins ↔ Bennet家 ↔ Charlotte
```
EVT-011  Collins造訪Longbourn（限嗣繼承制的人格化）
EVT-012  求婚對象：Jane → 改為Elizabeth
EVT-017  Elizabeth拒絕求婚
EVT-018  Mr Bennet站在Elizabeth一邊
EVT-020  Charlotte接受Collins → 與Elizabeth的友情出現裂痕
```

## 3. 階級／經濟關係圖

### 財產排名
```
［最高層］
  Mr Darcy           年收入10,000英鎊＋Pemberley莊園＋倫敦連棟住宅
  Lady Catherine     Rosings莊園＋巨額財產

［上層］
  Mr Bingley         繼承約100,000英鎊（年收入4,000～5,000）／尚未購買莊園
  Miss de Bourgh     Rosings繼承人（金額不詳，極其豐厚）
  Georgiana Darcy    30,000英鎊嫁妝

［中層］
  Mr Bennet          年收入2,000英鎊（限嗣繼承對女兒們不利）
  Mr Collins         Hunsford教區收入（相當穩定）

［下層］
  Mr Wickham         身無分文＋負債（由Darcy救助）
  Bennet姊妹每人     母親去世後約200英鎊（4%公債）
```

### 庇護（Patronage）關係
```
Lady Catherine ──［聖職推薦權］──→ Mr Collins
Mr Darcy ──［多個教區的聖職推薦權］──→ 多名牧師
Mr Darcy ──［購買少尉軍職］──→ Mr Wickham（秘密斡旋）
```

### 限嗣繼承制（Entail）鏈條
```
Mr Bennet的莊園（Longbourn）
    │
    └──［限嗣繼承］──→ Mr Collins（男性繼承人）
                         │
                         └──→ Charlotte Lucas Collins（未來女主人）

※ Mrs Bennet及其五個女兒被排除在繼承之外
```

## 5. 按情感／關係類型分類

### 愛情／婚姻關係
| 關係 | 性質 | 結果 |
|---|---|---|
| Darcy ↔ Elizabeth | 超越傲慢與偏見的真愛 | 結婚 |
| Bingley ↔ Jane | 純粹的彼此好感，因外部干預受到考驗 | 結婚 |
| Wickham ↔ Lydia | 衝動與輕率，以金錢促成 | 結婚（不幸） |
| Collins ↔ Charlotte | 現實妥協、沒有感情的契約 | 結婚（穩定但乏味） |
| Mr Bennet ↔ Mrs Bennet | 年輕時的錯覺，早已消失 | 結婚（冷嘲式共存） |

### 敵對／競爭關係
| 關係 | 性質 |
|---|---|
| Miss Bingley → Elizabeth | 圍繞Darcy的單向嫉妒與敵意 |
| Miss Bingley → Jane | 表面友情＋暗中挑撥 |
| Lady Catherine → Elizabeth | 階級蔑視＋維護與Miss de Bourgh的政治婚姻 |
| Darcy ↔ Wickham | 從童年夥伴變為仇敵（Georgiana事件） |

### 友情／信任關係
| 關係 | 性質 |
|---|---|
| Elizabeth ↔ Jane | 最深厚的姊妹情誼，彼此分享秘密 |
| Elizabeth ↔ Charlotte | 因婚姻觀衝突產生裂痕，後來部分修復 |
| Elizabeth ↔ Mrs Gardiner | 明智的顧問與情感支持者 |
| Darcy ↔ Bingley | 性格相反，卻是最親近的朋友 |
| Darcy ↔ Colonel Fitzwilliam | 表兄弟，也是Georgiana的共同監護人 |

### 庇護／依附關係
| 關係 | 性質 |
|---|---|
| Lady Catherine → Collins | 以聖職推薦權為基礎的絕對庇護與奉承 |
| Darcy → Wickham | 父輩遺產 → 背叛 → 秘密救助 |
| Darcy → Bingley | Darcy占據判斷優勢的友情（挑撥／撤銷） |

## 6. 供譯者參考的關係變化核心時間線

人物之間的稱呼、語氣和態度會隨事件改變，翻譯時請參考以下時間線：

| 時點 | Darcy → Elizabeth | Elizabeth → Darcy | Bingley → Jane | Wickham → Elizabeth |
|---|---|---|---|---|
| 第1～6章 | 漠不關心 → 暗中被吸引 | 單純反感 | 好感開始 | 尚未登場 |
| 第7～13章 | bewitched＋克制 | 維持反感 | 真誠擔憂 | 尚未登場 |
| 第14～17章 | 保持距離 | 偏見固化（Wickham的證詞） | 好感達到頂點 | 迷人的受害者 |
| 第18～23章 | 痛苦的吸引 | 敵意最大化 | 被迫分開 | 轉向Miss King |
| 第24～34章 | 克制失敗 → 求婚 | 「last man in the world」 | 分離＋絕望 | 關係消失 |
| 第35～40章 | 透過信件自我辯護 | 意識到偏見＋羞愧 | 無變化 | 徹底幻滅 |
| 第41～45章 | 態度徹底改變 | 尊敬＋感謝＋尚未命名的吸引 | 保持思念 | 鄙視Wickham |
| 第46～54章 | 秘密營救 | 意識到愛＋「all vain」 | 歸來＋重新接近 | 逃亡者／騙子 |
| 第55～61章 | 再次求婚 | 接受 | 訂婚＋結婚 | 家族之恥 |
