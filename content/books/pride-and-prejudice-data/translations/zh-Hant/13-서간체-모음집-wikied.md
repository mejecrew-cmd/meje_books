## 書信文體類型分類

### 類型1：柯林斯書信體（L-01、L-06、L-18、L-19）
```
特點：
- 一個句子長達5—8行
- 分號與插入子句過度嵌套
- “I flatter myself”／“I venture to say”等自負與謙遜的矛盾
- 每一段都提到凱瑟琳夫人
- 反覆使用“olive branch”（意義從和解轉為懷孕）

翻譯原則：
- 絕不可縮短篇幅。長度＝愚蠢
- 使用過度正式的語氣：「鄙人斗膽自許」／「雖屬冒昧，仍敢……」
- 保留同一封信中謙遜與自負交替出現的特點
```

### 類型2：達西書信體（L-10）
```
特點：
- 格調最高的複合句，精密構成讓步子句→條件子句→主要子句
- 情緒克制逐漸崩解（賓利部分＝冷靜 → 威克姆部分＝激動）
- “Be not alarmed, madam”——開頭第一句製造讀者的緊張
- 雖是自我辯護，卻有所節制，不責怪對方
- 以“God bless you”結尾——唯一一次流露情感

翻譯原則：
- 使用與口語明確區分的書信體：「謹望您……」／「現特此說明……」
- 最大限度呈現賓利部分（第一部分）與威克姆部分（第二部分）的語調變化
- 保留“I must now mention a circumstance which I would wish to forget myself”在揭開痛苦前的遲疑
```

### 類型3：賓利小姐書信體（L-04、L-05）
```
特點：
- 誇張的愛意修辭：“dearest friend”、“I depend on you”
- 用柔和表達包裝離間的核心內容
- “dare to entertain the hope”——假裝謙遜，同時把刀刺下
- 既正式又假裝親密的雙重性

翻譯原則：
- 原樣保留誇張的親密：「致我最親愛的朋友」
- 讓虛偽在重讀時顯現——初讀像真心，重讀則顯得工於心計
```

### 類型4：珍的書信體（L-03、L-07—09、L-11—12）
```
特點：
- 平時：平靜、克制、使用淡化表達
- L-11→L-12：從鎮定驟然轉為混亂
- L-12的“I hardly know what I have written”＝承認自己失去控制
- “I take up my pen again to do, what I have just told you I would not”＝剛說不再寫，卻又提筆

翻譯原則：
- 最大限度呈現L-11與L-12的語調差異
- 保留L-12斷裂又接續的呼吸
```

### 類型5：嘉丁納家的書信體（L-13—14、L-17、L-21）
```
嘉丁納先生（L-13—14）：
- 事務性、簡潔，法律與財務用語準確
- 克制情感。即使“I hope it will give you satisfaction”也相當乾澀

嘉丁納太太（L-17）：
- 比丈夫溫暖，也更健談
- 巧妙混合幽默與暗示：“If he had another motive, I am sure it would never disgrace him”
- 按時間順序展開故事的敘事性書信

翻譯原則：
- 嘉丁納先生：「現將……告知如下」（商人實務文體）
- 嘉丁納太太：「我想大概是……」（親切而有教養的女性文體）
```

### 類型6：莉迪亞書信體（L-15、L-16）
```
特點：
- 無視文法，濫用底線（斜體）
- “was obliged to leave off in a violent hurry”——倉促中斷
- 沒有內容卻很長（寄給吉蒂的信「到處都在詞語下方畫線」）
- L-15（私奔信）：“Don’t be alarmed”→馬上接著令人震驚的內容

翻譯原則：
- 原樣保留青少年不負責任的書信語氣
- 用斜體、底線、粗體等**排版標記**保留底線／強調
- 用破折號「——」呈現倉促中斷
```
