# 《傲慢與偏見》——逐章翻譯指南

> 本指南為全書61章逐一整理**摘要、在場人物、地點、時間、關鍵台詞、誤譯風險與參考設定表**。
> 譯者處理任何一章時，只要開啟**這一份文件**，即可取得所需的全部脈絡。
>
> **圖例：**
> - 🔑 = 對整部小說至關重要的關鍵台詞（翻譯時須特別謹慎）
> - ⚠️ = 容易誤譯的表達
> - 📎 = 應參考的設定表

## Volume I（Chapters 1～23）

### Chapter 1 — Bingley的消息＋Bennet夫婦的首次對話

| 條目 | 內容 |
|---|---|
| **摘要** | 消息傳來：一位富有的年輕人即將入住Netherfield。Mrs Bennet要求丈夫登門拜訪，Mr Bennet則以冷嘲迴避。 |
| **地點** | Longbourn客廳 |
| **人物** | Mr Bennet、Mrs Bennet（只有兩人對話；女兒們被提及） |
| **EVT** | EVT-001 |
| **🔑 關鍵台詞** | “It is a truth universally acknowledged, that a single man in possession of a good fortune must be in want of a wife.” — 開篇奠定全書的反諷：表層與實際意義彼此倒置。 |
| | “You want to tell me, and I have no objection to hearing it.” — Mr Bennet帶著譏諷的讓步 |
| | “I must throw in a good word for my little Lizzy.” — 偏愛的第一個訊號 |
| **⚠️ 誤譯風險** | “mean understanding” = 才智**貧乏**，不是卑鄙／“fancied herself nervous” = **自稱**神經衰弱，並非當下真的緊張／“business of her life” = 她一生的**事業**，帶經營隱喻 |
| **📎 參考** | 時代詞典（nervous, mean）、語氣清單（NUA-001, NUA-002） |

### Chapter 2 — Mr Bennet公開祕密拜訪

| 條目 | 內容 |

| **地點** | Longbourn客廳 |

### Chapter 3 — Meryton舞會：“tolerable”

| 條目 | 內容 |
|---|---|
| **摘要** | Meryton集會舞會。Bingley一行五人入場。Darcy起初備受矚目，卻因傲慢舉止招致厭惡，並以“tolerable”羞辱Elizabeth。Bingley與Jane跳了兩次舞。 |
| **地點** | Meryton集會廳 |
| **人物** | 幾乎所有人（Wickham尚未登場） |
| **EVT** | EVT-003 |
| **🔑 關鍵台詞** | “She is tolerable; but not handsome enough to tempt me; and I am in no humour at present to give consequence to young ladies who are slighted by other men.” — 小說的起點句，包含三層傲慢。 |
| **⚠️ 誤譯風險** | “tolerable” = 還過得去／勉強可以，不是讚美／“to tempt” = 吸引他，預設自己值得別人來吸引／“give consequence” = 當作重要人物對待／“discovered to be proud” = 不是「發現」，而是**被判定**為傲慢，帶倉促裁決的反諷／“His character was decided” = 被定下的是**名聲**，不是性格 |
| **📎 參考** | 時代詞典（tolerable, consequence, character）、語氣清單（NUA-005, NUA-006）、反覆母題（“tolerable”） |

### Chapter 4 — Jane與Elizabeth評價Bingley／Darcy

| 條目 | 內容 |
|---|---|
| **摘要** | 姊妹回顧舞會。Jane稱讚Bingley，對照Bingley與Darcy的性格。敘述者說明Bingley家族背景：商業出身，繼承近10萬英鎊。 |
| **地點** | Longbourn（姊妹房間） |
| **人物** | Jane、Elizabeth；敘述者說明Bingley與Darcy的背景 |
| **⚠️ 誤譯風險** | “not been of age two years” = 成年（21歲）未滿兩年，即約23歲／“the liberty of a manor” = 不只是房屋，而是狩獵權等**莊園權利**／“acquired by trade” = 透過商業**賺得**的財產，“trade”帶階級烙印 |
| **📎 參考** | 制度與經濟用語（manor, trade）、完整群體清單（trade的含義） |

### Chapter 5 — 定義“Pride vs Vanity”

| 條目 | 內容 |
|---|---|
| **摘要** | Lucas家來訪，眾人談論舞會。Charlotte提出務實婚姻論，Mary定義“pride vs vanity”，一個年幼的Lucas幻想飼養獵狐犬。 |
| **地點** | Longbourn（Lucas家來訪） |
| **人物** | Elizabeth、Charlotte、Mrs Bennet、Mary、Sir William Lucas |
| **🔑 關鍵台詞** | “I could easily forgive his pride, if he had not mortified mine.” — 首次揭示Elizabeth自己也有pride。 |
| | “Pride relates more to our opinion of ourselves; vanity to what we would have others think of us.” — 對書名核心概念的定義。Mary雖賣弄學問，內容卻準確。 |
| **⚠️ 誤譯風險** | “mortified” = **傷害**了她的自尊，比一般的令其難堪更強 |
| **📎 參考** | 反覆母題（pride, vanity）、時代詞典（pride vs vanity詞組） |

### Chapter 6 — Darcy的“fine eyes”＋在Lucas Lodge拒舞

| 條目 | 內容 |
|---|---|
| **摘要** | Darcy不知不覺被Elizabeth吸引，並承認她有“fine eyes”。Sir William在Lucas Lodge撮合共舞，Elizabeth拒絕。Miss Bingley用“fine eyes”挖苦Darcy。 |
| **地點** | Lucas Lodge |
| **人物** | Darcy、Elizabeth、Miss Bingley、Sir William Lucas、Charlotte |
| **EVT** | EVT-004, EVT-005 |
| **🔑 關鍵台詞** | “Miss Elizabeth Bennet.” — Darcy第一次向Miss Bingley說出Elizabeth姓名的瞬間 |
| | “Mr. Darcy is all politeness.” — Elizabeth的諷刺。表層是禮貌，實際是挖苦。 |
| **⚠️ 誤譯風險** | 自由間接引語：“Mr. Darcy had at first scarcely allowed her to be pretty”——敘述者用Darcy本人的語氣傳達其內心。“allowed”指他**允許自己承認**她漂亮。 |
| **📎 參考** | 語氣清單（NUA-008）、反覆母題（“fine eyes”）、稱謂指南（Miss Eliza vs Miss Elizabeth） |

### Chapter 7 — Jane患病＋Elizabeth徒步前往

| 條目 | 內容 |
|---|---|
| **摘要** | 說明Bennet家的財務狀況，首次提到Entail。民兵抵達。Miss Bingley邀請Jane共進晚餐，Mrs Bennet盼望下雨，故意讓Jane騎馬前往；Jane病倒。Elizabeth徒步三英里去照料她。 |
| **地點** | Longbourn → Netherfield |
| **人物** | Bennet家、Bingley一行 |
| **EVT** | EVT-006, EVT-007 |
| **🔑 關鍵台詞** | Mrs Bennet：“you had better go on horseback, because it seems likely to rain” — 計策的核心句。表面是觀察，實際是希望。 |
| **⚠️ 誤譯風險** | “entailed, in default of heirs male” = **沒有男性繼承人時**生效的限嗣繼承／“apothecary” = 藥劑師兼當地醫療人員，不只是普通藥師／“petticoat, six inches deep in mud” = 襯裙沾了六英吋深的泥，是違反階級規範的視覺證據 |
| **📎 參考** | 制度與經濟用語（Entail）、時代詞典（apothecary）、攝政時期百科（醫療體系）、場景圖（SCENE 03） |

### Chapter 8 — 才藝之爭（Accomplishment debate）

| 條目 | 內容 |
|---|---|
| **摘要** | Netherfield晚餐。Miss Bingley與Mrs Hurst背後批評Elizabeth。Darcy承認她的“fine eyes”因運動而明亮。眾人嘲弄Cheapside的舅舅。Elizabeth、Darcy與Miss Bingley爭論何謂「有才藝的女性」。 |
| **地點** | Netherfield餐廳 → 客廳 |
| **人物** | Elizabeth、Darcy、Miss Bingley、Mrs Hurst、Mr Bingley、Mr Hurst |
| **EVT** | EVT-007-001 |
| **🔑 關鍵台詞** | “Not at all; they were brightened by the exercise.” — Darcy簡短而堅決的反擊，正面擊破Miss Bingley對“fine eyes”的攻擊。 |
| | “to all this she must yet add something more substantial, in the improvement of her mind by extensive reading.” — Darcy真正的標準在內在心智而非外表。 |
| **⚠️ 誤譯風險** | “playing high” = 下**高額賭注**／“ragout” = 法式燉菜，屬於階級訊號；“plain dish”則是樸素英式菜／“Cheapside” = 不只是地名，更是**階級鄙視的符碼**／“accomplished” = 有教養，並非現代意義的「有成就」 |
| **📎 參考** | 物品詞典（ragout, loo）、時代詞典（accomplished）、語氣清單（NUA-010, NUA-011） |

### Chapter 9 — Mrs Bennet在Netherfield失態

| 條目 | 內容 |
|---|---|
| **摘要** | Mrs Bennet、Kitty與Lydia拜訪Netherfield。Mrs Bennet炫耀、傳閒話並攻擊Darcy，令Elizabeth羞愧。隨後討論詩與愛情，Lydia催促Bingley兌現舉辦舞會的承諾。 |
| **地點** | Netherfield早餐室 → 客廳 |
| **人物** | Mrs Bennet、Lydia、Kitty、Elizabeth、Darcy、Bingley、Miss Bingley |
| **EVT** | EVT-008 |
| **🔑 關鍵台詞** | Darcy：“I have been used to consider poetry as the food of love.”／Elizabeth：“of a fine, stout, healthy love it may. But if it be only a slight, thin sort of inclination, I am convinced that one good sonnet will starve it entirely away.” — 精彩的智性交鋒 |
| | Mrs Bennet：“I cannot see that London has any great advantage over the country... I know we dine with four-and-twenty families.” — 建立在無知上的炫耀 |
| **⚠️ 誤譯風險** | “fancied she had gained a complete victory” = 她**自以為**獲勝，“fancied”是訊號／“stout” = **強健的**，不是肥胖 |
| **📎 參考** | 語氣清單（NUA-012）、稱謂指南（Mrs Bennet的說話方式） |

### Chapter 10 — 里爾舞＋“bewitched”＋Miss Bingley的散步計策

| 條目 | 內容 |
|---|---|
| **摘要** | Darcy寫信，Miss Bingley不斷插話。Bingley與Darcy爭論性格。Darcy邀跳里爾舞，Elizabeth拒絕，他卻已“bewitched”。Miss Bingley邀請Elizabeth同行以吸引Darcy注意，Darcy分析她們的「兩種動機」。 |
| **地點** | Netherfield客廳 |
| **人物** | Darcy、Elizabeth、Miss Bingley、Bingley、Mr Hurst |
| **EVT** | EVT-009 |
| **🔑 關鍵台詞** | “He had never been so bewitched by any woman as he was by her. He really believed that, were it not for the inferiority of her connections, he should be in some danger.” — 敘述者揭露Darcy的內心。“bewitched”（著魔般迷戀）＋“danger”把愛情定義為威脅。 |
| **⚠️ 誤譯風險** | “bewitched” = **彷彿中了魔法般著迷**，強於普通的吸引／“precipitance” = 急躁，Darcy用以批評Bingley的優柔寡斷／“mend a pen” = **削**羽毛筆，不是修理 |
| **📎 參考** | 物品詞典（mend a pen, piquet, reel）、場景圖（SCENE 04）、語氣清單（NUA-008） |

### Chapter 11 — 坦承缺點＋“wilfully to misunderstand”

| 條目 | 內容 |
|---|---|
| **摘要** | Jane開始康復，Bingley細心照料。晚間讀書、打牌。Miss Bingley接近Darcy的嘗試失敗。Elizabeth與Darcy討論彼此的缺點，Darcy察覺自己興趣帶來的“danger”，轉而警戒。 |
| **地點** | Netherfield客廳 |
| **人物** | Darcy、Elizabeth、Bingley、Jane、Miss Bingley |
| **EVT** | EVT-009-001, EVT-009-002 |
| **🔑 關鍵台詞** | “My good opinion once lost is lost for ever.” — 雙重伏筆：（1）Wickham事件的基礎；（2）Elizabeth後來把這句自白當作**武器**。 |
| | “And yours is wilfully to misunderstand them.” — 精準預言全書主題 |
| **⚠️ 誤譯風險** | “wilfully” = **故意地**，並非失誤，而是選擇性誤解／“resentful” = 懷恨，Darcy的自我分析 |
| **📎 參考** | 語氣清單（NUA-013, NUA-014）、反覆母題（pride, “implacable resentment”） |

### Chapter 13 — Collins的信與來訪

| 條目 | 內容 |
|---|---|
| **摘要** | 朗讀Collins的信：提出和解、奉承Lady Catherine，並獻上“olive branch”。Collins於11月18日星期一四點抵達，稱讚Longbourn家具，暗含它們將來歸自己的意味。 |
| **地點** | Longbourn |
| **人物** | Bennet全家、Mr Collins |
| **EVT** | EVT-011 |
| **🔑 關鍵台詞** | “olive branch” — 和解姿態；Chapter 57中改指**後代／懷孕**。 |
| | “No, my dear; I think not. I have great hopes of finding him quite the reverse.” — Mr Bennet期待Collins恰恰是個蠢人。 |
| **⚠️ 誤譯風險** | “demean myself” = 當時指**行事／自處**，與現代「貶低自己」正相反／“condescension” = **屈尊施予的恩惠**，並非單純傲慢／“rectory” = 牧師住宅與相關收入的整體 |
| **📎 參考** | 時代詞典（condescension, demean）、制度與經濟用語（Entail, Patronage, living）、書信體彙編（L-01） |

### Chapter 14 — Collins長篇大論＋Fordyce《布道集》

| 條目 | 內容 |
|---|---|
| **摘要** | Collins頌揚Lady Catherine，描述Rosings及其800英鎊壁爐飾件，並介紹Miss de Bourgh。他朗讀Fordyce《布道集》，被Lydia打斷後改玩西洋雙陸棋。 |
| **地點** | Longbourn餐廳 → 客廳 |
| **人物** | Collins、Mr Bennet、Bennet姊妹、Mrs Bennet |
| **🔑 關鍵台詞** | “They arise chiefly from what is passing at the time” — 回答Mr Bennet關於奉承是即興還是預先計畫的問題；毫無自覺的虛偽臻於完成。 |
| **⚠️ 誤譯風險** | “Fordyce's Sermons” = 1766年出版的真實書籍，需加註／“protested that he never read novels” = **絕不**讀小說，屬於牧師的道德姿態 |
| **📎 參考** | 物品詞典（Fordyce's, backgammon）、稱謂指南（Collins的長篇大論體） |

### Chapter 15 — Collins更換對象＋Wickham登場

| 條目 | 內容 |
|---|---|
| **摘要** | Collins背景：教育不足、自負且卑躬。受Mrs Bennet暗示，把求婚對象從Jane改為Elizabeth。前往Meryton散步時Wickham首次出現，並在街上與Darcy相遇，雙方變了臉色。 |
| **地點** | Longbourn → Meryton街道 |
| **人物** | Collins、Mrs Bennet、Elizabeth、Lydia、Kitty、Wickham、Denny、Darcy、Bingley |
| **EVT** | EVT-012, EVT-013 |
| **🔑 關鍵台詞** | “This was his plan of amends—of atonement—for inheriting their father's estate” — 敘述反諷：Collins把求婚自我合理化為**補償**。 |

### Chapter 16 — Wickham的虛假證詞

| **地點** | Philips家客廳 |

| **EVT** | EVT-014, EVT-014-001 |
| **🔑 關鍵台詞** | “I have no right to give my opinion” — 偽裝謙遜的話術：聲稱無權評價，卻完成最嚴厲的譴責。 |
| | “I had not thought Mr. Darcy so bad as this” — Elizabeth的偏見固化之刻 |
| **⚠️ 誤譯風險** | “living” = **聖俸／教區收入**，不是生活；本章出現十餘次／“presentation” = **聖職推薦權**／“port wine” = Uncle Philips身上的氣味與階級訊號 |
| **📎 參考** | 制度與經濟用語（living, presentation, patronage）、時代詞典（living）、物品詞典（whist, lottery tickets, port wine）、場景圖（SCENE 05）、語氣清單（NUA-016, NUA-017） |

### Chapter 18 — Netherfield舞會（前半部高潮）

| 條目 | 內容 |
|---|---|
| **摘要** | Wickham缺席。Elizabeth與Collins跳了屈辱的一舞，與Darcy共舞時就Wickham正面交鋒。Miss Bingley警告她，Collins向Darcy自我介紹，Mrs Bennet公開預言，Mary唱歌後被Mr Bennet制止，Bennet家留到最後。 |
| **地點** | Netherfield大宴會廳＋餐廳 |
| **人物** | 除Wickham外幾乎所有人 |
| **EVT** | EVT-016, EVT-016-001～004 |
| **🔑 關鍵台詞** | “It is your turn to say something now, Mr. Darcy.” — Elizabeth嘲弄交談規則。 |
| | “I remember hearing you once say, Mr. Darcy, that you hardly ever forgave” — 把Chapter 11的自白當作**武器**。 |
