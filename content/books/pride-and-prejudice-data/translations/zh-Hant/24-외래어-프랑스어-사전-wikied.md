# 《傲慢與偏見》外來詞與法語表達辭典

> 收集、分類小說中出現的所有非英語表達，主要為法語，並整理翻譯指南。
> 在攝政時期的英語中，使用法語是教養、階級與精緻品位的標誌。
> 本指南提供音譯與意譯之間的選擇標準。

---

## 1. 使用法語的社會意義

### 時代背景（1810年代）

- 英國正處於**拿破崙戰爭期間**（1803～1815年），但法國文化仍是上流社會的典範。
- 法語是**有教養女性的必修語言**，第8章達西對“accomplished women”的定義也有所暗示。
- 可以推定喬治安娜正在學習法語。
- **義大利語**也透過音樂與歌劇出現。

### 小說中的使用模式

| 使用者 | 頻率 | 含義 |
|---|---|---|
| **敘述者** | 經常 | 格調＋細膩表達 |
| **達西** | 偶爾 | 自然流露的教養 |
| **賓利小姐** | 經常 | 炫耀時髦 |
| **柯林斯** | 不用 | 缺乏教養 |
| **伊麗莎白** | 偶爾 | 自然得體 |
| **貝內特太太** | 不用 | 無知 |

## 3. 從外語使用分析人物

### 達西—自然流露的教養

```
使用模式：
- 正式書信中用詞準確
- 日常交談中也恰到好處
- 不炫耀
- 能使用法語、拉丁語和古典引文

翻譯時：
- 自然的譯文＋適當保留外語詞
- 保持格調
```

### Miss Bingley—炫耀時髦

```
使用模式：
- 經常使用
- 發音準確，但刻意追求效果
- 偏愛“fashionable”的詞

翻譯時：
- 保留外語詞，同時帶出些許矯飾感
- 可以酌情加入“à la mode”一類表達
```

### Mr. Collins—缺乏教養

```
使用模式：
- 幾乎不用外語詞
- 僅用英語也足夠笨拙
- 只在宗教語境中使用拉丁語

翻譯時：
- 少用外語詞，以過度正式的詞彙補足
- 使用相當於“恕我斗膽自許”“雖屬冒昧”的表達
```

### Lydia—無知

```
使用模式：
- 外語表達為零
- 連“Boulanger”一類詞也由母親代為說出

翻譯時：
- 使用生硬發音，或避開外語詞
- 改述為“舞會最後那支舞”
```

### 伊麗莎白—機智＋自我調侃

```
使用模式：
- 以自我調侃的方式使用“éclat”“adieu”等詞
- 用於幽默，而非炫耀

翻譯時：
- 保留外語詞＋略帶距離感的語調
- 讓“永別了！”一類感嘆稍顯誇張
```

## 4. 翻譯決策矩陣

### 音譯與意譯的選擇標準

```
[建議音譯]
- 專有名詞：地點、人名、遊戲名、舞蹈名
- 以斜體強調的詞：hauteur, éclat
- 目標語言中沒有準確對應詞
- 需要保留時代格調

[建議意譯]
- 敘述中的普通詞：per annum → 每年
- 目標語言中有自然的對應詞
- 外來詞會損害可讀性

[混合使用]
- 首次出現時意譯＋括號內音譯
- 第二次起只用音譯
- 或首次出現時以註釋說明
```

| 原詞 | 建議譯法 | 理由 |
|---|---|---|

| ennui | 倦怠／ennui | 可用自然對應詞 |
| tête-à-tête | 兩人私談／tête-à-tête | 首次意譯＋原詞 |
| ragout | 拉古 | 保留菜名 |
| vingt-un | vingt-un（二十一點） | 保留原詞＋數字 |
| piquet | 皮克 | 保留遊戲名 |

## 5. 翻譯中處理外來詞的一致性

### 全書標準

```
[原則1：一致性]
- 同一部作品中，同一個外來詞使用同一種處理方式

[原則2：必要時加註]
- 首次出現時解釋含義與文化暗示
- 此後只用音譯或意譯

[原則3：時代準確性]
- 不使用較晚才進入目標語言的外來詞
- 將現代的“style”換成符合時代的“樣式”／“方式”

[原則4：格調]
- 外來詞是格調的標誌
- 只在自然的位置使用
- 賓利小姐多用，柯林斯少用
```

## 6. 法語之外—拉丁語、義大利語與希臘語

### 拉丁語（學識的標誌）

- per annum（每年）
- post script（附言）
- ad libitum（隨意）
- carte blanche（全權委託）
- modus operandi（運作方式）

這些詞暗示達西與貝內特先生的教養。

### 義大利語（音樂修養的標誌）

- aria（詠歎調）
- duet（二重唱）—已融入英語
- forte / piano（樂譜指示）
- soprano / alto / tenor / bass

第10章賓利小姐的“Italian songs”體現音樂修養。

### 希臘語（神話與哲學）

- olive branch—間接源於希臘的和平象徵
- Hermitage—隱士住處，源自希臘語

## 8. 影視與舞臺中的外來詞發音

### 演員與配音演員指南

```
[法語]
- 模仿英國演員，採用英語化的英式發音
- 過分準確的法語發音會顯得不自然
- 例：“hauteur”採用英式讀法，而非完整法語[oh-TUR]

[拉丁語]
- 採用英國學校式發音：Etonian Latin
- “per annum” = [per AN-um]

[義大利語]
- 音樂術語採用準確的義大利語發音
- 賓利小姐的“Italian songs”應演唱真正的義大利歌曲
```

## 10. 結論：翻譯外來詞的五項原則

1. **斜體仍用斜體**—原文強調的內容，譯文也要強調。
2. **音譯＋註釋通常最佳**—首次出現時說明含義，此後保留該詞。
3. **保留人物之間的頻率差異**—賓利小姐經常使用，柯林斯不用。
4. **只用時代中已有的外來詞**—排除十九世紀以後進入的表達。
5. **保持一致**—同一部作品中，同一個詞使用同一種拼寫。
