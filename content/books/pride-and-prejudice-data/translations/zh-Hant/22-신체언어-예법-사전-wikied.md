# 《傲慢與偏見》肢體語言與禮儀辭典

> 攝政時期身體禮儀、社交規範與動作方式指南，也是影視與舞臺調度的最終基準。
> 一個錯誤的動作，就足以摧毀時代劇的可信度。

---

## 1. 問候禮

### Bow（男性鞠躬禮）

#### 正式程度

| 類型 | 動作 | 用途 |
|---|---|---|
| **輕微點頭禮** | 頭部微微下傾4～5度 | 社會地位相當者之間的日常問候 |
| **頸部鞠躬禮** | 僅將頸部下傾10～15度 | 標準問候 |
| **腰部鞠躬禮** | 從腰部彎身30～45度 | 正式場合；面向貴婦或上位者 |
| **深鞠躬禮** | 彎身60度以上，一隻腳略向後撤 | 最高規格；面見王室或初次見面 |

#### 小說中的例子
- 第18章舞會上，**達西 → 柯林斯**：“slight bow（輕微鞠躬）”—刻意表現居高臨下
- **柯林斯 → 所有人：**過分鄭重的深鞠躬，顯露他的虛偽
- 第15章街頭，**威克姆 → 達西**：“touched his hat（碰了碰帽子）”—幾乎等於不予理會

#### 翻譯說明
- “Bowed”應當被呈現為一種**身體動作**，不能簡化為“打了招呼”。
- 鞠躬的正式程度標示著人物關係。

### Curtsey（女性屈膝禮）

#### 類型

| 類型 | 動作 | 用途 |
|---|---|---|
| **Bob curtsey** | 輕輕屈膝 | 日常 |
| **Half curtsey** | 屈膝更深並低頭 | 標準 |
| **Full court curtsey** | 一隻腳大幅後撤，雙膝深屈 | 宮廷或最高規格場合 |

#### 動作細節
- **雙手：**輕提裙身兩側，或自然垂放
- **視線：**略微向下，禮節越正式，目光越低
- **時長：**保持1～2秒

#### 小說中的例子
- 第30章，**伊麗莎白 → 達西**：“merely courtesied to him, without saying a word（只是向他行了屈膝禮，一言不發）”—**最低限度的屈膝禮＋沉默，表示冷淡**
- **瑪麗亞·盧卡斯 → 凱瑟琳夫人：**緊張地深行屈膝禮

### 握手—**幾乎不用**

- 在攝政時期，握手錶示**親密**，並不像現代社會那樣屬於日常問候。
- 初次見面絕不握手。
- 第61章，賓利小姐與伊麗莎白“even shook hands（甚至握了手）”—**表示讓步**。

## 2. 與人交談時的行為規範

### 初次見面後的正式介紹

```
[規則]
- 把地位較低者介紹給地位較高者
- 把男性介紹給女性，由女性掌握是否交往的選擇權
- 未經介紹便開口交談，屬於無禮

[流程]
1. 介紹人站在兩人之間
2. “Mr. Darcy, may I introduce Miss Elizabeth Bennet?”
   - 先說地位較高者的名字
3. 雙方互相致意：bow + curtsey
4. 隨後可以進行簡短交談

[小說中的例子]
- 第18章，柯林斯自行向達西介紹自己—無禮；達西以輕蔑回應
- 舞會上，司儀可以為賓客進行介紹
```

### 帽子禮儀

| 情形 | 動作 |
|---|---|
| 向女性問候 | **抬帽致意：**觸碰後提起 |
| 進入室內 | **脫帽** |
| 遇見熟人 | “Touched his hat（碰了碰帽子）”—第15章的威克姆與達西 |
| 與女性交談 | 開始時抬帽，之後拿在手中或夾在腋下 |

## 3. 舞會禮儀

### 入場

```
[順序]
1. 在衣帽間寄存外套、帽子和手套
2. 在前廳稍候片刻
3. 向司儀通報來賓：“Mr. and Mrs. Bennet, with their daughters”
4. 進入主舞廳
5. 向男女主人問候
```

### 邀舞流程

```
[男性走近女性]
1. 雙方必須經過正式介紹
2. 上前鞠躬，並說：“May I have the honour of the next dance, Miss Bennet?”
3. 如果女性接受，男性站在她身旁，交談至音樂響起
4. 音樂開始後，兩人一同走入舞列
```

### 拒絕規則

```
[規則]
- 女性若拒絕一名男性，依照慣例，她在這一組舞中就不能再與任何人跳舞
- 得體的理由：“I am engaged（我已有舞伴）”或“I am too tired（我太累了）”
- 明確拒絕是一種社交侮辱

[小說中的例子]
- 第6章：伊麗莎白拒絕達西—“I have not the least intention of dancing”
  = 帶有侮辱性的拒絕，反而令達西著迷
- 第18章：伊麗莎白“without knowing what she did（不知道自己做了什麼）”便接受了達西的邀請
  = 無意識的同意
```

### 舞步

#### Country Dance（最常見的舞蹈）

```
[隊形]
- 男士一列、女士一列，彼此相對而立
- 一對舞伴從佇列末端開始，逐漸沿佇列向前推進

[動作]
1. 面對彼此行禮：bow + curtsey
2. 雙手相接，轉一圈
3. 向前四步，再後退四步
4. Do-si-do—背對背繞行一圈
5. 與佇列中的下一對重複動作
6. 一組舞會透過反覆變換舞式持續30分鐘至1小時
```

#### 牽手方式

```
[正式—兩人]
- 掌心相對，只有指尖輕觸
- 絕不十指相扣
- 戴著手套，皮膚不會直接接觸

[更正式—僅用指尖]
- 只輕輕握住食指和中指
- 用於正式舞會

[小說中的含義]
- 僅觸指尖 = 距離
- 整個掌心接觸 = 親密
- 第18章達西與伊麗莎白共舞時如何牽手，具有視覺意義
```

### Reel（蘇格蘭舞）

- 節奏更快、更活潑
- 由四人或更多人共舞
- 旋轉動作很多
- 第10章達西提議與伊麗莎白共舞的舞種—**活潑的舞蹈令這一邀請帶有挑釁意味**

### Quadrille（四對舞伴的正式舞蹈）

- 四對舞伴排成方形
- 舞步複雜
- 約1815年開始在英國流行；在小說所處年代才剛傳入

### Boulanger（終場舞）

- 舞會的最後一曲
- 所有人圍成圓圈
- 活潑而富有社交性

## 4. 餐桌禮儀

### 座位安排

```
[正式晚宴]

[男主人]                              [女主人]
   ↓                                     ↓
   ━━━━━━━ 餐桌 ━━━━━━━

   女主人身旁 = 首席賓客
   男主人身旁 = 第二位賓客
   餐桌下首 = 地位最低者或親屬

[小說中的例子]
- 第29章，羅新斯：柯林斯坐在桌尾的最低席位
- 第18章舞會的夜宵：伊麗莎白與達西相對而坐—刻意安排的位置
```

### 上菜方式（à la française）

```
[方法]
1. 一次把所有菜餚擺上餐桌
2. 男主人開始切分主菜
3. 每位用餐者從附近的菜盤自行取食
4. 需要遠處的食物時，請鄰座傳遞：
   “Could I trouble you for the salt?”
5. 一餐持續1～2小時
```

### 基本禮儀

| 動作 | 禮儀 |
|---|---|
| 餐巾 | 放在膝上，不塞進衣領 |
| 餐叉 | 英國用法為**左手**持叉 |
| 餐刀 | 右手 |
| 切食物 | 每次只切一口的分量 |
| 口中有食物 | **不要說話** |
| 葡萄酒 | 與鄰座行共飲禮：“I drink to you” |
| 用餐中途離席 | 極其無禮 |

### 甜點之後

```
1. 女主人起身時，所有女性一同起身
2. 女性們退往客廳
3. 男性們留在餐桌旁，飲用波特酒並談論政治
4. 30分鐘至1小時後，男性前往客廳，與女性們共飲茶或咖啡
```

## 5. 客廳禮儀

### 座位安排

```
[標準安排]
- 女主人：茶桌和茶壺旁
- 女性賓客：女主人附近
- 男性：可自由站立或落座
- 老年人：壁爐旁的扶手椅

[活動]
- 飲茶
- 玩牌
- 鋼琴演奏
- 刺繡和縫紉
- 可以閱讀，但會顯得脫離社交
- 在花園散步
```

### 倒茶

```
[女主人的儀式]
1. 有些家庭會先把牛奶倒入小茶杯
2. 倒茶
3. 賓客自行加糖
4. 將茶杯和茶碟遞給賓客

[賓客禮儀]
- 左手託茶碟，右手持茶杯
- 小口啜飲
- 不要發出聲響
```

## 6. 拜訪禮儀

### Morning Call（上午拜訪）

```
[規則]
- 雖稱“morning call”，實際拜訪時間為上午11點至下午3點
- 只停留15～30分鐘；久留屬於無禮
- 在客廳接待訪客，絕不能在臥室或餐廳

[流程]
1. 到達時遞上名片，或由僕人通報本人來訪
2. 被引入客廳
3. 女主人入內，向訪客問候
4. 簡短談論天氣、健康和傳聞
5. 女主人奉茶，訪客接受或婉拒
6. 稍坐片刻後起身告辭

[名片的含義]
- “我曾來訪”—來訪通知
- 折起一角，表示訪客本人到場，而非派僕人送來
- 產生一週內回訪的義務
```

### 小說中的運用
- 第3章：賓利回訪貝內特先生，停留十分鐘—正式
- 第55章：賓利停留一小時—**親密的標誌**
- 第55章：賓利小姐進行短暫而冷淡的回訪—**保持距離**

## 7. 上下馬車

### 馬車禮儀

```
[上車順序]
1. 地位最高的女性先上車
2. 其他女性
3. 男性最後上車，或坐上車伕座／騎馬隨行

[座位]
- 面朝行進方向的前座 = 上座
- 背朝行進方向的後座 = 地位較低者的座位

[下車]
1. 男性先下車
2. 男性伸手協助女性
3. 女性握住他的手邁下車—難得的身體接觸機會
```

### 小說中的含義

- 第43章離開彭伯裡時，“Mr. Darcy handed the ladies into the carriage”—**達西的手**會觸碰伊麗莎白或加德納夫人的手。
- 在身體接觸極少的時代，**協助上下馬車是合乎禮法的親密時刻**。

## 8. 書寫、摺疊與封緘書信

### 寫信姿勢

```
[在書桌前]
1. 將信紙斜放
2. 右手持羽毛筆，靠近墨水瓶
3. 左手壓住紙張
4. 寫滿一張約需20～30分鐘

[姿勢錯誤 = 失禮]
```

### 摺疊書信

```
[傳統方法—不用信封]
1. 在一面書寫，另一面留白
2. 橫向折成三折
3. 再縱向摺疊
4. 在外側寫上地址
5. 用封蠟和印章封緘，如有家族印章則使用家徽

[小說中的例子]
- 第35章達西的信：“two sheets of letter paper, written quite through, in a very close hand”
- 達西破例使用了**信封**：“envelope containing two sheets”
- 這表示秘密性＋考究
```

### 接收書信

```
[禮儀]
- 不要立刻閱讀；稍候片刻以表現情緒剋制
- 私人信件應在獨處時閱讀
- 在家人面前閱讀時，只可簡短檢視
- 出於禮貌，應立即回信
```

### 小說中的運用
- 第21章，賓利小姐的信：簡上樓獨自閱讀—**預感到壞訊息**
- 第46章，簡的信：伊麗莎白立即閱讀—**緊急情況**

## 9. 臥室與私人空間禮儀

### 進入規則

```
[可以來訪]
- 臥室：僅限家人（姊妹、父母）
- 設有書桌的boudoir：親密的女性朋友
- 書房：家人＋親密朋友

[禁止]
- 未婚男女單獨共處一室 = 醜聞
- 異性進入臥室 = 絕對禁止

[小說中的例子]
- 達西＋伊麗莎白單獨留在客廳（第32、34章）= 已經嚴重違反正式禮法
- 達西進入客廳時，賓利小姐、赫斯特太太等人的目光
```

### 更衣

```
[由Lady’s Maid協助]
1. 解開長袍
2. 解開緊身胸衣
3. 更換襯裙
4. 穿上新長袍
5. 整理頭髮
6. 換鞋

[獨自更衣時]
- 姊妹互相幫助（貝內特姊妹）
- “Sarah, never mind Miss Lizzy’s hair”（第55章）—時間不足
```

## 10. 手與手臂的使用

### 挽臂同行（Walking Arm in Arm）

```
[規則]
- 女性站在男性右側
- 女性把手搭在男性手臂上，而不是抓住他的手臂
- 表示親密，但仍在正式禮法範圍內

[小說中]
- 第43章，彭伯裡：加德納夫人–伊麗莎白 → 達西–加德納夫人 → 達西–伊麗莎白
  = 同行組合的變換 = 意味深長的瞬間
```

### 吻手禮—**幾乎不用**

- 英國幾乎沒有吻手禮；這是歐洲大陸的習俗。
- 若由威克姆一類人物嘗試，會顯得**油滑做作**。

### 扇子的使用（女性）

```
[扇子的語言—即使無意識使用，也會傳達含義]
- 開啟並輕輕搖動：平靜
- 快速扇動：緊張／興奮
- 用扇子遮臉：害羞
- 合起扇子：結束談話的訊號

[小說中可能出現的場面]
- 舞會上賓利小姐扇動扇子 = 觀察達西的工具
```

## 11. 目光禮儀

### 注視規則

```
[允許]
- 每次與談話物件對視1～3秒
- 接受女主人／男主人的目光

[禁止]
- 公然凝視，尤其是凝視女性
- 從頭到腳打量他人 = 極其無禮
- 用餐時盯著別人的食物

[小說中的例子]
- 達西經常注視伊麗莎白—伊麗莎白在第10章察覺
- “She hardly knew how to suppose that she could be an object of admiration”
- 無法判斷他的目光究竟無禮，還是出於愛慕
```

### 微笑

```
[正式程度]
- 嘴角微微上揚 = 標準
- 露齒微笑 = 非常親密
- 放聲大笑 = 無禮，尤其對女性而言

[伊麗莎白的獨特之處]
- “lively, playful disposition, which delighted in anything ridiculous”
- 她經常發笑，比一般女性更活潑，因此輕微逾越了社會規範
```

## 12. 距離（Personal Space）

### 標準距離

```
[普通交談]
- 1～1.5米（超過一臂長度）
- 距離更近 = 親密

[正式問候]
- 2米（正式距離）
- 為行鞠躬禮先後退，再上前

[跳舞]
- 只有牽手時靠近
- 舞式之間至少相隔一步

[小說中]
- 達西把椅子移近伊麗莎白（第32章），之後又拉開距離
- 情緒迸發 → 自我剋制
```

## 13. 身體接觸頻率（低）

### 允許的接觸

| 接觸 | 情形 |
|---|---|
| 握手 | 表示親密（少見） |
| 挽臂 | 散步時 |
| 協助上下馬車 | 握手扶下車 |
| 跳舞 | 牽手 |
| 家人 | 姊妹之間擁抱、親吻 |

### 小說中的身體接觸時刻（稀少）

- 達西＋伊麗莎白：**共舞（第18章）**＋**協助上下馬車（第43章）**＋**再次求婚時的散步（第58章）**
- 除此之外幾乎沒有接觸，因此每一次觸碰都具有**情感上的爆發力**。

## 14. 悲傷與憤怒的表達（剋制）

### 禮儀社會中的情緒控制

```
[允許]
- 輕輕嘆息
- 轉過頭去
- 沉默
- 離開房間

[應當剋制]
- 放聲哭泣（僅可獨處時）
- 憤怒地喊叫
- 誇張的手勢

[小說中的違反情形]
- 貝內特太太的神經發作 = 缺乏剋制
- 莉迪亞的興奮 = 違反禮儀
```

### 真正的情緒爆發場面（剋制崩潰）

- 第34章，達西：“walked with quick steps across the room”—憤怒爆發
- 第34章，伊麗莎白：拒絕後“cried for half an hour”—但只在獨處時
- 第46章，伊麗莎白：在達西面前哭泣—危機使她失去自制

## 16. 各人物的特殊肢體語言

### 伊麗莎白—活躍＋略微打破規範

- 步伐輕快
- 偶爾大聲發笑
- 手勢活潑
- 目光直接，帶有挑戰意味

### 達西—剋制＋沉默

- 動作緩慢
- 長時間沉默
- 目光深沉而持久
- 即使說話，也幾乎不動雙手

### 貝內特太太—誇張

- 經常扇扇子
- 大聲嘆息
- 手勢誇張
- “神經”發作時靠在椅背上，用扇子遮住臉

### 貝內特先生—漠然的姿態

- 深深坐進椅子
- 幾乎不把目光從書本上移開
- 有時背對家人的談話

### 莉迪亞—毫無剋制

- 到處奔跑（“flew about the house”）
- 放聲大笑
- 無視禮儀

### 凱瑟琳夫人—威壓

- 姿勢筆直
- 直視對方，施加壓力
- 使用命令式手勢

### 柯林斯—過度

- 頻繁低頭

### 威克姆—迷人的禮儀

- 溫柔微笑
- 站在親密距離，違反慣例
- 長久注視，帶有誘惑意味

## 17. 關鍵場面的肢體語言指南

### 第34章：求婚場面

```
[達西的行動路線]

- 最後快步穿過房間

[伊麗莎白的行動路線]
- 因驚訝而坐著不動
- 不起身，維持原有姿勢
- 即使拒絕時也保持姿勢
- 因憤怒而漲紅臉
- 達西離開後才崩潰
```
