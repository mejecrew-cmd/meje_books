# 《傲慢與偏見》——選單與飲食文化（推測／創作）

> ⚠️ **本文屬於推測與創作。** 文中結合小說明確提及的食物（法式燉肉、鹿肉、溫室水果、白湯、波特酒）、攝政時期的標準飲食以及各家的經濟實力，重構可能出現的實際餐桌配置。

---

## 攝政時期餐食的基本結構

### 正餐（Dinner）的上菜結構

| 次序 | 內容 |
|---|---|
| **First Course** | 湯＋魚＋肉，同時擺上餐桌 |
| **Second Course** | 禽肉＋蔬菜＋酥皮點心，同時擺上 |
| **Dessert** | 水果＋堅果＋小蛋糕，重新佈置餐桌後另行端上 |

> **方式：** “à la française”——所有菜餚同時鋪陳在餐桌上，各人取用離自己較近的菜。

### 各家的水準
- **彭伯里／羅新斯：** 每餐都有完整正式的上菜次序和華麗裝飾
- **內瑟菲爾德：** 採用正式次序，但反映賓利樸素的口味
- **浪博恩：** 平日簡單，待客時講究禮儀
- **漢斯福德：** 節儉，減少上菜輪次
- **威克姆與莉迪亞：** 極為節儉

## 1. 彭伯里（Pemberley）——達西家的正餐（推測選單）

### 平日家庭正餐（達西＋喬治安娜）

```
[First Course]
- White Soup（白湯，以白肉禽類為底）
- Roasted Pheasant（烤野雞，來自莊園狩獵）
- Boiled Salmon with Lobster Sauce（龍蝦醬汁水煮鮭魚）
- Vegetable Side: Asparagus, Cauliflower

[Second Course]
- Roast Beef
- Stewed Mushrooms
- Apple Tart
- Plum Pudding

[Dessert]
- Fruit Pyramid: Grapes, Nectarines, Peaches（★ 溫室水果）
- Walnuts, Almonds
- Madeira Wine
```

### 第45章 嘉丁納夫婦＋伊莉莎白的正餐（推測）

```
[First Course]
- Turtle Soup（正式款待賓客）
- Glazed Ham
- Stewed Trout

[Second Course]
- Roast Capon
- Savoury Pie
- French Beans

[Dessert]
- Grapes, Nectarines, and Peaches ★（小說明確提及）
- Cheese, Biscuits

[Wine]
- Claret（波爾多高階紅葡萄酒）
- Madeira（配甜點）
- Port（男士餐後飲用）
```

### 飲料
- **Tea**（下午飲用，印度產高階茶）
- **Coffee**（正餐後）
- **Hot Chocolate**（早餐時可能飲用）

## 2. 羅新斯（Rosings）——凱瑟琳夫人的正餐

### 第29章首次正餐的推測選單

原著明確寫道：“exceedingly handsome dinner... all the servants, and all the articles of plate”。

```
[First Course]
- Spring Soup with Green Peas
- Salmon with Sauce Hollandaise
- Boiled Turkey with Oyster Sauce
- Side Dishes: Spinach, Carrots, Pickles

[Second Course]
- Roast Lamb（時令為春季）
- Sweetbreads
- Veal Olives
- Lemon Cheesecakes
- Apricot Tart

[Dessert]
- Pyramid of Fruits（溫室水果）
- Italian Cream
- Sweet Biscuits
- Various Sugared Confections
```

### 特點
- **凱瑟琳夫人的炫耀**——柯林斯逐一稱讚每道菜
- **安妮·德·包爾幾乎不吃東西**——詹金森太太監看並勸她嘗別的菜
- **柯林斯與盧卡斯爵士過度稱讚**

## 3. 內瑟菲爾德（Netherfield）——賓利家的正餐

### 第8章伊莉莎白抵達當天的正餐（推測）

原著線索：出現賓利姊妹喜歡的“ragout”。

```
[First Course]
- Soup à la Reine
- Boiled Carp
- Ragoût of Veal（★ 小說明確提及，法式燉小牛肉）
- Glazed Onions

[Second Course]
- Roast Partridge
- Pigeon Pie
- French Beans
- Charlotte Russe（相當於甜點）

[Dessert]
- Fruit, Cheese, Wine

※ 赫斯特先生只喜歡ragout，拒絕樸素菜餚
※ 伊莉莎白偏愛plain dish
```

### 第18章內瑟菲爾德舞會夜宵（Supper）

原著明確提及“white soup”，這是舞會夜宵不可或缺的菜。

```
[Supper Spread——午夜前後]
- White Soup ★（由女管家尼科爾斯製作）
- Cold Ham
- Cold Chicken（小說明確提及：“cold ham and chicken”）
- Trifle（英式甜點）
- Jellies, Creams
- Fruit
- Wine, Negus（葡萄酒＋水＋糖＋香料）
```

## 4. 浪博恩（Longbourn）——班奈特家的正餐

### 平日家庭正餐（推測）

```
[First Course]
- Pea Soup（依季節而定）
- Boiled Mutton with Caper Sauce（羊肉＋刺山柑醬）
- Side: Mashed Potatoes, Greens

[Second Course]
- Roast Chicken
- Apple Pudding

[Dessert]
- Cheese
- Seasonal Fruit（本地產）
- Port Wine（供男士飲用）
```

### 第13章歡迎柯林斯的正餐——正式擺設

原著說這頓飯受到柯林斯“highly admired”。

```
[升級選單]
- White Soup
- Boiled Fish
- Roast Beef + Yorkshire Pudding
- Pigeon Pie
- Mince Pies（柯林斯在11月抵達，因此可能出現）
- Apple Pie

[Dessert]
- Fruit, Cheese
- Madeira, Port
```

> **班奈特太太的自尊：** 第13章中，柯林斯詢問是誰做的飯，她強調“her daughters had nothing to do in the kitchen”——這是家中僱有專業廚師所帶來的自豪。

### 第55章歡迎達西與賓利的正餐

原著明確提及“venison”。

```
[最高階擺設]
[First Course]
- Vermicelli Soup
- Boiled Trout
- Fricassée of Chicken

[Second Course]
- Roast Venison ★（小說明確提及，鹿後腿肉＝haunch）
- Roast Duck
- Asparagus
- French Beans
- Game Pie

[Dessert]
- Apple Tart
- Custards
- Fruit（英國產蘋果、梨）
- Madeira, Port

※ 班奈特太太承諾“two full courses”（第39章）——完整的正式餐食
```

## 5. 菲利普斯家（Mrs. Philips）——梅里屯之夜（第16章）

原著明確提及“coffee and muffin”、“hot supper”以及威克姆喝的酒等。

```
[Hot Supper——打牌後、午夜前]
- Cold Roast Pork
- Tongue
- Cold Meat Pies
- Pickled Vegetables
- Jelly
- Cheese

[Drinks]
- Port Wine ★（與菲利普斯姨父身上的氣味相聯絡）
- Sherry
- Negus（供女士飲用）
- Tea, Coffee, Muffin

[遊戲間歇]
- Almonds, Raisins
- Sweet Biscuits
```

## 10. 各類飲料的社會意義

| 飲料 | 階級／語境 | 小說中 |
|---|---|---|
| **Champagne** | 最上流社會 | 彭伯里、羅新斯 |
| **Claret（Bordeaux）** | 上流社會 | 推測用於彭伯里正餐 |
| **Madeira** | 上流社會的甜點酒 | 所有正式餐食 |
| **Port Wine** | 中層男士 | 與菲利普斯姨父的氣味相聯絡；餐後飲用 |
| **Sherry** | 中層，女士也飲用 | 清淡開胃酒 |
| **Negus** | 溫熱葡萄酒飲料 | 舞會夜宵 |
| **Tea（Indian）** | 各階層普遍飲用，但有等級差異 | 所有家庭；品質體現財富 |
| **Coffee** | 正餐後 | 所有正式餐食 |
| **Beer/Ale** | 下層日常飲料 | 僕役 |
| **Cider** | 鄉村 | 下層 |
| **Hot Chocolate** | 奢侈早餐飲品 | 推測用於彭伯里 |

## 11. 各季食材

### 秋季（小說開篇，9月至11月）
- 狩獵季開始：野雞、鷓鴣、野兔
- 英國產蘋果、梨、李子
- 南瓜、捲心菜

### 冬季（12月至2月）
- 儲存肉類：鹽醃、熏製
- 冬季蔬菜：蕪菁、捲心菜、洋蔥
- 儲存水果：果醬、醃漬品
- **鹿肉（venison）**——狩獵季

### 春季（3月至5月）——漢斯福德時期
- 春季蔬菜：蘆筍、豌豆
- 羔羊肉
- 新鮮雞蛋

### 夏季（6月至8月）——彭伯里時期
- 新鮮蔬菜豐富
- **溫室水果：** 葡萄、桃子、油桃
- 魚類豐富
- 草莓、覆盆子

## 12. 餐桌禮儀（翻譯／改編參考）

| 專案 | 規則 |
|---|---|
| **座次** | 女主人坐在餐桌一端，主賓坐在她右側 |
| **上菜** | 所有菜餚同時擺在桌上，各人取用附近的菜 |
| **女士先離席** | 甜點後女士們前往客廳，男士們留下喝波特酒 |
| **重新會合** | 約一小時後，男士到客廳與女士會合，進入茶點時間 |
| **柯林斯的過度表現** | 每一道菜都稱讚，是有失體面的行為 |
| **赫斯特先生的無禮** | 只關心食物；伊莉莎白說喜歡“plain dish”後便中斷交談 |

## 運用

| 情景 | 運用方式 |
|---|---|
| **影視美術** | 設計餐桌陳設 |
| **翻譯** | 保證食物描寫的時代準確性 |
| **改編** | 透過用餐場景將階級與性格視覺化 |
| **補充世界觀** | 以“他們吃了什麼”的細節增加立體感 |
