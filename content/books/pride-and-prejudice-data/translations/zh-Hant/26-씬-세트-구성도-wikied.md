# 《傲慢與偏見》場景與佈景構成圖

> 從原作提取並整理各關鍵場景的**空間佈局、視覺資訊、照明、天氣、人物位置、聲音與感官細節**。
> 確保翻譯中場景描寫準確，並為改編時的佈景構成提供依據。

---

## SCENE 01：浪搏恩客廳—賓利的訊息（第1章）

| 項目 | 原文依據 |
|---|---|
| **空間** | 推測為客廳，只有貝內特夫婦 |
| **時間** | 白天—“one day” |
| **氛圍** | 日常夫妻談話。推測貝內特先生看書或報紙，貝內特太太做針線 |
| **動線** | 貝內特太太單方面說個不停，貝內特先生幾乎不動。最後“he left the room, fatigued with the raptures of his wife” |
| **核心視覺** | 貝內特先生面無表情，與貝內特太太的興奮形成對比 |

## SCENE 02：梅里屯公共舞廳—舞會（第3章）

| 項目 | 原文依據 |
|---|---|
| **空間** | 公共舞廳，一側跳舞，另一側交談與觀看 |
| **時間** | 傍晚至夜間，燭光照明 |
| **人數** | 整個鄰里數十人。賓利一行五人入場：賓利先生、兩位姊妹、姊姊的丈夫、另一名青年 |
| **聲音** | 樂隊、舞步、談笑聲 |
| **核心佈局** | 達西在房間角落踱步，不與任何人跳舞。伊麗莎白有兩支舞沒有舞伴，只能坐著。賓利來勸達西，伊麗莎白在**足以偷聽兩人談話的距離**。 |
| **核心視覺** | 達西的孤立與賓利的活躍。“She is tolerable”時伊麗莎白雖受辱，卻以幽默化解的表情 |
| **服裝** | 賓利姊妹：“fine women, with an air of decided fashion”。達西：“fine, tall person, handsome features, noble mien” |

## SCENE 03：內瑟菲爾德清晨—伊麗莎白抵達（第7～8章）

| 項目 | 原文依據 |
|---|---|
| **空間** | 內瑟菲爾德早餐室，眾人正在用早餐 |
| **時間** | 清晨 |
| **天氣** | 前夜暴雨，早晨田野泥濘 |
| **伊麗莎白的狀態** | “weary ancles, dirty stockings, and a face glowing with the warmth of exercise”；襯裙沾了六英吋泥，頭髮凌亂 |
| **反應** | 賓利小姐＋赫斯特太太：輕蔑，背後說她“almost wild”／賓利：讚賞，“remarkably well”／達西：“divided between admiration and doubt”／赫斯特：只想著早餐 |
| **核心視覺** | 滿身泥的伊麗莎白容光煥發，與衣著整潔的賓利姊妹冰冷目光形成對比 |

## SCENE 04：內瑟菲爾德夜晚—才藝爭論＋里爾舞（第8、10章）

| 項目 | 原文依據 |
|---|---|
| **空間** | 內瑟菲爾德客廳：壁爐、鋼琴、牌桌、沙發 |
| **照明** | 晚餐後的燭光 |
| **佈局—第8章** | 赫斯特夫婦、賓利小姐、賓利坐在盧牌桌旁。伊麗莎白拒絕參加並讀書，隨後移到牌桌旁觀看，與達西爭論女性才藝。 |
| **佈局—第10章** | 達西寫信，賓利小姐不斷插話。賓利與赫斯特玩皮克。伊麗莎白做針線。賓利小姐彈鋼琴，又彈蘇格蘭小調，引出達西的里爾舞邀請。 |
| **聲音** | 鋼琴、筆尖刮紙、賓利小姐無休止的奉承 |
| **核心視覺** | 賓利小姐察覺達西的目光反覆轉向伊麗莎白的時刻 |

## SCENE 05：菲利普斯家夜晚—威克姆的陳述（第16章）

| 項目 | 原文依據 |
|---|---|
| **空間** | 菲利普斯家客廳。兩張牌桌：柯林斯和成年人玩惠斯特，年輕人抽彩券 |
| **時間** | 夜晚，燭光 |
| **感官** | 菲利普斯姨父的波特酒氣味，咖啡與鬆餅 |
| **核心佈局** | 威克姆坐在伊麗莎白身旁。莉迪亞沉迷抽彩，無法獨佔威克姆，使他能與伊麗莎白長談。 |
| **氛圍** | 廚房暖食的香氣＋紙牌聲＋威克姆的低語 |
| **核心視覺** | 威克姆英俊的臉；其他軍官相比之下像“broad-faced stuffy uncle Philips” |

## SCENE 06：內瑟菲爾德舞會（第18章）

| 項目 | 原文依據 |
|---|---|
| **空間** | 內瑟菲爾德大舞廳，另有夜宵室 |
| **時間** | 傍晚至深夜。白湯、晚餐、紙牌；貝內特一家最後離開 |
| **照明** | 大量燭光，華麗明亮 |
| **人數** | 眾多軍官“cluster of red coats”＋鄰里各家＋賓利一行。威克姆**缺席**。 |
| **核心場面佈局** | (1) 柯林斯與伊麗莎白頭兩支舞：“dances of mortification”，柯林斯走錯方向；(2) 達西與伊麗莎白在相對兩列中邊舞邊談，出現“hauteur”時刻，威廉爵士插話；(3) 夜宵時二人**相對而坐**，達西聽見貝內特太太的預言；(4) 瑪麗在鋼琴前唱歌，眾人聽著；(5) 等候離場時賓利小姐與赫斯特太太疲憊不堪，賓利與簡兩人單獨站著交談。 |
| **核心視覺** | 最擁擠華麗的場面，其中並存達西的孤獨與伊麗莎白的羞恥 |

## SCENE 07：浪搏恩早餐室—柯林斯求婚（第19章）

| 項目 | 原文依據 |
|---|---|
| **空間** | 早餐室，早餐剛結束 |
| **人物動線** | 柯林斯請求與貝內特太太“private audience” → 貝內特太太和吉蒂離開 → 只剩伊麗莎白 → 柯林斯長篇大論 → 伊麗莎白試圖逃走 → 柯林斯追趕 |
| **伊麗莎白的動作** | 用“incessant employment”隱藏情緒，假裝專心做針線 |
| **核心視覺** | 貝內特太太在門外**徘徊偷聽**：“dawdled about in the vestibule to watch” |

## SCENE 08：浪搏恩書房—“你母親……我……”（第20章）

| 項目 | 原文依據 |
|---|---|
| **空間** | 書房，貝內特先生的專屬空間，四周是書 |
| **佈局** | 貝內特先生從書上抬頭，以“calm unconcern”看著妻子 |
| **關鍵臺詞位置** | 伊麗莎白被叫進書房，站在父母之間聽取裁決 |

## SCENE 09：羅新斯盛宴—初見凱瑟琳夫人（第29章）

| 項目 | 原文依據 |
|---|---|
| **空間** | 羅新斯餐廳 → 客廳。門廳有“fine proportion and finished ornaments” |
| **規模** | “all the servants, and all the articles of plate”；盛大晚餐 |
| **穿過公園** | 從漢斯福德步行約半英里：“every park has its beauty” |
| **佈局** | 凱瑟琳夫人居中。柯林斯坐桌尾下席。伊麗莎白在夏洛特與德包爾小姐之間，德包爾小姐一言不發。 |
| **視覺** | 凱瑟琳夫人：“tall, large woman, strongly-marked features”。德包爾小姐：“pale and sickly... insignificant”。詹金森太太在她面前放遮光屏。 |
| **聲音** | 凱瑟琳夫人不間斷的獨白＋柯林斯附和＋威廉爵士敬畏的沉默 |

## SCENE 10：漢斯福德牧師住宅—第一次求婚（第34章）

| 項目 | 原文依據 |
|---|---|
| **空間** | 牧師住宅客廳。伊麗莎白獨自一人，其他人去了羅新斯 |
| **時間** | 夜晚，一支蠟燭或最低限度照明 |
| **天氣** | 未明說；伊麗莎白因頭痛而留下，精神不佳 |
| **達西動線** | 門開後獨自進入 → 問候健康 → 坐下又起身，“walked about the room” → “in an agitated manner”走近 → 求婚 → 倚在壁爐架聽她拒絕 → “walked with quick steps”穿過房間 → 離開 |
| **伊麗莎白** | 坐著不起身 → 拒絕後“grew more angry every moment” → 達西離開後“sat down and cried for half an hour” |
| **核心視覺** | 黑暗房間中火光照著達西“pale with anger”的臉，與伊麗莎白漲紅的雙頰相對 |
| **聲音** | 只有兩人的聲音、達西腳步、門關上及前門開啟的聲音 |

## SCENE 11：羅新斯公園清晨—遞信（第35章）

| 項目 | 原文依據 |
|---|---|
| **空間** | 公園木柵欄旁步道 → 入口門 |
| **時間** | 春日清晨 |
| **天氣** | “pleasantness of the morning”—晴朗春晨 |
| **動線** | 伊麗莎白從柵欄門看向公園 → 一位紳士從樹林走近 → 達西“holding out a letter” → 伊麗莎白“instinctively”接下 → 達西略行一禮，消失在樹林中 |
| **核心視覺** | “a look of haughty composure”—達西臉上被剋制的傷痛；遞信的手；消失在樹葉間的背影 |
| **之後** | 伊麗莎白邊走邊讀：“two sheets of letter paper, written quite through, in a very close hand” |

## SCENE 12：初見彭伯里（第43章）

| 專案 | 原文依據 |
|---|---|
| **接近** | 馬車經門房進入莊園，穿過“a beautiful wood stretching over a wide extent”，上行半英里 |
| **第一眼** | 山頂隔谷看見**彭伯里宅邸**：“a large, handsome stone building, standing well on rising ground, and backed by a ridge of high woody hills; and in front a stream of some natural importance was swelled into greater, but without any artificial appearance” |
| **核心視覺** | 最大限度保留自然美：“neither formal nor falsely adorned”，與羅新斯炫耀式華麗相反 |
| **內部** | 餐廳：“large, well-proportioned”。傢俱：“neither gaudy nor uselessly fine—with less of splendour, and more real elegance, than the furniture of Rosings” |
| **畫廊** | 大幅肖像。伊麗莎白久久站在達西全身像前：“such a smile over the face, as she remembered to have sometimes seen, when he looked at her” |
| **花園重逢** | 穿過草坪走向河邊時，“the owner of it himself suddenly came forward from the road which led behind it to the stables”。二人在二十碼外相遇，都深深臉紅。 |

## SCENE 13：蘭姆頓旅店—私奔訊息（第46章）

| 項目 | 原文依據 |
|---|---|
| **空間** | 旅店起居室，伊麗莎白獨自讀信 |
| **時間** | 早晨，姨父母外出散步 |
| **動線** | 伊麗莎白讀信後跳起 → 奔向門口 → 門開，**達西出現** → 達西因她臉色蒼白、神態急迫而吃驚 → 她坐下哭泣 → 達西皺眉踱步 → 溫聲安慰 → 留下“one serious parting look”後離開 |
| **核心視覺** | 緊握信件的顫抖手；達西踱步的影子；“I have not an instant to lose” |
| **聲音** | 伊麗莎白啜泣、達西謹慎腳步、關門聲 |

## SCENE 14：浪搏恩荒園—與凱瑟琳夫人對峙（第56章）

| 項目 | 原文依據 |
|---|---|
| **空間** | 浪搏恩花園的人工自然林（wilderness）＋裝飾性隱居亭（hermitage） |
| **接近** | 凱瑟琳夫人乘四馬馬車抵達 → 貝內特太太接待 → 將伊麗莎白帶到花園 |
| **時間** | 早晨 |
| **佈局** | 碎石步道 → 兩人在小樹林中單獨對峙。馬車停在門前，女僕在車內等候。 |
| **核心視覺** | 凱瑟琳夫人的威嚴體格與伊麗莎白堅定的臉；兩人之間的樹葉陰影 |
| **聲音** | 踩碎石聲、凱瑟琳夫人的命令語氣、反襯平靜的鳥鳴 |

## SCENE 15：浪搏恩步道—第二次求婚（第58章）

| 項目 | 原文依據 |
|---|---|
| **空間** | 浪搏恩附近、通往盧卡斯小屋方向的鄉間道路 |
| **時間** | 秋日早晨 |
| **動線** | 五人—賓利、簡、伊麗莎白、吉蒂、達西—出發 → 賓利與簡落後 → 吉蒂在盧卡斯家離開 → **只剩伊麗莎白與達西** |
| **風景** | 秋日鄉路、落葉、田野：“They walked on without knowing in what direction” |
| **核心視覺** | 兩人並肩走著交談的側影。伊麗莎白看不到他的表情：“she could not look, she could listen”，情感只由聲音傳達。 |
| **氛圍** | 與第一次求婚黑暗緊張的室內完全相反：明亮開闊的戶外，自由的選擇 |
