# 《傲慢與偏見》——改編與劇本創作指南

> ⚠️ **本文件用於輔助創作。** 它整理了按媒介、時代與視角改編時，哪些元素可以如何改變，以及哪些元素絕不能觸碰。

---

## 1. 絕對不可改變：小說的精髓

### 核心骨架——改變即失去作品身分

| 元素 | 理由 |
|---|---|
| **第一次求婚 → 拒絕 → 信件 → 自我認識**的序列 | 小說的結構脊柱 |
| **達西兩次求婚的結構** | 改變是核心主題 |
| **威克姆的謊言 → 揭露** | 偏見的運作機制 |
| **伊麗莎白的自我認識**（“I never knew myself”） | 主題頂點 |
| **參觀彭伯里 → 對達西印象逆轉** | 透過空間發現人格 |
| **莉迪亞私奔 → 達西秘密斡旋** | 以行動證明愛情 |
| **雙重婚姻結局** | 對型別的承諾 |

### 必須謹慎處理的元素

| 元素 | 注意事項 |
|---|---|
| **階級意識** | 現代化可能使動機消失，必須配合所選時代調整 |
| **婚姻在經濟上的迫切性** | 容易從現代視角被簡化為單純愛情故事 |
| **女性受到的社會約束** | 削弱約束也會削弱衝突 |
| **柯林斯的荒謬可笑** | 容易淪為單純的喜劇角色 |

## 2. 按媒介分類的改編指南

### A. 電影（90–150分鐘）

#### 壓縮策略
- **第一卷（第1–23章）**＝30–40分鐘，約佔全片三分之一
- **第二卷（第24–42章）**＝30–40分鐘
- **第三卷（第43–61章）**＝30–50分鐘

#### 優先保留
- 梅里屯舞會（EVT-003）
- 第一次求婚（EVT-028）
- 信件（EVT-029）
- 自我認識的散步（EVT-030）
- 彭伯里（EVT-035、EVT-036）
- 私奔（EVT-038）
- 第二次求婚（EVT-046）

#### 可以壓縮或省略
- 柯林斯求婚的細節（EVT-017）——簡短處理
- 部分社交場景，包括第5–6章的一部分
- 加德納夫婦的部分出場
- 瑪麗和凱蒂的部分場景

#### 視覺重點
- 第一次望見彭伯里（第43章）
- 漢斯福德求婚時的封閉空間
- 舞會的華麗場面

### B. 迷你劇（5–6小時）

#### 1995年BBC模式：全6集，約5小時
- **幾乎可以包含所有章節**
- 可以用畫外音處理自由間接引語
- 在畫面中補充達西未被直接描寫的行動，例如在倫敦搜尋威克姆

#### 優先增加或擴充
- EVT-039-001–005，達西的秘密斡旋——直接影像化
- EVT-019-001，拆散賓利與簡的協商——直接影像化
- 人物日常生活的細節

### C. 舞臺劇（2小時30分鐘–3小時）

#### 壓縮與象徵化
- **旋轉舞臺或極簡佈景**
- 調整人物數量，允許合併小角色
- 用合唱隊或旁白壓縮情節

#### 核心場景：必須保留
- 梅里屯舞會，作為開場
- 漢斯福德求婚，作為第一幕高潮
- 彭伯里，作為第二幕開端
- 私奔危機
- 第二次求婚，作為終場

### D. 音樂劇

#### 適合用歌曲展開的場景
- “It is a truth universally acknowledged”，開場群舞
- 瑪麗的歌曲，喜劇段落
- 舞會，合唱與群舞
- “I never knew myself”，伊麗莎白獨唱
- “In vain have I struggled”，達西獨唱
- “All love must be vain”，伊麗莎白獨唱
- 第二次求婚，二重唱
- 婚禮，終場

### E. 網漫或漫畫

#### 分鏡策略
- 第一印象＝大畫格
- 對話＝連續小畫格
- 求婚＝面部表情特寫的整頁畫面

#### 視覺母題
- 達西的目光，反覆出現
- 信件，作為經過設計的實體道具
- 彭伯里風景，寬幅畫格

## 3. 按時代分類的改編

### A. 忠於時代的改編：1811年
- 原樣使用全部設定表
- 準確再現服裝、建築與禮儀
- **現代觀眾的理解門檻：**階級制度、婚姻的迫切性與稱謂體系

### B. 移至維多利亞時代：1850–1860年代
- 服裝：裙撐，更加正式
- 婚姻法：條件發生變化，時間仍在1882年《已婚婦女財產法》之前
- 達西：可設定為伯爵或更高爵位
- 威克姆：調往殖民地軍隊

### C. 移至愛德華時代：1900年代
- 服裝：蹣跚裙；汽車登場
- 舞會 → 鄉間別墅聚會
- 達西：莊園領主的最後一代

### D. 1920–1930年代
- 現代主義
- 服裝：飛來波女郎對傳統
- 汽車與電話登場
- 婚姻意義減弱，需要另設衝突

### E. 當代韓國改編：示例

#### 轉換矩陣
| 原作 | 當代韓國版本 |
|---|---|
| 貝內特家 | 有五個女兒的中產階級自營業家庭 |
| 限嗣繼承 | 其他形式的財務危機，例如生意失敗 |
| 達西 | 大企業繼承人或江南富豪 |
| 賓利 | 達西的好友 |
| 威克姆 | 富有魅力的騙子 |
| 柯林斯 | 宗教與當代韓國不易對應——律師、教授或保險業務員？ |
| 凱瑟琳夫人 | 達西的母親或姨母 |
| 舞會 | 俱樂部派對、婚禮或公司派對 |
| 軍隊 | 職場或另一個富有吸引力的外部群體 |

#### 必須保留的核心
- 第一印象的陷阱
- 透過信件或訊息揭露真相
- 家庭對愛情的影響
- 自我認識的瞬間
- 兩次告白的結構

#### 現代化的風險
- **婚姻的迫切性減弱** → 需要其他迫切利害，如事業、家庭、簽證等
- **階級差異減弱** → 需要其他差異，如學歷、出身地、家庭背景
- **溝通手段改變** → KakaoTalk或其他即時通訊軟體取代信件

### F. 科幻改編
- 太空殖民地上的家族
- 達西＝大型艦隊繼承人
- 威克姆＝太空海盜
- 限嗣繼承＝家族行星封地的繼承製度

### G. 奇幻改編
- 魔法家族
- 達西＝強大魔法世家的成員
- 婚姻＝家族魔法的結合
- 舞會＝魔法儀式

## 4. 按視角分類的改編

### A. 達西視角：重述
- **優點：**完整展現達西的內心，並直接呈現秘密斡旋
- **風險：**神秘感消失，伊麗莎白的發現過程被削弱
- **先例：**《Mr. Darcy's Diary》等作品

### B. 威克姆視角
- 反英雄敘事或自我辯護
- 喬治安娜事件與莉迪亞私奔中，威克姆一方的動機
- **風險：**很容易成為富有魅力的敘述者，必須維持道德平衡

### C. 夏洛特視角
- 婚姻現實主義者的視角
- 漢斯福德日常生活的細節
- 與柯林斯的婚姻生活

### D. 瑪麗視角
- “隱形的第三個女兒”的視角
- 姊妹們的觀察者
- 逃入音樂和書籍

### E. 喬治安娜視角
- 適合輕小說
- 從創傷中復原，以及接納伊麗莎白為嫂子的過程

### F. 僕人視角：雷諾茲太太等
- 《唐頓莊園》風格
- 從樓下觀察“樓上發生的事”

### G. 賓利視角
- 優柔寡斷的友誼視角
- 對達西的依賴
- 無法忘記簡的一年

## 5. 按型別分類的改編

### A. 強化喜劇
- 擴大柯林斯、貝內特太太與凱瑟琳夫人的戲份
- 強調威克姆可笑的一面
- 強調貝內特先生的機智

### B. 強化劇情或情節劇
- 細化達西與伊麗莎白的感情
- 簡與賓利分離一年的痛苦
- 莉迪亞的悲劇性

### C. 強化懸疑
- 更緩慢地揭露威克姆的真面目
- 把達西的秘密斡旋改成推理情節
- 利用信件出現的時機制造懸念

### D. 恐怖或哥特風格：實驗性
- 把彭伯里變成可怕的地方
- 把威克姆變成真正的惡徒
- 強化凱瑟琳夫人的威脅感

### E. 動作或驚悚
- 把追蹤威克姆改成動作場面
- 增加決鬥場景；原作沒有發生，只是貝內特太太擔心過
- 強化逃亡情節

## 6. 核心場景的改編選項

### 第34章：第一次求婚

#### 正統版本
- 昏暗的客廳，兩人獨處
- 達西爆發式的告白
- 伊麗莎白的憤怒
- 激烈對峙

#### 現代化選項1：職場
- 在公司辦公室告白
- 保持同等強度的衝突

#### 現代化選項2：KakaoTalk
- 長訊息 → 否定的回覆
- 以手機螢幕提供視覺畫面

#### 影像化選項
- 以特寫為主
- 兩人視線交替
- 不使用音樂

### 第43章：第一次望見彭伯里

#### 正統版本
- 從馬車上第一次揭示主宅全景
- 音樂漸強

#### 現代化
- 第一次到訪江南頂層公寓
- 或進入鄉間別墅

#### 影像化
- 廣角鏡頭 → 特寫
- 達西的肖像畫，或照片

### 第46章：私奔訊息

#### 正統版本
- 兩封信加上達西登場
- 伊麗莎白崩潰

#### 現代化
- 電話響起，達西恰好在身邊
- 在社交媒體上曝光

## 7. 絕對應該避免的改變

### 人物變形的風險
- ❌ **讓伊麗莎白從一開始就喜歡達西**——偏見主題消失
- ❌ **讓達西從一開始就很親切**——改變主題消失
- ❌ **把威克姆塑造成值得同情的人**——道德上的模糊會沖淡主題
- ❌ **把柯林斯塑造成嚴肅人物**——諷刺被削弱
- ❌ **只把莉迪亞塑造成悲劇受害者**——不負責任的主題被削弱

### 結構變形的風險
- ❌ **達西只求婚一次便被接受**——否定作品核心
- ❌ **沒有信件便完成自我認識**——移除最重要的裝置
- ❌ **省略彭伯里之行**——移除發現人格的裝置
- ❌ **省略莉迪亞危機**——移除證明達西改變的裝置

### 氛圍變形的風險
- ❌ **過於黑暗**——喪失奧斯汀的機智
- ❌ **過於輕浮**——喪失社會批判
- ❌ **過於激情**——喪失剋制的情感

## 8. 按媒介決定語調的指南

### 120分鐘電影的銀幕時間分配

| 部分 | 章節 | 時長 |
|---|---|---|
| 匯入 | 第1–3章 | 10–12分鐘 |
| 開端 | 第4–17章 | 25–30分鐘 |
| 展開 | 第18–34章 | 30–35分鐘 |
| **第一次求婚** | 第34章 | 5–7分鐘，單獨賦予比重 |
| 危機 | 第35–46章 | 25–30分鐘 |
| 高潮與解決 | 第47–58章 | 15–20分鐘 |
| 結局 | 第59–61章 | 5–10分鐘 |

### 三幕舞臺劇的幕次分配
- **第一幕：**第1–19章，至柯林斯求婚
- **第二幕：**第20–46章，至傳來私奔訊息
- **第三幕：**第47–61章

## 9. 把故事移至新時代時的檢查清單

```
[ ] 婚姻仍有迫切的理由嗎？
[ ] 階級或地位差異仍有意義嗎？
[ ] 這個體系會讓第一印象出錯嗎？
[ ] 秘密資訊仍有價值嗎？（信件或訊息）
[ ] 家庭仍可能成為負擔嗎？
[ ] 外表與實質仍可能不同嗎？
[ ] 仍可能出現自我認識的瞬間嗎？
[ ] 仍能以行動證明愛情嗎？
```

八項全部回答“是”，便有可能完成優秀的改編。

## 10. 作品的核心訊息：不可改變

1. **第一印象往往是錯的。**傲慢與偏見同時存在於雙方。
2. **婚姻既是社會與經濟行為，也是愛情行為。**兩者會發生衝突。
3. **自我認識是最艱難的領悟。**“Till this moment, I never knew myself.”
4. **愛情以行動而非外表來證明。**達西的秘密斡旋證明了這一點。
5. **家庭不能決定一個人愛誰，但會影響其決定。**
6. **時間會揭示外表與實質。**“One has all the goodness, the other all the appearance of it.”

只要保留這六條訊息，任何改編都仍是《傲慢與偏見》。
