# 《傲慢與偏見》——人物作息表（推定/創作）

> ⚠️ **本文件包含推定與創作。** 結合攝政時期鄉紳與中產階級的標準作息、各人物性格和原作行為模式，重構人物日常時間表。可用於改編時描寫人物日常生活。

---

## 攝政時期鄉紳的標準一天（基本型）

| 時間 | 活動 |
|---|---|
| 早上7至8點 | 起床。僕人把溫水送到臥室。穿晨間服。 |
| 8至9點 | 散步或私人時間（寫信、閱讀） |
| 9至10點 | Breakfast（早餐）——全家一起 |
| 10至12點 | 散步、拜訪（morning calls）、家務、閱讀、音樂練習 |
| 12至15點 | 外出、購物、散步、接待客人 |
| 15:30至16點 | 換裝（dressing for dinner） |
| 16至17點 | Dinner（正餐，一天的主餐） |
| 17至19點 | 餐後休息（男性：波特酒；女性：客廳） |
| 19至21點 | Tea、紙牌、音樂、閱讀、談話 |
| 21至22點 | Supper（簡單夜宵，通常僅在舞會後） |
| 22至23點 | 就寢 |
