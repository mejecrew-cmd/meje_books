# 《傲慢與偏見》——時代用語誤譯防範辭典

> 奧斯汀時代（攝政時期，約1810年代）的英語中，有許多詞與現代英語**拼法相同，含義卻不同**。
> 本文整理了那些**依照現代含義翻譯便會造成誤譯的詞語**，以防止翻譯失誤。
>
> **風險分級：**
> - **［危險］**——與現代含義相反或完全不同。誤譯會顛倒整個句子的意思。
> - **［注意］**——與現代含義部分重疊，但核心語感不同。
> - **［參考］**——現代已消失的用法，或為了自然翻譯而必須瞭解的用法。

---

## A. ［危險］級——與現代含義相反／完全不同

### 1. sensible
| 項目 | 內容 |
|---|---|
| **奧斯汀時代的含義** | (1) 有判斷力的、明智的（= having good sense）／(2) 意識到……的（= aware of） |
| **現代英語含義** | 敏感的（= sensitive）← **不是這個意思** |
| **書中例句** | "a sensible, intelligent young woman"（第5章，夏綠蒂）= **有判斷力且聰明的**年輕女性 |
| | "I am very sensible of the honour"（第19章，伊莉莎白）= 我**非常清楚**這份榮幸 |
| **譯法** | 「有判斷力的」／「明智的」／「很清楚……」 |
| **誤譯示例** | ~~「敏感的年輕女性」~~ → 塑造出完全不同的人物形象 |

### 2. condescension / condescend
| 項目 | 內容 |
|---|---|
| **奧斯汀時代的含義** | 身分高貴者**降低身段給予恩惠**。屬於讚美用語。 |
| **現代英語含義** | 自以為是、居高臨下的態度 ← **正好相反** |
| **書中例句** | "such affability and condescension"（第14章，柯林斯讚美凱瑟琳夫人）= 凱瑟琳夫人**紆尊降貴的恩典** |
| | "She had even condescended to advise him to marry"（第14章）= 她甚至**屈尊親自**勸他結婚 |
| **譯法** | 「紆尊降貴」／「恩慈地」／「親自屈尊」 |
| **誤譯示例** | ~~「那種傲慢與盛氣凌人」~~ → 把柯林斯的奉承變成了批評 |
| **注意** | 奧斯汀也會以**反諷**方式使用這個詞。柯林斯是真心稱頌，讀者卻覺得可笑——譯文必須保留這種雙重性。 |

### 3. nervous / nerves
| 項目 | 內容 |
|---|---|
| **奧斯汀時代的含義** | 神經**衰弱**／患有神經性疾病（醫學概念） |
| **現代英語含義** | 緊張、不安 ← 語感不同 |
| **書中例句** | "You have no compassion on my poor nerves."（第1章，班奈特太太）= 你對我**可憐的神經衰弱**一點同情也沒有 |
| | "she fancied herself nervous"（第1章，敘述者）= 她自以為自己**神經脆弱** |
| **譯法** | 「神經（脆弱）」／「神經衰弱」——不是單純的「緊張」 |
| **要點** | 班奈特太太的 “nerves” 是醫學式的自我診斷（其實是她的臆想），並非單純的不安。 |

### 4. nice
| 項目 | 內容 |
|---|---|
| **奧斯汀時代的含義** | 精密的、挑剔的、能作細微辨別的（= precise, discriminating） |
| **現代英語含義** | 好的、友善的 ← 幾乎無關 |
| **書中例句** | 雖然直接用例不多，但 "a nice distinction" = **細微的**區別 |
| **譯法** | 「精密的」／「挑剔的」／「細微的」 |

### 5. wanting
| 項目 | 內容 |
|---|---|
| **奧斯汀時代的含義** | **不足的**、欠缺的（= lacking） |
| **現代英語含義** | 想要的（= desiring）← 完全不同 |
| **書中例句** | "No one admitted to the privilege of hearing you can think anything wanting."（第31章，達西）= 有幸聽您演奏的人，不會覺得**有什麼不足** |
| **譯法** | 「不足的」／「欠缺的」 |
| **誤譯示例** | ~~「不會認為有什麼東西是想要的」~~ |

### 6. address（名詞）
| 項目 | 內容 |
|---|---|
| **奧斯汀時代的含義** | (1) 談吐、舉止、待人方式／(2) 信件地址／(3) 求愛行為 |
| **現代英語含義** | 幾乎只指地址 |
| **書中例句** | "very pleasing address"（第15章，威克姆）= 十分討人喜歡的**談吐／舉止** |
| | "my addresses"（第20章，柯林斯）= 我的**求愛** |
| | "Jane had written the direction remarkably ill"（第46章）= 珍把**地址**寫得格外難認（這裡 direction = 地址） |
| **譯法** | 依語境譯作「談吐」／「舉止」／「求愛」／「地址」 |

### 7. direction
| 項目 | 內容 |
|---|---|
| **奧斯汀時代的含義** | 信件的**地址／收件地點**（= address on a letter） |
| **現代英語含義** | 方向、指示 |
| **書中例句** | "the wished-for direction"（第52章，威克姆的地址）= 盼望已久的**地址** |
| **譯法** | 「地址」／「下落」 |

## B. ［注意］級——核心語感不同

### 8. handsome
| 項目 | 內容 |
|---|---|
| **奧斯汀時代的含義** | (1) 人：英俊／美麗（男女通用）／(2) 行為、金額：**慷慨的、豐厚的**／(3) 建築：宏偉的 |
| **與現代含義的差異** | 現代幾乎只用於男性外貌 |
| **書中例句** | "a handsome young man"（賓利）= 英俊／"handsome modern building"（羅新斯）= **宏偉的**現代建築／"a very handsome thought"（柯林斯讚美達西）= **高尚而慷慨的**想法 |
| **譯法** | 人：「英俊／美麗」／建築：「宏偉／氣派」／行為：「高尚／慷慨」 |

### 9. civil / civility
| 項目 | 內容 |
|---|---|
| **奧斯汀時代的含義** | 有禮貌——但只是**最低限度的禮貌**，未必出於真心 |
| **與現代含義的差異** | 現代用法也近似，但奧斯汀明確區分 “civil” 與 “warm/cordial” |
| **書中例句** | "with cold civility"（第46章，伊莉莎白對達西）= 帶著**冷淡的禮貌** |
| | "civil disdain"（第18章，賓利小姐）= **禮貌的輕蔑**——有意的矛盾修辭 |
| **譯法** | 「有禮貌的」；需要表現不真誠時譯作「形式上的」／「禮節性的」 |
| **等級體系** | civil（最低）< polite（標準）< courteous（殷勤）< warm（熱情）< cordial（真誠） |

### 10. consequence
| 項目 | 內容 |
|---|---|
| **奧斯汀時代的含義** | **社會重要性／體面／地位**（= social importance） |
| **現代英語含義** | 結果（= result） |
| **書中例句** | "give consequence to young ladies who are slighted by other men"（第3章，達西）= 把被其他男人冷落的小姐們**當成重要人物看待** |
| | "a man of consequence" = **有體面／有地位的**人 |
| **譯法** | 依語境譯作「體面」／「重要性」／「地位」 |

### 11. intelligence
| 項目 | 內容 |
|---|---|
| **奧斯汀時代的含義** | **消息、資訊**（= news, information） |
| **現代英語含義** | 智力 |
| **書中例句** | "the great intelligence"（第30章，柯林斯宣布達西到來）= 這個**重大消息** |
| | "fresher intelligence of her former friends"（第25章）= 關於舊友的**更新消息** |
| **譯法** | 「消息」／「資訊」／「音訊」 |

### 12. design
| 項目 | 內容 |
|---|---|
| **奧斯汀時代的含義** | **意圖、計畫、圖謀**（= intention, plan, scheme） |
| **現代英語含義** | 設計 |
| **書中例句** | "Is that his design in settling here?"（第1章，班奈特先生）= 他在這裡定居就是這個**意圖**嗎？ |
| | "without any design"（第22章，珍）= 沒有任何**圖謀** |
| **譯法** | 「意圖」／「計畫」／「盤算」 |

### 13. countenance
| 項目 | 內容 |
|---|---|
| **奧斯汀時代的含義** | (1) **臉、氣色、表情**／(2) 支持、贊同 |
| **現代英語含義** | 面容（現已少用） |
| **書中例句** | "a fine countenance"（威克姆）= 出眾的**容貌** |
| | "Bingley could keep his countenance"（第9章）= 賓利能夠**保持表情不變** |
| **譯法** | 「表情」／「氣色」／「容貌」 |

### 14. character
| 項目 | 內容 |
|---|---|
| **奧斯汀時代的含義** | (1) 性格／(2) **名聲、世人的評價**——兩個含義可能同時發生作用 |
| **書中例句** | "His character was decided."（第3章）= 他的**名聲已經定了**（= 被認定為傲慢的人） |
| | "Wickham's character" = 威克姆的**真面目／本性**（在後半部被顛覆） |
| **譯法** | 依語境區分「性格」和「名聲」。二義並存時可譯為「為人」或「人物形象」 |

### 15. establishment
| 項目 | 內容 |
|---|---|
| **奧斯汀時代的含義** | (1) **出嫁／安頓下來的歸宿**（透過婚姻建立獨立家庭）／(2) 獨立生活的整體基礎 |
| **書中例句** | "what an establishment it would be for one of them"（第1章）= 對其中一個女兒而言，那會是多麼好的**歸宿** |
| | "the pure and disinterested desire of an establishment"（第22章，夏綠蒂）= 純粹為了獲得**安穩住所**的願望 |
| **譯法** | 「出嫁」／「歸宿」／「安穩的家庭」——不是「設施」 |

### 16. connexion(s) / connection(s)
| 項目 | 內容 |
|---|---|
| **奧斯汀時代的含義** | **親屬關係／人脈**（= relatives, family ties） |
| **現代英語含義** | 一般意義上的連結 |
| **書中例句** | "low connexions"（達西）= **地位低微的親屬／關係**（= 經商的舅舅、當律師的舅舅） |
| | "the connexion was a most unhappy one" = 那段**姻親關係**極其不幸 |
| **譯法** | 「人脈」／「姻親」／「親屬關係」 |

### 17. amiable
| 項目 | 內容 |
|---|---|
| **奧斯汀時代的含義** | **可愛、討人喜歡、善良**——比現代 “amicable”（友好的）更強烈的讚美 |
| **書中例句** | "she is one of the most tractable creatures... a very great favourite"（第33章，伊莉莎白評價喬治安娜） |
| | "the most amiable man of my acquaintance"（第24章，珍評價賓利）= 我認識的人中最**可愛可親的**男人 |
| **譯法** | 「可愛可親的」／「溫厚的」／「討人喜歡的」 |

### 18. agreeable
| 項目 | 內容 |
|---|---|
| **奧斯汀時代的含義** | **令人愉快的人，相處起來很舒服的人**（= pleasant company） |
| **書中例句** | "If he had been so very agreeable, he would have talked to Mrs. Long"（第5章，班奈特太太）= 如果他真那麼**隨和討喜**，就會和朗太太說話 |
| **譯法** | 「令人愉快的」／「討人喜歡的」／「隨和的」 |
| **等級** | disagreeable（討厭）< tolerable（尚可忍受）< agreeable（令人愉快）< amiable（可愛可親） |

## C. ［參考］級——現代已經消失的用法

### 19. complaisance / complaisant
| 項目 | 內容 |
|---|---|
| **含義** | 為取悅他人而表現出的**順從、遷就態度**／樂意配合 |
| **注意區別** | 與 “complacency”（自滿）不同 |
| **書中例句** | "his complaisance"（第6章）= 他那種**遷就配合的態度** |

### 20. propriety / impropriety
| 項目 | 內容 |
|---|---|
| **含義** | **得體／禮儀規範／合乎身分的行為**——不只是抽象的「適當」，而是遵守社會規範 |
| **書中例句** | "want of propriety"（達西的信中批評班奈特一家）= **缺乏禮儀規範** |
| **譯法** | 「禮儀」／「體面」／「端莊得體」 |

### 21. scheme
| 項目 | 內容 |
|---|---|
| **含義** | **計畫**（比現代帶貶義的「陰謀」更中性） |
| **書中例句** | "No scheme could have been more agreeable"（第27章，旅行計畫）= 再沒有比這更令人愉快的**計畫**了 |
| **譯法** | 「計畫」／「方案」——不一定含貶義 |

### 22. apprehension
| 項目 | 內容 |
|---|---|
| **含義** | (1) **理解、察覺**／(2) 不安——奧斯汀更常使用含義 (1) |
| **書中例句** | "I had no apprehension of his feeling a serious attachment"（第35章，達西的信）= 我並**沒有察覺**他懷有認真的感情 |
| **譯法** | 「察覺」／「理解」（只有表示不安時才譯作「不安」） |

### 23. complaint
| 項目 | 內容 |
|---|---|
| **含義** | (1) 抱怨／(2) **疾病、病症** |
| **書中例句** | "nervous complaints"（班奈特太太）= **神經疾病** |
| **譯法** | 應依語境判斷；班奈特太太的用例一律是「疾病」 |

### 24. prospect
| 項目 | 內容 |
|---|---|
| **含義** | (1) **景觀／景色**（literal view）／(2) 未來前景 |
| **書中例句** | "a charming prospect over that gravel walk"（第9章）= 碎石路那邊迷人的**景色** |
| **譯法** | 建築／窗戶語境譯作「景觀／景色」，未來語境譯作「前景／可能性」 |

### 25. natural
| 項目 | 內容 |
|---|---|
| **含義** | (1) 自然的／(2) 自然主義的（庭園）／(3) **非婚生的**（少見）／(4) 天生的 |
| **書中例句** | "natural beauty"（彭伯里庭園）= 保留**自然之美的**景致（並非人工雕琢） |
| | "a sort of natural self-consequence"（莉迪亞）= 一種**天生的**自負 |
| **譯法** | 依語境區分。彭伯里的 “natural” 是**自然主義美學**術語。 |

## D. 好感度形容詞等級表——翻譯時調整強度

奧斯汀描寫人物時使用的形容詞具有**精細的等級差異**：

### 正面等級（弱→強）
```
tolerable（尚可忍受、馬馬虎虎）← 最低——達西對伊莉莎白的評價
  ↓
civil（有禮貌的）← 禮節上的最低限度
  ↓
agreeable（令人愉快、相處舒服）
  ↓
pleasing（討人喜歡）
  ↓
amiable（可愛可親、善良）
  ↓
charming（迷人）
  ↓
delightful（令人愉悅、帶來喜悅）
  ↓
excellent（極其出色）← 最高
```

### 負面等級（弱→強）
```
not very agreeable（不怎麼討人喜歡）← 英式委婉表達
  ↓
disagreeable（令人不快）
  ↓
unpleasant（不愉快，更直接）
  ↓
ill-natured（脾氣壞、刻薄）
  ↓
proud（傲慢）← 用於達西
  ↓
insolent（無禮、放肆）
  ↓
abominable（可惡至極）
  ↓
despicable（卑劣可鄙）← 最強——伊莉莎白的自我批評
```

### 翻譯原則
- 在譯文中也要保持**同樣的強度差異**。
- 把 “tolerable” 和 “agreeable” 都譯作「還不錯」，會破壞等級。
- 把 “civil” 和 “amiable” 都譯作「親切」，會抹去關係溫度的差異。

## E. 英式委婉表達（Understatement）——誤譯最多的領域

奧斯汀極其頻繁地使用**雙重否定、弱化表達和委婉語**。直譯會使語感消失。

| 原文模式 | 表面含義 | 實際含義 | 翻譯指南 |
|---|---|---|---|
| "not unpleasing" | 並非令人不快 | **相當不錯** | 化解雙重否定，但保持較弱強度：「還不錯」 |
| "not without" | 並非沒有 | **確實存在** | 「確實有……」 |
| "she could not help" | 她忍不住不…… | **她做了……**（強烈衝動） | 目標語言自然時保留否定結構：「她忍不住……」 |
| "I dare say" | 我敢說 | **大概如此**（輕微推測） | 「大概」／「我想是吧」 |
| "I am sure" | 我確信 | **依我看**（常是習慣性修辭，並非真正確信） | 班奈特太太使用時譯成沒有根據的確信：「準是這樣」 |
| "something like regret" | 類似後悔的東西 | 是後悔，但**不願承認** | 保留「近似後悔的感覺」，不能直接斷定為「後悔」 |
| "not entirely hopeless" | 並非完全沒有希望 | **還有一點希望** | 「並非全無希望」 |
| "I fancy" | 我想像 | **我覺得似乎……**（輕微推測） | 「我看」／「似乎」 |
| "by no means" | 無論如何都不 | **絕對不**（強烈否定） | 「決不」／「絕非」 |
| "tolerable/tolerably" | 可以忍受 | **勉強過得去**（不是讚美） | 「馬馬虎虎」／「勉強可以」 |

## F. 動詞的時代語感

| 動詞 | 奧斯汀時代 | 現代 | 書中例句 | 譯法 |
|---|---|---|---|---|
| **to own** | 承認、坦白 | 擁有 | "I own I was surprised" = 我**承認**自己很驚訝 | 「承認」／「坦白」 |
| **to engage** | (1) 約定 (2) 僱用 (3) 吸引注意 | 主要指訂婚 | "she was engaged elsewhere" = 她在別處**已有約會** | 依語境判斷 |
| **to want** | **缺少**／需要 | 想要 | "want of propriety" = **缺乏**禮儀 | 「缺少」／「欠缺」 |
| **to wait on** | **拜訪**（禮節性） | 等待／服侍 | "to wait on Mr. Bingley" = **拜訪**賓利先生 | 「拜訪」／「登門致意」 |
| **to make out** | **理解、弄清** | 辨認／接吻 | "I cannot make him out" = 我**無法理解他** | 「理解」／「弄清」 |
| **to recollect** | **有意識地回想起來** | 現代與 remember 近義 | 在奧斯汀筆下有「主動喚起記憶」的語感 | 「回想」／「憶起」 |
| **to flatter oneself** | **自以為、自負**（往往是誤判） | 奉承自己 | "I flatter myself that..." = **恕我自信地說……** | 「自以為」／「自負」——常帶反諷 |
| **to trifle with** | **玩弄、輕率對待** | — | "You are too generous to trifle with me" = 你不會卑劣到**玩弄我的感情** | 「玩弄」／「戲弄」 |

## G. 翻譯時絕對不能混淆的詞語組（Pair）

| 詞語組 | 區別 | 混淆的後果 |
|---|---|---|
| **pride** vs **vanity** | pride = 自我評價；vanity = 希望別人如何看待自己（第5章，瑪麗的定義） | 小說標題與主題的核心區分崩解 |
| **sense** vs **sensibility** | sense = 理性／判斷力；sensibility = 感受力／情感 | 珍（sensibility）與伊莉莎白（sense）的對照消失 |
| **civil** vs **polite** vs **amiable** | civil = 最低限度禮貌；polite = 標準禮貌；amiable = 真誠好感 | 關係溫度的差異消失 |
| **fortune** vs **income** | fortune = 財產總額；income = 年收入 | 誤讀經濟規模：賓利的10萬英鎊財產與每年4,000～5,000英鎊收入 |
| **trade** vs **profession** | trade = 商業（地位較低）；profession = 專業職業（地位較高） | 階級偏見中的細微差異消失 |
| **living** vs **life** | living = 聖職俸祿／教區收入；life = 生命、人生 | 顛倒達西與威克姆矛盾的核心 |
| **interesting** | 奧斯汀時代：引人關注／令人感動（比現代用法更強） | 把 “most interesting” 只譯成「有趣」，會削弱情感強度 |
