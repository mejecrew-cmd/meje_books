# 《傲慢與偏見》——音樂與聲音環境（推測／創作）

## 2. 攝政時期的音樂種類

### 家庭音樂（Drawing Room Music）

| 種類 | 說明 |
|---|---|
| **Ballads（敘事歌）** | 簡短抒情歌曲，由女性獨唱 |
| **Italian Songs** | 歌劇詠歎調；賓利小姐等時髦女性喜愛 |
| **Scotch/Irish Airs** | 民歌風格，氣氛輕快；瑪麗演奏 |
| **English Folk Songs** | “Greensleeves”等傳統歌曲 |
| **Glees** | 三至四聲部合唱曲 |
| **Sonatas** | 貝多芬、莫扎特、海頓等人的古典音樂 |

### 舞會音樂（Ball Music）

| 舞蹈 | 音樂 |
|---|---|
| **Country Dance** | 活潑的二拍或四拍音樂 |
| **Cotillion** | 四對舞者共舞 |
| **Quadrille** | 四對舞者共舞；1815年開始流行 |
| **Reel** | 蘇格蘭風格的快舞 |
| **Boulanger** | 最後一支鄉村舞 |
| **Minuet** | 正式、適合年長者；當時已過時 |
| **Waltz** | 1812年傳入英國；小說時點剛剛出現，原著未提及 |

### 推測可能演奏的作曲家
- **莫扎特**（1791年去世，仍處於人氣頂峰）
- **海頓**（1809年去世，曾訪問英國）
- **貝多芬**（仍在創作，代表新音樂）
- **亨德爾**（已成為經典）
- **Thomas Arne**（英國作曲家）
- **Charles Dibdin**（英國敘事歌）

## 3. 各人物的音樂傾向

### 伊莉莎白·班奈特
- **鋼琴：** “by no means capital”（第6章）——水平尚可
- **歌唱：** 自彈自唱，“pleasing”
- **偏好：** 輕快的ballads
- **態度：** “rather not sit down before those who must be in the habit of hearing the very best performers”（第6章）——謙遜
- **翻譯要點：** 享受音樂卻不炫耀；重表達而非技巧

### 瑪麗·班奈特
- **鋼琴：** **練習很多但缺乏天賦**——“neither genius nor taste”（第6章）
- **歌唱：** 聲音微弱、舉止生硬
- **偏好：** 長協奏曲、宗教音樂、帶道德訓誡的歌詞
- **場景：** 第6章盧卡斯莊園（用Scotch/Irish airs讓姊妹們跳舞）；第18章內瑟菲爾德舞會（長時間唱歌令家人難堪）
- **翻譯要點：** 嚴肅卻滑稽的組合

### 卡羅琳·賓利
- **鋼琴：** 演奏出色，風格正式
- **偏好：** Italian songs、Scotch airs（用於吸引達西注意）
- **場景：** 第10章為了吸引達西，故意改變曲風
- **豎琴：** 可能會演奏

### 喬治安娜·達西
- **鋼琴：** **卓越**——“her performance on the pianoforte is exquisite”（第8章，卡羅琳）
- “she practises very constantly”（第31章，達西）
- **豎琴：** “delighted to hear of her improvement on the harp”（第10章）
- **偏好：** 嚴肅的古典音樂
- **新鋼琴：** 第43章，達西所贈

### 安妮·德·包爾
- **不會演奏**——“her health had not allowed her to apply”（第31章，凱瑟琳夫人）
- 母親炫耀說她若身體健康一定會演奏得很好

### 凱瑟琳夫人
- **自己不會演奏**——“If I had ever learnt, I should have been a great proficient”（第31章）
- **自以為是：** 仍會評論音樂

### 莉迪亞·班奈特
- **幾乎不參與音樂**——缺乏興趣
- 只喜歡在舞會上跳舞

### 男性人物
- **達西：** 不演奏，符合當時男性常態；會聆聽
- **賓利：** 不演奏；喜歡聽音樂
- **柯林斯：** 只會稱讚音樂是“可愛的消遣”
- **班奈特先生：** 不參與，待在書房
- **赫斯特先生：** 漠不關心

## 4. 舞會音樂的組織

### 梅里屯公共舞會（第3章）
- 小型外聘樂隊（4–6人）
- 小提琴、大提琴、長笛、大鍵琴或鋼琴
- 司儀（Master of Ceremonies）指定曲目

### 內瑟菲爾德私人舞會（第18章）
- 較大型樂隊（6–10人）
- 在獨立舞臺或樓廊演奏
- 曲目之間短暫休息

### 舞會流程
```
1. Country Dance（2–3首）
2. 休息＋飲料
3. Quadrille或Cotillion
4. 休息
5. Reel
6. 夜宵（supper）
7. 再次Country Dance
8. 最後：Boulanger（全體）
```

## 5. 日常家庭音樂（推測）

### 浪博恩的日常
- **晚餐後的客廳：** 瑪麗或伊莉莎白演奏
- **頻率：** 有客人時幾乎每天；只有家人時偶爾

### 彭伯里的日常
- **喬治安娜每天上午練習**
- **晚上：** 可能在家庭聚會中演奏

### 漢斯福德
- **沒有樂器**——第31章：“Mrs. Collins has no instrument”
- 凱瑟琳夫人允許使用“詹金森太太房裡的鋼琴”，這是帶侮辱性的提議

## 7. 關鍵場景聲音（改編用）

### 第3章梅里屯舞會
- **音樂：** 活潑的country dance
- **背景：** 閒談、腳步、扇子
- **達西說“tolerable”的瞬間：** 可讓音樂暫時減弱

### 第10章內瑟菲爾德客廳
- **賓利小姐的鋼琴：** 先是義大利風格，再轉為活潑的Scotch
- **達西邀請跳“Reel”的瞬間：** 突出音樂
- 達西寫信時的**鋼筆刮紙聲**

### 第18章內瑟菲爾德舞會
- **完整聲場：** 大型樂隊＋人群
- **達西與伊莉莎白共舞：** 音樂中的兩人對話
- **瑪麗唱歌：** 尷尬氣氛；可讓實際音高略有偏差

### 第34章漢斯福德求婚
- **寂靜**——沒有音樂
- **壁爐木柴聲**
- **達西腳步聲**（在房內踱步）
- **伊莉莎白的呼吸**

### 第36章讀信後散步
- **鳥鳴**、**樹葉沙沙聲**
- **伊莉莎白的呼吸**，表現情感激盪
- 沒有外部音樂——內心沉默

### 第43章彭伯里
- **馬車輪聲**（碎石車道）
- **河水聲**（第43章明確提及）
- **樓廊腳步聲**（石地面）
- 達西出現前的**園丁耙地聲**

### 第46章私奔訊息
- **信紙窸窣聲**
- **伊莉莎白抽泣**
- **達西腳步聲**（在房內踱步）

### 第58章第二次求婚散步
- 秋日早晨的**鳥鳴**
- **踩碎石聲**
- 兩人**柔和的聲音**

### 第61章婚禮
- **教堂鐘聲**
- **管風琴**（如有）
- **賓客低語聲**

## 8. 音樂主題動機（改編用）

### 各人物主題建議（影視配樂）

| 人物 | 主題氛圍 |
|---|---|
| **伊莉莎白** | 輕快、機智（莫扎特風格） |
| **達西** | 深沉、剋制（早期貝多芬風格） |
| **珍** | 柔和、抒情 |
| **賓利** | 明亮、活躍 |
| **威克姆** | 迷人卻膚淺（欺騙性的華麗） |
| **柯林斯** | 滑稽進行曲風格 |
| **凱瑟琳夫人** | 壓迫性的進行曲 |
| **莉迪亞** | 魯莽的舞曲風格 |

### 各事件的音樂變化
- **第一次求婚（第34章）：** 緊張＋爆發
- **書信（第35章）：** 低沉的悲傷
- **自我覺醒（第36章）：** 沉思
- **彭伯里（第43章）：** 驚歎＋發現
- **私奔訊息（第46章）：** 震驚
- **再次求婚（第58章）：** 平靜＋結合
