# 《傲慢與偏見》情感詞彙對應指南

> 小說中使用的英語情感詞彙、對應含義與強度等級。
> 即使表達同一種情感，也要根據強度和色彩選用不同詞語。這是翻譯中最困難的領域。

---

## 1. 情感詞彙分類體系

### 五級強度
- ⓟ 極輕（faint）
- ① 輕微（mild）
- ② 中等（moderate）
- ③ 強烈（strong）
- ④ 激烈（intense）
- ⑤ 極端（extreme）

### 色彩（情感的質地）
- 🌸 溫暖（warm）
- ❄️ 冷淡（cold）
- 🔥 激情（passionate）
- 🌧️ 憂鬱（melancholy）
- ⚡ 盛怒（furious）
- 🌫️ 模糊（ambiguous）

## 2. 好感與愛情繫列（按強度排列）

| 英語 | 含義／語感 | 強度 | 色彩 | 小說中的例子 |
|---|---|---|---|---|
| **acquaintance** | 相識、熟人關係 | ⓟ | 🌫️ | “her acquaintance”—僅僅認識 |
| **liking** | 好意、喜歡 | ① | 🌸 | “I give you leave to like him” |
| **partiality** | 偏愛、帶偏向的好感 | ① | 🌸 | “his partiality for Miss Bennet” |
| **inclination** | 吸引、內心傾向 | ① | 🌫️ | “a slight, thin sort of inclination” |
| **regard** | 尊重、情分 | ② | 🌸 | “his sincere, if not equal, regard” |
| **esteem** | 敬重、高度評價 | ② | 🌸 | “she respected, she esteemed” |
| **affection** | 愛意、溫情 | ② | 🌸 | “tender affection” |
| **fondness** | 溫情、愛意 | ② | 🌸 | “fond of his sister’s society” |
| **attachment** | 深情、依戀 | ③ | 🌸 | “stronger attachment than I had ever known” |
| **admiration** | 傾慕、讚歎 | ③ | 🔥 | “looked at with great admiration” |
| **love** | 愛 | ③～④ | 🔥 | “how ardently I admire and love you” |
| **passion** | 熱情、激情 | ④ | 🔥 | “the utmost force of passion” |
| **ardour** | 熱烈 | ④ | 🔥 | “spoken... with such ardour”（間接） |
| **adoration** | 仰慕、崇拜 | ⑤ | 🔥 | “the adoration of Mrs. Forster” |

### 對應詞指南

```
[輕微好感]
acquaintance：“相識”／“熟人關係”
liking：“喜歡”／“好意”
partiality：“帶偏向的好感”／“偏愛”
inclination：“吸引”／“內心傾向”

[中等好感]
regard：“情分”／“心意”／“尊重”
esteem：“敬重”／“傾慕”
affection：“愛意”／“溫情”
fondness：“溫柔的情意”
attachment：“深情”／“依戀”

[強烈愛情]
admiration：“傾慕”／“讚歎”
love：“愛”／“愛慕”
passion：“熱情”／“激情”
ardour：“熱烈的情感”
adoration：“崇拜”／“愛慕”
```

### 小說中的階段性演變（達西 → 伊麗莎白）

```
第6章   admiration（讚歎）—“fine eyes”
第10章  bewitched（著迷）—如同魔法般的吸引
第34章  ardent love（熱烈的愛）—“how ardently I admire and love you”
第58章  unchanged affections（不變的愛意）—平靜的深度
```

## 3. 負面情感—憤怒與憎惡系列

| 英語 | 含義／語感 | 強度 | 色彩 |
|---|---|---|---|
| **dislike** | 不喜歡、反感 | ① | ❄️ |
| **disapprobation** | 不贊同、不滿 | ① | ❄️ |
| **displeasure** | 不悅、不滿 | ② | ⚡ |
| **vexation** | 煩惱、惱怒 | ② | ⚡ |
| **annoyance** | 煩惱 | ② | ⚡ |
| **irritation** | 不快、刺惱 | ② | ⚡ |
| **resentment** | 怨恨、憤慨 | ③ | ⚡ |
| **anger** | 憤怒 | ③ | ⚡ |
| **indignation** | 義憤 | ③ | ⚡ |
| **abhorrence** | 憎惡 | ④ | ❄️ |
| **detestation** | 痛恨 | ④ | ⚡ |
| **fury** | 暴怒 | ⑤ | 🔥 |
| **wrath** | 帶有《聖經》色彩的憤怒 | ⑤ | 🔥 |

### 對應詞指南

```
[輕]
dislike：“不喜歡”／“反感”
disapprobation：“不贊同”／“不以為然”

[中]
displeasure：“不悅”／“心生不快”
vexation：“煩惱”／“惱怒”—避免過於現代的說法，“心煩意亂”更穩妥
annoyance：“煩惱”
irritation：“刺惱”／“心有芥蒂”

[強]
resentment：“怨恨”／“憤慨”／“積怨”
anger：“憤怒”／“怒氣”
indignation：“義憤”
abhorrence：“憎惡”／“厭惡”

[激烈]
detestation：“痛恨”／“憎惡”
fury：“暴怒”／“怒火上湧”
wrath：“盛怒”—幾乎不用
```

### 小說中達西與伊麗莎白的憤怒強度

```
第5章   dislike（反感）—輕微
第16章  resentment（怨恨）—聽過威克姆的陳述後
第33章  indignation（義憤）—菲茨威廉透露真相
第34章  ardent rejection（激烈拒絕）
第36章  shame（羞恥）—對自己的憤怒
```

## 11. 各人物的情感詞彙使用頻率

### Elizabeth（活力＋機智）

```
[常用詞彙]
- delight, amusement, vexation
- mortification（涉及自尊時）
- “lively, playful disposition”

[翻譯語調]
- 活潑＋略帶諷刺
- “有意思”“有趣”“煩人”
```

### Darcy（剋制＋強度）

```
[常用詞彙]
- regard, esteem, ardour
- shame, mortification
- “in vain have I struggled”

[翻譯語調]
- 剋制而深沉
- “情分”“傾慕”“羞恥”
```

### Mrs. Bennet（激烈的表層情緒）

```
[常用詞彙]
- joy, rapture, transport
- vexation, distress
- nerves（神經）

[翻譯語調]
- 情緒激烈卻流於表面
- “歡欣”“煩惱”“憂心”
```

### Mr. Bennet（譏誚的冷漠）

```
[常用詞彙]
- amusement（來自他人的愚蠢）
- 幾乎不直接表達情感

[翻譯語調]
- 保持距離的機智
- “我覺得頗有意思”／“真有趣”
```

### Jane（剋制的溫柔）

```
[常用詞彙]
- regard, esteem, attachment
- 剋制情感

[翻譯語調]
- 溫柔卻不失禮法
- “情分”“放在心上”
```

### Lydia（毫無剋制）

```
[常用詞彙]
- joy, rapture, ecstasy
- 全都流於表面

[翻譯語調]
- 興奮，但避免粗俗
- “太高興了！”／“真是的！”
```

### Lady Catherine（憤怒＋威壓）

```
[常用詞彙]
- displeasure, indignation
- arrogance, condescension

[翻譯語調]
- 權威式憤怒
- “不悅”“怒氣”
```

### Collins（過度正式）

```
[常用詞彙]
- gratification, satisfaction
- humility, civility（僅有表象）

[翻譯語調]
- 過分正式的詞彙
- “欣慰”“不勝感激”

## 12. 關鍵場面的情感強度追蹤

### 第34章：求婚場面

```
[達西]
1. agitation（動搖）—進門
2. ardent admiration（熱烈傾慕）—求婚
3. astonishment（驚愕）—遭到拒絕
4. anger（憤怒）—“pale with anger”
5. mortification（屈辱）—“ashamed of what my own have been”

[伊麗莎白]
1. astonishment（驚愕）—聽到求婚
2. compassion（同情）—短暫出現
3. resentment（憤慨）—聽到他的措辭後
4. anger（憤怒）—激烈拒絕
5. agitation（動搖）—拒絕後哭了半小時
```

### 第36章：自我認識場面

```
1. astonishment, apprehension, horror（閱讀書信時）
2. shame（認識自己）
3. humiliation（自我批判）
4. wretchedness（自身的悲慘）
5. resolution（決意）—“till this moment, I never knew myself”
```

### 第46章：私奔的訊息

```
[伊麗莎白]
1. anguish（悲痛）
2. wretchedness（悲慘）
3. self-reproach（自責）

## 13. 在翻譯中保留細微差別

### “regard” vs. “esteem” vs. “affection” vs. “love”

```
[小說中的達西]
第34章：“ardently admire and love”—愛（強）
第46章：“real, though unavailing concern”—深切關懷（家庭危機）
第58章：“affections and wishes are unchanged”—不變的愛意

[小說中的伊麗莎白]
第44章：“She respected, she esteemed, she was grateful to him”
       → 尊敬、傾慕、感激（愛情前一階段）
第46章：“could have loved him”
       → 首次使用“love”
第58章：接受愛情
```

### 譯法差異

```
原文：“regard”     → “情分”／“心意”（中性）
原文：“esteem”     → “尊敬”／“傾慕”
原文：“affection”  → “愛意”／“溫情”
原文：“love”       → “愛”／“愛慕”
原文：“passion”    → “熱情”
原文：“ardour”     → “熱烈的情感”
```

若全部譯成同一個“愛”，細微的情感階段便會崩塌。

## 15. 翻譯的黃金原則

### 1. 保留情感階段
- “regard → esteem → affection → love”的變化，在譯文中也必須呈現為遞進階段。

### 2. 區分人物用詞
- 達西的“ardent”≠柯林斯的“ardent”。
- 即使是同一個詞，也要符合人物的修養與格調。

### 3. 保持強度一致
- 不要把“rapture”弱化為普通的“喜悅”—應譯為“狂喜”。
- 不要把“fury”弱化為“生氣”—應譯為“暴怒”。

### 4. 保留情感色彩
- 溫暖的詞要譯得溫暖。
- 冷淡的詞要譯得冷淡。
- “esteem”＝溫暖的敬重；“respect”＝中性的尊重—必須區別。

### 5. 保持時代格調
- “愛”可以譯為“愛慕”／“傾心”。
- “高興”可以譯為“欣慰”／“歡欣”。
