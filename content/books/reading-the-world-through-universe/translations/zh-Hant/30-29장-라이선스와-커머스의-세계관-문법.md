# 第29章 授權與商業的世界觀語法

## 摘要

授權與商業看似發生在世界觀之外，實際上卻是對世界觀最嚴峻的考驗。當 IP 擴展到商品、合作、快閃店、數位物品、會員權益、AI 工具和品牌聯名時，世界觀會在外部合作夥伴與市場中被翻譯。此時需要的並不只是標誌使用規範，還要有一套世界觀語法，用來規定什麼可以出售，出售什麼會讓世界變得淺薄，哪些象徵可以使用、哪些必須保護，以及應當向粉絲提供什麼體驗。本章把授權與商業解釋為 IP 世界的經濟翻譯。

## 1. 商品是世界觀的試金石

粉絲透過商品觸摸一個世界。專輯、小卡、服裝、鑰匙扣、玩偶、數位物品、快閃店和聯名產品，會把世界轉化為物品與空間。此時，商品並不只是獲利手段。粉絲會透過商品判斷這個 IP 對自己的世界理解有多深。優秀的商品會強化世界的記憶，糟糕的商品則會讓世界變得淺薄。

商品之所以是世界觀的試金石，是因為選擇會在其中顯露出來。選擇什麼象徵？使用什麼顏色？寫上什麼文案？以什麼價格和數量出售？透過什麼通路出售？優先服務哪些粉絲？所有這些選擇，都在說明 IP 與粉絲建立了怎樣的關係。

沒有世界觀的商業出售表面。它貼上標誌，印上面孔，把圖像覆蓋在流行商品上。短期銷售或許能夠實現，但粉絲很快就會察覺。如果無法解釋這件商品為什麼屬於這個世界、為什麼必須現在出現、為什麼必須是這種物品，粉絲就會感到這是一種淺薄的商業化。

有世界觀的商業出售記憶。它把商品連結到某項活動的象徵、角色的物品、粉絲圈的儀式、世界的禁忌與社群的語言。粉絲感受到的不是購買一件物品，而是把世界的一部分帶入自己的生活。這一區別正是授權與商業的核心。

## 2. 授權不是出借權利，而是移轉語法

授權通常被理解為權利使用合約。合約規定哪些圖像、名稱和標誌可以使用，可以應用於哪些商品類別，以及如何分配收益。這套法律結構不可或缺。然而，對世界觀 IP 而言，授權遠不止於此。授權是把世界觀語法移轉給外部合作夥伴。

外部合作夥伴並不了解 IP 的所有語境，因此必須告訴他們應當守護什麼，哪些表達被禁止，粉絲重視哪些象徵，以及哪些商品化方式具有風險。只有標誌檔案和圖像使用規範遠遠不夠，還需要 LOREBOOK 與商業指南。

好的授權不會壓制合作夥伴的創造力，反而會讓創造力在世界內部發揮作用。如果合作夥伴知道哪些象徵可以改編，哪些象徵屬於核心正典、不可損壞，哪些角色特質適合製作周邊，哪些表達因保護真實人物而應當避免，他們就能創作得更加準確。

壞的授權只把 IP 當作表面元素交出去。外部合作夥伴雖然使用了熱門圖像和名稱，卻不知道粉絲圈為什麼喜歡它們。結果可能很漂亮，卻與世界格格不入。粉絲感受到的可能不是外部合作，而是外部入侵。授權既是權利移轉，也是世界觀翻譯。

權利不僅是合約條款，也是一種經年累月積累的資產。在音樂產業，一個世紀以來，藝人創作的歌曲權利通常歸唱片公司與音樂出版商所有。然而據傳，20世紀80年代的一位超級巨星聽取同行建議，一次性買下了其他藝人的完整歌曲目錄。此後數十年間，這批目錄大幅增值，成為龐大音樂出版事業的基礎。人們對確切的增值倍數說法不一，但倍數並不是重點。那位巨星買下的不是歌曲本身，而是決定誰可以如何改編、如何使用這些歌曲的權利。自2010年代末起，基金開始收購知名藝人的著作權，音樂也像金融產品一樣被交易。與此同時，一位藝人為了奪回在違背本人意願的情況下被出售的母帶，重新錄製了自己的歌曲，並於2025年親自買回這些母帶，為漫長的爭議畫上句號。

這些案例可以直接透過本章的兩條軸線來理解：可改編性與紅線。歌曲目錄的價值來自一首歌可以改編到何種程度、可以用於哪些地方，也就是可改編性的範圍。與此同時，為奪回母帶而重新錄音，則涉及一條不可輕率跨越的紅線：創作者的控制權究竟延伸到哪裡？在世界觀 IP 中，誰擁有什麼、什麼可以改編、什麼必須守護，最終同樣決定著這個世界能否持續。

不過，把權利資產化也有陰影。當著作權集中於大型資本與基金時，真正的創作者可能與自己的作品疏離；複雜的權利關係可能把作品長期鎖住、無法使用；昂貴的授權費也可能阻礙新的創作與粉絲接觸作品。把權利視為資產與維持創作生態，並不總是朝著同一方向。決定保護什麼固然重要，決定開放什麼對世界的永續性也同樣重要。

## 3. 商業必須連結粉絲活動

商業與粉絲活動相連時才會變得強大。粉絲買下商品之後能做什麼？可以收藏、展示、應援、與其他粉絲交換、發現世界觀線索，或參加某種特定儀式嗎？商品必須帶來購買之後的行動。

小卡會創造收藏、交換與曬圖打卡，應援棒會創造演出現場的共同行動，服裝與飾品會成為日常空間中的歸屬訊號。快閃店帶來造訪世界的空間體驗，數位物品則讓人在虛擬化身與線上社群中表達自我。商品既是物品，也是行動的種子。

當商業與粉絲活動分離，商品疲勞很快就會出現。粉絲必須不斷購買新物品，卻感受不到它與世界有什麼關係。商品越多，意義反而越少。好的商業擁有恰當的密度，應當讓粉絲在購買之後更深地感受這個世界。

訂閱經濟也可以和商業結合。會員服務可以提供特定商品的優先購買權或限定周邊。然而，這套結構若給粉絲施加過度壓力，就會變得危險。不能讓非訂閱者感到自己被排除在世界的基本體驗之外。商業可以創造歸屬的喜悅，也可能製造歸屬的階層。

## 4. 授權由紅線定義

在授權指南中，紅線和允許項目同樣重要。規定什麼不能做，才能守護 IP 的身分。必須梳理哪些角色不能以某種方式表現，哪些象徵不能被輕率使用，哪些文案可能在粉絲圈內引發衝突，以及哪些商品類別不符合保護真實人物的原則。

品牌和世界觀在某種程度上由禁忌定義。不做什麼，與做什麼同樣重要。進入所有商品類別的 IP 看似強大，卻很容易被稀釋。如果把所有迷因官方化，把所有象徵商品化，把所有粉絲反應變現，世界就會變得淺薄。紅線是維護世界格調的裝置。

紅線對真實人物 IP 尤其重要。真實人物的身體、私人生活、家人、健康、關係與未公開情感，都不應成為商品化的材料。AI 生成物、合成圖像、聲音模仿，以及出售過度親密感的商品，更需要謹慎。授權必須同時處理法律權利與倫理邊界。

紅線也能幫助合作夥伴。知道什麼應當避免，就能減少不必要的試錯。了解粉絲圈敏感的地方、官方守護的價值與世界的核心正典，外部合作就會更加安全。好的紅線並不阻止創作，而是讓創作發生在世界之內。

如果紅線只是一種抽象宣言，就無法發揮作用。必須把它寫成一行一條的詞條，讓合作夥伴可以立即使用。比如，某個偶像 IP 的一欄紅線可以這樣填寫。

> 詞條：成員單人特寫合成 / 適用：AI 生成、數位周邊 / 禁止：未經同意合成表情或身體，使用私人空間背景 / 許可：調整官方寫真照的色彩與構圖 / 例外核准：本人及經紀公司書面同意時 / 依據：自然人保護線（第20章四層結構中的自然人層）

這樣寫，合作夥伴就能一眼看出什麼被許可、什麼被禁止，又需要誰來核准。紅線範例的關鍵，不是只寫禁令，而是必須在禁令旁同時寫明許可內容與例外核准路徑。如此一來，紅線才不是封鎖創作的障礙，而是創作的安全工作範圍。

## 5. 快閃店是空間化的世界觀

快閃店是商業與世界觀交匯的代表性場域。粉絲前去購買物品，同時也在造訪一個世界。空間色彩、參觀動線、拍照點、展品、在售商品、員工語氣、排隊方式和線上打卡，全都會成為體驗。快閃店不只是銷售場所，而是空間化的世界觀。

好的快閃店會向粉絲提供一個能夠進入的世界。粉絲在真實空間中遇見曾在影像裡看到的象徵，把角色的物品拿在手中，與其他粉絲站在同一個地方，再用照片記錄自己的造訪。這種體驗連結了收藏、展示、歸屬與世界觀沉浸等多條體驗軸。

壞的快閃店會變成排隊買東西的場所。等待和購買讓人疲憊，空間只停留在拍照裝飾的層次，商品也與世界無關。粉絲會覺得自己只是營利對象。快閃店可以很好地展示粉絲圈的熱情，但設計薄弱時也會製造粉絲疲勞。

MEJE 式快閃店設計必須從 LOREBOOK 出發。這個空間是世界中的哪個地點？粉絲在這裡會成為什麼角色？能夠辨認哪些象徵？什麼商品與什麼記憶相連？會引導什麼粉絲行為？只有回答這些問題，空間才會成為世界觀。

## 6. AI 與數位物品的商業

AI 和數位物品讓授權與商業變得更加複雜。圖像、聲音、對話、虛擬化身物品、數位周邊與個人化內容都能迅速生成。因為沒有或只有少量實體庫存，其擴展性很強。但也正因如此，一旦缺乏標準，世界觀與權利就很容易動搖。

AI 生成周邊或個人化圖像，必須標示它是官方商品、粉絲參與結果，還是非官方遊戲。使用真實人物的面孔和聲音時，必須具備同意、權利與安全措施。即使是角色 IP，混合正典、假設與粉絲詮釋也會產生問題。數位商品雖然無形，責任卻不會因此變輕。

數位物品很適合粉絲活動。粉絲可以在虛擬化身空間中穿戴，在社群個人檔案中展示，也可以把它用作參加活動的證明。但若過度強調稀缺性與所有權，就可能產生投機感。粉絲經濟必須與投機經濟區分開來。粉絲究竟是為了與世界建立關係而擁有它，還是期待價格上漲而交易它，會讓社群呈現出不同性質。

沒有 LOREBOOK 與授權指南，AI 商業會非常危險。必須整理哪些元素可以用於生成，哪些元素需要保護，以及哪些結果可以商業化。技術發展得很快，權利與信任卻會在不知不覺中瓦解。商品越是數位化，就越需要世界觀語法。

## 7. 案例：透過世界觀解讀聯名商品

假設某個飲料品牌與偶像 IP 展開合作。簡單的聯名只會在瓶身貼上標誌與面孔，出售限定包裝。這樣也可能賣得出去。但世界觀合作會提出更深的問題：這種飲料與世界中的哪個場景相連？粉絲會在何處、以什麼方式飲用，拍攝什麼照片，又用什麼文案分享？這件商品屬於藝術家／歌手層、概念層，還是角色層？

如果這個 IP 的世界觀重視某種顏色、季節與應援儀式，聯名商品就可以把這些感受包含進來。包裝不採用簡單的面孔圖像，而使用對應活動時期的象徵；快閃空間讓粉絲感到自己進入了世界中的某個情境；數位活動則提供一條可供粉絲解讀的小線索。商品由行銷物品轉化為粉絲活動。

不過，聯名商品也有風險。外部品牌的價值可能與 IP 世界觀衝突，價格和品質可能不符合粉絲期待，真實人物的形象也可能給人一種被過度用於銷售商品的感覺。如果品牌脫離語境取用粉絲圈內部的象徵，就可能引發反彈。

好的合作會讓粉絲感到「這個品牌理解我們的世界」。壞的合作則會讓他們感到「我們的世界被賣掉了」。差異就在於是否理解世界觀語法。

## 8. MEJE 式授權與商業設計

MEJE 式授權與商業設計從 Vault 開始。必須記錄哪些象徵、角色和粉絲行為可以商品化，哪些詞條需要保護，以及哪些粉絲反應反覆出現。商品應當與 Vault 詞條相連。沒有連結的商品很難成為世界的一部分。

LOREBOOK 必須被翻譯成面向外部合作夥伴的指南。指南應當梳理世界的核心價值、可使用的顏色與象徵、禁止表達、角色語氣、真實人物保護標準，以及粉絲活動的推薦方式。合作夥伴需要的不是世界的所有秘密，而是能夠做出好商品的標準。

Storytelling100 可以測試商品情境。寫下粉絲收到商品的瞬間、展示的瞬間、與其他粉絲交換的瞬間、在快閃空間拍照的瞬間，以及透過訂閱權益檢視的瞬間，就能看見商品的意義。商品必須在使用情境中接受驗證。

粉絲活動的八大體驗軸是判斷商業品質的標準。這件商品是否帶來辨識？是否提供收藏與展示的樂趣？是否強化應援與歸屬？是否加深世界觀沉浸？還是只要求付款？無法透過體驗軸檢驗的商品，很難長久留在粉絲經濟中。

## 9. 不要把商品化劃分為善與惡

談論授權與商業時，最大的錯誤是只把粉絲圈視為購買力。粉絲固然是重要顧客，同時也是社群成員和世界的詮釋者。可以引導購買，但如果只用購買來衡量粉絲圈，世界的品質就會下降。

反過來，無條件否定商品化也是一種誤解。周邊與授權是幫助粉絲擁有、展示和記住世界的重要裝置。問題不在商品化本身，而在脫離語境的商品化、過度稀缺、侵犯真實人物，以及擅自利用粉絲勞動。

不給外部合作夥伴創造空間也是一種陷阱。過度僵硬的指南可能讓合作變得平庸。授權指南應當守住核心與紅線，同時讓合作夥伴發揮自身媒介和商品類別的優勢。世界觀語法不是控制，而是翻譯的標準。

最後，輕視數位商品也是錯誤。沒有實體庫存並不意味著責任更輕。AI 生成物、數位物品和個人化內容，與是否為官方、權利、粉絲情感和資料保護相連。商業越是數位化，就越需要標準。

## 10. 轉化為粉絲社群

當粉絲透過商品與世界建立關係時，IP 的世界便成為粉絲社群。周邊不再只是物品，而成為記憶裝置；快閃店不再只是賣場，而成為造訪世界的空間；授權合作則成為外部品牌理解粉絲世界的證據。商業並非世界觀的外部事物，而是粉絲體驗的一部分。

訂閱經濟與授權可以協同運作。會員服務可以提供進階周邊、搶先公開內容、檔案與數位物品。只要避免前文提到的壓力風險，這種結合就會成為粉絲把世界更深地帶入生活的通道。

創作者經濟也與授權相連。需要建立一種結構，允許粉絲創作，把其中一部分擴展為官方合作，並尊重創作者的權利。如果粉絲社群共同創造世界，授權就應當成為安全容納這種參與的容器。

## 11. 小結

授權與商業並不在世界觀之外。它們是把世界觀翻譯成商品、空間、數位物品、合作與訂閱權益的過程。只管理標誌與圖像的使用遠遠不夠。世界的語法、粉絲體驗軸、真實人物保護、紅線與使用場景，都必須一同設計。

要讓 IP 成為粉絲社群，商品必須成為世界的記憶。授權必須超越權利出借，成為世界觀語法的移轉；而這套語法的中心，是決定開放什麼、守護什麼的紅線。好的商品不會在賣出之後結束，而會留在粉絲圈的記憶之中。最後一章將彙集至今為止的所有分支，以透過世界觀解讀不同世界的方法結束本書。

## 核心概念

- 世界觀商業：商品、空間與數位物品連結到世界的象徵、記憶和粉絲行為的結構。
- 授權語法：為幫助外部合作夥伴安全翻譯世界觀而提供的核心標準與紅線。
- 紅線：一旦商品化，可能損害世界的信任或人的安全的表達與領域。
- 空間化的世界觀：如快閃店一般，粉絲以實體空間的形式造訪世界的體驗。
- 商品情境檢視：透過粉絲接收、使用、展示和分享商品的情境，確認商業意義的工作。
