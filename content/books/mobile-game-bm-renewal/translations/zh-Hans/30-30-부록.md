# 第30篇 附录——付费习惯迁移模式辞典＋新内容BM检查清单


## 附录1. 各行业付费习惯迁移模式对照表

本附录按行业整理本书讨论的主要BM迁移模式。


### 游戏行业迁移模式

- **20世纪70年代街机：** 游戏厅投币、台球厅、保龄球馆 → 投币计次的人均收费。小额反复支付让人觉得“这点钱不算什么”。
- **20世纪80年代家用主机：** 购买玩具、桌游 → 先买硬件＋购买软件。购买玩具作为礼物的付费习惯迁移了过来。
- **20世纪90年代PC盒装游戏：** 书店、软件商店 → 预先购买盒装软件。买书的习惯在新空间落地。
- **21世纪初网络游戏：** 网吧使用费、补习班 → 月费制、免费游玩加内购。从网吧计时收费转为按月付费。
- **2010年代手游：** iTunes单曲购买、应用购买 → 应用内购买、抽卡。从小额购买应用迁移到微交易。
- **2010年代GaaS：** 主机DLC购买 → 战斗通行证、赛季通行证。从等待新赛季迁移到预购通行证。
- **2020年代电子竞技：** 观看体育赛事、支持选手 → 购买皮肤、战队周边。体育粉丝的付费习惯迁移了过来。


### 音乐行业迁移模式

- **19世纪乐谱：** 书店、印刷品订阅 → 销售乐谱。买书的习惯在新空间落地。
- **20世纪初唱片：** 演出门票 → 购买SP／LP唱片。在家重现现场演出的体验。
- **20世纪80年代磁带：** 收听广播 → 录音、分享。家庭录音形成了免费使用的习惯。
- **20世纪90年代CD：** 购买磁带 → 按定价购买整张专辑。高音质、无损音质成为溢价理由。
- **21世纪初MP3：** 购买CD → 每首0.99美元。消除了被迫购买整张专辑的不满。
- **2010年代流媒体：** 购买MP3、CD → 按月无限收听。“既然是这个价格，就不用再为音乐支出费心。”
- **2010年代K-pop：** J-pop粉丝文化、偶像文化 → 小卡抽选、粉丝平台。收藏型付费习惯＋粉丝俱乐部会费。


### 视频行业迁移模式

- **1895年电影院：** 观看马戏、戏剧 → 购买入场券。观看现场演出的付费习惯迁移了过来。
- **20世纪50年代电视：** 收听广播、阅读报纸 → 广告支持的免费观看。以接受广告代替直接付费。
- **20世纪80年代录像带：** 图书馆、漫画租赁店 → 租赁录像带。租赁付费习惯迁移了过来。
- **21世纪初有线电视：** 地面电视 → 频道捆绑订阅。以套餐形式出售更多选择。
- **2007年流媒体：** DVD邮寄订阅 → 按月订阅流媒体。从没有滞纳金的租赁迁移到无限观看。
- **2019年OTT大战：** 订阅Netflix → 订阅IP组合包。从粉丝付费习惯迁移到平台订阅。
- **2022年AVOD回归：** 观看地面电视广告 → 含广告的低价OTT。“有广告也没关系，我要便宜版本”的需求。


### 网络漫画与网络小说迁移模式

- **21世纪初：** 漫画租赁店（每册100～200韩元）→ Naver免费连载。先以免费内容聚集读者，再转化为IP价值。
- **2013年：** 免费连载 → Kakao“等待即可免费”。把时间变成货币，为加速阅读付费。
- **2015年：** Kakao Cash → 网络小说按话付费。漫画租赁店的小额支付迁移为数字微交易。
- **2020年代：** 韩国网络漫画、网络小说 → 向全球输出IP。先验证粉丝群，再迁移到其他媒介。


## 附录2. 历史上的BM失败案例

每个案例都用一句话概括：“为什么没有可用空间？”或“为什么落地失败？”


**游戏领域**

1. **任天堂Virtual Boy（1995）：** 3D画面带来的不适大于它所解决的问题。
2. **Google Stadia（2019）：** 在云游戏付费习惯尚未形成时推出。
3. **Sega Dreamcast（1998）：** MIL-CD导致盗版扩散，收入模式随之崩溃。
4. **P2E游戏（Axie Infinity等，2021～）：** 未能完成“游戏＝劳动”的重新定义，代币经济随之崩溃。
5. **NCSoft《Lineage W》全球发行（2021）：** 主要在韩国、台湾、日本和东南亚上线，却完全没有进入北美和欧洲，从一开始就限制了全球扩张。
6. **Battlefield 2042（EA／DICE，2021）：** 尚未稳定便仓促发售，错误和服务器不稳损害了早期用户的信任，打乱赛季通行证和DLC销售计划。

**音乐领域**

7. **Soribada付费化（2004）：** 习惯免费分享的用户离开。
8. **Sony MiniDisc（1992）：** CD习惯过于牢固，新格式未能在全球市场普及，但在日本国内长期生存。
9. **Tidal（2015）：** 与Spotify没有明确差异，却采取高价定位。
10. **Apple Ping（2010）：** 作为音乐社交网络无法与既有社交媒体习惯竞争，两年后关闭。
11. **MySpace Music（2008）：** 在音乐流媒体付费习惯形成之前免费提供服务。
12. **Dreamcatcher进军韩国主流市场：** 海外粉丝与韩国国内粉丝所处的付费习惯空间不同。
13. **Google Play Music（2011～2020）：** 与YouTube Music整合时出现播放列表和数据迁移问题，导致忠实用户流失。
14. **Prince拒绝流媒体平台（2010年代）：** 排除Spotify等主要平台并转向自有网站销售，降低了可及性，加速粉丝流失。

**视频领域**

15. **Quibi（2020）：** 以付费服务进入TikTok、YouTube占据的免费短视频空间。它筹集了**17.5亿美元**，却在上线六个月后关闭，并向投资者返还约3.5亿美元。
16. **CNN+（2022）：** 以付费订阅进入习惯免费新闻的空间，三十天后关闭。
17. **YouTube Red Originals（2015～2018）：** 未能在免费平台上实现独家内容付费化。
18. **Vine（2016）：** 没有向创作者分配收入，明星创作者离开，平台随之关闭。
19. **早期Google TV（2010）：** 既没有内容，又与有线电视运营商发生冲突。
20. **MoviePass（2019）：** 每月9.95美元无限观影的定价不可持续。降价消息发布后两天内涌入15万人，而重度用户的月均观影成本高于订阅费，负毛利不断累积。
21. **Warner在HBO Max同步上映（2021）：** 与院线生态发生冲突，引发导演反对。
22. **早期NBCUniversal Peacock（2020）：** 免费含广告版本与付费无广告版本的差异不清晰，亏损持续到2023年。
23. **Paramount+（由CBS All Access转型，2021）：** 更名造成混乱并导致既有订阅者流失，除Star Trek粉丝群外也缺少王牌内容。

**平台领域**

24. **MySpace（2005～2011）：** 封闭、不灵活的UI失去竞争力，最终被Facebook取代。
25. **Second Life（2003～）：** 元宇宙付费习惯尚未形成，未能大众化。
26. **Google Glass（2013）：** 遭遇社会对穿戴式设备的排斥。
27. **Amazon Fire Phone（2014）：** 进入了iOS和Android习惯已经固化的智能手机市场。
28. **Clubhouse（2020～2021）：** 邀请制音频社交网络，后来被Twitter Spaces取代。
29. **MSN Messenger／Windows Live Messenger（2013年关闭）：** 未能适应智能手机即时通信工具，PC端习惯没有迁移到移动端。
30. **Path（2010～2018）：** 以少数亲密关系社交网络起步，却没有付费模式；在无广告经营中用户增长停滞，最终关闭。
31. **Google+（2011）：** 在缺少社交网络付费习惯的空间里，无法取代既有平台。

**韩国特有案例**

32. **Cyworld（2010年代）：** 移动化转型失败，Dotori付费习惯未能迁移到移动端。
33. **Pandora TV（2004～）：** YouTube进入后，韩国本土视频平台市场整体陷入危机。
34. **Daum Kakao TV Pot：** 独立应用缺乏竞争力，最终被YouTube吸收。
35. **Watcha（2023）：** 尝试《好好公司》等小规模原创与独家内容，却无力承受大资本推动的原创内容数量竞争，与Netflix、Wavve竞争的能力达到极限。
36. **PSY未能延续全球热度：** “Gangnam Style”之后未能重新建立全球粉丝群。
37. **Naver TV（2016～）：** 作为独立视频平台起步，却在争夺创作者时落后于YouTube。由于未能建立自身生态，事实上转型为嵌入YouTube链接的服务。

**其他行业**

38. **Kodak数字相机（1975～）：** 在公司内部发明数字相机后，为保护胶片业务而中止开发。
39. **Blockbuster拒绝流媒体（2000）：** 拒绝以5,000万美元收购Netflix的提议。
40. **Dr. Dre的Beats Music（2012～2014）：** 在竞争中败给Spotify。Apple于2014年以30亿美元收购整个Beats业务（耳机＋音乐服务），随后将Beats Music并入Apple Music。
41. **Snapchat上市后增长停滞：** Instagram复制Stories功能，导致其失去差异化。
42. **Lezhin Comics成人内容付费危机（2016）：** 受到监管直接打击，同时面临免费竞争者。
43. **AOL拨号上网（2000年代）：** 宽带转型迟缓，因而失去市场。
44. **Yahoo搜索（2009年与Microsoft开展搜索合作）：** 外包搜索技术，削弱了核心能力。
45. **Enron能源交易：** 没有真实资产支持的虚拟交易在会计造假中崩溃。
46. **WeWork（2019年撤回IPO）：** 追求无盈利的增长，提供了超出付费习惯空间承受范围的昂贵服务。
47. **Terra–Luna崩溃（2022）：** 算法稳定币的信任基础崩溃。
48. **Twitter Blue（2022～2023）：** Elon Musk收购后推出每月8美元的付费认证蓝标。原本免费的功能变为付费，机器人账号也能购买蓝标，导致信任下降；付费订阅人数远低于目标。
49. **Jawbone UP（2011）：** 大规模召回摧毁信任，市场被Fitbit夺走。


## 附录3. 新内容BM检查清单

这是阅读本书后检查自身内容BM的实用工具。请在每一项标记✓或✗。


### A. 确认原有习惯空间

- [ ] 我已经为自己的内容明确定义至少一个原有习惯空间。
- [ ] 我已经了解当前在该空间中金钱如何流动。
- [ ] 我知道谁已经占据该空间。
- [ ] 我已经找出先占者未能解决的不便。
- [ ] 我能够说明自己的内容如何消除这种不便。


### B. 设计付费结构

- [ ] 已明确定义由谁付费。
- [ ] 已定义用户为何付费（便利／稀缺／身份／归属感／安心感／能力提升／乐趣）。
- [ ] 已设计付费时机（预付／后付／自动续费／条件式付费）。
- [ ] 已尽量降低付费方式的摩擦。
- [ ] 价格不超过现有参照对象的两倍（或有必须超过的有力理由）。
- [ ] 存在重复消费结构（或清楚说明为什么不需要）。


### C. 防止失败模式

- [ ] 不以付费服务进入免费习惯很强的空间（或已经获得足够多的免费用户）。
- [ ] 如果空间中已有免费的先占者，就有清晰的商业化策略。
- [ ] 如果是平台，就为供应者（创作者／卖家）设计了收入结构。
- [ ] 收入不过度依赖单一IP或单一内容。
- [ ] 过度扩张计划不会稀释原有粉丝群的价值。


### D. 首批100人策略

- [ ] 已找到首批100人所在的具体空间（社群、平台、聚会）。
- [ ] 清楚知道最初要向这100人提供什么。
- [ ] 有从100人发展到1,000人的路径。
- [ ] 已设计促成第一次付费购买的触发点（trigger）。


### E. 御宅策略（适用于粉丝群驱动的BM）

- [ ] 已了解自身内容的核心粉丝聚集在哪里。
- [ ] 已知道他们在类似领域花费多少。
- [ ] 真实性（authenticity）来自创作者真正的热情，而不是营销策划。
- [ ] 有让粉丝共同生产价值的结构（社群、允许粉丝创作等）。
- [ ] 已考虑如何避免被视为商业剥削。


### 最终检查：能否用一句话说明？

> “身处［原有习惯空间］的［目标客户］因为［现有不便］而苦恼。我的内容通过［解决方式］消除这种不便，并以［付费方式］收取［金额］。”

如果无法填完这句话，就需要继续完善BM结构。
