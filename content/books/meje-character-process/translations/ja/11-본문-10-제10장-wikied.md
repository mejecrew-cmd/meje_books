# 第10章．関係網とキャスト：キャラクターのそばの五人

前章までに一人の人物の駆動力はすべて植えられ、シード・鏡・日常断面・ひねり・ドラマ・欠点・トラウマ・目標・葛藤が一人の中に整列しました。しかし一人の人物は、ひとりで生きて動くことはできません。他の人物のあいだに置かれて初めて、その動きは意味を持ちます。

今日は、キャラクターのそばに他の人物を配置する段階です。四人をどのように分配するか、その四人がキャラクターとどのような機能的役割を持つか、四人すべてを識別できるかを点検します。

## キャラクターは関係の中でのみ生きて動く

まず、姿勢を確認しておきます。

一人の人物が単独で作品を引っ張れる場合は、きわめてまれです。一人称の短編やモノローグなら別ですが、一般的な作品は複数人物の関係の中で位置づき、その関係が一人の人物を立体的にします。一人の人物がどのような人間かをもっとも鮮明に表すのは、その人物の行動そのものではなく、他者と結ぶ関係の質感です。

私たちはキャラクター一人を作るとき、その人物のそばにいる五人を同時に作ります。五人を単に並べるのではなく、五軸のマトリクスで分配します。同じマスに二人が重ならないように散らします。分散がうまくいけば、五人は互いに異なる質感を持ち、識別可能になります。

今日はその分配の手順を扱います。

## 関係網の二つの様式

関係網を描くときは、二つの様式のどちらかを採用します。作品の性格によって枝分かれが変わります。

**家系図様式。** 血縁中心です。2〜3世代を一か所に描きます。大規模な家族叙事・王朝もの・シリーズものに適しています。長所は家の歴史と人物間の権力関係が一目で入ること。短所は血縁以外の関係（友人・同僚・恋人・ライバル）がつかみにくいことです。

**『インサイド・ヘッド』型マインドマップ様式。** キャラクター一人を中央に置き、周辺人物をキーワードごとに配置します。家族でなくても構いません。メンター・ライバル・親友・宿敵・恋人のようなキーワードで置きます。一般作品に適しています。長所は多様な関係が一目で入ること、短所は世代間の関係がつかみにくいことです。

作品に応じて二つの様式の一方を採るか、両方を使います。Mの作品にはマインドマップ様式が適しています。家族関係は核ですが、母は不在で家系図がほぼ空になり、同僚・検事・後輩のような職場関係が核なので、マインドマップのほうが自然です。

```
Mのマインドマップ（要約）：
 [母、不在]
 │
 [同僚監察医] ── M ── [検事ライバル]
 │
 [後輩] ── Mの日常 ── [AI検視補助]
```

五人がMを囲んでいます。各人物の横に二行のメモを付けます。一行目はMとの関係キーワード、二行目は核心となる葛藤を一行で記します。

## エグリの三次元・距離点検

五人を配置した後、一つ確認します。各人物がMからどのような距離にいるかです。

三つの次元で測定します。

生理的次元は性別・年齢・身長・外見・健康・遺伝を、社会的次元は出身・教育・宗教・政治的傾向・趣味・国籍・家族を、心理的次元は性格・複合・強迫・トラウマ・熱望・幻滅・道徳律を捉えます。各次元でMとその人物の距離を、近い・中間・遠いの三段階で示します。M（55歳女性）と同僚監察医（55歳男性）の距離は、生理的次元では中間、同じ大学病院の検視科で30年ともに働いた社会的次元では近く、私的領域を避けるM対私的領域に積極的な同僚という心理的次元では遠いです。

この表が、Mと同僚監察医の関係の質感を決めます。社会は近く、心理は遠い一組です。30年ともに働いたのに、私的領域では正反対の姿勢。この違いが二人のあいだに葛藤の可能性と相乗効果の可能性を作ります。

一つの原則です。**一つの次元でも距離が大きい次元がなければ、葛藤の可能性はありません。** すべての次元が近ければ、二人は同じ人になります。同一人物の鏡でしかありません。一つの次元ででも距離が開いていれば、二人は識別されます。

この点検はAIの仕事仲間とともに一次評価を受け、プロデューサーが検収します。

## 関係統合シート

夫婦・兄弟・同僚のように、二人がいつも一緒に行動する場合があります。このような二人には、個別シートのほかに統合シートを置きます。

```
[キャラクターシート]/
├── A.md ← 個別シート（真実の源）
├── B.md ← 個別シート（真実の源）
└── A_B（統合名）.md ← 統合シート（共同叙事のみ）
```

統合シートのヘッダー様式：

```markdown
# A & B（統合名）：関係統合シート

> 共同叙事の役割のみ。個別シートが真実の源。
> 個別：[A.md] / [B.md]
```

原則：個別情報を重複させないこと。統合シートには、二人がともに行動するときの質感だけを入れます。各自の個別の質感は個別シートに置き、統合では参照だけにします。

Mの作品には統合シートはあまり合いません。同僚監察医とは30年ともに働きましたが、私的には距離が遠いからです。統合シートより二つの個別シートとして置くほうが自然です。

Eの作品には統合シートが合います。Eと、死亡した兄弟のようなCは、7歳の時点ではいつもともに行動していました。Eの個別シート、Cの個別シート、そして「E・C 7歳時点統合シート」の三つをともに組みます。

## 人物の5分類

キャラクターのそばにいる人物を、すべてフルビルドするわけではありません。5分類に分け、各分類を異なる深さで組みます。

A 主・助演（作品の主軸人物、主人公・ライバル・メンター）はフルビルド（12段階すべて）で処理します。B 端役（1〜2場面の登場、通行人・職員）は短縮経路です。C 言及のみ（本文には登場せず会話で言及、例「母がいつも言っていた…」）は一行メモです。D 神話・歴史人物（作品外部の実在、アポロン・ナポレオン）は外部参照リンクです。E アレゴリー・象徴（抽象概念の擬人化、運命・死）は別キーワードとして登録します。

5分類に分けると、作業時間が合理化されます。5分類のうちAだけをフルビルドし、Bは短縮、Cはメモ、Dは参照、Eは別枠です。30人が登場する作品でAが5人程度なら、フルビルドは5人だけに適用されます。

Mのキャスト：

```
A 主・助演：
- 同僚監察医（Foil + Confidant）
- 検事ライバル（Opponent）
- 後輩（Innocent）
- AI検視補助システム（Wildcard、第13章の非定型事例）

B 端役：
- 遺体の遺族（一場面の登場）
- 検視科長（一〜二場面の登場）

C 言及のみ：
- 母（作品を通して言及されるが、本文には登場しない）

D・E：（該当なし）
```

Aが4人、Bが2人、Cが1人。M自身を含めるとキャストは8人です。フルビルドはM以外の4人だけに適用されます。

## キャラクターの機能的役割：三つの物語モデル

キャラクターを関係網に配置するとき、その人物が作品内でどのような機能的役割を果たすかを決めます。三つの物語モデルのうち、作品ジャンルに合う一つを選び、マッピングします。

原型（アーキタイプ、archetype）とは、複数の物語に繰り返し現れる人物の基本的枠組みです。師、門番、影、トリックスターなどです。心理学者カール・ユングが人類共通の無意識のイメージを指して用いた言葉ですが、作劇では人物の「機能的役割」を素早くつかむ欄として借ります。一人の人物が二つの原型を兼ねることもあります。

よく組まれたキャストでは、人物ごとに明確な機能的役割があります。『SLAM DUNK』の湘北の五人の先発を見ると、桜木花道は無鉄砲ながら盤面をひっくり返す主人公、流川楓は彼とぶつかることで輝くライバル、赤木剛憲はチームを支える柱、三井寿は戻ってきた放蕩息子、宮城リョータは流れをつなぐ調律者です。五人が同じコートに立つと、それぞれが空白を埋めて一つのチームになります。

**グレマスの6行為項**は、もっとも単純で普遍的な道具です。Subject（主体、追求する者）、Object（対象、追求されるもの）、Sender（発信者、追求を始めさせる者）、Receiver（受信者、結実を受け取る者）、Helper（助力者）、Opponent（敵対者）の六つです。Mの作品では、M=Subject+Receiver、母の最期=Object（不在そのもの）、遺体の手の結婚指輪=Sender、同僚監察医・後輩=Helper、検事ライバルとM自身の30年の姿勢=Opponentです。一人のキャラクターが二役を同時に占めることもでき（Mの自己敵対者）、マッピング後に空いている役割があれば、作品の駆動力は弱くなります。

**プロップの7領域**は、英雄叙事・童話・ゲームNPCに適しています。Hero（=グレマスのSubject）、Villain（=Opponent）、Helper、Donor（魔法の道具を与える者、Helperの一部）、Sought-for Person（=Objectの一部）、Dispatcher（=Sender）、False Hero（英雄を装う者）の七つです。Mの作品は英雄叙事ではないため6行為項のほうが合い、Eの作品（戦争世界観）には7領域が合います。

**ヴォグラーの7 archetype**は、キャンベルの英雄の旅から抽出された心理的原型で、ハリウッド・ウェブ小説・ゲームシナリオの業界標準です。Hero（主人公）、Mentor（師匠・教師）、Threshold Guardian（門番・最初の関門）、Herald（使者・招集者）、Shapeshifter（変身者、両義的な人物）、Shadow（影、英雄の暗い面を映す鏡）、Trickster（トリックスター、混乱・笑い・転換）の七つです。同じ敵対者でも、グレマスのOpponentとヴォグラーのShadowは質感が異なります。Shadowは単純な外部の敵ではなく、英雄の暗い面を映す鏡です。一人のキャラクターが二役を占めることも可能です（例：『スター・ウォーズ』のヨーダ = Mentor + Trickster）。英雄叙事・ファンタジー・ウェブ小説に強く、一般ドラマにはグレマス／プロップ＋補助archetype 7種で十分です。

Eの作品をヴォグラーでマッピングすると、E=Hero、肩に星を付けた男=Mentor（前世のメンター、回帰後に再登場）、組織のボス=Shadow（Eの暗い面の極端）、7歳の人質の子=Herald+Threshold Guardianです。Shapeshifter・Tricksterは作品コンセプト上空いていますが、意図された偏りです。一つの作品が七役すべてを埋める必要はありません。

三つのモデルすべてについて、マッピングはAIの仕事仲間とともに一次評価を受け、プロデューサーが検収します。

## 補助 archetype 7種：定型外のキャラクター

典型的な主人公・悪役以外に、作品を豊かにする七つの補助archetypeがあります。

『ルパン三世』の五人は、群像設計の教科書とよく言われます。飄々とした泥棒ルパン、寡黙な名射手次元、堅物の剣客五ェ門、つかみどころのない不二子、粘り強い追跡者銭形。五人はそれぞれ独立した個性を持ちながら、互いの隙間を埋めます。一人でも欠ければ均衡が崩れる、よく分散したアンサンブルです。シットコム『フレンズ』の六人の友人も、強迫的なモニカ、風変わりなフィービー、冗談で武装したチャンドラーのように、各自が異なるマスを占め、誰も重ならない均衡を作ります。

> **クローズアップ・『マスエフェクト』のノルマンディー分隊** RPG『マスエフェクト』で主人公シェパードは、宇宙船ノルマンディーに異星種族の仲間を一人ずつ乗せます。狙撃に長けたトゥーリアン、機械を扱うクォリアンの整備士、古代知識のアサリ学者、好戦的なクローガンの戦士。種族も性格も価値観も重ならないように分散され、同じ事件の前でそれぞれ異なる反応が出ます。しかもプレイヤーは彼らの関係と忠誠心を直接管理します。五軸でキャストを散らして配置し運用するという本章の作業を、ゲームが丸ごとシステム化した事例です。

Foilは主人公と正反対の性向で対比し、主人公の特質を強調します（主人公が慎重なら衝動、節制なら耽溺）。Confidantは主人公が秘密を打ち明ける人物です（聞く・助言する・沈黙する）。Comic Reliefは緊張を和らげ笑いを供給します（滑稽な行動・言葉）。Catalystは事件を触発しますが、自身は影響を受けない人物です（事件の中心に登場して去る）。Witness/Narratorは事件の観察者で、直接行動しません（記録・証言・観照）。Innocentは道徳的中心を保つ者です（純粋・一貫性）。Wildcardは予測不能で陣営を頻繁に変える人物です（即興的行動・裏切り・救済）。

七種のうち、キャラクターの作品に合う2〜4種をキャストに配置します。Mのキャスト：

```
- 同僚監察医：Confidant + Foil。30年の同行者であり、私的領域では正反対。
- 検事ライバル：Opponent。Mの手続き違反を察知する。
- 後輩：Innocent。新しく赴任した監察医。道徳的中心を保つ。
- AI検視補助システム：Wildcard。本文に登場するたび異なる役割。
- 母（不在）：Catalyst。すべての事件の原因だが、本人は不在。
```

Mのキャストの五人は、全員が異なるarchetypeを持ちます。Confidant・Foil・Opponent・Innocent・Wildcard・Catalyst。五人はそれぞれの役割から、Mの一面を照らします。

## Opponentの4機能：敵対者の役割分解

キャストにはOpponentが一人以上必要だという原則を確認しました。そのOpponentがどのような役割で働くかを、もう一度分解します。

### 背景

アメリカの脚本家ロバート・マッキーが『Story』で整理したOpponentの役割分析です。Opponentは単純な敵ではなく、マッキーが**主人公のTruthを取り出す助産師**として置いた存在です。この段階でOpponentの4機能が分かれます。

本当の敵対者は単なる悪役ではなく、主人公のもっとも深い価値を脅かす鏡です。『DEATH NOTE』の夜神月と名探偵Lがそうです。二人とも天才で、二人とも自分の正義を確信しています。まさにその似姿ゆえ、互いが互いのもっとも危険な敵になります。Lは月の外に立つ敵であると同時に、月がなり得たもう一人の自分です。このような敵対者が、物語をもっとも遠くまで押し出します。

### 導入の趣旨

キャストにOpponentが一人いるからといって、その人物がOpponentのすべての機能を占める必要はありません。4機能のうち一人が二つを占め、残り二つを別の人物が占めることもあります。マッキーの分解を通すと、その分担が一目でつかめます。

### 4機能

Opponentの4機能は、1）目標を阻む（主人公のWantを直接止める外部障害）、2）Lieを強化する（主人公が自分のLieをさらに強く信じるようにする内的罠）、3）Truthを強制的に見せる（主人公のLieが嘘であることを暴く役割）、4）鏡になる（主人公の暗い面を映すドッペルゲンガー、反転した自己）です。

Opponent一人が4機能すべてを占めれば、その人物は作品の核心的敵対者になります。二人が分担することもでき、ある機能は別のarchetype（Mentor・Foilなど）が変形して占めることもあります。

Opponentが4機能のうち二つ以上を果たせば、敵対の役割はより立体的に働きます。一機能だけを果たす人物は単純な障害物として設計することもあり、作品の必要に応じて比重を調整します。

Mの作品に適用してみます。

```
- 検事ライバル：1番（目標を阻む）＋3番（手続き違反を察知しTruthを強制的に見せる）
- M自身の30年の姿勢：2番（母の不在の領域に自分を縛り、Lieを強化する）
- 遺体の、母ではない別の人：4番（鏡、Mが30年間不在だと信じてきた領域の反転）
```

4機能は三人に分散されます。検事ライバルが外部敵対者の二機能を、M自身が内部敵対者の一機能を、遺体が鏡の一機能を担います。一作品の敵対役割が一人物に集中せず、分散して立体的に働きます。

## キャスト分配の3原則：対照・補完・調和

五人を分配するとき、三つの原則を適用します。

対照原則は正反対の性向を組ませるため、ペア（2人）・三角（3人）のパターンに強く、補完原則は弱点・強みが補い合う4〜6人のアンサンブルに強く、調和原則は同じ価値を共有しながら表現方式だけが異なる同一陣営・同僚グループに強いです。

キャスト全体では3原則をすべて使います。どの二人も一原則だけで結ばれるなら、キャスト全体が平板になります。一組に二原則以上がかかってこそ立体感が生まれます。

複数人物を一マスずつ異なる場所に置く発想は、合唱とゲームパーティ編成に似ています。合唱はソプラノ、アルト、テナー、ベースがそれぞれの音域を担ってこそ和音が満ち、ロールプレイングゲームのパーティはタンク、ダメージディーラー、ヒーラー、サポートが分かれてこそ動きます。五人の性格も価値も感情も重ならないよう散らせば、同じ舞台に上がったとき初めて和音になります。

Mのキャストに適用：

```
- M ↔ 同僚監察医：対照（私的領域で正反対）＋調和（職業価値は同じ）
- M ↔ 検事ライバル：対照（Mは手続きを回避し、ライバルは手続きに強迫的）
- M ↔ 後輩：補完（Mの30年の熟練と後輩の新鮮さ）
- M ↔ AI検視補助：補完（Mの直観とAIの正確さ）
```

三原則がすべて使われました。キャストが平板化しない結果です。

## N別パターン

キャストの人数によって、推奨パターンは異なります。

単独（1人）はフルビルドだけ（キャストモジュールは使わない）。ペア（2人、Foil pair）は正反対のMBTIに同じホグワーツ、またはその逆の分配。三角形（3人）は異なるホグワーツ三つで葛藤→和解または三者均衡。アンサンブル（4〜6人）は4ホグワーツ分配に外部者1〜2人。大規模キャスト（7〜12人）は4ホグワーツ×2にアンチヒーロー・ジョーカー1〜4人。群衆＋核心（13人以上）は主・助演4〜6人をフルビルドし、残りは短縮経路です。

Mのキャストは5人（M＋4）なので、4〜6人アンサンブルのパターンです。4ホグワーツが分配されるべきです。

```
Mのキャストのホグワーツ分配：
- M：ハッフルパフ
- 同僚監察医：グリフィンドール（直接的・決断的）
- 検事ライバル：スリザリン（戦略・野心）
- 後輩：レイブンクロー（知識・創造性）
- AI検視補助：（ホグワーツ外部）、第13章の非定型
```

4ホグワーツがすべてカバーされています。一つのホグワーツに二人以上が集まれば、キャストは平板になります。

## 5軸マトリクス：分配点検

キャスト分配の核心ツールが5軸マトリクスです。五人が五軸でどう重なり、どう分かれるかを点検する様式です。

```
五つの軸：
1. MBTI 4象限（NT・NF・SF・ST）
2. ホグワーツ4寮
3. 変化アーク（Positive・Flat・Negative）
4. 核となるステークス（物理・職業・アイデンティティ・関係・存在）
5. 五感情の優勢（喜び・悲しみ・怒り・嫌悪・恐れ）
```

Mのキャスト5軸マトリクス：

MはSF、ハッフルパフ、Positiveアーク、アイデンティティのステークス、五感情のうち悲しみです。同僚監察医はST、グリフィンドール、Flatアーク、関係のステークス、嫌悪。検事ライバルはNT、スリザリン、Flatアーク、職業のステークス、怒り。後輩はNF、レイブンクロー、Positiveアーク、存在のステークス、喜び。AI検視補助は外部人物で、物理のステークス、恐れです。

五軸で五人はすべて異なるマスを占めます。空くマスもありますが、二人が一マスを独占しません。キャストが立体的に分散された結果です。

このマトリクス分配は、AIの仕事仲間とともに草案候補を素早く比べるのに適した部分です。シードと五つの尺度の結果を入力すれば分配候補を作れ、プロデューサーは作品のトーンに合うかを検収して採用します。

一つの原則です。**5軸分配は推奨であり、強制ではありません。** 作品コンセプトによって一軸に集まることがあります（例：全キャラクターがスリザリンである政治ドラマ）。意図した偏りならOKです。意識して分配するかが重要です。

## キャスト一貫性の7チェック

キャスト分配が終わったら、七つを点検します。

```
1. 各キャラクターがWalks-into-a-room Testを通過（5秒で識別）
2. キャスト全体が4ホグワーツをすべてカバー（4人以上の場合）
3. 少なくとも1 Positive + 1 Negative arcが共存（3人以上の場合）
4. 各キャラクターの核となるステークスが異なる（5人以上なら5階層すべてをカバーするよう試みる）
5. キャラクター間の三次元距離を点検：すべての組に一つ以上、距離が大きい次元がある
6. One-Line Test：各キャラクターは一行で他のキャラクターと区別できるか
7. キャスト内にOpponentが1人以上（Lieを揺さぶる人物）
```

この7チェックはAIの仕事仲間とともに一次評価を受け、プロデューサーが決めます。七つのうち二つ以上が✕なら、分配をやり直します。

## モジュール5との結合

キャスト分配は、アイデンティティ・多様性モジュール（モジュール5）と結びつく部分でもあります。5軸に加えて、七次元の多様性をキャストに分布させる点検を追加します。

```
キャストの多様性点検：
- 性別分布：M（F）、同僚（M）、検事（M）、後輩（F）、AI（N）、均衡OK
- 年齢分布：55・55・40・28・N、多様
- 社会経済的背景：M（中産）、同僚（労働者）、検事（上流）、後輩（中産）、AI（N）、多様
- 神経多様性：人間キャラクター4人は全員、別設定なし、単一（追加検討が必要）
```

このような点検を意識しておくと、キャストの多様性が自然に分布します。一人の人物にすべての多様性を集めず、五人に七次元が適切に分布するようにします。

## Eのキャスト抜粋

もう一つ別の例です。EのキャストはMと異なります。

```
Eのキャスト（3人）：
- C（死亡、兄弟）、三次元：すべての次元が近い → 葛藤なし（関係終了）
- 「肩に星を付けた男」（メンター）、社会・心理の距離が大きい
- 7歳の人質の子（Phaseの核心衝突）、すべての次元で距離が大きい

ホグワーツ分布：
- E：スリザリン
- 「肩に星を付けた男」：スリザリン（同じ陣営）
- 7歳の子：（幼いためホグワーツ分類は弱い）

5軸マトリクス：ホグワーツが二人同じ。意図した偏り、同じ陣営内の権力階層。
```

Eのキャストは5軸分配が推奨と異なります。意図した偏りです。作品コンセプト（権力階層の中の悲劇）に合わせ、一軸が集まります。
