# 第8章　欠点・欠如・トラウマ――道徳的グレーの領域まで

前章までに、キャラクターには肉づけがなされました。行動・身体アンカー・前史・能力・嗜好という五つの道具によって外形は整いましたが、肉がついただけでは、まだ生きて動く人物にはなりません。動き出すには動力が必要です。

本章から第三の段階群、「動力を植える」作業が始まります。動力は、欠点・トラウマ（第7段階）、目標（第8段階）、葛藤（第9段階）の三段階で植えられます。今日はその最初、欠点・欠如・トラウマを扱います。

## 欠点はキャラクターの魅力です

まず、欠点に向き合う姿勢を確認しておきます。

この作業を初めて行う人は、しばしば同じ間違いを犯します。キャラクターに欠点を与えることをためらうのです。自分の手で作った人物だからこそ、傷ひとつない人であってほしい。その思いが強いほど、キャラクターは平板になります。

欠点のない人を愛する読者はいません。人は、どこか一面に傷のある人物を愛します。その傷こそがその人の魅力です。どこまでが欠点で、どこからが魅力なのか、その境界が曖昧になる場所でキャラクターは生きて動き始めます。

長く愛される人物は、ほとんど例外なくどこか角があります。『高慢と偏見』のダーシーは題名どおり高慢で、米国ドラマ『Dr.HOUSE』のハウス医師は天才的でありながらひどく無礼です。『プラダを着た悪魔』のミランダは冷たく非情です。傷ひとつない滑らかな人物より、尖ったところがはっきりした人物のほうが長く記憶に残ります。その角が、人物をつかむ取っ手になるからです。『カールじいさんの空飛ぶ家』のカール・フレドリクセンのように、妻を失った傷から無愛想になった頑固な老人がかえって愛らしいのも同じ理由です。

私たちは欠点を積極的に設計します。一人の人物に二つの欠点を意図して植え込みます。それが後続段階の「目標のずれ」と葛藤設計の動力になります。欠点がなければ葛藤はなく、葛藤がなければ変化はなく、変化がなければ生きた動きもありません。

欠点を設計するとき、あらかじめ二つのことを定めます。

第一に、**欠点は絶対悪ではありません。** 状況によって長所へ転換されます。嘘の巧みな人物が斥候になれば、その嘘は生存の資産になります。優柔不断な人物が危機交渉に入れば、その優柔不断さは全員の意見を聞く姿勢へと変わります。欠点の両面性を先に定めておけば、キャラクターの立体感が保たれます。

第二に、**欠点は欠如から生まれます。** そして欠如はトラウマから生まれます。この三つは一つの鎖で結ばれます。トラウマ → 欠如 → 欠点。欠点を単独で決めず、鎖の末端に置くのです。そうして初めて、欠点は作品内で一貫性を持ちます。

## 欠点の4軸――欲求・信念・道徳律・価値観

欠点は四つの軸で捉えます。欲求軸では何を得たいのか（承認・統制・安全・自由など）、信念軸ではどの行動を道徳的／非道徳的と考えるのか（正直・服従・反抗・実利）、価値観軸では何を最上の価値とするのか（家族・仕事・自己実現・正義）、克服方法軸ではどの人物や状況が欠点を崩すのか（信頼できるメンター・過去事件の再現）を点検します。

欠点を一つ決めたら、四つの軸を埋めます。この作業はAIの仕事仲間と進める部分です。シードと人物の質感を入力すれば、欠点候補と4軸の一次案が得られます。プロデューサーは欠点を一つ採用し、各軸の細部を検収します。

## 8つの欠点カタログ

よく使われる八つの欠点をカタログとして用意しています。

```
1. 不正直：嘘が巧み
2. 反抗：権威に無条件で抵抗する
3. 依存：自尊心が低く、他者の承認を渇望する
4. 傲慢：承認欲求が過度
5. 反社会性：共感の欠如
6. 偏向：自分の集団にだけ忠誠を尽くす
7. 怠惰：推進力の欠如
8. 虚栄心：他者の視線を過剰に意識する
```

八つを一か所に置くのは、欠点の決定を速めるためです。シードと質感を見て、最も合う一つか二つを採用します。

**Mの欠点・第1候補：** 私的領域の回避（反社会性＋怠惰の変奏）。三十年ともに働いた同僚の名さえ知らないほどです。母の不在が、私的関係の回避として固まった結果です。

**Eの欠点・第1候補：** 冷淡さ（反社会性＋依存の変奏）。依頼が入れば、動機や事情を無視して遂行します。私的関係も避けます。

**Jの欠点・第1候補：** 強迫的な完璧主義（依存＋虚栄心の変奏）。文章を書く前に手が赤くなるまで洗います。成果物は面白くなければ価値がない、という自己不信です。

三人の欠点はいずれも、カタログの一項目からそのまま出てきたものではありません。カタログを足場に、キャラクターに合う変奏を組み立てています。カタログは出発点であり、終着点ではありません。

## 両面性――欠点が長所へ転換される地点

欠点を一つ決めたら、その両面性を一行で整理します。

```
Mの欠点の両面性：
- 欠点：私的領域の回避
- 長所への転換：解剖作業への絶対的な集中。同僚の私的領域にも踏み込まないため、三十年間の同行が可能だった。

Eの欠点の両面性：
- 欠点：冷淡さ
- 長所への転換：請負任務を遂行する中核資産。依頼人の動機や事情に流されない。

Jの欠点の両面性：
- 欠点：強迫的な完璧主義
- 長所への転換：推敲で大きな利点となる。音読による検収、冗長な文体の自己点検。
```

この一行がキャラクターの立体感を保ちます。欠点だけを強調すれば人物は否定形になり、長所への転換も同時に見せれば、人物は立体的に生き始めます。

この一行はプロデューサーの直接作業です。AIの仕事仲間から候補を受けることはできますが、どの両面性が人物に合うかを判断するのは、作品のトーンを知るプロデューサーです。

## Big Tとsmall t――二種類のトラウマ

トラウマは二種類に分けて設計します。Big Tは日常を越える巨大事件（戦争・災害・レイプ・幼少期の性的虐待）、small tは自尊心を損なう日常的経験（繰り返されるからかい・教室での失敗・いじめの目撃）です。

『セッション』のアンドリュー・ニーマンを見ると、トラウマと欠点、才能が一つの根から育つことがわかります。「偉大になれなければ愛される資格もない」という承認の欠如が、彼を狂気じみた完璧主義へ追い込みます。しかし、その執着こそが手から血が出るまでドラムを叩かせる力でもあります。傷が欠点を生み、その欠点が同時に強みとなる典型です。

この分類は本作業以前からトラウマ臨床で使われてきました。smallは頻度が高いという意味であり、苦痛が小さいという意味ではありません。小さな自尊心の損傷も、繰り返されれば巨大事件に劣らない重いトラウマになります。

三人のトラウマは次のとおりです。

**M：** Big T。七歳のときに母がいなくなった事件。母がどこへ行ったのかわからないまま三十年を過ごしました。Mのあらゆる欠点はこの事件から始まります。

**E：** Big T。七歳のとき、きょうだい同然だったCが敵軍に殺される場面を目撃。自分がもっと強ければCは生きていた、という偽りの信念の出発点です。

**J：** small t。繰り返される自己不信。書き始めたころから「自分の文章には価値がないかもしれない」という疑いが儀式化しました。一度の巨大事件ではなく、日々の小さな疑いが蓄積した結果です。

トラウマの分類はプロデューサーの直接作業です。AIの仕事仲間から候補は得られますが、Big Tとsmall tのどちらが人物に合うかは、キャラクターの質感を知るプロデューサーが判断します。

## トラウマ・欠如・欠点の鎖

トラウマと欠点が決まったら、その間に欠如を差し込み、鎖を組みます。一行の型があります。

> **「キャラクターは［トラウマ］によって［欠如］を抱え、それを隠すために［欠点］を発達させた。」**

この一行にキャラクターの動力が集まります。

**Mの鎖：**

> Mは七歳で母がいなくなった事件によって私的関係への信頼を欠くようになり、それを隠すために私的領域の回避という欠点を発達させた。

**Eの鎖：**

> Eは七歳でCが殺される場面を目撃したため、誰かを愛せば失うという欠如を抱え、それを隠すために冷淡さという欠点を発達させた。

**Jの鎖：**

> Jは繰り返される自己不信によって成果物の価値を自分自身の価値から切り離せないという欠如を抱え、それを隠すために強迫的な完璧主義という欠点を発達させた。

三つの鎖はそれぞれ一行ですが、人物のあらゆる行動・決定・関係の根がそこに集まっています。この一行が決まらなければ、第8段階の目標と第9段階の葛藤がずれます。決まれば、次の段階は自動的に整列します。

ドゥニ・ヴィルヌーヴ監督『ブレードランナー 2049』の主人公Kは、自分がレプリカントであるという事実を抱えて生きています。それは彼を縛る欠点であると同時にアイデンティティの核であり、物語を最後まで運ぶエンジンでもあります。欠点は消すべきものではなく、人物を動かす動力として扱うべきだとよく示しています。

> **クローズアップ――『The Last of Us』のジョエル** ゲーム『The Last of Us』のジョエルは、世界が崩壊した日に娘を失います。その傷が欠如（誰にも情を移さない）を生み、欠如が欠点（他者を突き放す無愛想さと、ためらいのない暴力）として固まります。トラウマから欠如へ、欠如から欠点へ続く鎖が、一人の人物にそのまま刻まれています。少女エリーとの出会いによって鎖が揺らぎ始めると、ジョエルのあらゆる選択に重みが生まれます。欠点こそが人物を動かす動力だという本章の命題を、最も重厚に示す例です。

この一行の一次圧縮は、AIの仕事仲間と組み立てる部分です。トラウマ・欠如・欠点の三要素を入力すれば一次案が出ます。プロデューサーは検収し、キャラクターの質感と一致するかを確認します。

## 安易なトラウマ利用を避ける

トラウマを設計するときには落とし穴があります。トラウマを刺激の道具として消費することです。巨大事件を刺激的に投げ込み、その結果を便利な動力として使う姿勢が、トラウマの安易な利用です。

私たちは次の六項目を点検します。

```
1. トラウマは刺激的だが、キャラクターに統合されていないか？
2. 加害者が動機のない絶対悪になっていないか？
3. トラウマがあらゆる行動を説明する万能鍵になっていないか？
4. 克服するか、克服しないかを意図して選んだか？
5. 被害者性を装飾として消費していないか？
6. 「極端な危険に陥れ、そこから脱出させる」だけに単純化していないか？
```

一つでも「はい」なら設定を補正しなければなりません。安易に使われたトラウマは必ず見抜かれます。読者は想像以上に敏感です。見抜かれればキャラクターへの信頼が崩れ、その人物は生きて動けなくなります。

この点検はプロデューサーの直接作業です。AIの仕事仲間から一次評価は得られますが、作品のトーン・時代・読者に対する責任はプロデューサーが負います。

## 道徳的グレー――一般的な欠点だけでは足りないとき

一般的な欠点の4軸で十分な人物もいれば、それでは足りない人物もいます。悪役・アンチヒーロー・道徳的にグレーな人物は、通常のカタログだけでは捉えきれません。その場合は二つの追加尺度を呼び出します。

**アンチヒーロー（anti-hero）** は主人公の位置にいながら、英雄らしくない人物です。正義感や高潔さではなく、欠陥・私益・冷笑を抱えて物語を導きます。『タクシードライバー』のトラヴィス・ビックルやデッドプールが例です。観客は応援するというより理解するようになり、その微妙な距離が道徳的グレーの魅力です。

**Dark Triad：** 自己愛・マキャベリズム・精神病質の三次元。各0～10点。合計15以上なら明白な悪役、5～15なら道徳的グレー、5以下なら一般的な欠点の次元で十分です。

**HEXACO：** Big Fiveに正直さ・謙虚さ（H次元）を加えた6因子モデル。Hが低いほど道徳的グレーの領域に入ります。倫理的葛藤が中心の作品に適しています。

Eに二つの尺度を適用してみます。

**EのDark Triad：** 自己愛3点（自己承認欲求が弱い。生き残るのは自分のためではなく、再び失わないため）、マキャベリズム7点（依頼人の動機を無視し、結果だけを出す。戦略的な無道徳）、精神病質6点（共感と身体接触を避けるが、衝動性は低い）。合計16点で、明白な悪役に近い位置です。

**EのHEXACO H次元：** 4点。正直さも謙虚さも弱いものの、限りなく低いわけではありません。道徳的グレーの中央です。

この二つの尺度により、Eは単純な殺し屋ではなく道徳的にグレーな人物になります。合計16点でも、0～10のうち自己愛が3、正直さ・謙虚さが4という細部が、Eを別の殺し屋から区別します。

二つの尺度は全キャラクターには適用しません。通常の主人公なら一般的な欠点の4軸で十分です。悪役・アンチヒーロー・道徳的グレーの人物だけに呼び出します。

## Schwartzの10の普遍的価値――肯定的価値の優先順位

欠点が否定形の動力なら、価値は肯定形の動力です。一人の人物は何を最も重要だと考えるのか。心理学者シュワルツが六十数か国のデータから抽出した十の普遍的価値は円環状に配置され、隣接する価値は両立でき、向かい合う価値は衝突します。価値葛藤劇（家族対自己実現、伝統対改革など）では、表を開けば葛藤の輪郭が一目でわかり、第8段階の目標と第9段階の葛藤の足場になります。

十の価値は、Power（権力、社会的地位・統制）、Achievement（達成、個人的成功）、Hedonism（快楽、楽しみ・感覚的満足）、Stimulation（刺激、興奮・新規性）、Self-Direction（自律、自律性・創造）、Universalism（普遍、万人の福祉・自然）、Benevolence（慈善、身近な人の福祉）、Tradition（伝統、慣習・宗教・文化）、Conformity（順応、規範の遵守）、Security（安全、安定・秩序）です。向かい合う組が作品の葛藤軸になります。Self-Direction ↔ Conformity、Universalism ↔ Power、Tradition ↔ Stimulation。

Mの価値の優先順位：

```
中核価値：Tradition（伝統、三十年間の解剖室で守った正確な手順）＋Benevolence（慈善、遺族への責任）
拒否価値：Stimulation（刺激、変化を避ける）＋Power（権力、社会的野心が弱い）
葛藤軸：Tradition vs Self-Direction（母の指輪という私的領域で、手順違反を自ら決める）
```

Mの作品内葛藤は、まさにこの衝突から起こります。三十年間守ったTraditionから離れ、Self-Directionの決断を初めて下す瞬間が作品の中核動力です。

この尺度も全キャラクターには適用しません。価値の衝突が中心の作品だけに呼び出します。一般的なドラマやロマンスなら欠点の4軸で十分です。

## Sol Stein――出血する傷

最後にもう一つ道具があります。トラウマは一回限りの事件ではなく、毎日新たに出血する傷でなければならない、という作法です。

塞がった傷（弱い）と出血する傷（強い）を比べてみます。「Mは幼いころ母を失った」という塞がった記述は弱く、「毎年、母がいなくなった日の明け方に、解剖台の前で一時間ただ立っている」は出血する記述です。Eなら「七歳でCが殺されるのを見た」という塞がった記述ではなく、「雨の日に肩の火傷がうずくたび、その部分を手で押さえる」が出血です。Jなら「自分の文章への疑いを克服した」ではなく、「毎日書く前に、疑いのため手が赤くなるまで洗う」が出血です。

塞がった傷と出血する傷の違いは一語、時制です。塞がった傷は過去で、出血する傷は現在です。トラウマが作品の中で生きて動くには、現在進行形でなければなりません。

人物の傷を作品全体で繰り返し変奏する方法は、音楽のライトモティーフに似ています。ワーグナーのオペラや映画音楽では、特定の人物や感情が現れるたび、同じ旋律が少しずつ姿を変えて流れます。『スター・ウォーズ』でダース・ベイダーの登場ごとに響く「帝国のマーチ」のようなものです。出血する傷も、異なる場面で同じ痛みを別の形で響かせる点で同じ装置です。

この一行、すなわち出血する傷の型を人物ごとに明示します。

```
Mの出血：「毎年、母がいなくなった日の明け方に、解剖台の前で一時間ただ立っている。」
Eの出血：「雨の日ごとに、左肩の火傷の部分を手で押さえる。」
Jの出血：「書く前ごとに、手が赤くなるまで洗う。」
```

これらの一行は、日常断面シナリオ・葛藤設計・5幕の変遷でたびたび現れます。トラウマの痕跡は作品内の一か所に固定されず、毎回新しい場所から出血します。

## モジュール5が有効なら

前章で整理した五つの事前準備モジュールのうち、モジュール5「アイデンティティ・多様性の次元」が有効なら、トラウマ段階でもう一度開きます。

特にニューロダイバーシティ（自閉症・ADHD・ディスレクシア）、慢性疾患、障害がキャラクターに含まれる場合、その次元とトラウマの関係を点検します。ニューロダイバーシティを欠点ではなく差異として扱うことが本作業の原則です。自閉症の人物の社会的困難を欠点だけとして置けば人物は否定形になり、差異として置けば立体的になります。この区別がモジュール5の核心です。

## 次章へ

本章をまとめます。トラウマ・欠如・欠点は一つの鎖としてキャラクターの動力となります。一般的な欠点の4軸に加え、道徳的グレーが必要な人物にはDark TriadとHEXACOを呼び出します。トラウマは塞がった傷ではなく、毎日出血する傷でなければなりません。

AIの仕事仲間が強い部分は、欠点4軸の一次案、トラウマ分類候補、鎖の一行圧縮、Dark Triad得点の一次評価です。プロデューサーが直接決める部分は、欠点一つの採用、両面性の一行、鎖の正確な表現、安易な利用を避ける点検、出血する傷の一行です。

これで12段階中、第7段階まで終わりました。キャラクターに動力が植えられ始めています。次章では、動力の第二部分である目標を定めます。キャラクターが意識して望むもの、本当に必要なもの、本人が信じている嘘という三行が、そこに置かれます。
