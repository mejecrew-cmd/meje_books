# 第9章．目標と葛藤：望むもの・必要なもの・信じる嘘

前章では、欠点・欠乏・トラウマの連鎖を組みました。キャラクターに、駆動力の第一の部分が植えられました。今日はその第二の部分である目標と、その二つを運用する段階である葛藤を、一章であわせて扱います。

目標と葛藤は対になっています。目標があって初めて葛藤が可能になり、葛藤があって初めて目標は意味を持ちます。どちらか一方だけを組むと他方が弱くなるため、一章にまとめました。

今日は、外的目標と内的目標のずれ、Want・Need・Lieの三行、キャラクターアークの三類型、葛藤の八類型、ステークスの五種、Fun Engineの五種を扱います。量が多く見えるかもしれませんが、すべて一つの連鎖でつながっています。

## 変化のない物語は物語ではない

まず、姿勢を確認しておきます。

キャラクターを作る人は、その人物に変化してほしくないと願いがちです。変化には危険があります。変化の途中で人物が壊れるかもしれず、作者が制御できない領域へ進むかもしれません。だから作者は人物を安全な場所に置いておきたくなります。

その気持ちは理解できます。しかし、変化のない物語は物語ではありません。一人の人物が同じ場所にとどまれば、もはや生きて動くことはなく、生きて動かない人物は作品の中でも生きられません。キャラクターの変化は作者の統制を超える領域だからこそ、いっそう必要です。

変化は葛藤から来ます。葛藤は目標が阻まれることから来ます。目標が阻まれたとき、人物は決断しなければならず、決断の瞬間に変化します。この因果、すなわち目標 → 障害 → 決断 → 変化が、本段階の作業の流れです。

## 外的目標の4問

目標作業の第一は、外的目標を決める段階です。キャラクターが意識して望むもの、外から観察できる実体のある目標です。

外的目標は四つの問いで決めます。

```
1. 何を望むのか？
2. なぜそれを望むのか？（決定的な理由を一つ）
3. それを得るための代価は何か？
4. 獲得した後、その人はどのような姿になるのか？
```

四つの問いは単純に見えますが、作品の質感を決めます。同じ外的目標を持っていても、二つ目の問い、すなわち「なぜ」によって人物は異なります。「金持ちになりたい」という外的目標を持つ二人がいるとき、一人は貧しい幼少期のトラウマゆえ、もう一人は軽んじられた自尊心ゆえだとすれば、二人は同じ目標へ別の道を進みます。

**Mの外的目標：**

```
1. 何を：母の結婚指輪が見つかった遺体の身元を明らかにする。
2. なぜ：母の最期を知りたいから。30年の不在がどこへ行ったのかを確かめたいから。
3. 代価：検視手続きへの違反。自らの職業上の立場を失う危険。
4. 獲得後の姿：母の最期を知った人。その後30年間、死者のそばにとどまる資格を失った人。
```

四つの問いを埋めると、外的目標の質感が一か所に集まります。目標を得る瞬間にその人がどんな姿になるかを、あらかじめ見ておくのが4番目です。この4番が、後続作業であるWant・Need・Lieのずれの出発点になります。

この4問はAIの仕事仲間とともに進められる部分です。シード・欠点・トラウマを入力すれば一次案が出ます。プロデューサーが検収します。

## 内的目標

外的目標が決まったら、内的目標を組みます。外的目標が外部から観察できる領域なら、内的目標はその人物の内面が本当に向かう段階です。

内的目標は一行で整理します。

**Mの内的目標：** 母の不在を受け入れること。不在の場所に、不在の形をそのまま置いておくこと。

この内的目標は、外的目標（母の最期を突き止めること）とずれます。Mは意識して母の最期を知りたがっていますが、本当に必要なのは、最期を知らないまま不在を受け入れる姿勢です。このずれが、Mを作品の終わりまで動かします。

内的目標を組んだ後は、その人物が内的目標に対してどの段階にいるかを決めます。

段階は四つに分かれます。直面段階は、内的目標の存在さえ知らない段階です。受容段階は、知ってはいるが受け入れられない段階です。解決意志段階は、受け入れているが解決する意志が弱い段階です。解決方法段階は、意志はあるが方法を知らない段階です。

作品の始まりにキャラクターがどの段階にいるかを決めます。作品の終わりまでにどの段階へ移動するかが、作品の変化の型になります。

**M：** 始まりは直面段階（自分が母の不在を受け入れられないまま30年を生きてきたこと自体に気づいていない）。終わりは解決意志段階（受け入れるべきだという意志は生まれるが、まだ方法はわからない）。この二段階の差が、作品の変化の幅です。

## Want / Need / Lie

外的目標と内的目標を一か所に置くと、三行が組まれます。

Want（外的）はキャラクターが意識して望むと言うこと、Need（内的）はキャラクターが本当に必要とすること（たいてい自覚していないこと）、Lie（信じる嘘）はキャラクターが自分と世界について信じている誤った信念です。

『クリスマス・キャロル』のスクルージは、意識的には一ペニーでも多く貯めること（Want）を望みます。しかし幽霊たちが見せた末に、彼が本当に必要としていたのは財産ではなく、人とのつながりと温かな一食でした（Need）。外側で叫ぶ欲望と、内側で飢えている本当の必要がずれるとき、その隔たりが人物を最後まで動かします。よく組まれた物語の結末は、たいていWantではなくNeedを満たします。『レミーのおいしいレストラン』のレミーも同じです。意識的には偉大な料理人として認められたいと望みますが（Want）、本当に必要だったのは、ネズミという出自にひるまず自分の才能どおりに生きることでした（Need）。

> **クローズアップ・『レッド・デッド・リデンプション2』のアーサー・モーガン** ゲーム『レッド・デッド・リデンプション2』のアーサー・モーガンは、無法者ギャングの忠実な実行役です。彼が意識して望むもの（Want）は、ギャングの生存とボスへの義理です。しかし肺結核で死にゆくなか、彼が本当に必要としたもの（Need）は、「俺は悪人で、それが俺のすべてだ」という嘘（Lie）を手放し、残りの時間を正しく使うことでした。Wantを追ううちにLieが崩れ、Needに届くキャラクターアークの教科書を、一人の無法者の最期で見せる作品です。

キャラクターアークの本質は、次の一行です。**Lieを壊し、Needに到達する過程。** Wantは餌です。Wantへ走るあいだにLieが崩れ始め、Lieが壊れるとNeedが見えてきます。

三行を整理すると、次のとおりです。

**MのWant・Need・Lie：**

```
Want：母の結婚指輪が見つかった遺体の身元を明らかにすること
Need：母の不在を、不在の形のまま受け入れること
Lie：「母の最期を知らなければ、母は生きて戻ってくるかもしれない」
```

**EのWant・Need・Lie：**

```
Want：敵軍の首領を殺すこと
Need：Cの死について自分を許すこと
Lie：「私がもっと強ければ、Cは生きていた」
```

**JのWant・Need・Lie：**

```
Want：ベストセラー作家になること
Need：書く行為そのものの中で意味を失わないこと
Lie：「作品は面白くなければ価値がない」
```

三人の三行を見ると、Wantが餌で、Lieが本当の駆動力だとわかります。MはLieを壊してこそNeedに届きます。Lieが壊れない限り、Wantは無限に追われ、満足は訪れません。

この三行の一次圧縮は、AIの仕事仲間とともに組める部分です。外的目標の4問・内的目標・欠点・トラウマを入力すれば、三行の一次圧縮が出ます。プロデューサーが検収し、Lieの正確な表現をつかみます。Lieの一行は作品の核心的な駆動力なので、他人が代わりに決めることはできません。

## Needの二重分解：自分のための変化と他者のための変化

Want・Need・Lieの三行を定めた後、もう一つ確認します。Needには実は二種類ある、ということです。

### 背景

アメリカの脚本家ジョン・トゥルービーが提案した分解です。[^c9n1] Needが一行に見えるとき、その一行の中には二種類の変化がともに埋まっている、という観察です。

### 導入の趣旨

Needを一行だけで扱うと、キャラクターが自己変化だけに集中してしまいます。そうして作られた人物は、自分の認識の中では生きていても、他者との関係では生きられません。トゥルービーは、強いキャラクターには二種類のNeedがともにあると指摘します。

Psychological Need（心理的必要）は自己認識・自尊感情・トラウマの克服へ向かう自分のための変化であり、Moral Need（道徳的必要）は他者をよりよく扱うこと・傷つけた人への償いへ向かう他者のための変化です。

### 期待効果

二つのNeedがともにあれば、キャラクターは自己と他者のあいだで立体的に生きて動きます。Psychological Needだけなら自己中心的な人物になり、Moral Needだけなら自己変化なしに他者のための聖人の位置にとどまります。

Mに適用してみます。

```
Psychological Need：母の不在を、不在の形のまま受け入れること（自分の姿勢の変化）
Moral Need：30年間ともに働いた同僚に、自分を初めてさらすこと（関係の変化）
```

二つのNeedは対になって働きます。自己変化だけなら、Mが同僚と30年間ともに働いた事実は作品で意味を失い、関係変化だけなら、母の不在への姿勢が作品の後ろへ押しやられます。二つのNeedがともにあって初めて、Mの変化は立体的になります。

この分解は第8段階の目標作業の最後に置きます。Want・Need・Lieの三行を定めた後、Needを二行にほどいてみます。AIの仕事仲間とともに一次分解を受け、プロデューサーが採用します。

## キャラクターアークの3類型

三行が定まると、その人物のアーク（変化の型）が決まります。型は三つです。

キャラクターアークとは、一人の人物が物語を通して描く内面の変化曲線を指します。弓（arc）のように曲がる軌跡という意味です。誤った信念から出発して真実に届けば成長アーク、最後まで信念に閉じ込められて崩れれば没落アーク、自分は変わらないまま世界を変えればフラットアークです。

Positive Changeアークは、Lieの認知から拒絶と危機を経て、Lieの廃棄とTruthの採用、Needの達成へ向かいます。Flatアークは、最初からTruthを知っており、世界のLieを壊して変わるのが世界である型です。Negative Changeアークは、Lieを壊せないか、さらに深くはまり、悲劇・没落へ向かう型です。

イ・ミンジンの長編小説『パチンコ』のソンジャは、「生き残ること」という切実なWantを抱え、一世代を渡ります。その長い歳月のあいだ、彼女のNeedは、恥を耐えて踏みとどまることから、恥じることなく堂々と存在することへとゆっくり育ちます。一冊の中の短い変化ではなく、複数世代にわたって展開する成長アークのよい例です。

三つの型のどれを採るかが、作品のトーンを決めます。

**Mが採るアーク：** Positive Change。始まりではLieを強く信じていますが、作品を通じてLieが崩れ、最後にLieを捨ててNeedに届きます。

**Eが採るアーク：** Negative Change。最後までLieを壊せません。7歳の人質の子を救えず、自己処罰へさらに深く沈みます。悲劇です。

**Jが採るアーク：** Flat。最初からNeedを知っていますが、外の世界がWantへとたびたび引っ張ります。変わるのはJではなく、Jの周囲の世界です。

三人が異なるアークを持つと、作品のトーンも異なります。Mの作品は回復の物語、Eの作品は悲劇、Jの作品は自己意識の一貫性の物語です。同じ手順が三人に異なる運命を与えます。

アークの採用は、プロデューサーが直接行う作業です。作品のトーン・ジャンル・結末を決める段階なので、外部ツールは代行できません。

## Story Spine：7行圧縮

三行とアークが決まると、その人物の全行程を7行に圧縮できます。

```
Once upon a time, [キャラクターの日常]。
Every day, [反復する行動・習慣]。
One day, [発端となる事件]。
Because of that, [反応1]。
Because of that, [反応2]。
Until finally, [クライマックス]。
And ever since that day, [変化の結果]。
```

この7行が、キャラクターの作品内の行程を一か所に集めます。Mに適用してみます。

```
Once upon a time, Mは30年目の監察医として、遺体のそばでだけ生きていた。
Every day, 解剖台の遺体の指のたこに触れるたび、自分の人差し指のたこで一度確認した。
One day, ある遺体の左手の人差し指に、母の結婚指輪が静かにはめられていた。
Because of that, Mは手続きに違反し、その指輪を自分の鞄に入れた。
Because of that, Mは遺体の身元を明らかにするため、30年間避けてきた私的領域をのぞき始めた。
Until finally, Mはその遺体が母ではないと知った。母はなお不在だった。
And ever since that day, Mは母の不在の場所に、不在の形そのものを置いて生きるようになった。
```

7行がMの作品全体を一ページに圧縮します。この圧縮が、後続段階である5幕の変遷と葛藤設計の足場になります。

人物の行程を数段階に圧縮する道具は複数あります。ストーリースパインは即興劇作家ケン・アダムスが整理した形式で、ピクサーの創作者たちが活用した例としても広く知られています。『リック・アンド・モーティ』を作ったダン・ハーモンのストーリーサークルも代表的です。この「出発と帰還」の構造をもっとも遠くまでさかのぼると、神話学者ジョーゼフ・キャンベルが『千の顔をもつ英雄』で整理した「英雄の旅」に行き着きます。平凡な日常を離れ、試練を経て変わり、戻るという普遍的構造です。『オズの魔法使い』のドロシーのように、家を離れて冒険の末に変わって帰る多くの主人公が、この構造の上にあります。7行圧縮は、その大きな構造を一人の人物の大きさへ縮めた道具です。

7行圧縮は、AIの仕事仲間とともに進めると強い部分です。シード・三行・アークを入力すれば、7行の一次圧縮が短時間で出ます。プロデューサーが検収します。

## 始まりのLieと終わりのTruthの対応点検

7行圧縮が終わったら、始まりのLieと終わりのTruthが鏡のように対応しているかを点検します。『トイ・ストーリー』の「私はもっとも愛されるおもちゃでなければならない」が「愛は独占ではない」へ、『インサイド・ヘッド』の「喜びだけが正しい感情だ」が「悲しみも必要だ」へ、『カールじいさんの空飛ぶ家』の「私の旅は終わった」が「私の旅は今始まる」へ変わるようにです。MのLie「母の最期を知らなければ、母は生きて戻ってくるかもしれない」は、Truth「最期を知らなくても、母の不在を抱いて生きられる」と対応します。Lieの単純な否定形ではなく、同じ問題を別の姿勢で見る言明でなければ、キャラクターアークは完結しません。

## 葛藤の8類型

目標が決まったら、葛藤を組みます。葛藤は目標の障害なので、目標がわかれば葛藤の輪郭がつかめます。

まず、外的・内的の分類です。

外的葛藤は、人対人、または人対環境のように、実体を持つ存在のあいだの葛藤です。内的葛藤は、一人の人物の内面の葛藤です。

外的葛藤の四つの細目は、個人対個人／個人対社会／個人対運命／個人対自然です。

八類型の対立を表に置いています。

八類型は次のとおりです。1番は「真実だと信じるもの対実際の真実」（外的）、2番は「望むもの対持っているもの」（内的）、3番は「望むもの対他者が期待するもの」（外的）、4番は「自らの限界」（内的）、5番は「内面目標対外面目標」（統合）、6番は「恐れ対究極目標」（内的）、7番は「対立勢力」（外的）、8番は「対立勢力対慈悲」（外的）です。

八類型のうち、キャラクターに適用される3〜5類型を採用します。すべてを埋める必要はありません。核となる三類型があれば十分です。

**Mへの8類型の適用：** 1番（信じる真実対実際）は、母が生きていると30年間信じてきたのに、遺体の指輪がその信念を揺さぶる葛藤です。5番（内面対外面目標）は、外面では遺体の身元を明らかにしたいが、内面では母の不在を受け入れなければならない葛藤です。6番（恐れ対究極目標）は、母の最期を知る恐れと、不在を受け入れたい究極目標の対決です。

三類型がMの葛藤の核をつかみます。AIの仕事仲間とともに8類型マッピングの一次評価を受け、プロデューサーが採用します。

## ステークスの5種

葛藤の強度を測る道具があります。**ステークス**とは、キャラクターが葛藤の中で失い得るものの種類と量です。

五種に分類します。

物理的階層は生存・負傷・経済的損失（18kgの荷物、酸素の残り4時間）、職業的階層は解雇・失格・免許剥奪（監察医の資格剥奪の危険）、アイデンティティ階層は名前の消滅・種族分類不能（自己同一性の崩壊）、関係的階層は裏切り・黙認・保護（30年間ともに働いた同僚の信頼）、存在論的階層は記憶・次元・存在そのもの（30年間の自分の姿勢が無化される危険）を失う葛藤です。

強い葛藤は、複数層のステークスを同時に賭けます。『ダイ・ハード』のジョン・マクレーンは、自分の命（物理）だけでなく、別居中の妻の安全（関係）、警察官としての自尊心（アイデンティティ）まで一度に賭ける状況に置かれます。『ノートルダムの鐘』のカジモドは、愛（関係）と唯一の安息所である鐘楼（存在）、命（物理）を一度に失う危機にさらされます。失うものが一層のときより、複数層のときのほうが緊張は何倍にも高まります。

強い葛藤は、通常三階層以上のステークスを同時に賭けます。Mのステークスは次のように整理されます。

```
物理的：（該当なし）
職業的：手続き違反による監察医資格剥奪の危険
アイデンティティ：30年間「静かな証人」だった自己同一性の崩壊
関係的：30年間ともに働いた同僚に初めて自分をさらさなければならない
存在論的：母が不在だった30年間の、自分の姿勢そのものが無化される危険
```

Mは四階層のステークスを同時に賭けます。強い葛藤の質感です。

```
核となるステークスの一行：「Mがもっとも失うことを恐れるのは、30年間、死者のそばにとどまる資格である。」
```

ステークスの五種は、AIの仕事仲間とともに一次マッピングを受け、プロデューサーが検収します。

## Fun Engineの5種

葛藤の強度がステークスでつかめるなら、葛藤の面白さはFun Engineでつかめます。作品にどのような面白さのエンジンが働いているかを五種に分類します。

バディダイナミクスは二人の関係が物語を牽引し、具体の奇跡は世界観のメカニクスが驚異感を生み、逆説の美学は論理的矛盾が物語的な力となり、時限爆弾は時間制限が緊張感を生み、裏切り・反転は期待を覆す転換を作ります。

五つのエンジンのうち、二つ以上の搭載を勧めます。上位作品は三〜四つです。五つすべては過積載で散漫になります。

```
MのFun Engine：
- 逆説の美学：死者のそばにとどまるため弱さを偽装してきた強さが、母の死の前でついに露出する。
- 時限爆弾：検視手続き上、遺体は24時間以内に遺族へ引き渡される。その間に身元を明らかにしなければならない。
- 裏切り・反転：結局、その遺体は母ではない。母はなお不在である。
```

Mには三つのエンジンが搭載されています。上位圏です。

このマッピングは、AIの仕事仲間とともに一次評価を受け、プロデューサーが採用します。

## 葛藤 → 変化への転換

最後に、葛藤が変化へどうつながるかを一段落で整理します。

```
Mの核となる葛藤：母の最期を知ろうとする姿勢 対 不在の場所に不在の形をそのまま置く姿勢。
変化：遺体が母ではないと知った後、母の不在の形をあらためて見る。
方向：Positive Change Arc。
耐えなければならないこと：30年間の自分の姿勢の無化。手続き違反の責任。同僚に初めて自分をさらすこと。
```

この一段落が、キャラクターの駆動力全体を一か所に集めます。これまで組んだすべての要素、Want・Need・Lie・アーク・葛藤・ステークスが、この段落で一つの流れとしてつながります。

## さらに深く：補助ツール（選択）

本文の核心ツール（Want・Need・Lie、アーク、葛藤・ステークス・Fun Engine）の横に、さらに精密に磨きたいときだけ使う補助ツールを集めました。初回ビルドでは飛ばしても構いません。

**マズローの欲求階層**（自己実現・承認・社会的・安全・生理的）。キャラクターのWantがどの段階に属するかを見ます。M・Jは承認段階、Eは自己実現段階です。

**ダン・ハーモンのストーリーサークル8段階**（You・Need・Go・Search・Find・Take・Return・Change）。キャラクターが旅のどの段階にいるかを点検します。短編は通常4〜7段階、長編一冊は1〜8段階すべてを通ります。

**ピクサーの22ルール**（エマ・コーツ、2011）。キャラクターに直結するものだけを挙げると、#6（得意なことの正反対を投げ込め）は第7・8段階、#16（ステークス）は第9段階、#19（偶然でキャラクターを窮地から救うな）は第9段階の葛藤の絶対原則です。とくに#19は、第14章のSexy Lamp Testと同じ文脈で、キャラクターの能動性を守る原則です。

## 次章へ

今日の章を整理すると、こうなります。外的目標（Want）、内的目標（Need）、信じる嘘（Lie）の三行がキャラクターの駆動力を決め、三行のずれが作品の変化を作り、その変化の型がアーク（Positive・Flat・Negative）として組まれます。葛藤は八類型に分類され、その強度はステークスの五種で、面白さはFun Engineの五種で測られます。

今日までに12段階のうち9段階が終わりました。キャラクターの駆動力はすべて植えられました。次章からは、その人物を他の人物のあいだに置く段階、すなわち関係網へ入ります。

次章は、キャラクターのそばの五人。一人の人物を中央に置き、五人を配置するキャスト分配の5軸マトリクスが、この段階の核心です。
