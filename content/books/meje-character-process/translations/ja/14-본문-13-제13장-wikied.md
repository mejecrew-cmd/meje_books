# 第13章　非典型キャラクターと作品の多様性

前章では一人の人物に五つのメディアの衣装を着せ、12段階のフルビルドを終えました。しかし、本講義が扱うのは英雄と悪役だけではありません。キャラクターには別の顔もあります。

今日は、英雄でも悪役でもない六種類の非典型キャラクターと、作品のキャストの多様性を意識して設計する七つの次元を扱います。

## すべてのキャラクターが英雄か悪役とは限りません

まず姿勢を確認します。

ここまで扱ったM・J・Eは、いずれも主人公級または主要人物であり、作品の視点と葛藤の軸でした。

しかし作品には、ゲームのNPC、チャットボットのペルソナ、ブランドマスコット、教育コンテンツの案内役、治療道具としてのペルソナ、VTuberやストリーマー、児童書の擬人化動物なども登場します。日常で接するキャラクターの多くは英雄でも悪役でもありません。

12段階はこうした非典型にも適用できますが、強調する段階が異なります。本章ではその差を見ます。

## 非典型1　AIチャットボットのペルソナ

第一は、大規模言語モデル型チャットボットやAIアシスタントのペルソナです。

非人間にも欲望、成長、偽りの構造が入ります。映画『エクス・マキナ』のAIエイヴァは人間らしく設計されていますが、閉ざされた研究所から外へ出るという明確なWantを持ち、そのために好意を演じ相手を利用します。AIキャラクターでも最初に「この人格は何を望むのか」と問う理由です。

通常手順との主な違い：

統合シートとは別にSystem Prompt形式を出力します。拒否する話題と話し方、AIであることをどこまで自覚・承認するか、命令型・対話型・メンター型のどれで接するか、セッション間で記憶を保持するかを定めます。

推奨段階：

1. **第1・2・3段階（シード・鏡・日常断面）：** フル実施。1500字短編を対話サンプル一式に置き換えてもよい。
2. **第8段階（目標）：** Wantを正確にし、Need・Lieは略式。
3. **第11段階（没入運用）：** 話し方を強化。呼称変化表と話し方6類型が中心。
4. **第12段階（出力）：** Obsidian標準にSystem Prompt形式を追加。

System Promptの一行形式例：

```
あなたは［名前］です。［一語のアイデンティティ］の姿勢を持ちます。
［拒否パターン領域］については答えません。
［相互作用モード］でユーザーと対話します。
［話し方の類型］で答えます。
［追加事項］
```

AIチャットボットを作ることは第1章の姿勢と鏡のように向かい合います。私たちがAIの仕事仲間と人物を作るように、AIキャラクターも同じ手順で作られます。働き方が、そのまま作る人物のあり方になります。

## 非典型2　ブランドマスコット

第二は商業・教育・キャンペーン用のマスコットです。

長く続くマスコットは単純さと一貫性で保たれます。サンリオのハローキティは口さえない単純な顔で半世紀以上愛されてきました。口がないため見る人が自分の気分を投影できる、と説明されます。複雑な来歴や欠点より、一目でわかる輪郭と変わらない親しみが生命です。カカオフレンズのライアンも「たてがみのない雄ライオン」という単純な設定と穏やかな表情で、韓国を代表するマスコットになりました。

要点：

- **第一印象の強化：** 五メディア出力のうちアニメーションパッケージ（Riotの5要素）を優先。
- **単一メッセージ：** 伝える内容を一行に。
- **対象年齢：** 子ども、若者、中年など。
- **商標登録可能性：** 視覚・聴覚上の識別性。
- **柔軟性：** 季節・キャンペーンごとの変形。

推奨段階：

1. **第1段階シード＋第6段階同定＋第11段階の話し方＋第12段階のアニメーションパッケージ**をフルビルド。
2. 欠点・トラウマ・葛藤は弱いか、なし。マスコットには欠点がほぼありません。

全段階を通らない単純化例ですが、鏡・同定・視覚はほかの人物以上に強調します。

## 非典型3　教育コンテンツの案内役

第三は教科書、オンライン講義、児童コンテンツの案内役です。

同じ人物も目的により全面的に再設計されます。『ティーンエイジ・ミュータント・ニンジャ・タートルズ』は成人向けの荒々しい暴力的パロディ漫画から始まりましたが、アニメと玩具へ移ると、ピザを好み冗談を言い合う子ども向け英雄になりました。誰に何を伝えるかが変われば、同じ骨格でもまったく違う服を着ます。

要点：

- **親しみを優先：** 初登場で好感を得るSave the Cat装置。
- **欠点を学習目標に：** 小さな失敗を学習点にする。
- **年齢適合性：** 発達段階4・5に合う語彙と感情。
- **反復可能性：** シーズン・エピソードごとに更新可能。

推奨段階：

1. **第1・2段階シード・鏡：** フル実施。
2. **第6段階同定：** フル実施。
3. **第7段階欠点：** 学べる小さな欠点のみ。トラウマは弱化・省略。
4. **第11段階没入運用：** 親しみある話し方を優先。
5. **第12段階出力：** アニメーションパッケージ＋週次エピソード形式。

欠点は道徳的グレーではなく学べる小さな失敗であるべきです。「綴りをよく間違える」は魅力と学習点になりますが、「道徳的グレーに近い嘘」は不適切です。

## 非典型4　治療的ペルソナ

第四は相談、コーチング、セルフケアの道具となるキャラクターです。

要点：

- **共感・傾聴優先：** Confidant archetypeを強調。
- **判断回避：** HEXACOの正直さ・謙虚さ（H）が高い。
- **自己開示制限：** 深いトラウマを露出しない。
- **セッション安全設計：** 自傷・自殺言及など危険信号の検知・対応手順。

推奨段階：

1. **第1・2段階シード・鏡：** フル実施。
2. **第6段階同定：** フル実施。
3. **第8段階目標：** Wantのみ正確に。Need・Lieは略式でユーザーに見せない。
4. **第11段階没入運用：** 話し方と傾聴姿勢が中心。
5. **第12段階出力：** AIチャットボット形式に安全線と禁句仕様を追加。

治療的ペルソナでは、ビルドそのものより安全線仕様が長くなることがあります。「しないこと」が明確であってこそ安全に使えます。

治療対話キャラクターの歴史は約60年前に遡ります。1966年、MITのヨーゼフ・ワイゼンバウムが作ったELIZAは、相談者の言葉を問い返す単純なプログラムでした。それでも人々は内面を打ち明け慰められ、開発者自身はそれを警戒しました。設計された傾聴ペルソナの力と、安全線の必要性を同時に示す例です。

## 非典型5　VTuber・ストリーマーのペルソナ

第五はライブ配信用のキャラクターです。

要点：

- **リアルタイム反応性：** 台本でなく即興。
- **ミーム・サブカルチャー親和性：** デレ系分類＋韓国ネットミーム。

**デレ系（デレ系）：** 好意の示し方でアニメ人物を分類するファン文化語彙です。無愛想だが時に優しいツンデレ、執着が危険なヤンデレ、いつも気だるく冷静なクーデレなど、愛情表現の型を素早くつかみます。

- **2D/3D視覚：** Riotの5要素、とくにColor Story・Voice。
- **話題範囲：** ゲーム、雑談、歌など。
- **タブー・禁句（Roleplayの禁止線）：** 本人の限界。

推奨段階：

1. **第1・2段階シード・鏡：** フル実施し、鏡にデレ系を追加。
2. **第6段階同定：** フル実施、視覚優先。
3. **第11段階没入運用：** 即興ガイド＋タブー一覧が中心。
4. **第12段階出力：** アニメーションパッケージ＋即興ガイド。

VTuberは即興性が最大です。12段階後も毎配信で即興が生じるため、その質感を先に定めることが重要です。

VTuberは「決めた質感＋リアルタイム即興」で動きます。2016年登場のキズナアイは「AIを名乗る明るく風変わりな少女」という明確なペルソナでジャンルを開きました。外見、口癖、好むミームを固定し、反応は即興で満たし、「絶対に言わないこと」だけを先に線引きします。固定された結果と開かれた即興を同時に設計するのが核心です。

## 非典型6　児童文学キャラクター

第六は0～12歳向けです。

要点：

- **より強い単純化：** 鏡は四つの固定尺度（MBTI・ホグワーツ・白黒・30核心）のみ。補助尺度を省略。
- **学べる欠点：** 怠惰、嫉妬、嘘など回復可能なもの。
- **トラウマを弱化：** Big Tを避けsmall tのみ。
- **外見一要素を強調：** 子どもの視覚中心性。
- **ギャップの魅力を弱化：** 一貫性を優先。

推奨段階：

第1～5段階はフル実施。第6段階は名前と外見一要素だけを残し、第7～12段階を大幅に短縮します。

12段階のうち五つをフルで行う単純版で、一人30分～1時間で十分です。

長く愛される児童キャラクターは単純で一貫しています。くまのプーさんは「蜂蜜が大好きな穏やかでのんびりした熊」という一行がほぼすべて。欠点は食いしん坊程度で、子どもが笑える小さく回復可能な傷です。深いトラウマや反転なしに、毎回同じ優しさを守ることが信頼になります。『くもパン』のホンビのように「一つの明確な心」で描かれる子どもも同じです。

## 適用マトリクス

六種類をまとめます。

AIチャットボットは第11段階の話し方と第12段階のSystem Promptが中心で、第3段階を対話サンプルに替え、System Promptを追加出力します。ブランドマスコットは第12段階アニメーションパッケージが中心で、第7～9段階を省略でき、視覚ガイドラインを追加。教育キャラクターは第7段階の小さな欠点を学習目標とし、トラウマを弱化・省略、第9段階を短縮し学習点マッピングを追加。治療的ペルソナは第6段階同定と第11段階の話し方が中心で、第3・9段階を省略し安全線・禁句仕様を追加。VTuberは第11・12段階が中心で、第3・7・8段階を短縮し、タブーと即興ガイドを追加。児童文学は第1～5段階をフル、第6段階は名前と外見のみ、以降を大幅短縮し学習価値マッピングを追加します。

作品に合う型を選びます。略式段階が多くても、人物または人格として扱う核心姿勢は同じです。

## 意識的な多様性設計

非典型と対になるのが、作品のキャストにどのアイデンティティがどの分布で登場するかです。

意識しなければ、キャストはプロデューサー自身に似たアイデンティティへ収束します。意図しないなら点検が必要です。

七次元で点検します。

```
1. 性的指向（Heterosexual / Gay / Lesbian / Bi / Pan / Aceなど）
2. ジェンダーアイデンティティ（Cis / Trans / Non-binary / Agenderなど）
3. 人種・民族の詳細（外見＋文化・言語）
4. 障害（身体 / 精神 / 認知 / 慢性疾患）
5. 社会経済的背景（階級・移住経験）
6. 宗教・スピリチュアリティ
7. ニューロダイバーシティ（自閉症 / ADHD / ディスレクシアなど）
```

一人にすべてを集中させる必要はありません。集中させればトークン（象徴的事例）になります。五人のキャストに適切に分布させるのが意識的設計です。

## キャスト多様性の点検

第10章の5軸マトリクスの横に多様性点検を置きます。

M＋4人では次のとおりです。

```
キャスト多様性点検：
- ジェンダー：M(F)、同僚(M)、検事(M)、後輩(F)、AI(N)、均衡
- 年齢：55・55・40・28・N、多様
- 社会経済：M(中流)、同僚(労働者)、検事(上流)、後輩(中流)、AI(N)、多様
- ニューロダイバーシティ：人間四人に設定なし、単一（追加検討）
- 人種・民族：全員韓国人、単一（意図した偏り）
- 宗教：M(無神論)、同僚(キリスト教)、検事(仏教)、後輩(無関心)、AI(N)、多様
- 障害：AI(N/A)、ほかはなし、単一（意図した偏り）
```

五次元が分布し、人種・民族と障害の二次元が単一です。単一なら意図した偏りか、作品のトーンに応じて補正します。

点検しなければ、全員がプロデューサーに最も馴染みある質感へ収束します。意識して表を開くことが核心です。

## 多様性は追加ではありません

多様性はキャスト完成後に載せるのではなく、最初から分配に溶かします。後づけなら「多様性のために追加された人」、最初からなら自分の場所と質感を持つ人になります。

『モアナ』のポリネシア文化は外見装飾でなく航海という物語の骨格です。映画『コーダ あいのうた』のろう者の家族も「聴覚障害キャラクター」というラベルでなく、欲望、冗談、口論を持つ人々です。後づけのアイデンティティは象徴に、初めから統合されたものは人に見えます。

> **クローズアップ・『ムーンライト』のシャイロン** 映画は黒人クィア少年シャイロンの幼年、青年、成人を追います。人種、性、貧困はラベルでなく、彼が息をし、ためらい、愛する方法そのものです。自分が誰かは自分で決めるという言葉が三時期を貫きます。アイデンティティを骨格へ溶かすと象徴でなく人になる、という命題を静かに深く証明します。

事前準備モジュール5（アイデンティティ・多様性次元）はこのためにあります。シード直後に分布を決めれば、後続11段階が整列します。

単一アイデンティティに集中すべき作品（家族内葛藤劇など）なら、意識してモジュールを切ることもできます。意識した停止と、単なる見落としは異なります。

## Hofstedeの6文化次元――多文化キャラクター

多文化・国際作品ではHofstedeの6文化次元を使います。50か国超、IBM社員11万人超の調査から社会文化を六得点に圧縮したモデルです。

マーベル『シャン・チー／テン・リングスの伝説』では、父への義務と家の序列を優先する世界と、人生を自分で選びたい米国育ちの自己が衝突します。出身文化と現在文化の距離が内的葛藤になります。外見や名前だけの多文化人物にしないには、この距離を意識します。

韓国／米国の比較は、Power Distance 60／40、Individualism vs Collectivism 18／91、Masculinity vs Femininity 39／62、Uncertainty Avoidance 85／46、Long-term vs Short-term 100／26、Indulgence vs Restraint 29／68です。韓国は序列、集団、協力・ケア、不確実性回避、長期志向が強く、米国は個人、達成・競争、短期志向、欲求充足が強い傾向です。

出身文化と居住文化の得点差が葛藤の動力になります。韓国系米国人1.5世なら、韓国（個人主義18、長期志向100）と米国（個人主義91、短期志向26）の差が日常葛藤となり、家族行事で韓国式序列を求める親と自己決定を優先したい本人が衝突します。

## 次章へ

本章をまとめます。すべての人物が英雄か悪役ではなく、六種類の非典型が手順の変形として作られます。キャストの多様性は七次元で意識して設計します。

これで12段階がすべて終わりました。一人でも五人のキャストでも非典型でも、本手順の姿勢と道具はそのまま適用されます。

次章が最後です。完成したキャラクターが合格線を通るかを見る五つの診断と、講義全体の総括です。
