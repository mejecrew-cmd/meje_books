# 第14章　五つの診断、そして最後の挨拶

13章にわたり、キャラクターをゼロから出力まで作る手順を追ってきました。シード、自己客観化、日常断面、ひねり・ドラマ・同定、欠点・トラウマ、目標・葛藤、関係網、没入、五メディア出力、非典型キャラクターまで、12段階はすべて完了しました。

本章では、完成した人物が合格線を通るかを見る五つの診断と、講義全体の総括・最後の挨拶を扱います。

## キャラクターは作品に貢献しているか

最後に問うのは、その人物が作品へ貢献するかです。ビルドが優れていても、作品内で固有の場所と質感を持つとは限りません。

批評と制作現場で使われる五つの診断を用います。すべて通れば強い証拠になりますが、作品によって一、二項目の不通過は許容されます。一方、即時再設計が必要な不通過もあります。

## 診断1　Bechdel Test

Alison Bechdelの1985年の漫画に由来します。

作品単位の三条件：

1. 女性キャラクターが二人以上いる。
2. 二人が会話する。
3. 会話の話題が男性以外である。

三つすべてで通過です。これは低い基準で、通過しても優れた女性表象を保証しませんが、不通過は女性を十分扱っていない可能性を示します。

Mは女性で、女性の後輩法医病理医がいます。二人が解剖や同僚関係について話す場面があれば通過します。これは一人でなく作品全体の診断です。

## 診断2　Mako Mori Test

2013年の映画『パシフィック・リム』のマコ・モリに由来します。彼女は男性主人公の道具でなく、家族を失った過去を越えイェーガー操縦士になるという自分の物語を持ちます。

原型の三条件：

1. 女性が一人以上いる。
2. 自分の物語アークを持つ。
3. そのアークが他者の道具に還元されない。

本作業では性別・アイデンティティを問わず一般化します。

> 第11段階の5幕変遷は、他者のための道具でなく本人の変化か？

Mは母の不在を受け入れ、Eは七歳の人質を救えなかったトラウマと向き合い、Jはベストセラーメンターを拒み自分の姿勢を守ります。三人とも自分のアークを持ち通過します。

## 診断3　Sexy Lamp Test

Kelly Sue DeConnickが提案した診断です。

条件は一つ：

> キャラクターをセクシーなランプに替えても筋が同じなら不通過。

能動的な作品貢献を見ます。Mをランプに替えれば、遺体の指輪の発見、袋に入れる決断、手順違反は起きません。Mの決定が事件を動かすため通過です。

人物を除いても話が変わらないなら即時再設計します。『エイリアン』のリプリーをランプに替えれば、生死を決める判断が消え、物語は止まります。人物が事件の主語か目的語かを分ける試験です。

## 診断4　Walks-into-a-room Test

業界の非公式な道具です。

> 初登場から5秒以内に、誰で何を望む人物か識別できるか？

支えるのは外見一要素、身体アンカー、話し方です。三つとも初登場で働く必要があります。

Mの場合：

```
- 外見：解剖服の下の白いシャツ。○
- 身体アンカー：左手人差し指二関節内側のたこ。○
- 話し方：「二度なら保存できる」のような短く正確な一言。○
```

Mは通過します。不通過なら初登場の効率が低く、観客が人物を素早く認識できません。

## 診断5　One-Line Test

完成した人物は長い説明なしでも読めます。WALL-Eはほとんど話しませんが、地球でごみを処理し、小物を集め、イヴを見る行動と視線で、孤独、愛、好奇心を伝えます。

条件は一つ：

> 一行で説明したとき、「主人公」以外に他者と区別する特質があるか？

「三十年目の法医病理医。失踪した母の結婚指輪を遺体から発見する」は良く、「暗い過去を持つ主人公」は弱い。「冷淡な殺し屋。七歳の人質の前で初めて命令に背く」は良く、「有能な刑事」は弱い例です。

第12段階統合シートの最初の一行が通過しなければなりません。Mの一行はほかの人物にそのまま移せないため通過です。

## 診断6（補助）　Pixar Truth Test

第9章の補助診断で、冒頭のLieと末尾のTruthが鏡のように対応し、結末と始まりの因果が整合するかを見ます。

```
Mの冒頭Lie：「母の最後を知らなければ、母は生きて戻るかもしれない。」
Mの末尾Truth：「母の最後を知らなくても、その不在を抱えて生きられる。」
```

二つは鏡のように対応し、Mは通過します。対応しなければ作品全体の因果鎖が切れます。

## 五つ＋一つの統合点検

Mの結果：

```
Mの診断結果：
- Bechdel：○（作品に複数の女性）
- Mako Mori：○（自分のアーク）
- Sexy Lamp：○（事件がMの決定に依存）
- Walks-into-a-room：○（外見・アンカー・話し方が作動）
- One-Line：○（代替不能な一行）
- Truth Test：○（LieとTruthが鏡対応）
```

六つすべて通過です。

道徳的グレーのEは異なります。

```
Eの診断結果：
- Bechdel：×（戦争世界に女性がほぼいない）。偏りが意図的か点検する警告。
- Mako Mori：○（Cの死をめぐる自己赦免という自分のアーク）
- Sexy Lamp：○（命令拒否・救出などEの選択が事件を動かす）
- Walks-into-a-room：○（肩の火傷と冷たい話し方で5秒識別）
- One-Line：○（「冷淡な殺し屋、七歳の人質の前で初めて命令に背く」）
- Truth Test：○（Truthは明確だが到達しない。意図した悲劇的未到達）
```

診断は全欄を○にするチェックリストでなく鏡です。一人芝居ではBechdelが非該当のこともあります。ただし二つは強制通過です。

- **Sexy Lamp不通過：** 即時再設計。作品貢献がゼロ。
- **One-Line不通過：** キャッチフレーズを書き直す。他者と区別できない。

この二つを通らない人物は、そのまま作品へ入れてはいけません。

> **クローズアップ・『ハンガー・ゲーム』のカットニス** 弓、マネシカケスのピン、「妹の代わりに志願した少女」という一行で即座に識別され、選択が事件を動かし、妹を守る少女から革命の象徴へ育つ自分のアークを持ちます。試験のために作らなくても、強い主人公は自然に通過します。診断は目標ではなく鏡です。

## 誰が何をするか

AIは統合シートとキャストから六診断の一次評価を出すことに強みがあります。

通過か再設計かを決めるのはプロデューサーです。トーン、企画、読者への責任は人間にあり、自動評価が通過でも作品に合わなければ再設計します。

## 講義全体の総括

アーシュラ・K・ル＝グウィンは『闇の左手』序文で、小説は予言や証明でなく思考実験だと述べました。人物をある状況に置き、「そうしたらどうなるか」を見るものです。キャラクター作りも、結論を強いるのでなく、良い問いを設計し答えを見守る仕事です。

14章は、姿勢、12段階地図、4階層と5モジュールから始まり、シード、五つの鏡、日常断面へ進みました。第7章はひねり・ドラマ・同定、第8章は欠点・欠如・トラウマ、第9章は目標・葛藤、第10章は関係網、第11章は質問・手紙・日記・呼称・話し方・5幕による没入、第12章は五メディア出力、第13章は非典型と多様性、そして本章は診断と総括です。

道具は多いものの、二つの姿勢の上でのみ意味を持ちます。**キャラクターを一人の人間として扱うこと。** **プロデューサーがAIの仕事仲間とともに導くこと。** 前者はデータへの還元を拒み、後者は迅速な一次出力と人間の決定・圧縮・合格判断を分けます。

## 限界を認める

本手順だけが正解ではありません。12段階は案内であり命令ではありません。

シードから直接1500字短編へ行き鏡を飛ばす作家も、カードでなく頭の印象から始める作家もいます。それでも生きた人物は作れます。一部だけ借りても、まったく使わなくてもよいのです。本講義の主張は「そのまま従え」ではなく「こうした手順も可能だ」です。

8～15時間の予算はAI活用を前提とします。手作業のみなら二、三倍かかることがありますが、それは非効率ではなく別の深さです。時間予算をそのまま適用できないだけです。

## 姉妹講義との接点

本講義はゼロからの**新規キャラクタービルド**を扱いました。

隣には二つの仕事があります。

第一は、**外国語テキストの翻訳準備工程。** 既存作品を深く読み、翻訳前に整理します。本講義が作る仕事なら、これは読む仕事です。

第二は、**世界観IPの統合ライブラリ構築。** 完成した人物と世界の資料を一か所へ集めます。本講義が一人のフォルダを作るなら、これは全フォルダを収める図書館です。

三つは「作る、読む、集める」と一列につながります。本講義は第一を扱いました。

## ともに働いた仲間へ

姿勢の整理、12段階、五尺度と五メディア形式、AIとの分担を正直に置く作業まで、多くの手が14章を支えました。「怒りでなく愛情で」という姿勢も協働の現場から生まれました。

14章を読んだ方にも感謝します。必要な部分だけ持ち帰っても、何も持ち帰らなくてもかまいません。ただ、人物を人間として扱うこと、プロデューサーがAIの仕事仲間と導くこと、この二つは手に残していただければ幸いです。

## 終わりに――作家でなくプロデューサーと呼ぶ私たちへ

第1章へ戻ります。キャラクターを作る人を、作家だけでなくプロデューサーと呼びました。作家は文章を直接書き、プロデューサーはシードから出力まで責任を持ちます。二つは重なることも分かれることもあります。

本手順は「一行を美しく書くには」ではなく「一人をゼロから出力までどう導くか」というプロデューサーの意識に沿います。

各章で、プロデューサーの決定とAIの一次作業を分けました。この正直な分離が効率と品質を同時に支えます。

講義は14章で終わりますが、仕事は終わりません。人物が作品に入り、作品が出版され、読者が出会い、その名を再び口にします。本講義がそこまで届く道具を渡せたなら、講義を行った意味があります。
