# 第6章　日常切面情境：角色書寫自己

上一章以五項尺度提取出單詞身分。M是「安靜的見證者」，J是「理想主義者」。我們說過，一個詞看不見人物的一半。本章要第一次替那片空白穿上散文的衣服。

我們讓角色寫一篇短篇。採第一人稱主角視角，韓文1500至2000字，模擬角色坐到作家的位置，寫下自己日常的一個切面。經過這項模擬，一個詞抓不住的另一半，才會在散文中顯形。

這是本流程最大的驗證工具。它直接用散文測試：只有一句種子與一個詞身分的人物，是否真的能像活人一樣行動。

## 日常切面情境是什麼

先釐清這種形式。

維吉尼亞・吳爾芙的《戴洛維夫人》，只用一名女子為晚宴出門買花的一天，展開她的一生。沒有重大事件；她走過街道、想起舊愛、聽見鐘聲，克拉麗莎整個內在便浮現出來。證明人物的不一定是事件，也可以是普通的一小段生活。日常切面情境追求的正是這一點。

> **特寫・安倍夜郎《深夜食堂》的常客** 漫畫《深夜食堂》以午夜營業的巷弄小店為舞台，每一回透過一位客人的一道「固定菜色」展開人生片段。有人點章魚造型維也納香腸，有人點奶油拌飯。沒有宏大事件，只憑「吃什麼、怎麼吃」的日常切面，一個人物便完整浮現。它把本章命題——不是事件，而是平凡片段也能證明人物——壓縮成一只餐盤的大小，反覆演示。

**日常切面情境：** 以第一人稱主角視角描寫人物日常片段的短篇形式，在本工作中用作角色驗證工具。

這種形式借用日文所稱的「私小說」、英語研究中譯為I-novel的第一人稱自傳小說。借用方式如下。

原本的私小說，是作家以第一人稱坦白自身私密經驗的日本近代文學類型，由作家把自己角色化，講述自己的故事。本流程加以改造：不是作家寫自己，而是假設**正在建構的角色坐上作家的位置，以第一人稱寫自己的日常**。

成果是一篇韓文1500至2000字的短篇。它不是正篇的一部分，而是角色驗證文件，不直接進入作品。

## 與鄰近形式的差異

我們要區分它與其他短散文形式。

本流程的日常切面情境採第一人稱主角視角，由角色本人敘述，篇幅為韓文1500～2000字。Slice of Life可以自由使用第一或第三人稱，也可能偏重外部觀察，因此未必符合本流程的驗證條件。開場情境是作品起點，不是驗證工具。角色獨白是日常切面的壓縮版，時間不足時可代用。Vignette與之相近，但事件因果可能較弱。

**為何選日常切面情境？** 因為它最直接暴露角色內在如何運作。Slice of Life可能著重外部觀察；開場是作品第一幕，不是角色測試；短獨白不易容納充分因果；Vignette雖有印象，因果可能薄弱。此處定義的日常切面，同時要求第一人稱、角色本人敘述與一件事件的因果。

角色的調性、說話方式、在世界觀中的位置，會在此階段一次成形。這項測試確認由五項尺度抽象化的人物，能否在實際散文中存活；若不能，就代表必須修正尺度結果。

## I remember / I don't remember二十五句

正式寫作前，先取出一項工具：進入角色，寫下二十五個短句，作為短篇材料。

十五句以「角色記得」開始，十句以「角色不記得」開始，排列二十五個簡短而具體的完整句子。

```
M記得母親的結婚戒指是什麼顏色。
M記得七歲時，母親戴著的戒指掉在哪裡。
M記得第一次站在解剖台前，手腕如何顫抖。
M不記得母親最後說了什麼。
M不記得嬰兒爽身粉氣味最初來自哪裡。
```

這裡只展示五句，實際要寫二十五句。這是AI特別擅長的部分。輸入種子與五項尺度結果，很快就能得到初步候選。製作人選出最有吸引力的二十五句，或親手重寫一兩句。

記住兩項原則。

- 寫完整句，不用「母親結婚戒指的顏色」之類名詞片語。
- 仔細選詞。句子雖短，仍須包含具體名詞、數字或專有名詞。

二十五句中一兩句會成為開頭，其餘自然進入正文各處。

連續排列「我記得」句式，本來也是文學技法。美國藝術家兼作家喬・布雷納德在《I Remember》中串起數百個「我記得」，描繪一個時代與一個人；法國作家喬治・佩雷克再以《我記得》承接。其核心想法是：記憶清單本身就能成為一個人。

## 五個開頭候選

接著決定短篇第一句。它決定全文調性，所以不直接寫死，而從五個候選中挑選。

事件直進型採「那天，［事件］發生了」，例如「那天，我再次看見母親的戒指。」事件中段描寫型從事件內部切入：「戒指安靜地套在遺體左手食指上。」結局描寫型從事後景象開始：「解剖台上放著一只塑膠袋。」因果連接詞型以「所以」「因為」開頭：「所以母親從一開始就戴著戒指。」記憶型直接取二十五句之一：「M記得母親的結婚戒指是什麼顏色。」

五個候選從不同位置抓住同一事件。事件直進型節奏快，中段描寫型慢，結局描寫型帶回顧調性，因果連接詞型偏思辨，記憶型則接近意識流。

產生五候選是AI工作。輸入二十五句與種子一行即可快速獲得；從中採用最吸引人的一句，是製作人的工作。

## 參照作家組合

開始寫作前再決定一件事：**參照作家組合**。以一位外國作家加一位韓國作家，預先抓住短篇調性。

把人物從「類型」變成「個人」的最快方法，是使用具體名詞。雷蒙・卡佛的極簡短篇精確寫下人物職業、居住街區與車款，讓讀者看見的不是「一般勞工」，而是眼前這個人。一個精確車名，比十個抽象形容詞更能立起人物，如同漫畫的一件道具或電影的一套服裝。

```
M的參照作家：戈馬克・麥卡錫＋金薰
「乾燥的勞動、帶體溫的克制。適合三十年資歷法醫手勢的調性。」

J的參照作家：卡夫卡＋吳貞姬
「官僚式荒謬＋內在克制。適合陷入自我懷疑的作家。」
```

預先準備約八種組合，再依種子採用一組，速度最快。

麥卡錫＋金薰以乾燥勞動與帶體溫的克制，適合肉體勞動、生存、死亡；泰德・姜＋吳貞姬以技術精密與內在感覺，適合科幻與職業日常；卡夫卡＋金愛爛以官僚荒謬與存在重量，適合制度、移民、身分。波赫士＋成碩濟以邏輯迷宮與黑色幽默處理悖論、系統；勒瑰恩＋殷熙耕以平靜觀察與結構批評處理種族、分類、身分。布考斯基＋趙世熙以無表情的憤怒與勞動物性處理工廠、階級；歐威爾＋片惠英以制度控制與內在裂縫處理監視、規訓；卡爾維諾＋朴玟奎以形式實驗與流行感處理後設敘事、框架故事。

採用組合是製作人的直接工作。判斷哪一組符合種子調性，是外部工具不擅長的領域。

組合確定後，逐行整理文體參數。

```
M的文體參數：
- 句長：短（韓文8～15語節），盡量少用連接詞。
- 修辭方式：以轉喻為主，節制比喻，以物性替代。
- 感官優先：觸覺 > 視覺 > 聽覺。
- 對話比例：15～25%，短而乾燥。
- 情感表達：不命名，只以行動與物性呈現。
- 敘述速度：慢，一個動作用3～5句。
```

這些參數是初步寫作指南。

**轉喻：** 以與某物相鄰、接觸的另一物指代它。例如「白宮宣布」以建築指政府。隱喻依靠相似（「人生是航海」），轉喻依靠鄰接。它特別適合不直接命名情緒，而用旁邊的物件呈現。下文M的短篇不用「悲傷」一詞，只寫「折了兩次的塑膠袋」，就是轉喻。

## 寫一篇1500字短篇

現在寫一篇韓文1500至2000字的短篇。視角是第一人稱主角，敘述者就是角色本人。

寫作程序：

1. 從五個候選採用第一句。
2. 從二十五句中選5～8句放進正文。
3. 決定一件在短篇中具有開始與結束的事件。
4. 輸入種子、五尺度、二十五句、第一句、參照作家與文體參數，與AI工作夥伴取得初稿。
5. 校正不符合角色調性、偏離文體參數或因果薄弱之處。
6. 判定合格線；合格便採用，不合格則重新要求或改寫。

這是本階段最耗時的部分。AI可以快速產出初稿，但審核與補正需要時間。每一句都要確認是否帶有角色調性，不能草率完成。

## 四項附加技法

短篇另使用四項技法。

卡夫卡《變形記》以一個句子開場：格里高爾・薩姆沙某天早晨從不安的夢中醒來，發現自己在床上變成巨大昆蟲。沒有冗長解釋，只把一個衝擊細節植入普通早晨，就翻轉整個世界。選得好的細節，是支撐世界的楔子。

**對話比例指南。** 正文20～40%為合適範圍。超過40%，加入行動、移動、感官場面；低於10%，警惕變成獨白散文。禁止只為傳遞資訊而存在的對話，每段對話都要揭示關係、衝突或性格。

**結尾落地。** 最後一段以感官而具體的影像收束。禁止抽象宣言（「那才是真正的自由」）、整理領悟（「現在我終於明白……」）、主題總結與情緒命名（「我很悲傷」）。合格模式有六種：感官餘韻、行動轉換、物性對比、回到錨點、不在場的感覺、未解決物件。

**數字場景化。** 時間、距離、費用、機率、重量等數字，必須成為行動或判斷依據，而非說明。禁止「M平均每具遺體花47分鐘」；允許「47分鐘處理完一具。那是今早第七具。」

**寫作原則。** 讓角色質地滲入文體，讓角色選擇左右結局。禁止為湊篇幅描寫，優先追求壓縮之美。

四項技法也是審稿清單，初稿若偏離一兩項，就補正該處。

## 範例：M的短篇摘錄

完整1500字太長，只展示首尾兩段。

> 那天，我再次看見母親的戒指。它安靜地套在遺體左手食指上。三十年了。食指兩個指節內側長繭的，不只我一人。遺體食指同一位置，也有相同的繭。解剖台上的日光燈，以相同角度照亮兩處繭。
>
> ［中略］
>
> 我把戒指放進塑膠袋。袋子一角破了。我取出新袋子，把戒指放進去。把袋口折兩次。兩次就能保存。一次不足，三次過多。兩次才精確。

即使只看片段，也能看出M的調性：短句、轉喻、觸覺優先、不命名情緒，以及末段未解決的物件與數字場景化。麥卡錫＋金薰的組合活在其中。

## 範例：J的短篇摘錄

J有不同調性。

> 旋轉木馬左側頭部附近，有一長條油漆剝落。J不記得那裡原本是什麼顏色，但確定旋轉木馬轉動時，一直摸著那裡。被摸過之處的顏色，只留下觸摸的感覺。沒有顏色的感覺。
>
> ［中略］
>
> J坐在桌前，望著手腕很久。手腕上什麼也沒有，也沒有像旋轉木馬油漆那樣、不記得原本顏色的位置。那麼J失去了什麼？沒有失去位置的失去，可能存在嗎？

這裡活著卡夫卡＋吳貞姬的組合：抽象思辨、自我意識反轉、荒謬的問題。調性與M不同。

## 範例：E的短篇摘錄

E又是另一種調性。身處虛構戰爭世界的殺手，以壓抑情感的乾燥語言處理暴力。

> 下雨了。左肩燒傷發作時，我用手掌壓住那裡。壓住也不會消失，只會讓疼痛換個位置。我長久以來都在移動那些不會消失的東西。
>
> ［中略］
>
> 目標家門前坐著一個孩子，看來約七歲。我繞路，決定改天再來。有人教過我，只有弱者才會這樣決定。所以今天，我依然是弱者。雨水流過肩上的燒傷。

E的身體錨點（左肩燒傷）與隱藏的慈悲同時滲出。在目標面前轉身的一個動作，反轉呈現「假裝軟弱以隱藏強大」的種子。它也不同於M的克制與J的思辨。

同一流程把三人展開為調性各異的短篇。這證明種子一行與五尺度，會依人物生成不同散文。

## 七項檢查

短篇完成後，以七個問題再次確認是否充分驗證角色。

```
1. 角色對事件結局造成了什麼影響？
2. 角色哪一種質地引導了特定選擇？
3. 哪項選擇最影響事件？若做相反選擇呢？
4. 若同一情況發生在角色的好友身上呢？
5. 角色行動的動機是什麼？
6. 角色在事件中最大的欲望是什麼？
7. 結尾滿足了欲望與動機嗎？若沒有，原因是什麼？
```

把問題實際套用於M的短篇。

```
1. M在遺體手上發現戒指，不按家屬移交程序，而親手裝袋保存；由M直接決定處理方式。
2. 「必須精確保存」的三十年職業質地，引導她把袋口折兩次。
3. 最大選擇是不按規定移交而親自收起。若照程序，短篇張力便消失。
4. 同事遇到同樣事件會立刻報告。這項差異凸顯M的質地。
5. 動機：不願失去母親最後的痕跡。
6. 最大欲望：以精確形式抓住不在場。
7. 只滿足一半。戒指保存了，母親行蹤仍未知；未解決之處成為下一階段動力。
```

七項答案會流入第7階段缺點、匱乏、創傷與第8階段目標。短篇不只完成本階段，也替後續製造材料。

## 前往下一章

本章可整理如下：讓已確定種子與五尺度的人物寫一篇1500字短篇，便能一次抓住其調性、說話方式與世界觀位置。AI擅長二十五句初步候選、五個第一句與1500字初稿；製作人擅長選擇參照作家、審核、判定合格線與七項檢查。

十二階段最初三項——種子、鏡子、日常切面——至此完成。一個人物已被壓縮一次。下一章開始，在壓縮之上添加血肉。

下一章是扭轉與戲劇：故意把角色放上陳腔濫調的座標，再扭轉座標；並以身體錨點，把角色質地固定於身體某一處的感覺。兩者是立體化的第一批工具。
