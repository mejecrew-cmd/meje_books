# 第8章　缺點、匱乏與創傷：走入道德灰色地帶

截至上一章，角色已逐漸有了血肉。行動、身體錨點、前史、能力與偏好這五項工具，塑造出角色的外在輪廓；然而，長出血肉並不等於他已經活了起來。要真正行動，角色還需要動力。

本章開始進入第三組階段：植入動力。動力會在三個階段中建立——缺點與創傷（第7階段）、目標（第8階段）、衝突（第9階段）。今天先處理第一項：缺點、匱乏與創傷。

## 缺點就是角色的魅力

首先，我們要釐清面對缺點時的態度。

初次進行這項工作的人經常犯一個錯誤：猶豫要不要賦予角色缺點。因為角色是自己親手創造的，便希望他毫無瑕疵。這份願望越強，角色就越扁平。

讀者不會愛上一個毫無缺陷的人，而會愛上一個某處帶著傷痕的人。那道傷痕正是此人的魅力。當我們難以分辨哪裡是缺點、哪裡是魅力時，角色便在那條模糊的邊界上活了起來。

長久受到喜愛的角色，幾乎都有某個稜角。《傲慢與偏見》的達西如書名所示十分傲慢；美國影集《怪醫豪斯》的豪斯醫師才華橫溢，卻極其無禮；《穿著Prada的惡魔》的米蘭達冷酷無情。相較於光滑無瑕的人物，稜角鮮明的角色更令人難忘，因為那些稜角正是讓人抓住他的把手。《天外奇蹟》的卡爾・費迪遜因喪妻之痛變得沉默而固執，卻反而格外可愛，也是同樣的道理。

我們主動設計缺點，並有意識地在一個人物身上植入兩項缺點。它們會成為後續目標錯位與衝突設計的動力。沒有缺點，就沒有衝突；沒有衝突，就沒有變化；沒有變化，就沒有活生生的行動。

設計缺點時，我們先確定兩件事。

第一，**缺點絕不是絕對的惡。** 它會依情境轉化為優點。善於說謊的人若成為斥候，謊言便是求生資產；優柔寡斷的人若參與危機協商，那份猶豫可能轉化為傾聽所有意見的態度。預先界定缺點的兩面性，才能保存角色的立體感。

第二，**缺點源自匱乏。** 匱乏又源自創傷。三者連成一條鎖鏈：創傷 → 匱乏 → 缺點。我們不單獨決定一項缺點，而把它放在鎖鏈的末端。如此，缺點才能在作品中保持一致。

## 缺點的四軸：欲求、信念、道德律與價值觀

我們以四條軸線設定缺點。欲求軸檢視角色想得到什麼（認可、控制、安全、自由等）；信念軸檢視他認為哪些行為合乎或違反道德（誠實、服從、反抗、實利）；價值觀軸檢視他把什麼視為最高價值（家庭、工作、自我實現、正義）；克服方法軸則檢視哪個人物或情境能瓦解這項缺點（值得信任的導師、過去事件重演）。

決定一項缺點後，填滿四條軸線。這部分可以與AI工作夥伴共同進行：輸入種子與既有質地，就能獲得缺點候選與四軸初稿。製作人選定一項缺點，再審核四軸細節。

## 八項缺點目錄

我們備有八項常用缺點作為目錄。

```
1. 不誠實：擅長說謊
2. 反抗：無條件抵抗權威
3. 依賴：自尊低落，渴求他人認可
4. 傲慢：過度追求認可
5. 反社會：缺乏同理心
6. 偏私：只忠於自己的群體
7. 懶惰：缺乏驅動力
8. 虛榮：過度在意他人目光
```

把八項集中在一起，是為了加快決策。觀察種子與人物質地後，選出最合適的一至兩項。

**M的首要缺點：** 逃避私人領域（反社會＋懶惰的變奏）。她甚至不知道共事三十年的同事名字。母親的失蹤，最終凝固成對私人關係的逃避。

**E的首要缺點：** 冷漠（反社會＋依賴的變奏）。一旦收到委託，便無視動機與來歷直接執行，也會逃避私人關係。

**J的首要缺點：** 強迫式完美主義（依賴＋虛榮的變奏）。寫作前會把手洗到發紅，因為懷疑作品只有夠有趣才有價值。

三人的缺點都不是直接取用某一個目錄項目，而是以目錄為踏板，設計適合角色的變奏。目錄是起點，不是終點。

## 兩面性：缺點轉化為優點的時刻

決定一項缺點後，以一行整理它的兩面性。

```
M的缺點兩面性：
- 缺點：逃避私人領域
- 轉化為優點：對解剖工作保持絕對專注，也不侵犯同事的私人領域；正因如此，才能共事三十年。

E的缺點兩面性：
- 缺點：冷漠
- 轉化為優點：執行殺手委託的核心資產，不受委託人的動機或故事影響。

J的缺點兩面性：
- 缺點：強迫式完美主義
- 轉化為優點：在修訂時極具優勢，包括朗讀校對與自我檢查冗長句式。
```

這一行保存了角色的立體感。若只強調缺點，人物會成為完全負面的形象；若同時看見它如何轉化為優點，人物便會立體地活起來。

這一行是製作人的直接工作。AI工作夥伴可以提出候選，但只有了解作品調性的製作人，才能判斷哪種兩面性真正適合角色。

## Big T與small t：兩種創傷

設計創傷時，我們將其分為兩種。Big T是超越日常的重大事件（戰爭、災難、強暴、童年性侵）；small t則是損害自尊的日常經驗（反覆嘲弄、課堂失誤、目睹霸凌）。

《進擊的鼓手》的安德魯・尼曼，讓我們看見創傷、缺點與才華如何從同一根源生長。「若不能成為偉大的人，就不值得被愛」這種認可匱乏，把他推向瘋狂的完美主義；然而，也正是這份執著，讓他能把鼓打到雙手流血。傷口生出缺點，缺點同時成為優勢，這是典型案例。

這項分類在本流程之前，便已用於創傷臨床。「small」指事件較日常或反覆出現，不代表痛苦較小。微小的自尊損傷若持續累積，也可能成為與重大事件同樣沉重的創傷。

三個人物的創傷如下：

**M：** Big T。七歲時母親失蹤。她不知道母親去了哪裡，就這樣過了三十年。M所有缺點都始於此事。

**E：** Big T。七歲時目睹情同手足的C遭敵軍殺害。這是「如果自己更強，C就不會死」這個虛假信念的起點。

**J：** small t。反覆的自我懷疑。從開始寫作時起，「我的文章可能沒有價值」便逐漸儀式化。這不是一次重大事件，而是每天細小懷疑累積的結果。

創傷分類是製作人的直接工作。AI工作夥伴可以提出候選，但必須由理解角色質地的製作人判斷Big T或small t何者適合。

## 創傷、匱乏與缺點的鎖鏈

創傷與缺點確定後，在兩者之間插入匱乏，組成一條鎖鏈。我們使用一個單句格式。

> **「角色因為［創傷］而產生［匱乏］，並為了掩蓋它而發展出［缺點］。」**

這一行會把角色的動力集中到一處。

**M的鎖鏈：**

> M因為七歲時母親失蹤，而缺乏對私人關係的信任；為了掩蓋它，她發展出逃避私人領域的缺點。

**E的鎖鏈：**

> E因為七歲時目睹C遭殺害，而產生「只要愛上誰，就會失去誰」的匱乏；為了掩蓋它，E發展出冷漠的缺點。

**J的鎖鏈：**

> J因為反覆自我懷疑，而無法把作品價值與自身價值分開；為了掩蓋它，J發展出強迫式完美主義的缺點。

三條鎖鏈各只有一句話，卻聚集了每個人物所有行動、決定與關係的根源。若沒有確定這一行，後續第8階段的目標與第9階段的衝突便會錯位；一旦確定，下一階段就會自動對齊。

丹尼・維勒納夫《銀翼殺手2049》的主角K，背負著自己是複製人的事實而活。這件事既是束縛他的缺點，也是其身分認同的核心，更是把故事推到最後的引擎。這個人物清楚顯示：缺點不是應被抹除的東西，而應被視為推動人物的動力。

> **特寫・《最後生還者》的喬爾** 遊戲《最後生還者》的喬爾在世界崩潰那天失去了女兒。那道傷口產生匱乏（不願再對任何人產生感情），匱乏又凝固成缺點（以沉默粗暴推開他人，並毫不猶豫地使用暴力）。從創傷到匱乏、再從匱乏到缺點的鎖鏈，完整刻在一個人物身上。當少女艾莉的出現開始撼動這條鎖鏈時，喬爾的每一項選擇都變得沉重。它有力地證明了本章的命題：缺點正是推動人物的動力。

這一行的初步壓縮可與AI工作夥伴共同完成。輸入創傷、匱乏與缺點三部分，便能得到初稿。製作人再審核它是否符合角色質地。

## 避免草率地使用創傷

設計創傷時有一個陷阱：把創傷當成刺激工具消費。僅為了衝擊而丟入重大事件，再把結果當成方便的角色動力，就是草率使用創傷。

我們以以下六項問題進行檢查。

```
1. 創傷是否只具刺激性，卻沒有融入角色？
2. 加害者是否成了毫無動機的絕對惡？
3. 創傷是否成了解釋所有行為的萬能鑰匙？
4. 是否有意識地選擇讓角色克服或不克服？
5. 是否把受害者身分當成裝飾消費？
6. 是否簡化成「令角色陷入極端危險，再讓他脫身」？
```

只要任一項答案為「是」，就必須修正創傷設定。草率使用的創傷一定會被發現，讀者比我們想像中敏銳。被識破後，讀者對角色的信任便會崩塌，角色也無法再像活人般行動。

這項檢查由製作人直接負責。AI工作夥伴可以提出初步評估，但作品的調性、時代與讀者所涉及的責任，仍由製作人承擔。

## 道德灰色：一般缺點不足以描述時

有些角色使用一般缺點四軸便已足夠，有些則不然。反派、反英雄與道德灰色人物，無法完全由一般缺點目錄捕捉。對這些人物，我們會啟用兩項額外尺度。

**反英雄（anti-hero）** 位於主角位置，卻不像傳統英雄。他以缺陷、私利與犬儒取代正義感與高尚品格，帶領故事前進。《計程車司機》的崔維斯・比克爾與死侍便是例子。觀眾與其說支持他，不如說逐漸理解他；這種微妙距離，正是道德灰色人物的魅力。

**Dark Triad：** 自戀、馬基維利主義與精神病態三個向度，各以0～10計分。總分15以上屬明確反派，5～15屬道德灰色，5以下則用一般缺點向度便已足夠。

**HEXACO：** 在Big Five之外加入誠實—謙遜（H向度）的六因子模型。H越低，越接近道德灰色地帶。適合倫理衝突居於核心的作品。

現在把兩項尺度套用於E。

**E的Dark Triad：** 自戀3分（自我認可欲望不高；活下來不是為了自己，而是為了不再失去），馬基維利主義7分（無視委託人動機，只追求結果；策略性的非道德），精神病態6分（逃避同理與身體接觸，但衝動性低）。總分16，接近明確反派。

**E的HEXACO H向度：** 4分。誠實與謙遜都偏低，但並非低到極限。位於道德灰色的中央。

兩項尺度使E不再只是普通殺手，而成為道德灰色人物。雖然總分為16，但自戀在0～10中只有3分，誠實—謙遜為4分，這些細節使E有別於其他殺手。

我們不會對所有人物使用這兩項尺度。一般主角以缺點四軸便已足夠；只有反派、反英雄或道德灰色人物才會啟用它們。

## Schwartz十大普世價值：正向價值的優先順序

如果缺點是負向動力，價值便是正向動力。一個人最重視什麼？心理學家施瓦茨將六十多國的資料濃縮為十大普世價值。它們排列成圓環，相鄰價值可以並存，相對價值則彼此衝突。在以價值衝突為核心的作品中（如家庭對自我實現、傳統對改革），展開這張表便能一眼看清衝突輪廓，也能成為第8階段目標與第9階段衝突的初步踏板。

十大價值為Power（權力，社會地位與控制）、Achievement（成就，個人成功）、Hedonism（享樂，愉悅與感官滿足）、Stimulation（刺激，興奮與新奇）、Self-Direction（自主，自主性與創造）、Universalism（普世，所有人的福祉與自然）、Benevolence（仁慈，身邊之人的福祉）、Tradition（傳統，習俗、宗教與文化）、Conformity（遵從，遵守規範）、Security（安全，穩定與秩序）。相對的組合會成為作品的衝突軸：Self-Direction ↔ Conformity、Universalism ↔ Power、Tradition ↔ Stimulation。

M的價值優先順序：

```
核心價值：Tradition（傳統，三十年來在解剖室遵守的精確程序）＋Benevolence（仁慈，對死者家屬的責任）
拒絕價值：Stimulation（刺激，逃避變化）＋Power（權力，缺乏社會野心）
衝突軸：Tradition vs Self-Direction（在母親戒指這項私事上，自主決定違反程序）
```

M的作品衝突正發生在這條軸線上。她離開守了三十年的Tradition，第一次做出Self-Direction的自主決定，這一刻就是作品的核心動力。

我們也不會把此尺度套用於所有人物。只有價值衝突居於核心的作品才會啟用；一般戲劇與愛情作品使用缺點四軸即可。

## Sol Stein：流血的傷口

最後還有一項工具：創傷不該只是一次性事件，而必須是每天重新流血的傷口。

比較已癒合的傷口（弱）與仍在流血的傷口（強）。「M童年時失去母親」是較弱的癒合式陳述；「每年母親失蹤那天的凌晨，她都會在解剖台前站整整一小時」則是流血式陳述。E若只寫「七歲時看見C被殺」便是癒合式；改成「每逢雨天，肩上的燒傷隱隱作痛時，就會用手按住那裡」才是流血式。J若寫「克服了對自己文章的懷疑」是癒合式；「每天寫作前，懷疑都使J把手洗到發紅」才是流血式。

癒合傷口與流血傷口的差別只有一個詞：時態。癒合屬於過去，流血屬於現在。創傷若要在作品中活著運作，就必須是現在進行式。

讓人物的傷口在整部作品裡反覆變奏，近似音樂中的主導動機。在華格納歌劇或電影配樂中，每當特定人物或情緒出現，同一旋律就以略微不同的形態響起，如《星際大戰》中達斯・維達出現時響起的〈帝國進行曲〉。流血的傷口也是相同裝置：同一份痛苦在不同場景中，以不同形態發聲。

我們為每個人物明確寫下一句流血傷口。

```
M的流血：「每年母親失蹤那天的凌晨，她都會在解剖台前站整整一小時。」
E的流血：「每逢雨天，就會用手按住左肩燒傷的位置。」
J的流血：「每次寫作前，都把手洗到發紅。」
```

這些句子會頻繁出現在日常切面情境、衝突設計與五幕變化中。創傷痕跡不會固定在作品的一個位置，而會每次從不同地方重新流血。

## 若已啟用模組5

若上一章整理的五個前置模組中，模組5「身分認同與多樣性向度」已啟用，便在創傷階段再次展開它。

尤其當角色帶有神經多樣性（自閉症、ADHD、閱讀障礙）、慢性疾病或身心障礙時，要檢查這個向度如何與創傷產生關係。本流程的原則，是把神經多樣性視為差異，而非缺點。若只把自閉症角色的社交困難視為缺點，人物便會成為負面形象；若視為差異，人物便能保持立體。這項區分正是模組5的核心。

## 前往下一章

本章可整理如下：創傷、匱乏與缺點連成一條鎖鏈，成為角色的動力。除了通常的缺點四軸，需要道德灰色的人物還會啟用Dark Triad與HEXACO。創傷必須是每天流血的傷口，而不是早已癒合的傷口。

AI工作夥伴擅長的部分，是缺點四軸初稿、創傷分類候選、鎖鏈單句壓縮，以及Dark Triad分數的初步評估。製作人必須直接決定的部分，則是採用哪一項缺點、兩面性的一句話、鎖鏈的精確表述、避免草率使用創傷的檢查，以及流血傷口的一句話。

至此，十二階段中的第7階段已完成。動力開始植入角色。下一章將確定動力的第二部分——目標。屆時會寫下三句話：角色有意識地想要什麼、真正需要什麼，以及本人深信的謊言。
