# 第13章　非典型角色與作品的多樣性

上一章替一個人物穿上五種媒介的服裝，完成了十二階段完整建構。但本講義不只處理英雄與反派，角色還有其他面貌。

今天要處理六種既非英雄也非反派的非典型角色，以及有意識地設計作品卡司多樣性的七個向度。

## 並非所有角色都是英雄或反派

先確認我們的態度。

至今處理的M、J、E都是核心人物，是主角或主要角色，也是作品視角與衝突的軸線。

然而作品還有遊戲NPC、聊天機器人人格、品牌吉祥物、教育內容主持人、治療性人格、VTuber與實況主、兒童讀物的擬人動物。這些角色往往更貼近日常，我們每天接觸的大多數角色既非英雄也非反派。

十二階段仍適用於它們，只是強調的階段不同。本章要辨認這些差異。

## 非典型1：AI聊天機器人人格

第一種是大型語言模型聊天機器人或AI助理的人格。

非人角色同樣可以具有欲望、成長與謊言結構。《人造意識》的AI艾娃雖被設計成人類外形，卻有自己的明確Want：離開封閉實驗室。她甚至為此表演好感、利用他人。因此，建構AI角色時，第一個問題仍是：「這個人格想要什麼？」

與一般流程的主要差異：

除了普通整合表，另行輸出System Prompt格式。明定拒絕的主題與語氣、對自己是AI的自覺與承認程度、命令型／對話型／導師型互動模式，以及跨工作階段是否保存記憶。

建議階段：

1. **第1、2、3階段（種子、鏡子、日常切面）：** 完整執行。可用一組對話樣本取代1500字短篇。
2. **第8階段（目標）：** 精確設定Want，Need與Lie採簡式。
3. **第11階段（沉浸運用）：** 強化語氣，稱呼變化表與六種說話類型是核心。
4. **第12階段輸出：** 在Obsidian標準之外增加系統提示格式。

系統提示單行格式範例：

```
你是［名字］。你採取［單詞身分］的態度。
你不回答［拒絕模式領域］的問題。
你以［互動模式］與使用者對話。
你以［說話類型］回答。
［追加事項］
```

建構AI聊天機器人人格，與第1章的態度形成鏡像。正如我們和AI工作夥伴共同建構人物，AI角色也經由相同流程誕生。我們的工作方式，正是我們建構角色的方式。

## 非典型2：品牌吉祥物

第二種是商業、教育或活動用吉祥物。

長久存在的吉祥物靠簡單與一致性維持。三麗鷗Hello Kitty以連嘴巴都沒有的簡單臉孔，受全世界喜愛超過半世紀。常見解釋是，沒有嘴巴能讓觀看者把自己的心情投射上去。比起複雜前史或缺點，一眼可辨的輪廓與不變的親切感才是生命。Kakao Friends的Ryan也以「沒有鬃毛的雄獅」與始終平和的表情，成為韓國最熟悉的吉祥物。

重點：

- **強化第一印象：** 五媒介輸出中優先製作動畫套件（Riot五要素）。
- **單一訊息：** 用一句話寫出吉祥物傳達的內容。
- **目標年齡：** 兒童、青年、中年等。
- **可註冊商標性：** 視覺與聽覺辨識度。
- **彈性：** 可依季節與活動變化。

建議階段：

1. 完整執行**第1階段種子＋第6階段身分固定＋第11階段語氣＋第12階段動畫套件**。
2. 缺點、創傷、衝突很弱或不存在；吉祥物幾乎沒有缺點。

品牌吉祥物是簡化建構，不走完所有階段，但鏡子、身分固定與視覺比其他角色更受強調。

## 非典型3：教育內容主持人

第三種是教科書、線上課程與兒童內容的引導者。

同一角色會依目的全面改造。《忍者龜》原為面向成人的粗獷暴力諷刺漫畫，轉入動畫與玩具後，成為喜愛披薩、互相開玩笑的兒童英雄。當「向誰傳達什麼」改變，同一骨架也會穿上完全不同的衣服。

重點：

- **親切優先：** 首次登場必須有取得好感的Save the Cat裝置。
- **把缺點變成學習目標：** 讓小錯誤成為學習點。
- **年齡適切性：** 使用符合發展階段4、5的語彙與情緒。
- **可重複性：** 可依季度與單集更新。

建議階段：

1. **第1、2階段種子與鏡子：** 完整執行。
2. **第6階段身分固定：** 完整執行。
3. **第7階段缺點：** 只使用可學習的小缺點，弱化或省略創傷。
4. **第11階段沉浸運用：** 優先友善語氣。
5. **第12階段輸出：** 動畫套件＋每週單集格式。

教育角色的缺點應是可學習的小失誤，而非道德灰色。「常寫錯字」可同時是魅力與學習點；「接近道德灰色的謊言」則不合適。

## 非典型4：治療性人格

第四種是諮商、教練或自助工具中的角色。

重點：

- **同理與傾聽優先：** 強調Confidant archetype。
- **避免評判：** HEXACO誠實—謙遜（H）偏高。
- **限制自我揭露：** 不展示人格的深層創傷。
- **工作階段安全：** 建立自傷、自殺言及等危險信號的偵測與應對手冊。

建議階段：

1. **第1、2階段種子與鏡子：** 完整執行。
2. **第6階段身分固定：** 完整執行。
3. **第8階段目標：** 只精確設定Want；Need、Lie採簡式且不向使用者揭露。
4. **第11階段沉浸運用：** 語氣與傾聽態度是核心。
5. **第12階段輸出：** 類似AI聊天機器人格式，另加安全線與禁語規格。

治療性人格的安全規格可能比角色建構本身更長。只有明確規定它不該做什麼，使用者才能安全運用。

治療對話角色可追溯至六十年前。1966年，MIT約瑟夫・魏森鮑姆製作的ELIZA只是模仿治療師、重述使用者話語的簡單程式，人們卻向機器傾訴並獲得安慰，開發者本人反而對此警戒。它同時展示良好傾聽人格的力量與安全線的必要。

## 非典型5：VTuber與實況主人格

第五種是直播用角色。

重點：

- **即時反應：** 不是腳本，而是即興。
- **迷因與次文化親和：** 萌系分類＋韓國網路迷因。

**萌系（デレ系）：** 依角色表達好感的方式分類，例如外冷內熱的傲嬌、執著到危險的病嬌、慵懶冷靜的冷嬌。這是粉絲文化中快速掌握愛情表達模式的詞彙。

- **2D/3D視覺：** Riot五要素，尤其Color Story與Voice。
- **主題範圍：** 遊戲、聊天、唱歌等。
- **禁忌與禁語（Roleplay禁止線）：** 人格的界線。

建議階段：

1. **第1、2階段種子與鏡子：** 完整執行，鏡子加入萌系分類。
2. **第6階段身分固定：** 完整執行，視覺優先。
3. **第11階段沉浸運用：** 即興指南＋禁忌清單是核心。
4. **第12階段輸出：** 動畫套件＋即興指南。

VTuber人格的即興性最高。完成十二階段後，每場直播仍會產生即興，因此須預先固定即興的質地。

VTuber以「預先設定的質地＋即時即興」運作。2016年登場的絆愛，以「自稱AI、開朗古怪的少女」這個鮮明人格開啟了整個類型。固定外形、口頭禪與常用迷因，讓每場反應保持即興，只預先畫出「這個人物絕不會說什麼」的界線。共同設計固定結果與開放即興，便是核心。

## 非典型6：兒童文學角色

第六種面向0～12歲讀者。

重點：

- **更強簡化：** 鏡子只用四項固定尺度（MBTI、霍格華茲、黑白、30核心），省略輔助尺度。
- **可學習缺點：** 懶惰、嫉妒、說謊等可恢復缺點。
- **弱化創傷：** 避免Big T，只用small t。
- **強調一項外形：** 符合兒童的視覺中心性。
- **弱化反差魅力：** 一致性更重要。

建議階段：

第1～5階段完整執行。第6階段只保留名字與一項外形因素，第7～12階段大幅縮短。

這是十二階段的簡化版：五階段完整，其餘簡式。每個角色三十分鐘至一小時即可。

長久受喜愛的兒童角色簡單且一致。小熊維尼幾乎可由一句話概括：「太喜歡蜂蜜、溫和又悠閒的熊。」缺點只是貪吃，是孩子可以跟著笑的小而可恢復的瑕疵。沒有深層創傷或反轉，每一集保持相同溫柔，反而建立信任。《雲朵麵包》的洪妃等以「一顆清楚的心」描繪的孩子，也是同樣設計。

## 適用矩陣

六種類型可整理如下。

AI聊天機器人以第11階段語氣與第12階段System Prompt為核心，第3階段可換成對話樣本，並追加系統提示輸出。品牌吉祥物以第12階段動畫套件為核心，可省略第7～9階段並追加視覺指南。教育角色把第7階段的小缺點作為學習目標，弱化或省略創傷，縮短第9階段，追加學習點對應。治療性人格以第6階段身分與第11階段語氣為核心，可省略第3、9階段並追加安全線、禁語。VTuber以第11、12階段為核心，縮短第3、7、8階段，追加禁忌與即興指南。兒童文學完整使用第1～5階段，第6階段只留名字與外形，後續大幅縮短並追加學習價值對應。

展開矩陣，選擇適合作品的類型。即使多個階段採簡式，核心仍相同：把角色當成一個人，或至少一個完整人格。

## 有意識的多樣性設計

非典型角色的另一面，是作品卡司的多樣性：哪些身分以何種分布登場。

若不有意識設計，卡司會收斂到與製作人相似的身分。除非那是刻意結果，否則必須檢查。

以七個向度檢查：

```
1. 性傾向（Heterosexual / Gay / Lesbian / Bi / Pan / Ace等）
2. 性別認同（Cis / Trans / Non-binary / Agender等）
3. 種族與族群細節（外形＋文化、語言）
4. 障礙（身體 / 心理 / 認知 / 慢性病）
5. 社會經濟背景（階級、移民經驗）
6. 宗教與靈性
7. 神經多樣性（自閉症 / ADHD / 閱讀障礙等）
```

不必把七向度全放進一人。集中所有多樣性，會讓人物成為象徵性Token。把七向度適當分布於五人卡司，才是有意識的設計。

## 卡司多樣性檢查

在第10章五軸卡司矩陣旁，加上多樣性檢查。

M加四人的結果如下：

```
卡司多樣性檢查：
- 性別：M(F)、同事(M)、檢察官(M)、後輩(F)、AI(N)，平衡
- 年齡：55、55、40、28、N，多樣
- 社會經濟：M(中產)、同事(勞工)、檢察官(上層)、後輩(中產)、AI(N)，多樣
- 神經多樣性：四名人類均無特別設定，單一（須追加檢討）
- 種族與族群：全為韓國人，單一（有意偏向）
- 宗教：M(無神論)、同事(基督教)、檢察官(佛教)、後輩(無關心)、AI(N)，多樣
- 障礙：AI(N/A)，其餘無，單一（有意偏向）
```

五個向度已有分布，種族／族群與障礙兩項單一。單一向度應是有意偏向，或依作品調性檢查後補正。

若不檢查，所有人物會收斂到製作人最熟悉的質地。有意識地展開表格，就是多樣性設計的核心。

## 多樣性不是追加物

多樣性不是卡司完成後才覆蓋上去，而要從一開始融入分配。後加的角色會被讀成「為了多樣性新增的人」；從起點融入，人物才有自己的位置與質地。

對《海洋奇緣》的莫娜而言，玻里尼西亞文化不是外形裝飾，而是航海故事的骨架。《樂動心旋律》的聾人家庭，也不是「聽障角色」標籤，而是各有欲望、笑話、爭執的人。後加身分讀起來像象徵，從一開始融入則像人。

> **特寫・《月光下的藍色男孩》的夏隆** 電影追隨黑人酷兒少年夏隆的童年、青春與成年。他的種族、性別認同與貧窮不是附加標籤，而是呼吸、遲疑與愛人的方式。「你必須自己決定你是誰」貫穿三個時期。它安靜而深刻地證明：當身分融入骨架而非事後添加，角色會成為人，而非象徵。

五個前置模組中的模組5（身分與多樣性向度）正為此存在。種子確定後立即決定分布，後續十一階段便會依此對齊。

關閉模組也可是一項有意決定。若作品概念必須聚焦單一身分，例如家庭衝突劇，就關閉它。有意關閉與單純遺漏，結果不同。

## Hofstede六個文化向度：多文化角色

多文化與國際作品可啟用Hofstede六個文化向度。此模型調查五十多國、超過十一萬名IBM員工，以六個分數濃縮社會文化質地。

Marvel《尚氣與十環傳奇》把文化差異放在戲劇核心：重視父親義務與家族階序的世界，和在美國長大、想自行選擇人生的自我不斷衝突。出身文化與目前文化的距離，正是內在衝突。要避免多文化角色只剩外形與名字，就須有意識設計這段距離。

韓國／美國六項比較為：Power Distance 60／40；Individualism vs Collectivism 18／91；Masculinity vs Femininity 39／62；Uncertainty Avoidance 85／46；Long-term vs Short-term 100／26；Indulgence vs Restraint 29／68。韓國較傾向階序、集體、合作照顧、迴避不確定與長期取向，美國較傾向個人、成就競爭、短期取向與欲望滿足。

出身文化與居住文化的分數差成為衝突動力。以韓裔美國1.5代為例，韓國（個人主義18、長期取向100）與美國（個人主義91、短期取向26）的差距成為日常衝突；家庭活動中，要求韓式階序的父母，會與優先自主決定的本人不斷碰撞。

## 前往下一章

本章可整理如下：並非所有角色都是英雄或反派；六種非典型透過流程變形建構。作品卡司的多樣性則以七個向度有意識設計。

至此十二階段全部完成。無論是一個角色、五人卡司或非典型人物，本流程的態度與工具仍然適用。
