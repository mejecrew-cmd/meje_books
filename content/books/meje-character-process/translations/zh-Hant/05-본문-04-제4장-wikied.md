# 第4章　決定種子：30張卡片與三條簡式路徑

前三章建立了態度與骨架：把角色當成一個人，由製作人與AI工作夥伴共同帶領，十二階段、層級與五個前置模組。現在，我們手上已有地圖。

實作從今天開始。第一階段是決定種子，把一個人物的起點壓縮成一句話。這句一旦確定，後續十一階段便向它對齊；它若搖晃，後續也會搖晃。

本章處理兩條路：整理外部關鍵字，或從三十張種子卡抽五張；另有三種快速簡式，以及把種子壓縮成一句話的技法。

## 種子不是封閉命令

種子不是決定人物一切的命令，只是起點。後續十一階段中，一部分會在第4階段扭轉，一部分於第7階段缺點工作中補強，也有部分被捨棄。

我們把種子分兩類。Hard種子是不改變的1～3個核心詞；Soft種子是可替換的輔助詞。

Hard直接連結作品概念，例如「三十年資歷的法醫」這種職業、「朝鮮後期」這種時代、「雙胞胎敘事」這種核心關係，後續階段不更換。

Soft是初始印象，可換成更好的表達。例如「左肩燒傷」在第5階段變成「左手食指指紋消失」，對作品影響不大。

預先分開固定核心與可變枝節，後續決定會更快。遊戲開發稱不動搖的核心價值為design pillar，劇本的premise也類似。先明定不可協商的中心，其他部分才能自由實驗。

## 外部種子：製作人已擁有的關鍵字

第一條路適合製作人心中已有角色印象，只需整理成格式。

少數外部條件就能提示人物的可能空間。《廣告狂人》的Don Draper只憑「1960年代紐約廣告公司的成功高階主管」，便讓人想見時代欲望、虛偽與隱藏真實身分的人。外部種子先畫出人物站立的舞台。

格式如下：

```markdown
# Seed（外部）

## 已給關鍵字
- 類型：
- 世界觀：
- 職業・角色：
- 母題：
- 關係種子：
- 事件種子：
- 時代：

## 調性・情調
## 限制
## 企畫意圖（一句）

## 人物層級
- 主角 / 主要 / 配角 / 臨時角色
```

填寫空格是製作人的直接工作，不是AI工作，因為它整理的是製作人腦中的既有印象。卡住的欄位可以空著，最後只有壓縮所需的詞會存活。

但企畫意圖「想透過此人說什麼」不能空白；人物層級也不能空白。前者決定方向，後者決定後續十一階段的深度。

## 30種種子卡：從隨機問答開始

第二條路適合印象薄弱、需要刺激發想時。

《班傑明的奇幻旅程》以「生來是老人、逐漸年輕的人」這一內部種子，立刻裝上外形與年齡的不一致，故事與衝突便從中湧出。

三十張卡分五類，每類六張。A卡刺激職業與環境，B卡是決定性事件，C卡是核心關係，D卡是招牌感官與影像，E卡是矛盾與雙面性。例如A03「工作精確，私下笨拙」、B05「童年失去的東西成年後回來」、E01「假裝軟弱，隱藏強大」。

隨機卡有著名先例。Brian Eno與Peter Schmidt於1975年製作《Oblique Strategies》，以「刪掉最重要的東西」等句子扭轉卡住的思考。TRPG隨機表與塔羅圖像也有相似作用。

程序：

1. 從30張抽五張，標準是每類各一張。
2. 對每個關鍵字以一兩句回答，使用未經審查的第一個念頭。
3. 嘗試把五個答案綁成一個人。
4. 捨棄或替換無法結合的一兩個答案。
5. 把結合的答案壓縮成一句話。

工具負責洗牌。AI工作夥伴為每張卡快速提出**正好五個**初步答案候選。製作人採用最吸引人的一項，或以候選為踏板另寫答案。

結合與捨棄需要理解作品概念，由製作人決定。AI可提出結合草案，但不能代替採否。

最後的一句壓縮也由製作人直接完成，因它將成為後續十一階段的合格線。

## 範例：從五張卡到一句話

這位後續會再出現的角色，先用代號**M**稱呼。

抽卡結果：

```
A03：「工作精確，私下笨拙」
B05：「童年失去的東西成年後回來」
C02：「沒有血緣的家人」
D04：「聞到特定氣味就想吐」
E01：「假裝軟弱，隱藏強大」
```

假設每張卡取得五個候選，再各選一項：

```
A03 → 三十年資歷的法醫。面對遺體沉著，卻總在超市自助結帳迷路。
B05 → 七歲時遺失的母親結婚戒指。三十年後，在解剖中的遺體手上發現。
C02 → 同部門同事。共事三十年，仍不知道彼此名字。
D04 → 嬰兒爽身粉氣味。原因不明。
E01 → 刻意維持軟弱名聲，因為相信強大一旦暴露，就不能再留在死者身旁。
```

在答案間加入因果：

> M是三十年資歷法醫。她在遺體手上發現七歲時遺失的母親戒指；與共事三十年的同事仍不知道彼此名字；爽身粉使她作嘔。她隱藏強大，因為私下相信只有弱者能留在死亡身旁。

五項自然結合，無須捨棄。最後壓縮：

> **M。三十年資歷的法醫。在遺體上發現母親遺失的戒指，並面對自己刻意隱藏的強大。**

這就是M的種子，後續十一階段向它對齊。

## 範例：只從外部種子開始

另一條路不用卡片。整理代號**J**的外部種子：

```markdown
# Seed（外部）：J

## 已給關鍵字
- 類型：自傳小說、文學散文
- 世界觀：當代韓國，出版與媒體業
- 職業・角色：作家，同時主修電影導演與韓國文學
- 母題：強迫式完美主義，懷疑自己冒充作家
- 關係種子：寫作同事、無法斷開截稿循環的朋友
- 事件種子：寫作低潮、克服低潮、暢銷失敗與恢復
- 時代：當代

## 調性・情調
- 思辨與實務交融的平靜第一人稱。以愛而非憤怒。

## 限制
- 不得改變職業。作家的意識流是核心。

## 企畫意圖（一句）
- 一位作家不在成果價值，而在寫作行為本身尋找意義的某段時期。

## 人物層級
- 主角
```

一句壓縮：

> **J。渴望暢銷，卻努力不失去寫作行為本身樂趣的作家。**

J的外部詞已充分，不需要卡片。印象弱用卡，印象強用外部種子。

## 三條簡式種子：快速進入

完整建構需8～15小時，但不是每個人物都需要。

**1卡種子（5分鐘）。** 從附錄A隨機抽一張，以一兩句回答並壓縮成一行。用於臨時角色或發想刺激。

**3關鍵字種子（15分鐘）。** 製作人直接決定職業、母題、關係，整合為一兩句，再加入部分鏡子（MBTI主功能一字母＋黑白滑桿三軸）。適合配角或主要人物初稿。

**隨機1行種子（發想刺激）。** 抽五張但不回答，直接把五個詞接成一句，記錄即興浮現的角色。例如：「工作精確、私下笨拙，童年失物回來，原以為是敵人的其實是自己，只在雨天聽見聲音，並假裝軟弱以隱藏強大的人。」

不先決定答案，而以偶然組合為材料，與超現實主義的自動書寫相通。André Breton等人在意識審查介入前捕捉意外影像；凝視五個隨機詞也是同一原理。

三條路都適合AI產出初步答案、整合句與即興筆記，再由製作人審核採用。

時間指南：5分鐘得到臨時角色的一卡＋單詞身分；15分鐘得到配角／主要人物的種子＋部分鏡子＋名字；30分鐘以七步進入指南得到整合表初稿；2～3小時短路徑得到壓縮表；8～15小時完整建構得到主角完整表與日常切面。

三十人作品可預先分配：兩名主角各8小時、五名主要各4小時、十名配角各1小時、十三名臨時各30分鐘。

## 從簡式晉升為完整建構

原本的臨時角色在後半成為核心衝突軸，層級就會上升。

Camus《異鄉人》的Meursault從「不表露情感的男人」這顆小種子出發，母親葬禮不哭最終成為法庭判他死刑的重要根據。Melville《抄寫員巴特比》的「我寧願不做」同樣成為人物與作品全部結構。

> **特寫・尹胎鎬《未生》的張克萊** 「職業棋士入段失敗的青年，進入綜合商社當合約實習生」這一句幾乎包含人物全部。因只懂圍棋而不熟工作，他用棋局讀法理解職場每一刻。它證明一句種子確定後，後續會向它對齊。

晉升時，只需把層級由臨時改為主要，依序補上短路徑省略的階段；既有一卡答案或三關鍵字保留。

因此簡式也必須存成格式文件。若只放在腦中，晉升時就得重來。即使臨時角色，也保存一行文件。

## 九分類輔助提取

一句種子確定後，可展開九個分類，檢查核心旁的空白。種子壓縮核心，九分類填補或有意保留空缺。

**背景。** 只用外形、性格、職業或環境單一軸描寫，人物其他面向會乾涸。九分類是預先檢查空白的表。

**導入目的。** 一次展開九維，辨認與種子牢固結合及空白之處，決定後續發展或有意留白。

**預期效果。** 讓人物有立體行動的準備，並加快第5階段戲劇、第7階段缺點創傷、第9階段衝突的決策。

九分類為：1職業・角色、2家庭・關係、3外形・視覺（身體錨點候選）、4時間軸・年齡、5空間・地理、6社會・階級、7能力・才華、8缺點・匱乏、9矛盾・雙面性。

每類回答一兩行。種子已回答者保留，沒有者產生初稿。AI起草，製作人採用或捨棄。

M的例子：

```
1. 職業・角色：三十年資歷法醫（種子）
2. 家庭・關係：母親不在（種子）＋共事三十年的同事
3. 外形・視覺：解剖服內白襯衫＋左手食指繭（第5階段發展）
4. 時間軸・年齡：55歲 / Erikson第7階段（生產性 vs 停滯）
5. 空間・地理：首爾某大學醫院解剖部門
6. 社會・階級：中產專業人士 / 強烈職業自尊
7. 能力・才華：從遺體手指繭推測職業（種子發展）
8. 缺點・匱乏：逃避私人領域（第7階段）
9. 矛盾・雙面性：解剖台前精確，自助結帳前停住（種子）
```

一兩維空白可留待後續，但工具的本分是帶著對空白的意識進入。

**Erikson發展階段：** Erik Erikson把人生分為八期，每期配一組課題。第7階段是中年「生產性對停滯」，問人是否為下一代留下什麼，或停積於自身。

## 套用前置模組3與4

兩個模組常在種子階段運作。

若**模組4（世界觀一致性）**開啟，把種子與八項——世界一句摘要、科技／魔法水準、社會結構、核心衝突來源、重要年表、其他核心人物、調性手冊、禁止事項——逐一比對。一兩項不合則修正，五項以上不合則捨棄重來。三十分鐘內完成，可大幅節省後續十一階段。

若**模組3（參照角色對應）**開啟，選一兩名外國與韓國作品角色，寫五行：外國參照、韓國參照、借用的相似點、刻意扭轉的差異、陳腔避免備忘。調性偏移時回到此基準。
