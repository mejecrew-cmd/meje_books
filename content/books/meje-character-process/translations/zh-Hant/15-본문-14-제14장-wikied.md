# 第14章　五項診斷，以及最後的道別

十三章以來，我們從零一路建構角色到輸出：種子、自我客觀化、日常切面、扭轉、戲劇、身分固定、缺點與創傷、目標與衝突、關係網、沉浸、五媒介輸出與非典型角色。十二階段已全部完成。

本章合併兩項工作：以五項診斷檢查完成的角色是否通過合格線，並總結整部講義、做最後道別。

## 角色是否對作品有所貢獻

建構的最後工作，是檢查人物是否對作品有貢獻。角色建得好，不代表他在作品內必然擁有自己的位置與質地；這是另一個檢查領域。

我們使用批評與創作現場常見的五項診斷。全部通過是強證據；依作品性質，一兩項不通過仍可能合理，但有些失敗必須立即重設。

## 診斷1：Bechdel Test

源自Alison Bechdel於1985年的漫畫。

作品層級三條件：

1. 至少有兩名女性角色。
2. 她們彼此交談。
3. 話題不是男性。

三項全滿足才通過。這只是低基準，通過不保證女性描寫良好；不通過則是作品可能未給女性足夠空間的警訊。

M是女性，卡司中另有女性後輩法醫。兩人若談解剖工作或共事，便通過。這是作品診斷，不是單一角色診斷。

## 診斷2：Mako Mori Test

名稱來自2013年電影《環太平洋》的森真子。她不是為男性主角服務的工具，而有自己的創傷、目標與機甲駕駛員成長線。

原始三條件：

1. 至少一名女性。
2. 擁有自己的故事弧。
3. 故事弧不被化約成其他角色的工具。

本流程將它一般化，平等套用於所有性別與身分。

> 第11階段的五幕變化，是人物自己的改變，還是服務別人的工具？

M接受母親的不在；E面對未能救下七歲人質的創傷；J拒絕暢銷導師、守住自己的態度。三人都有自己的弧線，因此通過。

## 診斷3：Sexy Lamp Test

由Kelly Sue DeConnick提出。

唯一條件：

> 若把角色換成一盞性感檯燈，情節仍照常進行，便不通過。

它檢查主動貢獻。把M換成檯燈，就不會有人發現戒指、把它裝袋或違反程序；M的決定推動事件，因此通過。

移除人物而故事不變，就要立即重設。《異形》的Ripley若換成檯燈，左右生死的判斷會消失。這項測試區分人物是事件的主詞，還是單純受詞。

## 診斷4：Walks-into-a-room Test

這是業界非正式工具。

> 角色首次走進場景時，能否在五秒內辨認他是誰、想要什麼？

三項工具支撐辨識：一項外形因素、身體錨點、說話方式。三者都應在首次登場運作。

M的結果：

```
- 外形：解剖服內的白襯衫。○
- 身體錨點：左手食指兩個指節內側的繭。○
- 說話方式：「折兩次就能保存」之類短而精確的一句。○
```

M通過。不通過代表首次登場效率不足，讀者不能迅速辨認人物。

## 診斷5：One-Line Test

完成度高的角色不需長篇說明。瓦力幾乎不說話，卻透過清理地球垃圾、收藏小物、望向伊芙的眼神，傳達孤獨、愛與好奇。

唯一條件：

> 用一句話描述角色時，除了「主角」之外，是否有能和其他角色區分的特質？

「三十年資歷的法醫，在遺體上發現失蹤母親的結婚戒指」是好例；「有黑暗過去的主角」很弱。「冷漠殺手在七歲人質面前第一次違抗命令」是好例；「能幹的刑警」很弱。

第12階段整合表第一行必須通過。M的一句話不能原封不動換給其他人物，因此通過。

## 診斷6（輔助）：Pixar Truth Test

這是第9章的輔助診斷，檢查開頭Lie與結尾Truth是否如鏡子相對，亦即結局是否與開端因果一致。

```
M的開頭Lie：「只要不知道母親最後如何，她就可能還會活著回來。」
M的結尾Truth：「即使不知道母親最後如何，我仍能帶著她的不在生活。」
```

兩句互相映照，M通過。若Truth不回答Lie，作品因果鏈便斷裂。

## 五項＋一項診斷整合檢查

M的結果：

```
M的診斷結果：
- Bechdel：○（作品有多名女性）
- Mako Mori：○（擁有自己的故事弧）
- Sexy Lamp：○（事件取決於M的決定）
- Walks-into-a-room：○（外形、錨點、語氣即時運作）
- One-Line：○（不可替代的一句話）
- Truth Test：○（Lie與Truth鏡像對應）
```

六項全部通過。

道德灰色的E則不同：

```
E的診斷結果：
- Bechdel：×（戰爭世界幾乎沒有女性）。須檢查偏向是否有意。
- Mako Mori：○（關於C之死與自我原諒的獨立故事弧）
- Sexy Lamp：○（拒絕命令、嘗試救援等選擇推動事件）
- Walks-into-a-room：○（肩部燒傷與冷淡語氣可在五秒辨認）
- One-Line：○（「冷漠殺手在七歲人質面前第一次違抗命令」）
- Truth Test：○（Truth清楚但E未能抵達，是刻意的悲劇）
```

診斷是鏡子，不是要求全○的清單。一人短篇的Bechdel可能不適用。然而兩項必須強制通過：

- **Sexy Lamp不通過：** 立即重設，角色對作品的貢獻為零。
- **One-Line不通過：** 重寫一句話，角色與他人無法區分。

兩者任一失敗，人物都不應原樣進入作品。

> **特寫・《飢餓遊戲》的凱妮絲** 弓、學舌鳥胸針與「代替妹妹自願參賽的少女」一句話，讓她立刻可辨；她的選擇推動事件，也有從保護妹妹到成為革命象徵的獨立故事弧。強大主角不必為了考試而設計，也會自然通過。診斷是鏡子，不是目標。

## 誰負責什麼

AI擅長根據整合表與卡司進行六項初步評估。

是否通過或重設，由製作人決定。作品調性、企畫、讀者的責任在人；即使自動評估通過，不合作品也要重設。

## 整部講義的總結

娥蘇拉・勒瑰恩在《黑暗的左手》序言中說，小說不是預言或證明，而是一種思想實驗：把人放進情境，觀察會發生什麼。角色建構也不是強加預定答案，而是設計好問題，看人物如何回答。

十四章依序建立態度、十二階段地圖、四層級與五模組，再處理種子、五面鏡子、日常切面。第7章是扭轉、戲劇、身分固定；第8章是缺點、匱乏、創傷；第9章是目標、衝突；第10章關係網；第11章以問答、信件、日記、稱呼、語氣、五幕完成沉浸；第12章五媒介輸出；第13章非典型與多樣性；第14章診斷與總結。

所有工具只有放在兩種態度上才有意義：**把角色當成一個人。****由製作人與AI工作夥伴共同帶領。** 前者拒絕把人化約為資料，後者區分快速初稿與人的決定、壓縮、合格判斷。

## 承認限制

本流程不是唯一正解。十二階段是指南，不是命令。

有些作家從種子直接進入1500字短篇，跳過鏡子；有些不用隨機卡，而從腦中印象出發，都能產生活人。可以只借一部分，也可以完全不用。本講義說的不是「照做」，而是「這樣的流程也可能」。

八至十五小時預算以AI協作為前提。全手工可能需要兩三倍時間，那不是低效率，而是另有深度；只是不能直接套用本預算。

## 與姊妹講義的接點

本講義處理從零開始的**新角色建構**。

旁邊有兩項工作。

第一，**外語文本的翻譯準備：** 深入閱讀既有作品，翻譯前先整理。本講義是建構，它則是閱讀。

第二，**世界觀IP整合資料庫建構：** 把完成的角色與世界資料集中一處。本講義建立一個人物資料夾，它建立容納所有資料夾的圖書館。

三者連成一線：建構、閱讀、彙集。本講義處理第一項。

## 致共同工作的夥伴

從態度、十二階段、五尺度與五媒介格式，到誠實區分製作人與AI工作，許多雙手共同完成十四章。「以愛而非憤怒」也先在協作現場誕生。

感謝讀完十四章的讀者。可以只帶走有用部分，也可以什麼都不帶；但希望留下兩種態度：把角色當人，由製作人與AI工作夥伴共同帶領。

## 結語：致稱自己為製作人，而不只是作家的我們

回到第1章。我們把建構角色的人稱為製作人。作家直接寫文字；製作人從種子到輸出負責。兩者可以重疊，也可分開。

本流程遵循製作人的意識：不是「如何寫出漂亮一句」，而是「如何把一個人物從零帶到輸出」。

每章都區分製作人的決定與AI的初步工作。這種誠實分工同時支撐效率與品質。

講義在十四章結束，工作卻未結束。人物進入作品，作品出版，讀者與人物相遇，再次說出他的名字。若本講義交出的工具能抵達那些階段，它便完成了使命。
