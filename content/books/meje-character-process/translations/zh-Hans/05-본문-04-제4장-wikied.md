# 第4章　决定种子：30张卡片与三条简式路径

前三章建立了态度与骨架：把角色当成一个人，由制作人与AI工作伙伴共同带领，十二阶段、层级与五个前置模块。现在，我们手上已有地图。

实作从今天开始。第一阶段是决定种子，把一个人物的起点压缩成一句话。这句一旦确定，后续十一阶段便向它对齐；它若摇晃，后续也会摇晃。

本章处理两条路：整理外部关键字，或从三十张种子卡抽五张；另有三种快速简式，以及把种子压缩成一句话的技法。

## 种子不是封闭命令

种子不是决定人物一切的命令，只是起点。后续十一阶段中，一部分会在第4阶段扭转，一部分于第7阶段缺点工作中补强，也有部分被舍弃。

我们把种子分两类。Hard种子是不改变的1～3个核心词；Soft种子是可替换的辅助词。

Hard直接连结作品概念，例如「三十年资历的法医」这种职业、「朝鲜后期」这种时代、「双胞胎叙事」这种核心关系，后续阶段不更换。

Soft是初始印象，可换成更好的表达。例如「左肩烧伤」在第5阶段变成「左手食指指纹消失」，对作品影响不大。

预先分开固定核心与可变枝节，后续决定会更快。游戏开发称不动摇的核心价值为design pillar，剧本的premise也类似。先明定不可协商的中心，其他部分才能自由实验。

## 外部种子：制作人已拥有的关键字

第一条路适合制作人心中已有角色印象，只需整理成格式。

少数外部条件就能提示人物的可能空间。《广告狂人》的Don Draper只凭「1960年代纽约广告公司的成功高阶主管」，便让人想见时代欲望、虚伪与隐藏真实身份的人。外部种子先画出人物站立的舞台。

格式如下：

```markdown
# Seed（外部）

## 已给关键字
- 类型：
- 世界观：
- 职业・角色：
- 母题：
- 关系种子：
- 事件种子：
- 时代：

## 调性・情调
## 限制
## 企画意图（一句）

## 人物层级
- 主角 / 主要 / 配角 / 临时角色
```

填写空格是制作人的直接工作，不是AI工作，因为它整理的是制作人脑中的既有印象。卡住的栏位可以空着，最后只有压缩所需的词会存活。

但企画意图「想透过此人说什么」不能空白；人物层级也不能空白。前者决定方向，后者决定后续十一阶段的深度。

## 30种种子卡：从随机问答开始

第二条路适合印象薄弱、需要刺激发想时。

《班杰明的奇幻旅程》以「生来是老人、逐渐年轻的人」这一内部种子，立刻装上外形与年龄的不一致，故事与冲突便从中涌出。

三十张卡分五类，每类六张。A卡刺激职业与环境，B卡是决定性事件，C卡是核心关系，D卡是招牌感官与影像，E卡是矛盾与双面性。例如A03「工作精确，私下笨拙」、B05「童年失去的东西成年后回来」、E01「假装软弱，隐藏强大」。

随机卡有著名先例。Brian Eno与Peter Schmidt于1975年制作《Oblique Strategies》，以「删掉最重要的东西」等句子扭转卡住的思考。TRPG随机表与塔罗图像也有相似作用。

程序：

1. 从30张抽五张，标准是每类各一张。
2. 对每个关键字以一两句回答，使用未经审查的第一个念头。
3. 尝试把五个答案绑成一个人。
4. 舍弃或替换无法结合的一两个答案。
5. 把结合的答案压缩成一句话。

工具负责洗牌。AI工作伙伴为每张卡快速提出**正好五个**初步答案候选。制作人采用最吸引人的一项，或以候选为踏板另写答案。

结合与舍弃需要理解作品概念，由制作人决定。AI可提出结合草案，但不能代替采否。

最后的一句压缩也由制作人直接完成，因它将成为后续十一阶段的合格线。

## 范例：从五张卡到一句话

这位后续会再出现的角色，先用代号**M**称呼。

抽卡结果：

```
A03：「工作精确，私下笨拙」
B05：「童年失去的东西成年后回来」
C02：「没有血缘的家人」
D04：「闻到特定气味就想吐」
E01：「假装软弱，隐藏强大」
```

假设每张卡取得五个候选，再各选一项：

```
A03 → 三十年资历的法医。面对遗体沉着，却总在超市自助结帐迷路。
B05 → 七岁时遗失的母亲结婚戒指。三十年后，在解剖中的遗体手上发现。
C02 → 同部门同事。共事三十年，仍不知道彼此名字。
D04 → 婴儿爽身粉气味。原因不明。
E01 → 刻意维持软弱名声，因为相信强大一旦暴露，就不能再留在死者身旁。
```

在答案间加入因果：

> M是三十年资历法医。她在遗体手上发现七岁时遗失的母亲戒指；与共事三十年的同事仍不知道彼此名字；爽身粉使她作呕。她隐藏强大，因为私下相信只有弱者能留在死亡身旁。

五项自然结合，无须舍弃。最后压缩：

> **M。三十年资历的法医。在遗体上发现母亲遗失的戒指，并面对自己刻意隐藏的强大。**

这就是M的种子，后续十一阶段向它对齐。

## 范例：只从外部种子开始

另一条路不用卡片。整理代号**J**的外部种子：

```markdown
# Seed（外部）：J

## 已给关键字
- 类型：自传小说、文学散文
- 世界观：当代韩国，出版与媒体业
- 职业・角色：作家，同时主修电影导演与韩国文学
- 母题：强迫式完美主义，怀疑自己冒充作家
- 关系种子：写作同事、无法断开截稿循环的朋友
- 事件种子：写作低潮、克服低潮、畅销失败与恢复
- 时代：当代

## 调性・情调
- 思辨与实务交融的平静第一人称。以爱而非愤怒。

## 限制
- 不得改变职业。作家的意识流是核心。

## 企画意图（一句）
- 一位作家不在成果价值，而在写作行为本身寻找意义的某段时期。

## 人物层级
- 主角
```

一句压缩：

> **J。渴望畅销，却努力不失去写作行为本身乐趣的作家。**

J的外部词已充分，不需要卡片。印象弱用卡，印象强用外部种子。

## 三条简式种子：快速进入

完整建构需8～15小时，但不是每个人物都需要。

**1卡种子（5分钟）。** 从附录A随机抽一张，以一两句回答并压缩成一行。用于临时角色或发想刺激。

**3关键字种子（15分钟）。** 制作人直接决定职业、母题、关系，整合为一两句，再加入部分镜子（MBTI主功能一字母＋黑白滑杆三轴）。适合配角或主要人物初稿。

**随机1行种子（发想刺激）。** 抽五张但不回答，直接把五个词接成一句，记录即兴浮现的角色。例如：「工作精确、私下笨拙，童年失物回来，原以为是敌人的其实是自己，只在雨天听见声音，并假装软弱以隐藏强大的人。」

不先决定答案，而以偶然组合为材料，与超现实主义的自动书写相通。André Breton等人在意识审查介入前捕捉意外影像；凝视五个随机词也是同一原理。

三条路都适合AI产出初步答案、整合句与即兴笔记，再由制作人审核采用。

时间指南：5分钟得到临时角色的一卡＋单词身份；15分钟得到配角／主要人物的种子＋部分镜子＋名字；30分钟以七步进入指南得到整合表初稿；2～3小时短路径得到压缩表；8～15小时完整建构得到主角完整表与日常切面。

三十人作品可预先分配：两名主角各8小时、五名主要各4小时、十名配角各1小时、十三名临时各30分钟。

## 从简式晋升为完整建构

原本的临时角色在后半成为核心冲突轴，层级就会上升。

Camus《异乡人》的Meursault从「不表露情感的男人」这颗小种子出发，母亲葬礼不哭最终成为法庭判他死刑的重要根据。Melville《抄写员巴特比》的「我宁愿不做」同样成为人物与作品全部结构。

> **特写・尹胎镐《未生》的张克莱** 「职业棋士入段失败的青年，进入综合商社当合约实习生」这一句几乎包含人物全部。因只懂围棋而不熟工作，他用棋局读法理解职场每一刻。它证明一句种子确定后，后续会向它对齐。

晋升时，只需把层级由临时改为主要，依序补上短路径省略的阶段；既有一卡答案或三关键字保留。

因此简式也必须存成格式文件。若只放在脑中，晋升时就得重来。即使临时角色，也保存一行文件。

## 九分类辅助提取

一句种子确定后，可展开九个分类，检查核心旁的空白。种子压缩核心，九分类填补或有意保留空缺。

**背景。** 只用外形、性格、职业或环境单一轴描写，人物其他面向会干涸。九分类是预先检查空白的表。

**导入目的。** 一次展开九维，辨认与种子牢固结合及空白之处，决定后续发展或有意留白。

**预期效果。** 让人物有立体行动的准备，并加快第5阶段戏剧、第7阶段缺点创伤、第9阶段冲突的决策。

九分类为：1职业・角色、2家庭・关系、3外形・视觉（身体锚点候选）、4时间轴・年龄、5空间・地理、6社会・阶级、7能力・才华、8缺点・匮乏、9矛盾・双面性。

每类回答一两行。种子已回答者保留，没有者产生初稿。AI起草，制作人采用或舍弃。

M的例子：

```
1. 职业・角色：三十年资历法医（种子）
2. 家庭・关系：母亲不在（种子）＋共事三十年的同事
3. 外形・视觉：解剖服内白衬衫＋左手食指茧（第5阶段发展）
4. 时间轴・年龄：55岁 / Erikson第7阶段（生产性 vs 停滞）
5. 空间・地理：首尔某大学医院解剖部门
6. 社会・阶级：中产专业人士 / 强烈职业自尊
7. 能力・才华：从遗体手指茧推测职业（种子发展）
8. 缺点・匮乏：逃避私人领域（第7阶段）
9. 矛盾・双面性：解剖台前精确，自助结帐前停住（种子）
```

一两维空白可留待后续，但工具的本分是带着对空白的意识进入。

**Erikson发展阶段：** Erik Erikson把人生分为八期，每期配一组课题。第7阶段是中年「生产性对停滞」，问人是否为下一代留下什么，或停积于自身。

## 套用前置模块3与4

两个模块常在种子阶段运作。

若**模块4（世界观一致性）**开启，把种子与八项——世界一句摘要、科技／魔法水准、社会结构、核心冲突来源、重要年表、其他核心人物、调性手册、禁止事项——逐一比对。一两项不合则修正，五项以上不合则舍弃重来。三十分钟内完成，可大幅节省后续十一阶段。

若**模块3（参照角色对应）**开启，选一两名外国与韩国作品角色，写五行：外国参照、韩国参照、借用的相似点、刻意扭转的差异、陈腔避免备忘。调性偏移时回到此基准。
