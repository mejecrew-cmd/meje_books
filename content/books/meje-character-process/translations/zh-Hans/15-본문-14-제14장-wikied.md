# 第14章　五项诊断，以及最后的道别

十三章以来，我们从零一路建构角色到输出：种子、自我客观化、日常切面、扭转、戏剧、身份固定、缺点与创伤、目标与冲突、关系网、沉浸、五媒介输出与非典型角色。十二阶段已全部完成。

本章合并两项工作：以五项诊断检查完成的角色是否通过合格线，并总结整部讲义、做最后道别。

## 角色是否对作品有所贡献

建构的最后工作，是检查人物是否对作品有贡献。角色建得好，不代表他在作品内必然拥有自己的位置与质地；这是另一个检查领域。

我们使用批评与创作现场常见的五项诊断。全部通过是强证据；依作品性质，一两项不通过仍可能合理，但有些失败必须立即重设。

## 诊断1：Bechdel Test

源自Alison Bechdel于1985年的漫画。

作品层级三条件：

1. 至少有两名女性角色。
2. 她们彼此交谈。
3. 话题不是男性。

三项全满足才通过。这只是低基准，通过不保证女性描写良好；不通过则是作品可能未给女性足够空间的警讯。

M是女性，角色阵容中另有女性后辈法医。两人若谈解剖工作或共事，便通过。这是作品诊断，不是单一角色诊断。

## 诊断2：Mako Mori Test

名称来自2013年电影《环太平洋》的森真子。她不是为男性主角服务的工具，而有自己的创伤、目标与机甲驾驶员成长线。

原始三条件：

1. 至少一名女性。
2. 拥有自己的故事弧。
3. 故事弧不被化约成其他角色的工具。

本流程将它一般化，平等套用于所有性别与身份。

> 第11阶段的五幕变化，是人物自己的改变，还是服务别人的工具？

M接受母亲的不在；E面对未能救下七岁人质的创伤；J拒绝畅销导师、守住自己的态度。三人都有自己的弧线，因此通过。

## 诊断3：Sexy Lamp Test

由Kelly Sue DeConnick提出。

唯一条件：

> 若把角色换成一盏性感台灯，情节仍照常进行，便不通过。

它检查主动贡献。把M换成台灯，就不会有人发现戒指、把它装袋或违反程序；M的决定推动事件，因此通过。

移除人物而故事不变，就要立即重设。《异形》的Ripley若换成台灯，左右生死的判断会消失。这项测试区分人物是事件的主词，还是单纯受词。

## 诊断4：Walks-into-a-room Test

这是业界非正式工具。

> 角色首次走进场景时，能否在五秒内辨认他是谁、想要什么？

三项工具支撑辨识：一项外形因素、身体锚点、说话方式。三者都应在首次登场运作。

M的结果：

```
- 外形：解剖服内的白衬衫。○
- 身体锚点：左手食指两个指节内侧的茧。○
- 说话方式：「折两次就能保存」之类短而精确的一句。○
```

M通过。不通过代表首次登场效率不足，读者不能迅速辨认人物。

## 诊断5：One-Line Test

完成度高的角色不需长篇说明。瓦力几乎不说话，却透过清理地球垃圾、收藏小物、望向伊芙的眼神，传达孤独、爱与好奇。

唯一条件：

> 用一句话描述角色时，除了「主角」之外，是否有能和其他角色区分的特质？

「三十年资历的法医，在遗体上发现失踪母亲的结婚戒指」是好例；「有黑暗过去的主角」很弱。「冷漠杀手在七岁人质面前第一次违抗命令」是好例；「能干的刑警」很弱。

第12阶段整合表第一行必须通过。M的一句话不能原封不动换给其他人物，因此通过。

## 诊断6（辅助）：Pixar Truth Test

这是第9章的辅助诊断，检查开头Lie与结尾Truth是否如镜子相对，亦即结局是否与开端因果一致。

```
M的开头Lie：「只要不知道母亲最后如何，她就可能还会活着回来。」
M的结尾Truth：「即使不知道母亲最后如何，我仍能带着她的不在生活。」
```

两句互相映照，M通过。若Truth不回答Lie，作品因果链便断裂。

## 五项＋一项诊断整合检查

M的结果：

```
M的诊断结果：
- Bechdel：○（作品有多名女性）
- Mako Mori：○（拥有自己的故事弧）
- Sexy Lamp：○（事件取决于M的决定）
- Walks-into-a-room：○（外形、锚点、语气即时运作）
- One-Line：○（不可替代的一句话）
- Truth Test：○（Lie与Truth镜像对应）
```

六项全部通过。

道德灰色的E则不同：

```
E的诊断结果：
- Bechdel：×（战争世界几乎没有女性）。须检查偏向是否有意。
- Mako Mori：○（关于C之死与自我原谅的独立故事弧）
- Sexy Lamp：○（拒绝命令、尝试救援等选择推动事件）
- Walks-into-a-room：○（肩部烧伤与冷淡语气可在五秒辨认）
- One-Line：○（「冷漠杀手在七岁人质面前第一次违抗命令」）
- Truth Test：○（Truth清楚但E未能抵达，是刻意的悲剧）
```

诊断是镜子，不是要求全○的清单。一人短篇的Bechdel可能不适用。然而两项必须强制通过：

- **Sexy Lamp不通过：** 立即重设，角色对作品的贡献为零。
- **One-Line不通过：** 重写一句话，角色与他人无法区分。

两者任一失败，人物都不应原样进入作品。

> **特写・《饥饿游戏》的凯妮丝** 弓、学舌鸟胸针与「代替妹妹自愿参赛的少女」一句话，让她立刻可辨；她的选择推动事件，也有从保护妹妹到成为革命象征的独立故事弧。强大主角不必为了考试而设计，也会自然通过。诊断是镜子，不是目标。

## 谁负责什么

AI擅长根据整合表与角色阵容进行六项初步评估。

是否通过或重设，由制作人决定。作品调性、企画、读者的责任在人；即使自动评估通过，不合作品也要重设。

## 整部讲义的总结

娥苏拉・勒瑰恩在《黑暗的左手》序言中说，小说不是预言或证明，而是一种思想实验：把人放进情境，观察会发生什么。角色建构也不是强加预定答案，而是设计好问题，看人物如何回答。

十四章依序建立态度、十二阶段地图、四层级与五模块，再处理种子、五面镜子、日常切面。第7章是扭转、戏剧、身份固定；第8章是缺点、匮乏、创伤；第9章是目标、冲突；第10章关系网；第11章以问答、信件、日记、称呼、语气、五幕完成沉浸；第12章五媒介输出；第13章非典型与多样性；第14章诊断与总结。

所有工具只有放在两种态度上才有意义：**把角色当成一个人。****由制作人与AI工作伙伴共同带领。** 前者拒绝把人化约为资料，后者区分快速初稿与人的决定、压缩、合格判断。

## 承认限制

本流程不是唯一正解。十二阶段是指南，不是命令。

有些作家从种子直接进入1500字短篇，跳过镜子；有些不用随机卡，而从脑中印象出发，都能产生活人。可以只借一部分，也可以完全不用。本讲义说的不是「照做」，而是「这样的流程也可能」。

八至十五小时预算以AI协作为前提。全手工可能需要两三倍时间，那不是低效率，而是另有深度；只是不能直接套用本预算。

## 与姐妹讲义的接点

本讲义处理从零开始的**新角色建构**。

旁边有两项工作。

第一，**外语文本的翻译准备：** 深入阅读既有作品，翻译前先整理。本讲义是建构，它则是阅读。

第二，**世界观IP整合资料库建构：** 把完成的角色与世界资料集中一处。本讲义建立一个人物资料夹，它建立容纳所有资料夹的图书馆。

三者连成一线：建构、阅读、汇集。本讲义处理第一项。

## 致共同工作的伙伴

从态度、十二阶段、五尺度与五媒介格式，到诚实区分制作人与AI工作，许多双手共同完成十四章。「以爱而非愤怒」也先在协作现场诞生。

感谢读完十四章的读者。可以只带走有用部分，也可以什么都不带；但希望留下两种态度：把角色当人，由制作人与AI工作伙伴共同带领。

## 结语：致称自己为制作人，而不只是作家的我们

回到第1章。我们把建构角色的人称为制作人。作家直接写文字；制作人从种子到输出负责。两者可以重叠，也可分开。

本流程遵循制作人的意识：不是「如何写出漂亮一句」，而是「如何把一个人物从零带到输出」。

每章都区分制作人的决定与AI的初步工作。这种诚实分工同时支撑效率与品质。

讲义在十四章结束，工作却未结束。人物进入作品，作品出版，读者与人物相遇，再次说出他的名字。若本讲义交出的工具能抵达那些阶段，它便完成了使命。
