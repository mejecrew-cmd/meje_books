# 第6章　日常切面情境：角色书写自己

上一章以五项尺度提取出单词身份。M是「安静的见证者」，J是「理想主义者」。我们说过，一个词看不见人物的一半。本章要第一次替那片空白穿上散文的衣服。

我们让角色写一篇短篇。采第一人称主角视角，韩文1500至2000字，模拟角色坐到作家的位置，写下自己日常的一个切面。经过这项模拟，一个词抓不住的另一半，才会在散文中显形。

这是本流程最大的验证工具。它直接用散文测试：只有一句种子与一个词身份的人物，是否真的能像活人一样行动。

## 日常切面情境是什么

先厘清这种形式。

维吉尼亚・吴尔芙的《戴洛维夫人》，只用一名女子为晚宴出门买花的一天，展开她的一生。没有重大事件；她走过街道、想起旧爱、听见钟声，克拉丽莎整个内在便浮现出来。证明人物的不一定是事件，也可以是普通的一小段生活。日常切面情境追求的正是这一点。

> **特写・安倍夜郎《深夜食堂》的常客** 漫画《深夜食堂》以午夜营业的巷弄小店为舞台，每一回透过一位客人的一道「固定菜色」展开人生片段。有人点章鱼造型维也纳香肠，有人点奶油拌饭。没有宏大事件，只凭「吃什么、怎么吃」的日常切面，一个人物便完整浮现。它把本章命题——不是事件，而是平凡片段也能证明人物——压缩成一只餐盘的大小，反覆演示。

**日常切面情境：** 以第一人称主角视角描写人物日常片段的短篇形式，在本工作中用作角色验证工具。

这种形式借用日文所称的「私小说」、英语研究中译为I-novel的第一人称自传小说。借用方式如下。

原本的私小说，是作家以第一人称坦白自身私密经验的日本近代文学类型，由作家把自己角色化，讲述自己的故事。本流程加以改造：不是作家写自己，而是假设**正在建构的角色坐上作家的位置，以第一人称写自己的日常**。

成果是一篇韩文1500至2000字的短篇。它不是正篇的一部分，而是角色验证文件，不直接进入作品。

## 与邻近形式的差异

我们要区分它与其他短散文形式。

本流程的日常切面情境采第一人称主角视角，由角色本人叙述，篇幅为韩文1500～2000字。Slice of Life可以自由使用第一或第三人称，也可能偏重外部观察，因此未必符合本流程的验证条件。开场情境是作品起点，不是验证工具。角色独白是日常切面的压缩版，时间不足时可代用。Vignette与之相近，但事件因果可能较弱。

**为何选日常切面情境？** 因为它最直接暴露角色内在如何运作。Slice of Life可能着重外部观察；开场是作品第一幕，不是角色测试；短独白不易容纳充分因果；Vignette虽有印象，因果可能薄弱。此处定义的日常切面，同时要求第一人称、角色本人叙述与一件事件的因果。

角色的调性、说话方式、在世界观中的位置，会在此阶段一次成形。这项测试确认由五项尺度抽象化的人物，能否在实际散文中存活；若不能，就代表必须修正尺度结果。

## I remember / I don't remember二十五句

正式写作前，先取出一项工具：进入角色，写下二十五个短句，作为短篇材料。

十五句以「角色记得」开始，十句以「角色不记得」开始，排列二十五个简短而具体的完整句子。

```
M记得母亲的结婚戒指是什么颜色。
M记得七岁时，母亲戴着的戒指掉在哪里。
M记得第一次站在解剖台前，手腕如何颤抖。
M不记得母亲最后说了什么。
M不记得婴儿爽身粉气味最初来自哪里。
```

这里只展示五句，实际要写二十五句。这是AI特别擅长的部分。输入种子与五项尺度结果，很快就能得到初步候选。制作人选出最有吸引力的二十五句，或亲手重写一两句。

记住两项原则。

- 写完整句，不用「母亲结婚戒指的颜色」之类名词片语。
- 仔细选词。句子虽短，仍须包含具体名词、数字或专有名词。

二十五句中一两句会成为开头，其余自然进入正文各处。

连续排列「我记得」句式，本来也是文学技法。美国艺术家兼作家乔・布雷纳德在《I Remember》中串起数百个「我记得」，描绘一个时代与一个人；法国作家乔治・佩雷克再以《我记得》承接。其核心想法是：记忆清单本身就能成为一个人。

## 五个开头候选

接着决定短篇第一句。它决定全文调性，所以不直接写死，而从五个候选中挑选。

事件直进型采「那天，［事件］发生了」，例如「那天，我再次看见母亲的戒指。」事件中段描写型从事件内部切入：「戒指安静地套在遗体左手食指上。」结局描写型从事后景象开始：「解剖台上放着一只塑胶袋。」因果连接词型以「所以」「因为」开头：「所以母亲从一开始就戴着戒指。」记忆型直接取二十五句之一：「M记得母亲的结婚戒指是什么颜色。」

五个候选从不同位置抓住同一事件。事件直进型节奏快，中段描写型慢，结局描写型带回顾调性，因果连接词型偏思辨，记忆型则接近意识流。

产生五候选是AI工作。输入二十五句与种子一行即可快速获得；从中采用最吸引人的一句，是制作人的工作。

## 参照作家组合

开始写作前再决定一件事：**参照作家组合**。以一位外国作家加一位韩国作家，预先抓住短篇调性。

把人物从「类型」变成「个人」的最快方法，是使用具体名词。雷蒙・卡佛的极简短篇精确写下人物职业、居住街区与车款，让读者看见的不是「一般劳工」，而是眼前这个人。一个精确车名，比十个抽象形容词更能立起人物，如同漫画的一件道具或电影的一套服装。

```
M的参照作家：戈马克・麦卡锡＋金薰
「干燥的劳动、带体温的克制。适合三十年资历法医手势的调性。」

J的参照作家：卡夫卡＋吴贞姬
「官僚式荒谬＋内在克制。适合陷入自我怀疑的作家。」
```

预先准备约八种组合，再依种子采用一组，速度最快。

麦卡锡＋金薰以干燥劳动与带体温的克制，适合肉体劳动、生存、死亡；泰德・姜＋吴贞姬以技术精密与内在感觉，适合科幻与职业日常；卡夫卡＋金爱烂以官僚荒谬与存在重量，适合制度、移民、身份。波赫士＋成硕济以逻辑迷宫与黑色幽默处理悖论、系统；勒瑰恩＋殷熙耕以平静观察与结构批评处理种族、分类、身份。布考斯基＋赵世熙以无表情的愤怒与劳动物性处理工厂、阶级；欧威尔＋片惠英以制度控制与内在裂缝处理监视、规训；卡尔维诺＋朴玟奎以形式实验与流行感处理后设叙事、框架故事。

采用组合是制作人的直接工作。判断哪一组符合种子调性，是外部工具不擅长的领域。

组合确定后，逐行整理文体参数。

```
M的文体参数：
- 句长：短（韩文8～15语节），尽量少用连接词。
- 修辞方式：以转喻为主，节制比喻，以物性替代。
- 感官优先：触觉 > 视觉 > 听觉。
- 对话比例：15～25%，短而干燥。
- 情感表达：不命名，只以行动与物性呈现。
- 叙述速度：慢，一个动作用3～5句。
```

这些参数是初步写作指南。

**转喻：** 以与某物相邻、接触的另一物指代它。例如「白宫宣布」以建筑指政府。隐喻依靠相似（「人生是航海」），转喻依靠邻接。它特别适合不直接命名情绪，而用旁边的物件呈现。下文M的短篇不用「悲伤」一词，只写「折了两次的塑胶袋」，就是转喻。

## 写一篇1500字短篇

现在写一篇韩文1500至2000字的短篇。视角是第一人称主角，叙述者就是角色本人。

写作程序：

1. 从五个候选采用第一句。
2. 从二十五句中选5～8句放进正文。
3. 决定一件在短篇中具有开始与结束的事件。
4. 输入种子、五尺度、二十五句、第一句、参照作家与文体参数，与AI工作伙伴取得初稿。
5. 校正不符合角色调性、偏离文体参数或因果薄弱之处。
6. 判定合格线；合格便采用，不合格则重新要求或改写。

这是本阶段最耗时的部分。AI可以快速产出初稿，但审核与补正需要时间。每一句都要确认是否带有角色调性，不能草率完成。

## 四项附加技法

短篇另使用四项技法。

卡夫卡《变形记》以一个句子开场：格里高尔・萨姆沙某天早晨从不安的梦中醒来，发现自己在床上变成巨大昆虫。没有冗长解释，只把一个冲击细节植入普通早晨，就翻转整个世界。选得好的细节，是支撑世界的楔子。

**对话比例指南。** 正文20～40%为合适范围。超过40%，加入行动、移动、感官场面；低于10%，警惕变成独白散文。禁止只为传递资讯而存在的对话，每段对话都要揭示关系、冲突或性格。

**结尾落地。** 最后一段以感官而具体的影像收束。禁止抽象宣言（「那才是真正的自由」）、整理领悟（「现在我终于明白……」）、主题总结与情绪命名（「我很悲伤」）。合格模式有六种：感官余韵、行动转换、物性对比、回到锚点、不在场的感觉、未解决物件。

**数字场景化。** 时间、距离、费用、机率、重量等数字，必须成为行动或判断依据，而非说明。禁止「M平均每具遗体花47分钟」；允许「47分钟处理完一具。那是今早第七具。」

**写作原则。** 让角色质地渗入文体，让角色选择左右结局。禁止为凑篇幅描写，优先追求压缩之美。

四项技法也是审稿清单，初稿若偏离一两项，就补正该处。

## 范例：M的短篇摘录

完整1500字太长，只展示首尾两段。

> 那天，我再次看见母亲的戒指。它安静地套在遗体左手食指上。三十年了。食指两个指节内侧长茧的，不只我一人。遗体食指同一位置，也有相同的茧。解剖台上的日光灯，以相同角度照亮两处茧。
>
> ［中略］
>
> 我把戒指放进塑胶袋。袋子一角破了。我取出新袋子，把戒指放进去。把袋口折两次。两次就能保存。一次不足，三次过多。两次才精确。

即使只看片段，也能看出M的调性：短句、转喻、触觉优先、不命名情绪，以及末段未解决的物件与数字场景化。麦卡锡＋金薰的组合活在其中。

## 范例：J的短篇摘录

J有不同调性。

> 旋转木马左侧头部附近，有一长条油漆剥落。J不记得那里原本是什么颜色，但确定旋转木马转动时，一直摸着那里。被摸过之处的颜色，只留下触摸的感觉。没有颜色的感觉。
>
> ［中略］
>
> J坐在桌前，望着手腕很久。手腕上什么也没有，也没有像旋转木马油漆那样、不记得原本颜色的位置。那么J失去了什么？没有失去位置的失去，可能存在吗？

这里活着卡夫卡＋吴贞姬的组合：抽象思辨、自我意识反转、荒谬的问题。调性与M不同。

## 范例：E的短篇摘录

E又是另一种调性。身处虚构战争世界的杀手，以压抑情感的干燥语言处理暴力。

> 下雨了。左肩烧伤发作时，我用手掌压住那里。压住也不会消失，只会让疼痛换个位置。我长久以来都在移动那些不会消失的东西。
>
> ［中略］
>
> 目标家门前坐着一个孩子，看来约七岁。我绕路，决定改天再来。有人教过我，只有弱者才会这样决定。所以今天，我依然是弱者。雨水流过肩上的烧伤。

E的身体锚点（左肩烧伤）与隐藏的慈悲同时渗出。在目标面前转身的一个动作，反转呈现「假装软弱以隐藏强大」的种子。它也不同于M的克制与J的思辨。

同一流程把三人展开为调性各异的短篇。这证明种子一行与五尺度，会依人物生成不同散文。

## 七项检查

短篇完成后，以七个问题再次确认是否充分验证角色。

```
1. 角色对事件结局造成了什么影响？
2. 角色哪一种质地引导了特定选择？
3. 哪项选择最影响事件？若做相反选择呢？
4. 若同一情况发生在角色的好友身上呢？
5. 角色行动的动机是什么？
6. 角色在事件中最大的欲望是什么？
7. 结尾满足了欲望与动机吗？若没有，原因是什么？
```

把问题实际套用于M的短篇。

```
1. M在遗体手上发现戒指，不按家属移交程序，而亲手装袋保存；由M直接决定处理方式。
2. 「必须精确保存」的三十年职业质地，引导她把袋口折两次。
3. 最大选择是不按规定移交而亲自收起。若照程序，短篇张力便消失。
4. 同事遇到同样事件会立刻报告。这项差异凸显M的质地。
5. 动机：不愿失去母亲最后的痕迹。
6. 最大欲望：以精确形式抓住不在场。
7. 只满足一半。戒指保存了，母亲行踪仍未知；未解决之处成为下一阶段动力。
```

七项答案会流入第7阶段缺点、匮乏、创伤与第8阶段目标。短篇不只完成本阶段，也替后续制造材料。

## 前往下一章

本章可整理如下：让已确定种子与五尺度的人物写一篇1500字短篇，便能一次抓住其调性、说话方式与世界观位置。AI擅长二十五句初步候选、五个第一句与1500字初稿；制作人擅长选择参照作家、审核、判定合格线与七项检查。

十二阶段最初三项——种子、镜子、日常切面——至此完成。一个人物已被压缩一次。下一章开始，在压缩之上添加血肉。

下一章是扭转与戏剧：故意把角色放上陈腔滥调的座标，再扭转座标；并以身体锚点，把角色质地固定于身体某一处的感觉。两者是立体化的第一批工具。
