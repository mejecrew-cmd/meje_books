# 第13章　非典型角色与作品的多样性

上一章替一个人物穿上五种媒介的服装，完成了十二阶段完整建构。但本讲义不只处理英雄与反派，角色还有其他面貌。

今天要处理六种既非英雄也非反派的非典型角色，以及有意识地设计作品角色阵容多样性的七个维度。

## 并非所有角色都是英雄或反派

先确认我们的态度。

至今处理的M、J、E都是核心人物，是主角或主要角色，也是作品视角与冲突的轴线。

然而作品还有游戏NPC、聊天机器人人格、品牌吉祥物、教育内容主持人、治疗性人格、VTuber与实况主、儿童读物的拟人动物。这些角色往往更贴近日常，我们每天接触的大多数角色既非英雄也非反派。

十二阶段仍适用于它们，只是强调的阶段不同。本章要辨认这些差异。

## 非典型1：AI聊天机器人人格

第一种是大型语言模型聊天机器人或AI助理的人格。

非人角色同样可以具有欲望、成长与谎言结构。《人造意识》的AI艾娃虽被设计成人类外形，却有自己的明确Want：离开封闭实验室。她甚至为此表演好感、利用他人。因此，建构AI角色时，第一个问题仍是：「这个人格想要什么？」

与一般流程的主要差异：

除了普通整合表，另行输出System Prompt格式。明定拒绝的主题与语气、对自己是AI的自觉与承认程度、命令型／对话型／导师型互动模式，以及跨工作阶段是否保存记忆。

建议阶段：

1. **第1、2、3阶段（种子、镜子、日常切面）：** 完整执行。可用一组对话样本取代1500字短篇。
2. **第8阶段（目标）：** 精确设定Want，Need与Lie采简式。
3. **第11阶段（沉浸运用）：** 强化语气，称呼变化表与六种说话类型是核心。
4. **第12阶段输出：** 在Obsidian标准之外增加系统提示格式。

系统提示单行格式范例：

```
你是［名字］。你采取［单词身份］的态度。
你不回答［拒绝模式领域］的问题。
你以［互动模式］与用户对话。
你以［说话类型］回答。
［追加事项］
```

建构AI聊天机器人人格，与第1章的态度形成镜像。正如我们和AI工作伙伴共同建构人物，AI角色也经由相同流程诞生。我们的工作方式，正是我们建构角色的方式。

## 非典型2：品牌吉祥物

第二种是商业、教育或活动用吉祥物。

长久存在的吉祥物靠简单与一致性维持。三丽鸥Hello Kitty以连嘴巴都没有的简单脸孔，受全世界喜爱超过半世纪。常见解释是，没有嘴巴能让观看者把自己的心情投射上去。比起复杂前史或缺点，一眼可辨的轮廓与不变的亲切感才是生命。Kakao Friends的Ryan也以「没有鬃毛的雄狮」与始终平和的表情，成为韩国最熟悉的吉祥物。

重点：

- **强化第一印象：** 五媒介输出中优先制作动画套件（Riot五要素）。
- **单一讯息：** 用一句话写出吉祥物传达的内容。
- **目标年龄：** 儿童、青年、中年等。
- **可注册商标性：** 视觉与听觉辨识度。
- **弹性：** 可依季节与活动变化。

建议阶段：

1. 完整执行**第1阶段种子＋第6阶段身份固定＋第11阶段语气＋第12阶段动画套件**。
2. 缺点、创伤、冲突很弱或不存在；吉祥物几乎没有缺点。

品牌吉祥物是简化建构，不走完所有阶段，但镜子、身份固定与视觉比其他角色更受强调。

## 非典型3：教育内容主持人

第三种是教科书、线上课程与儿童内容的引导者。

同一角色会依目的全面改造。《忍者龟》原为面向成人的粗犷暴力讽刺漫画，转入动画与玩具后，成为喜爱披萨、互相开玩笑的儿童英雄。当「向谁传达什么」改变，同一骨架也会穿上完全不同的衣服。

重点：

- **亲切优先：** 首次登场必须有取得好感的Save the Cat装置。
- **把缺点变成学习目标：** 让小错误成为学习点。
- **年龄适切性：** 使用符合发展阶段4、5的语汇与情绪。
- **可重复性：** 可依季度与单集更新。

建议阶段：

1. **第1、2阶段种子与镜子：** 完整执行。
2. **第6阶段身份固定：** 完整执行。
3. **第7阶段缺点：** 只使用可学习的小缺点，弱化或省略创伤。
4. **第11阶段沉浸运用：** 优先友善语气。
5. **第12阶段输出：** 动画套件＋每周单集格式。

教育角色的缺点应是可学习的小失误，而非道德灰色。「常写错字」可同时是魅力与学习点；「接近道德灰色的谎言」则不合适。

## 非典型4：治疗性人格

第四种是谘商、教练或自助工具中的角色。

重点：

- **同理与倾听优先：** 强调Confidant archetype。
- **避免评判：** HEXACO诚实—谦逊（H）偏高。
- **限制自我揭露：** 不展示人格的深层创伤。
- **工作阶段安全：** 建立自伤、自杀言及等危险信号的侦测与应对手册。

建议阶段：

1. **第1、2阶段种子与镜子：** 完整执行。
2. **第6阶段身份固定：** 完整执行。
3. **第8阶段目标：** 只精确设定Want；Need、Lie采简式且不向用户揭露。
4. **第11阶段沉浸运用：** 语气与倾听态度是核心。
5. **第12阶段输出：** 类似AI聊天机器人格式，另加安全线与禁语规格。

治疗性人格的安全规格可能比角色建构本身更长。只有明确规定它不该做什么，用户才能安全运用。

治疗对话角色可追溯至六十年前。1966年，MIT约瑟夫・魏森鲍姆制作的ELIZA只是模仿治疗师、重述用户话语的简单程序，人们却向机器倾诉并获得安慰，开发者本人反而对此警戒。它同时展示良好倾听人格的力量与安全线的必要。

## 非典型5：VTuber与实况主人格

第五种是直播用角色。

重点：

- **即时反应：** 不是脚本，而是即兴。
- **迷因与次文化亲和：** 萌系分类＋韩国网路迷因。

**萌系（デレ系）：** 依角色表达好感的方式分类，例如外冷内热的傲娇、执着到危险的病娇、慵懒冷静的冷娇。这是粉丝文化中快速掌握爱情表达模式的词汇。

- **2D/3D视觉：** Riot五要素，尤其Color Story与Voice。
- **主题范围：** 游戏、聊天、唱歌等。
- **禁忌与禁语（Roleplay禁止线）：** 人格的界线。

建议阶段：

1. **第1、2阶段种子与镜子：** 完整执行，镜子加入萌系分类。
2. **第6阶段身份固定：** 完整执行，视觉优先。
3. **第11阶段沉浸运用：** 即兴指南＋禁忌清单是核心。
4. **第12阶段输出：** 动画套件＋即兴指南。

VTuber人格的即兴性最高。完成十二阶段后，每场直播仍会产生即兴，因此须预先固定即兴的质地。

VTuber以「预先设定的质地＋即时即兴」运作。2016年登场的绊爱，以「自称AI、开朗古怪的少女」这个鲜明人格开启了整个类型。固定外形、口头禅与常用迷因，让每场反应保持即兴，只预先画出「这个人物绝不会说什么」的界线。共同设计固定结果与开放即兴，便是核心。

## 非典型6：儿童文学角色

第六种面向0～12岁读者。

重点：

- **更强简化：** 镜子只用四项固定尺度（MBTI、霍格华兹、黑白、30核心），省略辅助尺度。
- **可学习缺点：** 懒惰、嫉妒、说谎等可恢复缺点。
- **弱化创伤：** 避免Big T，只用small t。
- **强调一项外形：** 符合儿童的视觉中心性。
- **弱化反差魅力：** 一致性更重要。

建议阶段：

第1～5阶段完整执行。第6阶段只保留名字与一项外形因素，第7～12阶段大幅缩短。

这是十二阶段的简化版：五阶段完整，其余简式。每个角色三十分钟至一小时即可。

长久受喜爱的儿童角色简单且一致。小熊维尼几乎可由一句话概括：「太喜欢蜂蜜、温和又悠闲的熊。」缺点只是贪吃，是孩子可以跟着笑的小而可恢复的瑕疵。没有深层创伤或反转，每一集保持相同温柔，反而建立信任。《云朵面包》的洪妃等以「一颗清楚的心」描绘的孩子，也是同样设计。

## 适用矩阵

六种类型可整理如下。

AI聊天机器人以第11阶段语气与第12阶段System Prompt为核心，第3阶段可换成对话样本，并追加系统提示输出。品牌吉祥物以第12阶段动画套件为核心，可省略第7～9阶段并追加视觉指南。教育角色把第7阶段的小缺点作为学习目标，弱化或省略创伤，缩短第9阶段，追加学习点对应。治疗性人格以第6阶段身份与第11阶段语气为核心，可省略第3、9阶段并追加安全线、禁语。VTuber以第11、12阶段为核心，缩短第3、7、8阶段，追加禁忌与即兴指南。儿童文学完整使用第1～5阶段，第6阶段只留名字与外形，后续大幅缩短并追加学习价值对应。

展开矩阵，选择适合作品的类型。即使多个阶段采简式，核心仍相同：把角色当成一个人，或至少一个完整人格。

## 有意识的多样性设计

非典型角色的另一面，是作品角色阵容的多样性：哪些身份以何种分布登场。

若不有意识设计，角色阵容会收敛到与制作人相似的身份。除非那是刻意结果，否则必须检查。

以七个维度检查：

```
1. 性倾向（Heterosexual / Gay / Lesbian / Bi / Pan / Ace等）
2. 性别认同（Cis / Trans / Non-binary / Agender等）
3. 种族与族群细节（外形＋文化、语言）
4. 障碍（身体 / 心理 / 认知 / 慢性病）
5. 社会经济背景（阶级、移民经验）
6. 宗教与灵性
7. 神经多样性（自闭症 / ADHD / 阅读障碍等）
```

不必把七维度全放进一人。集中所有多样性，会让人物成为象征性Token。把七维度适当分布于五人角色阵容，才是有意识的设计。

## 角色阵容多样性检查

在第10章五轴角色阵容矩阵旁，加上多样性检查。

M加四人的结果如下：

```
角色阵容多样性检查：
- 性别：M(F)、同事(M)、检察官(M)、后辈(F)、AI(N)，平衡
- 年龄：55、55、40、28、N，多样
- 社会经济：M(中产)、同事(劳工)、检察官(上层)、后辈(中产)、AI(N)，多样
- 神经多样性：四名人类均无特别设定，单一（须追加检讨）
- 种族与族群：全为韩国人，单一（有意偏向）
- 宗教：M(无神论)、同事(基督教)、检察官(佛教)、后辈(无关心)、AI(N)，多样
- 障碍：AI(N/A)，其余无，单一（有意偏向）
```

五个维度已有分布，种族／族群与障碍两项单一。单一维度应是有意偏向，或依作品调性检查后补正。

若不检查，所有人物会收敛到制作人最熟悉的质地。有意识地展开表格，就是多样性设计的核心。

## 多样性不是追加物

多样性不是角色阵容完成后才覆盖上去，而要从一开始融入分配。后加的角色会被读成「为了多样性新增的人」；从起点融入，人物才有自己的位置与质地。

对《海洋奇缘》的莫娜而言，玻里尼西亚文化不是外形装饰，而是航海故事的骨架。《乐动心旋律》的聋人家庭，也不是「听障角色」标签，而是各有欲望、笑话、争执的人。后加身份读起来像象征，从一开始融入则像人。

> **特写・《月光下的蓝色男孩》的夏隆** 电影追随黑人酷儿少年夏隆的童年、青春与成年。他的种族、性别认同与贫穷不是附加标签，而是呼吸、迟疑与爱人的方式。「你必须自己决定你是谁」贯穿三个时期。它安静而深刻地证明：当身份融入骨架而非事后添加，角色会成为人，而非象征。

五个前置模块中的模块5（身份与多样性维度）正为此存在。种子确定后立即决定分布，后续十一阶段便会依此对齐。

关闭模块也可是一项有意决定。若作品概念必须聚焦单一身份，例如家庭冲突剧，就关闭它。有意关闭与单纯遗漏，结果不同。

## Hofstede六个文化维度：多文化角色

多文化与国际作品可启用Hofstede六个文化维度。此模型调查五十多国、超过十一万名IBM员工，以六个分数浓缩社会文化质地。

Marvel《尚气与十环传奇》把文化差异放在戏剧核心：重视父亲义务与家族阶序的世界，和在美国长大、想自行选择人生的自我不断冲突。出身文化与目前文化的距离，正是内在冲突。要避免多文化角色只剩外形与名字，就须有意识设计这段距离。

韩国／美国六项比较为：Power Distance 60／40；Individualism vs Collectivism 18／91；Masculinity vs Femininity 39／62；Uncertainty Avoidance 85／46；Long-term vs Short-term 100／26；Indulgence vs Restraint 29／68。韩国较倾向阶序、集体、合作照顾、回避不确定与长期取向，美国较倾向个人、成就竞争、短期取向与欲望满足。

出身文化与居住文化的分数差成为冲突动力。以韩裔美国1.5代为例，韩国（个人主义18、长期取向100）与美国（个人主义91、短期取向26）的差距成为日常冲突；家庭活动中，要求韩式阶序的父母，会与优先自主决定的本人不断碰撞。
