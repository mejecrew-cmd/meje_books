# 第6編　実務編C　第2次統合CSVサンプル

この実務編は、第1次の散らばった行が第2次で見出し語単位に整理される姿を示す加工例です。第2次の目標は美しい文章を書くことではなく、Vaultが依存する代表キーワード、別名、世界観軸、関連キーワードを安定して確定することです。

例示IPは**「ヌビの地」**です。実務編Bの第1次抽出サンプルが、第2次でどのように見出し語へ束ねられるかを続けて示します。

## 13カラム定義

| カラム | 説明 |
|---|---|
| IDX | 見出し語の行番号 |
| 分類 | 9分類の一つ |
| 代表キーワード | Vaultファイル名になる公式見出し語 |
| 世界観軸 | プロファイルで許容した軸 |
| 異称_別称 | 同じ対象を指す代替表記 |
| 英語名 | 翻訳または国際化用表記 |
| ローマ字 | 韓国語見出し語のローマ字表記 |
| 定義 | 一〜二文の自己完結した定義 |
| 詳細説明 | 関係と作動文脈を含む短い説明 |
| 関連キーワード | 代表キーワード基準の関連見出し語一覧 |
| バージョン状態 | 現行 / 過去 / 廃棄 / 二重表記 / 未定 |
| 翻訳ノート | 翻訳判断と注意事項 |
| 出所リスト | 第1次行の出所束 |

## 第2次検収ポイント

| 項目 | 合格基準 |
|---|---|
| 代表キーワード重複 | 同じ代表キーワードが二行に存在しない。 |
| 世界観軸 | プロファイルの許容値だけを使う。 |
| 別称 | 同じ別称が異なる見出し語を指すなら手動検討する。 |
| 関連キーワード | 代表キーワード基準で書き、別名リンクを任意に入れない。 |
| バージョン状態 | 未定比率を全体の5%以下にする。 |
| 翻訳ノート | 禁訳語、原語併記、初出解説が必要な場合に明記する。 |

## 第2次で人が最終決定すること

- 同義語束
- 代表キーワード
- 世界観軸の新設または変更
- バージョン状態
- 翻訳判断

自動化は候補を作れますが、この五つはIPの意味を変える決定なので人が承認します。
