# 第6篇　實務篇 B．第一階段抽取 CSV 範例

本實務篇是展示第一階段抽取產物的加工範例。目的不在完成度，而在呈現態度。第一階段允許重複與不完整，優先選擇不要遺漏。

範例 IP 為**「努比之地」**。所有範例詞條都視為來自此加工 IP 的同一場景。

## 輸入範例

下列段落是為說明而寫的加工文本。

> 無紋者在舉行重新編織審判的灰色織堂後方，提著一盞舊燈。織女說，那盞燈的煤灰是努比偽造的痕跡。判決官允海攤開氏族帳冊，但擁有赤裸記憶的孩子清楚叫出了帳冊上沒有的名字。從那天起，灰色織堂禁止紅線。

## 五欄 CSV 範例

```csv
IDX,分類,關鍵字,說明,出處
1,人物,無紋者,無法讀取或編織紋樣但以赤裸記憶記得氏族事務的人物或階層,加工本文第1段
2,裝置,重新編織,重新編織努比以改變歷史與法律紀錄的運作原理,加工本文第1段
3,場所,灰色織堂,舉行重新編織審判的場所,加工本文第1段
4,物件,舊燈,無紋者提著的物件且煤灰成為線索,加工本文第1段
5,人物,織女,將燈的煤灰解釋為努比偽造痕跡的人物,加工本文第1段
6,事件,努比偽造,努比紀錄遭操弄的事件或嫌疑,加工本文第1段
7,人物,判決官允海,攤開氏族帳冊進行審判的判決官,加工本文第1段
8,物件,氏族帳冊,記載氏族名字與紀錄的帳冊,加工本文第1段
9,概念,赤裸記憶,不使用文字或紋樣、以人的身體與話語傳承的記憶方式,加工本文第1段
10,人物,孩子,說出帳冊上沒有名字的孩子，身分未定,加工本文第1段
11,價值,帳冊上沒有的名字,揭示被官方紀錄排除之存在的主題表達,加工本文第1段
12,場面調度,紅線,灰色織堂禁止的色彩象徵,加工本文第1段
```

## 判定備註

| 項目 | 判定 | 理由 |
|---|---|---|
| 允許重複 | PASS | 尚不必確定`無紋者`是人物還是階層。 |
| 允許未定 | PASS | `孩子`的身分在第二階段確認。 |
| 含意關鍵字抽取 | PASS | `帳冊上沒有的名字`不是明示術語，但作為價值分類候選保存。 |
| 禁止過度整理 | PASS | 尚不合併`努比偽造`與`重新編織`。 |

## 第一階段不做的事

- 不確定代表關鍵字。
- 不強行合併同義詞。
- 不最終確定世界觀軸。
- 不判斷 hub 與 leaf。
- 不決定翻譯名。

第一階段的合格不看「是否整理好」，而看「是否已放入足夠供下一階段判斷的材料」。
