# 第6篇　实务篇 B．第一阶段抽取 CSV 示例

本实务篇是展示第一阶段抽取产物的加工示例。目的不在完成度，而在呈现态度。第一阶段允许重复和不完整，优先选择不遗漏。

示例 IP 为**“努比之地”**。所有示例词条均视为来自这个加工 IP 的同一场景。

## 输入示例

下列段落是用于说明的加工文本。

> 无纹者在举行重新编织审判的灰色织堂后方提着一盏旧灯。织女说，那盏灯的煤灰是努比伪造的痕迹。判决官允海翻开氏族账册，但拥有赤裸记忆的孩子清楚叫出了账册上没有的名字。从那天起，灰色织堂禁止红线。

## 五列 CSV 示例

```csv
IDX,分类,关键词,说明,出处
1,人物,无纹者,无法读取或编织纹样但以赤裸记忆记得氏族事务的人物或阶层,加工正文第1段
2,装置,重新编织,重新编织努比以改变历史与法律记录的运作原理,加工正文第1段
3,场所,灰色织堂,举行重新编织审判的场所,加工正文第1段
4,物件,旧灯,无纹者提着的物件且煤灰成为线索,加工正文第1段
5,人物,织女,将灯的煤灰解释为努比伪造痕迹的人物,加工正文第1段
6,事件,努比伪造,努比记录遭操纵的事件或嫌疑,加工正文第1段
7,人物,判决官允海,翻开氏族账册进行审判的判决官,加工正文第1段
8,物件,氏族账册,记载氏族名字与记录的账册,加工正文第1段
9,概念,赤裸记忆,不使用文字或纹样、通过人的身体和话语传承的记忆方式,加工正文第1段
10,人物,孩子,说出账册上没有名字的孩子，身份未定,加工正文第1段
11,价值,账册上没有的名字,揭示被官方记录排除之存在的主题表达,加工正文第1段
12,场面调度,红线,灰色织堂禁止的色彩象征,加工正文第1段
```

## 判定备注

| 项目 | 判定 | 理由 |
|---|---|---|
| 允许重复 | PASS | 尚不必确定`无纹者`是人物还是阶层。 |
| 允许未定 | PASS | `孩子`的身份在第二阶段确认。 |
| 含意关键词抽取 | PASS | `账册上没有的名字`不是明示术语，但作为价值分类候选保存。 |
| 禁止过度整理 | PASS | 尚不合并`努比伪造`与`重新编织`。 |

## 第一阶段不做的事

- 不确定代表关键词。
- 不强行合并同义词。
- 不最终确定世界观轴。
- 不判断 hub 与 leaf。
- 不决定翻译名。

第一阶段的合格不看“是否整理好”，而看“是否已放入足够供下一阶段判断的材料”。
