# 第6篇　实务篇 F．验证检查清单

本清单是在将 Vault 移交给下一项工作之前需要确认的最低标准。各项以 `PASS / WARN / FAIL` 判定。只要还有任何 `FAIL`，就不能进行运营用 handoff。

## 判定标准

| 判定 | 含义 | 措施 |
|---|---|---|
| PASS | 符合标准 | 进入下一阶段 |
| WARN | 可以使用，但需要记录和后续处理 | 写入 handoff 备忘 |
| FAIL | 交给消费工作时错误风险很高 | 修正后重新验证 |

## 第1阶段：自动检查

| 项目 | 标准 | 判定 | 备注 |
|---|---|---|---|
| 第1阶段输入覆盖度 | 主要输入文档的前、中、后段来源全部存在 |  |  |
| 第1阶段九分类覆盖度 | 九分类均至少有一项；例外在 profile 中记录原因 |  |  |
| 第2阶段代表关键词重复 | 0项 |  |  |
| 第2阶段必填英文名缺失 | 翻译项目为0项 |  |  |
| 第2阶段版本状态未定比例 | 不超过5% |  |  |
| 第2阶段世界观轴未定比例 | 不超过10% |  |  |
| enum 违规 | category、worldbuilding_axis、version 的允许值违规均为0项 |  |  |
| 第3阶段文件数 | 第2阶段 CSV 词条行数与 `.md` 词条文件数一致 |  |  |
| YAML 解析 | frontmatter 解析失败为0项 |  |  |
| 文件名冲突 | 代表关键词文件名冲突为0项 |  |  |
| 失效链接 | wikilink 目标缺失为0项，或存在 stub |  |  |
| self-link | 指向自身的链接为0项 |  |  |
| LLM 完成率 | 运营 handoff 标准为95%以上 |  |  |
| manifest 一致性 | manifest 进度与实际文件状态一致 |  |  |

## 第2阶段：人工抽样

建议至少抽样30项，并混合 hub、中间层级、leaf、九分类和各世界观轴。

| 项目 | 标准 | 判定 | 备注 |
|---|---|---|---|
| 同义词组 | 代表关键词和别名指向同一对象 |  |  |
| 分类适合度 | 九分类符合叙事功能 |  |  |
| 世界观轴适合度 | 世界观轴符合 IP 结构 |  |  |
| 定义质量 | 用一两句话给出自足的含义 |  |  |
| 详细说明质量 | 展示相关词条和运行语境 |  |  |
| LLM 叙述质量 | 不与 hub 的语调标准冲突 |  |  |
| 翻译说明 | 有翻译风险的词条具备充分指引 |  |  |
| 来源逆向追溯 | 能回到原文或第1阶段来源 |  |  |
| 过度链接 | 相关关键词没有无意义膨胀 |  |  |
| 敏感冲突 | 不与已发行 LOREBOOK 或官方正文冲突 |  |  |

## 第3阶段：集成构建检查

| 项目 | 标准 | 判定 | 备注 |
|---|---|---|---|
| Obsidian 浏览 | Vault 能正常打开 |  |  |
| 图谱显示 | 主要 hub 和链接网络没有异常孤立 |  |  |
| 消费 skill 读取 | 写作、Lorebook、翻译工作可引用词条 |  |  |
| 辅助文档 | manifest 中登记的必需 `aux_docs` 存在 |  |  |
| 索引 | 完整词条列表或索引为最新 |  |  |
| handoff 记录 | 下一位工作者能理解状态 |  |  |

## Handoff 勾选清单

```markdown
## Handoff Checklist

- [ ] 自动检查 FAIL 为0项
- [ ] 人工抽样 FAIL 为0项
- [ ] 记录 WARN 项目的原因和后续措施
- [ ] LLM 完成率95%以上
- [ ] 必需辅助文档存在
- [ ] 确认 manifest `ready_for_consumption` 值
- [ ] 不与已发行 LOREBOOK 或官方正文冲突
- [ ] 与下一项消费工作共享版本号

handoff_status: ready_for_consumption / ready_with_warnings / not_ready
handoff_date:
handoff_owner:
handoff_note:
```

## 运营判断

个人练习用 Vault 即使带有 WARN 或部分未完成，也可以用于学习；但是移交给消费 skill 的运营 Vault 必须通过本清单。标准放松虽然能在前段省时，却会在后段以数倍的错误成本偿还。
